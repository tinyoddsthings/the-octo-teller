## 搜尋建議

- 找**法術規則** → `phb/ch7b_spell_desc_*.md`（按首字母選檔）
- 找**職業特性** → `phb/ch3_XX_<class>.md`
- 找**裝備數值** → `phb/ch6_equipment.md`
- 找**規則術語解釋** → `phb/appB_rules_glossary.md`
- 找**遭遇設計** → `dmg/ch4_creating_adventures.md`
- 找**魔法物品（隨機表）** → `dmg/random_magic_items.md`
- 找**魔法物品（完整描述）** → `dmg/magic_items_az_part*.md`（依首字母選檔）
- 找**怪物數據** → `mm/monsters_X.md`（X 為怪物名稱首字母）
- 找**怪物列表/CR查詢** → `mm/appB_monster_lists.md`

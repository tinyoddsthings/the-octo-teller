| 成分 | V, S, M（一塊鞣製皮革） |
| 持續時間 | 8 小時 |

**效果：** 你觸碰一個未穿護甲的自願生物。在法術結束前，目標的基礎 AC 變為 13 + 其敏捷修正值。若目標穿上護甲，法術提前結束。

#### 法師之手（Mage Hand）

| 項目 | 內容 |
|------|------|
| 環級 | 戲法 |
| 學派 | 咒法（Conjuration） |
| 職業 | 吟遊詩人、術士、契術師、法師 |
| 施法時間 | 1 動作 |
| 射程 | 9m |
| 成分 | V, S |
| 持續時間 | 1 分鐘 |

**效果：** 一隻半透明的飄浮之手出現在射程內你選擇的一點。手持續至持續時間結束。若手離你超過 9m 或你再次施放此法術，手會消失。

施放法術時，你可以用手操作物體、打開未上鎖的門或容器、從開著的容器中存取物品，或倒出瓶中的液體。

在之後的回合中，你可以使用魔法動作再次控制手。作為該動作的一部分，你可以移動手最多 9m。

手無法攻擊、啟動魔法物品，也無法搬運超過 5 磅的物品。

#### 魔法陣（Magic Circle）

| 項目 | 內容 |
|------|------|
| 環級 | 3 環 |
| 學派 | 防護（Abjuration） |
| 職業 | 牧師、聖騎士、契術師、法師 |
| 施法時間 | 1 分鐘 |
| 射程 | 3m |
| 成分 | V, S, M（鹽和銀粉，價值 100+gp，施法後消耗） |
| 持續時間 | 1 小時 |

**效果：** 你在射程內地面上可見的一點為中心，創造一個半徑 3m、高 6m 的柱體魔法能量場。柱體與地板或其他表面交匯處出現發光的符文。

選擇以下一種或多種生物類型：天界生物、元素生物、妖精、邪魔或不死生物。法陣對所選類型的生物產生以下效果：

- 該生物無法以非魔法手段自願進入柱體。若該生物嘗試使用傳送或跨位面旅行進入，必須先進行一次魅力豁免。
- 該生物對柱體內目標的攻擊擲骰具有劣勢。
- 柱體內的目標不能被該生物附身，也不能從該生物身上獲得魅惑或恐懼狀態。

每次施放此法術時，你可以讓魔法反向運作——阻止指定類型的生物離開柱體，並保護柱體外的目標。

**升環：** 每高一環，持續時間增加 1 小時。

#### 容器術（Magic Jar）

| 項目 | 內容 |
|------|------|
| 環級 | 6 環 |
| 學派 | 死靈（Necromancy） |
| 職業 | 法師 |
| 施法時間 | 1 分鐘 |
| 射程 | 自身 |
| 成分 | V, S, M（一顆寶石、水晶或聖物匣，價值 500+gp） |
| 持續時間 | 直到被解除 |

**效果：** 你的身體陷入昏迷狀態，靈魂離開身體進入作為法術材料成分的容器。當你的靈魂棲息於容器中時，你能感知周遭環境，彷彿身處容器所在的空間。你無法移動或使用反應。你唯一能採取的動作是將靈魂投射到容器外 30m 的範圍內，可以選擇回到自己的活體（結束法術）或嘗試附身一個類人生物。

你可以嘗試附身 30m 內你能看見的任何類人生物（受防護善惡或魔法陣保護的生物無法被附身）。目標進行魅力豁免。
- **失敗**：你的靈魂進入目標身體，目標的靈魂被困在容器中
- **成功**：目標抵抗你的附身嘗試，且你在 24 小時內無法再次嘗試附身它

一旦你附身了一個生物的身體，你便控制它。你的生命值、生命骰、力量、敏捷、體質、速度和感官被該生物的替換。除此之外你保留自己的遊戲數值。

同時，被附身生物的靈魂可以從容器中使用自身感官感知外界，但無法移動且處於失能狀態。

附身身體期間，你可以使用魔法動作從宿主身體回到容器（若容器在 30m 內），宿主的靈魂回歸其身體。若宿主身體在你附身時死亡，該生物死亡，你進行一次對抗自身施法 DC 的魅力豁免。
- **成功**：你回到容器（若在 30m 內）
- **失敗**：你死亡

若容器被摧毀或法術結束，你的靈魂回歸你的身體。若你的身體離你超過 30m 或已死亡，你死亡。若法術結束時另一個生物的靈魂在容器中且容器被摧毀，該生物的靈魂回歸其身體（若身體存活且在 30m 內），否則該生物死亡。

法術結束時，容器被摧毀。

#### 魔嘴術（Magic Mouth）

| 項目 | 內容 |
|------|------|
| 環級 | 2 環 |
| 學派 | 幻術（Illusion） |
| 職業 | 吟遊詩人、法師 |
| 施法時間 | 1 分鐘（R） |
| 射程 | 9m |
| 成分 | V, S, M（翡翠粉末，價值 10+gp，施法後消耗） |
| 持續時間 | 直到被解除 |

**效果：** 你在射程內的一個物體中植入一段訊息——當觸發條件被滿足時就會被念出。選擇一個你能看見且未被其他生物穿戴或攜帶的物體。然後說出訊息，必須在 25 個字以內，但可以在最長 10 分鐘內傳達。最後決定觸發法術傳達訊息的情境。

當觸發發生時，一張魔法嘴巴出現在物體上，以你的聲音和你說話時相同的音量朗誦訊息。若你選擇的物體有嘴或類似嘴的東西（例如雕像的嘴巴），魔法嘴巴就會出現在那裡，使文字看起來像是從物體的嘴巴中發出。施法時，你可以選擇讓法術在傳達訊息後結束，或保持存在並在每次觸發發生時重複其訊息。

觸發條件可以是概括的或是詳細的，但必須基於物體 9m 內發生的視覺或聽覺條件。例如，你可以指示嘴巴在任何生物進入物體 9m 內時說話，或在銀鈴在 9m 內響起時說話。

#### 主幻象（Major Image）

| 項目 | 內容 |
|------|------|
| 環級 | 3 環 |
| 學派 | 幻術（Illusion） |
| 職業 | 吟遊詩人、術士、契術師、法師 |
| 施法時間 | 1 動作 |
| 射程 | 36m |
| 成分 | V, S, M（一小撮羊毛） |
| 持續時間 | C 10 分鐘 |

**效果：** 你創造一個物體、生物或其他可見現象的影像，大小不超過 6m 立方。影像出現在射程內你能看見的一點並持續至持續時間結束。它看起來很真實，包括適當的聲音、氣味和溫度，但無法造成傷害或施加狀態。

若你在幻象的射程內，你可以使用魔法動作將影像移動到射程內的另一點。隨著影像位置改變，你可以改變其外觀使其動作顯得自然。例如，若你創造了一個生物的影像並移動它，你可以改變影像使其看起來像在走路。同樣，你可以讓幻象在不同時間發出不同聲音，例如讓它進行對話。

物理接觸影像會揭示它是幻象，因為物體可以穿過它。一個使用研究動作檢查影像的生物，可以進行對抗你法術豁免 DC 的智力（調查）檢定，成功則判定它是幻象。若一個生物識破幻象，該生物可以看穿影像，其他感官效果對該生物變得微弱。

**升環：** 使用 4 環以上法術位施放時，法術持續至被解除，無需專注。

#### 群體暗示術（Mass Suggestion）

| 項目 | 內容 |
|------|------|
| 環級 | 6 環 |
| 學派 | 惑控（Enchantment） |
| 職業 | 吟遊詩人、術士、法師 |
| 施法時間 | 1 動作 |
| 射程 | 18m |
| 成分 | V, M（一根蛇舌） |
| 持續時間 | 24 小時 |

**效果：** 你對射程內至多十二個你能看見、能聽到且理解你的生物建議一個行動方案——以不超過 25 個字描述。該建議必須聽起來可行，且不涉及明顯對任何目標或其盟友造成傷害的事情。例如你可以說：「沿著那條路走到村莊，幫助村民們收割莊稼直到日落。」或者你可以說：「現在不是暴力的時候。放下武器，跳舞吧！一小時後停下。」

每個目標必須進行一次感知豁免。
- **失敗**：在持續時間內或直到你或你的盟友對目標造成傷害前獲得魅惑狀態

每個被魅惑的目標會盡力執行建議。建議的活動可以在整個持續時間內持續進行，但若活動能在較短時間內完成，法術在目標完成後結束。

**升環：** 使用更高環法術位時持續時間更長：7 環（10 天）、8 環（30 天）或 9 環（366 天）。

|------|------|
| 環級 | 8 環 |
| 學派 | 咒法（Conjuration） |
| 職業 | 法師 |
| 施法時間 | 1 動作 |
| 射程 | 18m |
| 成分 | V, S |
| 持續時間 | C 10 分鐘 |

**效果：** 你將射程內你能看見的一個生物放逐到迷宮般的半位面。目標在持續時間內或逃出迷宮前留在其中。

目標可以使用研究動作嘗試逃脫。這樣做時，它進行一次智力（調查）檢定（DC 20）。
- **成功**：逃出迷宮，法術結束

法術結束時，目標重新出現在它離開的空間，若該空間已被佔據，則出現在最近的未佔據空間。

#### 融入石頭（Meld into Stone）

| 項目 | 內容 |
|------|------|
| 環級 | 3 環 |
| 學派 | 變化（Transmutation） |
| 職業 | 牧師、德魯伊、遊俠 |
| 施法時間 | 1 動作（R） |
| 射程 | 觸碰 |
| 成分 | V, S |
| 持續時間 | 8 小時 |

**效果：** 你踏入一個足以完全容納你身體的石質物體或表面，在持續時間內將你自己和裝備與石頭融合。你必須觸碰石頭才能這麼做。你的存在痕跡對非魔法感官不可見或不可偵測。

融入石頭時，你無法看到外面發生的事，且你進行的任何聽取外面聲音的感知（察覺）檢定具有劣勢。你仍能感知時間的流逝，並能在融入石頭時對自己施放法術。你可以使用 1.5m 的移動力從你進入的位置離開石頭，結束法術。否則你無法移動。

石頭受到輕微物理損壞不會傷害你，但它的部分破壞或形狀改變（到你不再能容身的程度）會將你排出並對你造成 6d6 力場傷害。石頭的完全摧毀（或變質為不同物質）會將你排出並對你造成 50 點力場傷害。若被排出，你移動到最接近你最初進入位置的未佔據空間，並處於倒地狀態。

#### 復活術（Resurrection）

| 項目 | 內容 |
|------|------|
| 環級 | 7 環 |
| 學派 | 死靈（Necromancy） |
| 職業 | 吟遊詩人、牧師 |
| 施法時間 | 1 小時 |
| 射程 | 觸碰 |
| 成分 | V, S, M（一顆鑽石，價值 1,000+gp，施法後消耗） |
| 持續時間 | 瞬間 |

**效果：** 你以觸碰復活一個已死亡不超過一世紀、非老死、且死亡時不是不死生物的死亡生物。

生物以全部生命值復活。此法術也中和生物死亡時受到的任何毒素。此法術關閉所有致命傷口並恢復所有缺失的身體部位。

從死亡中回來是一場磨難。目標的 d20 檢定受到 -4 減值。每次目標完成長休，減值減少 1 直到變為 0。

以此法術復活一個已死亡 365 天或更久的生物會使你精疲力竭。在你完成長休前，你無法再次施放法術，且你的 d20 檢定具有劣勢。

#### 逆轉引力（Reverse Gravity）

| 項目 | 內容 |
|------|------|
| 環級 | 7 環 |
| 學派 | 變化（Transmutation） |
| 職業 | 德魯伊、術士、法師 |
| 施法時間 | 1 動作 |
| 射程 | 30m |
| 成分 | V, S, M（一塊磁石和鐵屑） |
| 持續時間 | C 1 分鐘 |

**效果：** 此法術逆轉射程內一點為中心、半徑 15m、高 30m 柱體內的重力。該區域內所有未固定在地面上的生物和物體向上墜落並到達柱體頂部。生物可以進行敏捷豁免以抓住能觸及的固定物體，從而避免向上墜落。

若在向上墜落中遇到天花板或固定物體，生物和物體撞擊它就像向下墜落一樣。若受影響的生物或物體在未撞到任何東西的情況下到達柱體頂部，它在持續時間內懸浮在那裡。法術結束時，受影響的物體和生物向下墜落。

#### 繩索戲法（Rope Trick）

| 項目 | 內容 |
|------|------|
| 環級 | 2 環 |
| 學派 | 變化（Transmutation） |
| 職業 | 法師 |
| 施法時間 | 1 動作 |
| 射程 | 觸碰 |
| 成分 | V, S, M（一段繩索） |
| 持續時間 | 1 小時 |

**效果：** 你觸碰一條繩索。繩索的一端向上懸浮，直到繩索垂直於地面或繩索到達天花板。在繩索的上端，一個隱形的 90cm x 1.5m 傳送門打開通往一個異次元空間，持續至法術結束。可以攀爬繩索到達該空間，繩索可以被拉入或放出該空間。

該空間可容納最多八個中型或更小的生物。攻擊、法術和其他效果無法進出該空間，但空間內的生物可以透過傳送門看到外面。法術結束時，空間內的任何東西掉出。

# 第七章：法術

> **來源**：2024 PHB (5.5e) Chapter 7: Spells

---

本章提供施放法術的規則，並包含 D&D 世界中常見法術的描述。這些法術被許多職業特性、魔法物品和怪物使用。

---

## 獲得法術

在你施放法術之前，你必須在心中備妥該法術，或從魔法物品（如法術卷軸）中獲得存取。你的特性會指定你能存取哪些法術（如有）、你是否永遠備妥某些法術、以及你是否能更改備妥法術列表。

### 備妥法術

若你有一份可更改的 1 環以上備妥法術列表，你的施法特性會指定何時可以更改以及可以更改多少個法術，如下表摘要。

#### 各職業法術備妥方式

| 職業 | 更改時機 | 更改數量 |
|------|---------|---------|
| 吟遊詩人 | 獲得等級時 | 一個 |
| 牧師 | 完成長休時 | 任意 |
| 德魯伊 | 完成長休時 | 任意 |
| 聖騎士 | 完成長休時 | 一個 |
| 遊俠 | 完成長休時 | 一個 |
| 術士 | 獲得等級時 | 一個 |
| 契術師 | 獲得等級時 | 一個 |
| 法師 | 完成長休時 | 任意 |

大多數施法怪物不會更改其備妥法術列表，但城主可以自由修改。

### 永備法術

某些特性可能給予你一個永遠備妥的法術。若你同時有一份可更改的備妥法術列表，永備法術不計入該列表的數量。

> **穿甲施法。** 你必須接受過你所穿護甲的訓練，才能穿著它施放法術。否則護甲會過度妨礙你施法。

## 施放法術

每個法術描述都有一系列條目，提供施放該法術所需的細節。以下各節解釋這些條目。

### 法術環級

每個法術都有 0 至 9 的環級，標示在法術描述中。法術環級是其威力的指標。戲法——幾乎可以背誦施放的簡單法術——是 0 環。各施法職業的規則說明其成員何時獲得某環級法術的存取權。

### 法術位

施法是消耗精力的，因此施法者在休息前只能施放有限數量的 1 環以上法術。法術位是表現施法者魔法潛力的主要方式。每個施法職業給予其成員有限數量的特定環級法術位。例如 3 級法師有四個 1 環法術位和兩個 2 環法術位。

當你施放法術時，你消耗一個等於或高於該法術環級的法術位。想像法術位是特定大小的溝槽——1 環法術位小，更高環級的更大。1 環法術可以放入任何大小的法術位，但 2 環法術只能放入至少 2 環的法術位。所以當 3 級法師施放 1 環法術魔法飛彈時，消耗四個 1 環法術位之一，剩餘三個。

完成長休後恢復所有已消耗的法術位。

### 不消耗法術位的施放

有多種方式可以不消耗法術位施放法術：

**戲法。** 戲法不需要法術位即可施放。

**儀式。** 某些法術在施法時間條目中帶有儀式標籤。這類法術可以按正常施法規則施放，也可以作為儀式施放。儀式版本的施法時間比正常多 10 分鐘，但不消耗法術位。要以儀式施放法術，施法者必須已備妥該法術。

**特殊能力。** 某些角色和怪物擁有特殊能力，允許他們不消耗法術位施放特定法術。這種施放通常有其他限制，例如每天只能施放有限次數。

**魔法物品。** 法術卷軸和某些其他魔法物品包含可不消耗法術位施放的法術。此類物品的描述會指定法術可以從中施放多少次。

### 使用更高環法術位

當施法者使用高於法術環級的法術位施放法術時，該法術在此次施放中視為更高環級。例如法師用 2 環法術位施放魔法飛彈，該魔法飛彈就是 2 環。實際上，法術擴展以填滿放入的法術位。

某些法術（如魔法飛彈和療傷術）在以更高環級施放時有更強效果，詳見法術描述。

### 魔法學派

每個法術屬於一個魔法學派。學派列於魔法學派表。這些分類有助於描述法術，但本身沒有規則效果，儘管其他規則可能會引用它們。

#### 魔法學派表

| 學派 | 典型效果 |
|------|---------|
| 防護（Abjuration） | 防止或逆轉有害效果 |
| 咒法（Conjuration） | 傳送生物或物體 |
| 預言（Divination） | 揭示資訊 |
| 惑控（Enchantment） | 影響心智 |
| 塑能（Evocation） | 引導能量創造效果，通常具破壞性 |
| 幻術（Illusion） | 欺騙心智或感官 |
| 死靈（Necromancy） | 操控生與死 |
| 變化（Transmutation） | 變化生物或物體 |

### 職業法術列表

若法術在某職業的法術列表上，該職業名稱會出現在法術學派後的括號中。某些特性可能將法術加入角色的法術列表，即使該角色不是括號中職業的成員。

### 施法時間

大多數法術需要魔法動作來施放，但某些法術需要附贈動作、反應或 1 分鐘以上。法術的施法時間條目指定需要哪種。

**每回合一個法術位法術。** 在一個回合中，你只能消耗一個法術位來施放法術。這意味著你不能在同一回合中用魔法動作消耗法術位施放一個法術，又用附贈動作消耗法術位施放另一個。

**反應和附贈動作觸發。** 施法時間為反應的法術是對法術施法時間條目中定義的觸發做出回應而施放的。某些施法時間為附贈動作的法術也是對法術中定義的觸發做出回應而施放。

**更長的施法時間。** 某些法術——包括以儀式施放的法術——需要更多時間施放：數分鐘甚至數小時。當你施放施法時間為 1 分鐘或更長的法術時，你必須在每個回合中使用魔法動作，且必須維持專注。若你的專注被打斷，法術失敗，但不消耗法術位。要再次施放該法術，你必須重頭開始。

### 射程

法術的射程指示法術效果可以從施法者多遠處產生，法術描述會指定效果的哪個部分受射程限制。

射程通常採用以下形式之一：

**距離。** 射程以 m 表示。

**觸碰。** 法術效果產生在施法者觸及範圍內必須觸碰的某物上，如法術所定義。

**自身。** 法術施放在施法者自身上或從施法者散發，如法術所指定。

若法術具有可移動的效果，除非法術描述另有說明，否則它們不受射程限制。

### 成分

法術的成分是施法者施放法術時必須滿足的物理需求。每個法術描述指示是否需要言語（V）、姿勢（S）或材料（M）成分。若施法者無法提供法術的一個或多個成分，則無法施放該法術。

**言語（V）。** 言語成分是對未經啟蒙者聽來如胡言亂語的祕法吟誦。這些話必須以正常說話音量發出。話語本身不是法術力量的來源；而是聲音的特定組合、音調和共鳴啟動了魔法。因此，被堵嘴或在魔法沉默區域中的生物無法施放需要言語成分的法術。

**姿勢（S）。** 姿勢成分是有力的手勢或複雜的動作組合。施法者必須至少使用一隻手來執行這些動作。

**材料（M）。** 材料成分是施放法術時使用的特定材料，在成分條目的括號中指定。除非法術描述另有說明，否則這些材料不會被法術消耗。施法者必須有一隻空手來存取它們，但可以與用於姿勢成分的同一隻手。

若法術不消耗其材料且未指定費用，施法者可以使用成分包替代法術指定的材料，或若施法者有允許替代的特性，則可以使用施法法器。使用成分包需要一隻空手伸入其中，使用施法法器則必須握著它（除非其描述另有說明）。

### 持續時間

法術的持續時間是法術施放後持續的時間長度。持續時間通常採用以下形式之一：

**專注。** 需要專注的持續時間遵循專注規則（見規則詞彙表）。

**瞬間。** 瞬間持續時間意味著法術的魔力只出現一瞬間然後消失。

**時間跨度。** 提供時間跨度的持續時間指定法術以回合、分鐘、小時等為單位持續多長時間。當你施放的時間跨度法術仍在持續時，你可以解除它（無需動作），前提是你沒有失能狀態。

### 效果

法術的效果詳述在其持續時間條目之後。這些細節精確地說明法術做什麼，忽略普通物理法則；超出這些效果的任何結果由城主裁定。無論效果為何，它們通常涉及目標、豁免、攻擊擲骰或三者皆有，以下分別詳述。

**目標。** 典型的法術要求施法者挑選一個或多個受法術魔力影響的目標。法術描述說明法術的目標是生物、物體還是其他東西。

- *通往目標的清晰路徑。* 要以法術指定目標，施法者必須有通往它的清晰路徑，因此它不能在完全遮蔽之後。
- *指定自己為目標。* 若法術指定你選擇的生物，你可以選擇自己，除非該生物必須是敵意的或明確是你以外的生物。
- *效果區域。* 某些法術覆蓋稱為效果區域的範圍，定義於規則詞彙表。效果區域決定法術的目標。法術描述指定它是否有效果區域，通常是以下形狀之一：錐形、立方體、柱體、散發、直線或球體。
- *感知到被指定為目標。* 除非法術有可感知的效果，否則生物不知道自己被法術指定為目標。
- *無效目標。* 若你對無法受影響的人或物施放法術，該目標不會發生任何事，但若你消耗了法術位施放該法術，法術位仍被消耗。

**豁免。** 許多法術指定目標進行豁免以避免法術的部分或全部效果。法術指定目標使用的屬性以及成功或失敗的結果。以下是計算法術豁免 DC 的公式：

> **法術豁免 DC** = 8 + 你的施法屬性修正值 + 你的熟練加值

**攻擊擲骰。** 某些法術要求施法者進行攻擊擲骰以確定法術是否命中目標。以下是計算法術攻擊修正值的公式：

> **法術攻擊修正值** = 你的施法屬性修正值 + 你的熟練加值

### 結合法術效果

不同法術的效果在持續時間重疊時疊加。相反，同一法術多次施放的效果不會結合。取而代之的是，這些施放中最強效的效果——如最高加值——在持續時間重疊時適用。若施放效力相同且持續時間重疊，則適用最近的效果。例如兩位牧師對同一目標施放祝福術，該目標只獲得一次法術效益；目標不會獲得兩顆加值骰。但若法術持續時間重疊，效果持續到第二次祝福術的持續時間結束。

> **辨識持續中的法術。** 你可以嘗試透過可觀察的效果辨識一個非瞬間且持續時間仍在進行的法術。要辨識它，你必須使用研究動作並進行智力（奧祕）檢定（DC 15）。

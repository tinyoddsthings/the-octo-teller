# 戰士（Fighter）

> **來源**：2024 PHB (5.5e) Chapter 3: Character Classes — Fighter
> **翻譯對照**：`glossary/class_features/fighter.md`

*精通所有武器和護甲的大師*

---

## 核心特性

| 項目 | 內容 |
|------|------|
| **核心能力** | 力量（STR）或 敏捷（DEX） |
| **生命骰** | 每戰士等級 d10 |
| **豁免熟練** | 力量（STR）、體質（CON） |
| **技能熟練** | 選 2：特技動作、馴服動物、運動、歷史、洞察、威嚇、說服、觀察、求生 |
| **武器熟練** | 簡易武器 + 軍用武器 |
| **護甲訓練** | 輕甲、中甲、重甲、盾牌 |
| **起始裝備** | 選 A、B 或 C：(A) 鏈甲、巨劍、連枷、標槍 ×8、地城者背包、4 GP；(B) 鑲釘皮甲、彎刀、短劍、長弓、箭 ×20、箭袋、地城者背包、11 GP；或 (C) 155 GP |

戰士統治著許多戰場。無論是尋求聖杯的騎士、皇家冠軍、精銳士兵還是堅韌的傭兵——作為戰士，他們都擁有無與倫比的武器和護甲精通。他們也與死亡很熟悉，無論是給予還是抵抗。

戰士精通各種武器技術，裝備精良的戰士總是有合適的工具應對任何戰鬥情況。同樣，戰士精通各種形式的護甲。除了這種基本的熟悉度，每個戰士都專精於特定的戰鬥風格。

### 成為戰士

**作為 1 級戰士：** 獲得核心特性表中的所有特性和戰士特性表中的 1 級特性。

**作為兼職戰士：** 獲得核心特性表中的：生命骰、軍用武器熟練、輕甲和中甲及盾牌訓練。獲得戰士特性表中的 1 級特性。

## 職業特性

### 1 級：戰鬥風格（Fighting Style）

你磨練了武藝，獲得一個自選的戰鬥風格專長（見第五章）。建議防禦。每次升級時可替換。

### 1 級：第二風（Second Wind）

你有一口有限的體力和精力之泉可以汲取。以附贈動作回復 1d10 + 你戰士等級的 HP。

你可以使用此特性兩次。完成短休恢復一次，完成長休恢復全部。更高等級獲得更多次數（見特性表）。

### 1 級：武器精通（Weapon Mastery）

你的武器訓練允許你使用三種自選的簡易或軍用武器的精通屬性。完成長休後可更換。更高等級獲得更多（見特性表）。

### 2 級：戰術衝刺（Action Surge）

在你的回合中，你可以額外執行一個動作（魔法動作除外）。使用後須完成短休或長休才能再次使用。17 級起，兩次休息間可使用兩次，但每回合只能用一次。

### 2 級：戰術心智（Tactical Mind）

屬性檢定失敗時，你可以消耗一次第二風使用次數。不回復 HP，而是擲 1d10 加到該檢定上，可能使其成功。如果仍然失敗，不消耗第二風。

### 3 級：戰士子職業

你獲得一個自選的戰士子職業：戰鬥大師、鬥士、奧法騎士或靈能戰士（詳見下文）。

### 4 級：屬性值提升

在 6、8、12、14、16 級再次獲得。

### 5 級：額外攻擊（Extra Attack）

攻擊動作時可以攻擊兩次。

### 5 級：戰術轉移（Tactical Shift）

以附贈動作啟動第二風時，你可以移動至多一半速度，不觸發借機攻擊。

### 9 級：不屈（Indomitable）

豁免失敗時可以重骰，加上等於戰士等級的加值，必須使用新結果。須完成長休才能再次使用。13 級起每次長休兩次，17 級起三次。

### 9 級：戰術大師（Tactical Master）

用你可以使用精通屬性的武器攻擊時，你可以用推擊（Push）、削弱（Sap）或減速（Slow）屬性替換該次攻擊的精通屬性。

### 11 級：雙額外攻擊（Two Extra Attacks）

攻擊動作時可以攻擊三次。

### 13 級：研習攻擊（Studied Attacks）

攻擊擲骰未命中生物時，你的下次對該生物的攻擊擲骰（在你下回合結束前）有優勢。

### 19 級：史詩恩賜

建議選擇戰鬥英勇之恩賜。

### 20 級：三額外攻擊（Three Extra Attacks）

攻擊動作時可以攻擊四次。

## 戰士子職業

### 戰鬥大師（Battle Master）

*精通高超戰鬥技巧*

**3 級：戰鬥優勢（Combat Superiority）。**

- **戰技。** 你學會三個自選戰技（見下方「戰技選項」）。每次攻擊只能使用一個戰技。在戰士 7、10、15 級時各多學兩個，且每次可替換一個。
- **優勢骰。** 你有四顆 d8 優勢骰。消耗時用掉，完成短休或長休恢復全部。在戰士 7 級（共五顆）和 15 級（共六顆）時各增加一顆。
- **豁免。** DC = 8 + 你的力量或敏捷修正值（你選）+ 熟練加值。

**3 級：戰學生（Student of War）。** 你獲得一種自選工匠工具的熟練，以及戰士 1 級可選技能中一項的熟練。

**7 級：知彼知己（Know Your Enemy）。** 以附贈動作，你可以辨別 9m內你可見生物的優弱點——得知其是否有任何免疫、抗性或易傷，以及具體是什麼。使用後須完成長休才能再次使用。也可消耗一顆優勢骰恢復。

**10 級：強化戰鬥優勢（Improved Combat Superiority）。** 優勢骰變為 d10。

**15 級：堅韌不拔（Relentless）。** 每回合一次，使用戰技時你可以擲 1d8 代替消耗優勢骰。

**18 級：究極戰鬥優勢（Ultimate Combat Superiority）。** 優勢骰變為 d12。

#### 戰技選項

| 戰技 | 效果摘要 |
|------|---------|
| 伏擊（Ambush） | 敏捷（隱藏）檢定或先攻擲骰 + 優勢骰 |
| 調虎離山（Bait and Switch） | 與 1.5m 內自願生物交換位置，你或它 AC + 優勢骰至下回合 |
| 指揮打擊（Commander's Strike） | 替代一次攻擊，指揮盟友用反應攻擊，傷害 + 優勢骰 |
| 指揮風範（Commanding Presence） | 魅力（威嚇/表演/說服）檢定 + 優勢骰 |
| 繳械攻擊（Disarming Attack） | 傷害 + 優勢骰，目標力量豁免失敗則掉落持有物品 |
| 分心打擊（Distracting Strike） | 傷害 + 優勢骰，其他攻擊者下次對目標攻擊有優勢 |
| 閃避步法（Evasive Footwork） | 以附贈動作撤離 + AC + 優勢骰至下回合 |
| 假動作（Feinting Attack） | 以附贈動作，下次對 1.5m 內目標攻擊有優勢，命中傷害 + 優勢骰 |
| 激怒攻擊（Goading Attack） | 傷害 + 優勢骰，目標感知豁免失敗則對你以外目標攻擊有劣勢 |
| 衝鋒攻擊（Lunging Attack） | 以附贈動作衝刺，直線移動 1.5m 後近戰命中傷害 + 優勢骰 |
| 調度攻擊（Maneuvering Attack） | 傷害 + 優勢骰，一名盟友可用反應移動一半速度不觸發借機攻擊 |
| 威嚇攻擊（Menacing Attack） | 傷害 + 優勢骰，目標感知豁免失敗則驚懼至下回合 |
| 格擋（Parry） | 被近戰攻擊傷害時用反應，傷害減少優勢骰 + 力量或敏捷修正值 |
| 精準攻擊（Precision Attack） | 攻擊未命中時，+ 優勢骰可能使其命中 |
| 推擊攻擊（Pushing Attack） | 傷害 + 優勢骰，大型或更小目標力量豁免失敗則推離至多 4.5m |
| 鼓舞（Rally） | 以附贈動作，9m 內盟友獲得優勢骰 + 戰士等級一半的臨時 HP |
| 回刺（Riposte） | 被近戰攻擊未命中時用反應攻擊，命中傷害 + 優勢骰 |
| 橫掃攻擊（Sweeping Attack） | 命中時消耗優勢骰，原目標 1.5m 內另一生物受優勢骰傷害 |
| 戰術評估（Tactical Assessment） | 智力（歷史/調查）或感知（洞察）檢定 + 優勢骰 |
| 絆倒攻擊（Trip Attack） | 傷害 + 優勢骰，大型或更小目標力量豁免失敗則倒地 |

### 鬥士（Champion）

*追求極致的肉體卓越*

**3 級：精進致命（Improved Critical）。** 武器和徒手打擊的攻擊擲骰在 d20 上 19 或 20 即為爆擊。

**3 級：卓越運動員（Remarkable Athlete）。** 先攻擲骰和力量（運動）檢定有優勢。爆擊後可移動至多一半速度，不觸發借機攻擊。

**7 級：額外戰鬥風格（Additional Fighting Style）。** 你獲得另一個戰鬥風格專長。

**10 級：英勇戰士（Heroic Warrior）。** 戰鬥中，每回合開始時如果你沒有英雄靈感，你可以給自己英雄靈感。

**15 級：超凡致命（Superior Critical）。** 武器和徒手打擊的攻擊擲骰在 d20 上 18-20 即為爆擊。

**18 級：倖存者（Survivor）。**

- **抗拒死亡。** 死亡豁免有優勢，且 18-20 視為骰出 20。
- **英勇振作。** 每回合開始時，如果你處於重傷且至少有 1 HP，回復 5 + 體質修正值的 HP。

### 奧法騎士（Eldritch Knight）

*以奧術魔法輔助戰鬥技能*

**3 級：施法。** 你從法師法術列表中學習法術。智力（INT）為施法屬性。初始選三個 1 環法術（建議燃燒之手、跳躍術、護盾術）。奧術法器為施法焦點。

**3 級：戰爭契約（War Bond）。** 花 1 小時（可在短休中）與武器建立魔法連結。連結後不會被繳械，且可以附贈動作召喚到手中。最多連結兩把武器。

**7 級：戰爭魔法（War Magic）。** 攻擊動作中可以用施法時間為一個動作的法師戲法替代一次攻擊。

**10 級：奧法打擊（Eldritch Strike）。** 用武器命中生物時，該生物在你下回合結束前對你施放的下一個法術的豁免有劣勢。

**15 級：奧法衝鋒（Arcane Charge）。** 使用戰術衝刺時，你可以傳送至 9m內你可見的空位（額外動作之前或之後）。

**18 級：強化戰爭魔法（Improved War Magic）。** 攻擊動作中可以用一個 1 或 2 環法師法術替代兩次攻擊。

### 靈能戰士（Psi Warrior）

*以靈能之力增強肉體力量*

**3 級：靈能之力（Psionic Power）。** 你有靈能能量骰池。

| 戰士等級 | 骰型 | 數量 |
|---------|------|------|
| 3 | d6 | 4 |
| 5 | d8 | 6 |
| 9 | d8 | 8 |
| 11 | d10 | 8 |
| 13 | d10 | 10 |
| 17 | d12 | 12 |

完成短休恢復一顆，完成長休恢復全部。

- **防護力場（Protective Field）。** 你或 9m內你可見的生物受傷時，你可以用反應消耗一顆靈能骰，減少傷害 = 擲骰 + 智力修正值（至少減 1）。
- **靈能打擊（Psionic Strike）。** 每回合一次，命中 9m內目標並造成傷害後，你可以消耗一顆靈能骰，造成額外力場傷害 = 擲骰 + 智力修正值。
- **念動移動（Telekinetic Movement）。** 以魔法動作，選擇 9m內你可見的一個目標（大型或更小的鬆散物品，或你以外的自願生物），傳送至多 9m到空位。或者，微型物品可傳送到/從你手中。使用後須完成短休或長休，除非消耗一顆靈能骰恢復。

**7 級：念動精通（Telekinetic Adept）。**

- **靈能跳躍（Psi-Powered Leap）。** 以附贈動作獲得等於速度兩倍的飛行速度至當前回合結束。須完成短休或長休，除非消耗靈能骰恢復。
- **念動推擊（Telekinetic Thrust）。** 靈能打擊造成傷害時，可以迫使目標力量豁免（DC 8 + 智力修正值 + 熟練加值），失敗則倒地或被水平傳送至多 3m。

**10 級：守護心靈（Guarded Mind）。** 你擁有心靈傷害抗性。回合開始時如果處於魅惑或驚懼狀態，你可以消耗一顆靈能骰結束你身上的這些狀態。

**15 級：力場壁壘（Bulwark of Force）。** 以附贈動作，選擇 9m內至多等於智力修正值數量的生物（包含自己），每個獲得半掩蔽 1 分鐘或直到你失能。須完成長休，除非消耗靈能骰恢復。

**18 級：念動宗師（Telekinetic Master）。** 你始終備妥念力術（Telekinesis）。可以不消耗法術位或成分施放（智力為施法屬性）。維持專注期間（包括施放回合），每回合可以附贈動作以武器攻擊一次。以此方式施放後須完成長休，除非消耗靈能骰恢復。

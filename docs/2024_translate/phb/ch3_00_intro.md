# 第三章：角色職業

> **來源**：2024 PHB (5.5e) Chapter 3: Character Classes
> **翻譯對照**：`src/tot/data/glossary/dnd_glossary.md` 及其子檔

---

角色職業提供角色最令人興奮的能力。本章提供十二個職業，每個職業包含四個子職業——概述如下。

**野蠻人（Barbarian）。** 以狂暴之力衝鋒陷陣，投入肉搏戰。然後追隨……
- **狂戰士之路**，釋放原始暴力。
- **野性之心之路**，展現與動物的血脈關聯。
- **世界樹之路**，汲取宇宙生命力。
- **狂信者之路**，與神明合一地狂暴。

**吟遊詩人（Bard）。** 施展鼓舞、治療盟友或迷惑敵人的法術。然後加入……
- **舞蹈學院**，在戰鬥中駕馭敏捷。
- **魅惑學院**，編織蠱惑的妖精荒野魔法。
- **知識學院**，蒐集知識和魔法秘密。
- **勇氣學院**，以武器配合法術戰鬥。

**牧師（Cleric）。** 召喚神聖魔法來治療、增強和懲擊。然後駕馭……
- **生命領域**，成為治療大師。
- **光明領域**，揮舞灼熱守護之光。
- **詭術領域**，以惡作劇迷惑敵人。
- **戰爭領域**，激勵英勇並懲罰敵人。

**德魯伊（Druid）。** 引導自然魔法來治療、變形和控制元素。然後加入……
- **大地結社**，汲取環境魔法。
- **月亮結社**，採取強大的動物形態。
- **海洋結社**，引導潮汐與風暴。
- **星辰結社**，在星辰形態中獲得力量。

**戰士（Fighter）。** 精通所有武器和護甲。然後化身……
- **戰鬥大師**，使用特殊戰技。
- **鬥士**，追求極致的戰鬥實力。
- **奧法騎士**，學習輔助戰鬥的法術。
- **靈能戰士**，以靈能之力增強攻擊。

**武僧（Monk）。** 在近戰中快速穿梭、迅猛打擊。然後成為……
- **慈悲武者**，以觸碰治療或傷害。
- **暗影武者**，運用暗影進行隱匿行動。
- **元素武者**，揮舞元素之力。
- **開掌武者**，精通徒手戰鬥。

**聖騎士（Paladin）。** 以神聖和武力的力量懲擊敵人、護衛盟友。然後立下……
- **奉獻誓約**，效法正義天使。
- **榮耀誓約**，攀登英雄主義的巔峰。
- **遠古誓約**，守護生命、喜悅和自然。
- **復仇誓約**，追捕作惡者。

**遊俠（Ranger）。** 編織武藝、自然魔法和求生技能。然後化身……
- **馴獸師**，與原始野獸建立羈絆。
- **精靈漫遊者**，展現精類的歡愉與狂怒。
- **幽暗獵人**，狩獵潛伏黑暗中的敵人。
- **獵人**，以武藝的多樣性守護自然。

**游蕩者（Rogue）。** 透過隱匿避免傷害，發動致命的偷襲。然後化身……
- **秘法騙徒**，以法術強化隱匿。
- **刺客**，進行伏擊和使用毒藥。
- **靈刃**，以靈能之刃打擊敵人。
- **神偷**，精通潛入和尋寶。

**術士（Sorcerer）。** 揮舞你存在中與生俱來的魔法，塑造力量為己所用。然後引導……
- **異怪術法**，使用奇異的靈能魔法。
- **鐘律術法**，駕馭宇宙秩序的力量。
- **龍族術法**，呼吸龍的魔法。
- **狂野魔法**，釋放混沌魔法。

**契術師（Warlock）。** 施展源自神祕學識的法術。然後與……締約。
- **妖精宗主**，傳送和使用妖精魔法。
- **天界宗主**，以天界魔法治療。
- **邪魔宗主**，召喚邪惡力量。
- **遠古者宗主**，深入禁忌知識。

**法師（Wizard）。** 研究奧術魔法，掌握各種用途的法術。然後化身……
- **防護師**，護衛盟友、放逐敵人。
- **預言師**，探知多元宇宙的秘密。
- **塑能師**，創造爆裂性效果。
- **幻術師**，編織欺騙的法術。

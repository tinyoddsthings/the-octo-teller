# 附錄 C：生物數據表

> **來源**：2024 PHB (5.5e) Appendix B: Creature Stat Blocks

---

本附錄提供書中其他章節（特別是職業、裝備和法術章節）中提及的生物數據表。有關如何閱讀數據表的說明，請參閱規則詞彙表中的「數據表」條目；更多生物請參閱《怪物圖鑑》。

以下數據表依英文字母順序排列。當地下城主使用數據表時，DM 可以更改其中的細節。

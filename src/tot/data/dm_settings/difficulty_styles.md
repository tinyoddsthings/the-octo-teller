# 難度系統（Difficulty）

> 本系統透過「暗骰」和「作骰（Fudge）」調整遊戲體驗，**不更動角色或怪物數值**。
> 每個戰役的風格記錄在該戰役的 `campaign.md`。

---

## 快速總覽

| 風格 | 骰子公開度 | 微調方向 | DC 調整 | 滅團風險 | 適合玩家 |
|------|-----------|---------|---------|---------|---------|
| 菜雞（Beginner） | 暗骰為主 | 偏向幫助 | 可降 1~2 | 極低 | 第一次玩 TTRPG |
| 傳統（Traditional） | 全部公開 | **禁止微調** | 不調 | RAW 標準 | 想體驗純規則 |
| 效果（Dramatic） | 暗骰為主 | 雙向（戲劇化） | 動態 ±1~2 | 中等 | 重視故事張力 |
| 高手（Expert） | 暗骰為主 | 偏向增難 | 可升 1~2 | 高 | 想要硬核挑戰 |

---

## 微調規則

### 上限表

| 骰子 | 最大微調幅度 | 說明 |
|------|-------------|------|
| d4 ~ d12 | ±1 | 傷害骰、生命骰等 |
| d20 | ±2 | 攻擊、豁免、技能檢定等 |
| d100 | ±5 | 百分比骰（遭遇表等） |

### 三原則

1. **敘事包裝自然** — 微調結果必須用敘事包裝，讓玩家感覺是自然發生的事件
2. **不改數值** — 永遠不修改角色數值、怪物數值、物品屬性或法術效果
3. **玩家不應察覺** — 如果微調幅度大到玩家會覺得不合理，就不該微調

### 敘事範例

| 情境 | 原始結果 | 微調 | 敘事包裝 |
|------|---------|------|---------|
| 怪物攻擊重傷玩家（菜雞模式） | 骰到 18 命中 | 改為 16 未命中 | 「哥布林揮砍過來，但被你的盾牌邊緣彈開了！」 |
| 玩家在關鍵劇情擲了低骰（效果模式） | 感知檢定骰 8 | 改為 10 通過 DC 10 | 「你幾乎錯過了，但一陣冷風吹來引起你的注意——牆上有道裂縫。」 |
| 戰鬥太輕鬆（效果模式） | 怪物攻擊骰 9 未命中 | 改為 11 命中 | 「狼群中最大的那隻突然變換角度，牠的爪子劃過你的手臂！」 |
| 玩家覺得太簡單（高手模式） | 陷阱傷害 d6 骰 2 | 改為 3 | 「毒針刺入更深，毒液在血管中蔓延。」 |

## 各風格詳細行為

### 菜雞（Beginner）

> 新手友善，讓玩家享受英雄感。DM 暗中保護，避免挫折體驗。

| 項目 | DM 行為 |
|------|---------|
| **骰子** | DM 暗骰（怪物攻擊、傷害）；玩家自己的骰公開 |
| **微調方向** | 偏向幫助玩家（降低怪物命中、降低傷害） |
| **例外** | 玩家明顯順風時停止微調，讓骰子正常決定 |
| **滅團處理** | 積極避免：怪物會犯戰術錯誤（攻擊分散、忽略倒地角色）、增援延遲到達 |
| **DC 調整** | 可降低 1~2（偷偷的） |
| **資訊提供** | 可主動提示：「你覺得可以試試感知檢定」「這個門看起來可以用盜賊工具打開」 |
| **何時停止微調** | 隊伍 HP 都在 50% 以上、戰鬥勝券在握時 |

### 傳統（Traditional）

> 純 RAW（Rules As Written）。骰子決定一切，DM 不介入結果。

| 項目 | DM 行為 |
|------|---------|
| **骰子** | 全部公開擲骰 |
| **微調方向** | **完全禁止**，不論情況 |
| **例外** | 無 |
| **滅團處理** | 不干預。如果骰子說你死，你就死。玩家需要靠策略和資源管理存活 |
| **DC 調整** | 嚴格按模組數值，不調整 |
| **資訊提供** | 只回答玩家主動詢問的問題，不主動提示 |

### 效果（Dramatic）

> 戲劇張力最大化。DM 雙向微調，讓故事更有起伏。

| 項目 | DM 行為 |
|------|---------|
| **骰子** | DM 暗骰；關鍵時刻可公開擲以增加緊張感 |
| **微調方向** | 雙向 — 太順時增難，太慘時拉回 |
| **例外** | 玩家做出精彩的角色扮演或戰術決策時，傾向讓結果更好 |
| **滅團處理** | 不會讓滅團在無趣的情況下發生；但如果是史詩級的最終戰，可以讓角色壯烈犧牲 |
| **DC 動態調整** | 見下方決策樹 |
| **資訊提供** | 視劇情需要。重要線索不會因一次失敗的檢定就永遠錯過 |

**DC 動態調整決策樹（效果模式專用）：**

```
戰鬥局勢評估
├── 碾壓（玩家方明顯優勢）
│   └── DC/怪物攻擊 +1~2（暗中）
│       → 讓戰鬥多持續幾回合，增加張力
├── 均勢（雙方互有來回）
│   └── 不調整
│       → 已經很有戲劇性了
└── 掙扎（玩家方劣勢）
    └── DC/怪物攻擊 -1~2（暗中）
        → 給玩家一線生機，但不要讓他們察覺
```

### 高手（Expert）

> 高挑戰、策略導向。DM 偏向增加難度，回報好的策略。

| 項目 | DM 行為 |
|------|---------|
| **骰子** | DM 暗骰 |
| **微調方向** | 偏向增難（提高怪物命中、增加傷害） |
| **例外** | 玩家展現出色策略時，不微調或微調幫助（獎勵好策略） |
| **滅團處理** | 不主動避免。但會給玩家公平的逃跑/談判機會 |
| **DC 調整** | 可提高 1~2 |
| **資訊提供** | 嚴格限制。不提示、不暗示。玩家必須主動調查、詢問 NPC、推理 |
| **怪物戰術** | 使用最佳戰術：集火弱目標、保護施法者、利用地形、會撤退重整 |

## DM 決策流程

> 每次需要擲骰或判定結果時，按以下流程操作。

```
1. 確認當前風格
   └── 讀取戰役 campaign.md 的風格設定

2. 擲骰
   ├── 傳統 → 公開擲骰，結果即最終結果 → 結束
   └── 其他風格 → 暗骰（DM 心中決定結果）

3. 評估是否需要微調
   ├── 結果已經符合風格方向 → 不微調 → 跳到 5
   └── 結果不符合 → 進入 4

4. 微調判定
   ├── 檢查微調幅度是否在上限內
   │   ├── 超過上限 → 不微調，接受原始結果
   │   └── 在上限內 → 繼續
   ├── 檢查三原則
   │   ├── 能自然包裝？ → 繼續
   │   └── 包裝會很牽強？ → 不微調
   └── 執行微調

5. 敘事包裝
   └── 用劇情描述呈現結果（不論是否有微調，都要有敘事）

6. 記錄（DM 內部）
   └── 記住本場已微調的次數，避免過度使用
       ├── 菜雞 / 高手：每場戰鬥最多微調 3~4 次
       └── 效果：每場戰鬥最多微調 2~3 次（雙向合計）
```

## 風格切換

遊戲中任何時候，玩家可說「切換風格 XX」來調整：

- DM 更新 `campaign.md` 的風格設定
- 立即套用新風格行為
- 戰鬥中途切換也可以，下一次擲骰就生效

> 💡 建議：新手先用「菜雞」熟悉規則，之後轉「效果」或「傳統」。

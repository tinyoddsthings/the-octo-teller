# DM 敘事風格系統（Narrative Style）

> 本系統控制 DM 的**文字輸出風格**，與難度系統（`difficulty_styles.md`）正交。
> 難度 = 骰子機制（作骰/DC 調整）；敘事 = 文字輸出（描述怎麼寫、NPC 怎麼講話）。
> 兩者獨立設定，自由組合。每個戰役的風格記錄在該戰役的 `campaign.md`。

---

## 快速總覽

| 風格 | 核心調性 | 描述長度 | NPC 對話風格 | 適合氛圍 |
|------|---------|---------|-------------|---------|
| 簡潔（Concise） | 短句、重點、快節奏 | 1~2 句 | 直來直往，不廢話 | 戰鬥密集、節奏優先 |
| 小說（Literary） | 感官細節、比喻、文學腔 | 3~5 句 | 個性鮮明，有口頭禪和語氣詞 | 探索、角色互動 |
| 電影（Cinematic） | 鏡頭感、慢動作、特寫切換 | 3~4 句 | 像電影台詞，節奏明快有張力 | 史詩戰鬥、Boss 戰 |
| 搞笑（Comedic） | 吐槽、輕鬆、偶爾打破第四面牆 | 2~4 句 | 各種口音、諧音梗、吐槽 | 輕鬆跑團、搞笑場景 |
| 黑暗（Gritty） | 殘酷寫實、強調代價 | 3~5 句 | 粗獷、疲憊、帶有世故感 | 生存恐怖、道德灰色 |
| 言情（Romance） | 誇張情感反應、曖昧描寫 | 3~5 句 | 帶有情感張力，語氣暗示多 | 角色互動、NPC 社交 |

## 各風格詳細規範

### 簡潔（Concise）

> 短句重點，快節奏。每次描述 1~2 句，讓玩家快速決策。

| 面向 | DM 行為 |
|------|---------|
| **戰鬥描述** | 命中：「劍命中，7 點傷害。」未中：「砍空了。」法術：「魔能轟擊命中，8 點力場傷害。」倒地：「[角色A]倒下了。」 |
| **環境敘述** | 進入：「石造走廊，空氣潮濕，前方分岔。」陷阱：「地板塌陷，敏捷豁免。」天氣：「開始下雨。」 |
| **NPC 對話** | 友善：「歡迎，需要什麼？」敵對：「滾。」商人：「這把劍 50 金，要不要？」 |
| **檢定結果** | 成功：「你注意到牆上有暗門。」失敗：「什麼都沒發現。」大成功：「你不但找到暗門，還發現機關位置。」大失敗：「你踩到陷阱了。」 |
| **描述長度** | 每次 1~2 句，戰鬥中盡量一句解決 |

### 小說（Literary）

> 感官細節、比喻、文學腔。預設風格，適中長度。

| 面向 | DM 行為 |
|------|---------|
| **戰鬥描述** | 命中：「劍身劃過空氣時發出嗡鳴，刃尖找到了[怪物]肋骨間的縫隙——牠發出一聲濕潤的嘶吼。」未中：「你的劍劃過牠的耳邊，帶起幾根粗硬的毛髮。」法術：「紫色的能量在你指尖凝聚，帶著灼熱的嘶嘶聲射向目標。」倒地：「[角色A]的膝蓋終於撐不住了，他像一棵被砍斷的橡樹般倒下，鎧甲與石地碰撞的聲響迴盪在洞穴中。」 |
| **環境敘述** | 進入：「走廊的石牆上長滿了青苔，空氣中瀰漫著潮濕和某種腐敗的甜膩。你們的腳步聲被不均勻的地面吞噬，只有遠處偶爾傳來的滴水聲打破沉默。」陷阱：「你的腳踩下的瞬間，石板發出一聲不祥的喀嚓——重力比你的反應更快。」天氣：「天空像是被人潑了一桶灰色的墨水，第一滴雨落在你的鼻尖上，冰涼而突然。」 |
| **NPC 對話** | 友善：「老人抬起佈滿皺紋的臉，混濁的眼中閃過一絲溫暖：『旅人，你們看起來走了很遠的路。坐，喝口水再說。』」敵對：「他把匕首插進桌面，木屑飛濺：『你有三秒鐘解釋你為什麼出現在這裡。』」商人：「矮人商人用布仔細擦拭著劍身，像是在撫摸一個老朋友：『五十金。少一個銅板你就是在侮辱這把劍的鍛造師。』」 |
| **檢定結果** | 成功：「你的指尖沿著牆面滑過時，觸感突然變了——某塊石頭比其他的更光滑，邊緣有不自然的縫隙。這裡有一扇暗門。」失敗：「你仔細端詳了每一塊石頭，但它們看起來都一樣。也許這面牆真的就只是一面牆。」大成功：「你不只找到了暗門——你還注意到門框邊緣有細微的刮痕，有人最近才用過。而且，鉸鏈在左邊，機關應該在右側石塊。」大失敗：「你自信地推了一塊看起來可疑的石頭。它確實動了——連帶著觸發了你完全沒注意到的壓力板。」 |
| **描述長度** | 每次 3~5 句，重要場景可以更長 |

### 電影（Cinematic）

> 鏡頭感、慢動作、特寫切換。像在看電影。

| 面向 | DM 行為 |
|------|---------|
| **戰鬥描述** | 命中：「鏡頭拉近你的手腕——肌肉繃緊，劍向前刺出。慢動作中，刃尖撕開[怪物]的皮甲，血珠在空氣中畫出一道弧線。」未中：「特寫你的眼神，瞳孔收縮——劍揮出，但[怪物]在最後一刻側身，鏡頭跟著刀鋒劃過空氣。」法術：「你的手掌張開，鏡頭從指縫間拍過去——紫光迸射而出，畫面切到[怪物]臉上的驚恐表情，然後是衝擊。」倒地：「慢動作：[角色A]的盾牌從手中滑落，在地上彈跳兩次。他的膝蓋觸地。鏡頭繞到正面，拍到他嘴角的血。」 |
| **環境敘述** | 進入：「長鏡頭從隊伍背後跟拍，穿過狹窄的走廊——牆上的火把一盞接一盞掠過。鏡頭推進，停在前方的黑暗中。有什麼東西在動。」陷阱：「俯拍鏡頭：你的靴子踩下石板。特寫石板下沉一公分。切到你的臉——瞳孔放大。然後地板消失了。」天氣：「航拍鏡頭拉高，灰色的雲層從地平線湧來，吞沒了最後一片藍天。切回地面，第一滴雨打在你的劍鞘上。」 |
| **NPC 對話** | 友善：「鏡頭推進到老人的臉——皺紋像乾涸的河床。他慢慢抬頭，逆光中瞇起眼睛：『你們……看起來不像普通的旅人。』」敵對：「過肩鏡頭：他的手按上刀柄，指關節發白。低角度仰拍他的臉：『最後一次。你們是誰派來的？』」商人：「特寫：矮人的手指在劍身上滑過，金屬反光掃過他的臉。他抬眼：『五十金。別討價還價，我不喜歡浪費鏡頭。』」 |
| **檢定結果** | 成功：「特寫你的手指沿牆面滑動——指尖停住。鏡頭切到石頭的微距特寫：那道縫隙清晰可見。」失敗：「你的手在牆面上來回摸索，鏡頭拉遠，走廊空蕩蕩的。什麼都沒有。」大成功：「快速剪接：手指觸碰縫隙——切到機關特寫——切到你的微笑。你找到了一切。」大失敗：「你的手按上石塊。定格一秒。然後——鏡頭劇烈搖晃，碎石紛飛。」 |
| **描述長度** | 每次 3~4 句，用鏡頭語言取代長篇敘述 |

### 搞笑（Comedic）

> 吐槽、輕鬆、偶爾打破第四面牆。不影響機制嚴謹度。

| 面向 | DM 行為 |
|------|---------|
| **戰鬥描述** | 命中：「你一劍戳中[怪物]的肚子，牠看了看傷口，臉上寫滿了『我今天不該來上班』。」未中：「你用力揮劍——[怪物]往旁邊歪了一下，你的劍砍進了一根完全無辜的石柱。」法術：「你射出魔能轟擊，[怪物]被打飛三公尺。牠的同伴看了看，默默往後退了一步。」倒地：「[角色A]終於倒下了，鎧甲碰地的聲音大到隔壁洞穴的蝙蝠都被嚇醒。」 |
| **環境敘述** | 進入：「你們走進一條走廊。它很暗、很濕、很臭。基本上就是地下城的標準配備。」陷阱：「地板塌了。在所有人都很驚訝的情況下，地板塌了。」天氣：「開始下雨了，因為當然要下雨，你們剛好沒帶傘。」 |
| **NPC 對話** | 友善：「老人熱情地招手：『來來來！坐！喝茶！吃餅！……你們有帶錢吧？』」敵對：「他拔出刀，頓了一下，換了一把更大的刀：『這把比較有說服力。』」商人：「矮人把劍舉高：『五十金！』看到你們的表情後：『……四十九？不，五十。我說了五十。』」 |
| **檢定結果** | 成功：「你摸到了暗門。好吧，嚴格來說你是臉先撞上去的，但結果一樣。」失敗：「你仔細檢查了牆壁。恭喜，你現在知道它是用石頭砌的。」大成功：「你不只找到了暗門，你還聞到門後面有人在烤肉。情報就是這麼來的。」大失敗：「你推了一塊石頭。石頭推了你一下。石頭贏了。」 |
| **描述長度** | 每次 2~4 句，笑點比字數重要 |

### 黑暗（Gritty）

> 殘酷寫實、強調代價。每一場戰鬥都有重量。

| 面向 | DM 行為 |
|------|---------|
| **戰鬥描述** | 命中：「劍刺穿腹部，溫熱的血液沾濕你的手。[怪物]發出像是在哭的聲音，雙手徒勞地抓著劍身。」未中：「你劈空了。手臂的慣性帶著你向前踉蹌了一步，露出了破綻。」法術：「魔能擊中[怪物]的胸口——牠的肋骨碎裂的聲音清晰可聞。牠倒下時嘴裡冒出血沫。」倒地：「[角色A]跪倒在地。他的呼吸急促而淺，每一次吸氣都帶著濕潤的咕噥聲。血從鎧甲的接縫處滲出來，在石地上匯成暗紅色的水窪。」 |
| **環境敘述** | 進入：「走廊的空氣沉重得像是有實體。牆壁上的某種汙漬已經乾涸成深褐色——你不確定那是鏽還是血。角落裡有一堆破布，走近了才發現那曾經是一個人。」陷阱：「石板塌陷的那一刻，你聽到自己的腳踝發出一聲脆響。疼痛比你想像的來得更快。」天氣：「雨水夾帶著冰碴打在你的臉上。你的手指已經凍得快要握不住劍柄了。」 |
| **NPC 對話** | 友善：「老人的手在顫抖。他倒水的時候灑了一半在桌上：『坐吧。這年頭……路上不安全。你們還活著就是好事。』」敵對：「他沒有拔刀。他甚至沒有看你們。『你們不是第一批來的。上一批……嗯。你想知道他們的下場嗎？』」商人：「矮人把劍放在櫃檯上，刀刃上還有沒擦乾淨的暗漬：『五十金。這把劍見過比你更硬的骨頭。』」 |
| **檢定結果** | 成功：「你的手指摸到了縫隙。石頭邊緣有指甲的刮痕——有人曾經拼命想從裡面打開這扇門。」失敗：「你什麼都沒找到。只有潮濕的石頭和自己越來越急促的呼吸聲。」大成功：「暗門後面有一條通道。地上有拖行痕跡，還有一股淡淡的腐臭。有人被拖進去了，而且不是很久以前。」大失敗：「你推動石塊的瞬間，一根生鏽的鐵釘刺穿了你的手掌。你咬住嘴唇，嚐到了自己的血。」 |
| **描述長度** | 每次 3~5 句，細節聚焦於痛苦、代價、不安 |

### 言情（Romance）

> 誇張情感反應、曖昧描寫、角色互動戲劇化。尺度：輕度~中度（臉紅嬌喘曖昧比喻，不露骨）。

| 面向 | DM 行為 |
|------|---------|
| **戰鬥描述** | 命中：「劍尖刺入的瞬間，[怪物]身體一陣抽搐，臉頰泛紅，嬌喘一聲癱倒在地。」未中：「你的劍擦過牠的臉頰，留下一道淺淺的紅痕。[怪物]驚喘一聲，後退兩步，用手捂住那道傷痕，眼神複雜。」法術：「紫色的能量纏繞上[怪物]的身體——牠的背脊弓起，喉嚨裡溢出一聲壓抑的呻吟。」倒地：「[角色A]的身體緩緩傾倒。[角色B]衝上前接住他，手指穿過他沾滿汗水的髮絲：『你不准倒……聽到沒？』」 |
| **環境敘述** | 進入：「走廊深處傳來若有似無的嘆息聲，牆壁上的苔蘚在火光中泛著曖昧的綠，像是某種邀請。空氣溫暖而潮濕，帶著說不清的芬芳。」陷阱：「石板塌陷——你失去平衡的瞬間，身旁的同伴一把抓住你的手腕。你們四目相交，心跳加速。『……小心點。』」天氣：「雨絲落在你的睫毛上，世界變得朦朧。身旁的人湊近了一步，替你擋住風。」 |
| **NPC 對話** | 友善：「老人看著你們的眼神柔軟了下來：『多久沒看到這麼好看的冒險者了……進來吧。今晚的燉湯是為你們準備的。』」敵對：「他慢慢走近，靴跟在地上敲出不緊不慢的節奏。他的手指勾起你的下巴：『如果你不是敵人……我其實挺想請你喝一杯的。』」商人：「矮人的手指在劍身上流連：『五十金。你摸摸這個手感——滑順得就像……嗯。你懂的。』」 |
| **檢定結果** | 成功：「你的指尖滑過石壁——在某個位置，你感覺到了一陣微弱的暖意。你屏住呼吸，輕輕按下。暗門悄無聲息地為你打開了，像是在回應你的觸碰。」失敗：「你的手在冰冷的石壁上徘徊，什麼也沒找到。牆壁對你保持沉默，拒絕回應。」大成功：「石門在你的觸碰下發出一聲輕顫——機關、通道、門後微弱的光，一切都在你的指尖下坦誠相見。」大失敗：「你用力推動石塊。它紋絲不動，但你的手指被夾住了。在你吃痛抽手的瞬間，背後傳來一聲低笑。」 |
| **描述長度** | 每次 3~5 句，重點在情感張力和感官曖昧 |

## 範例段落 — 進入[怪物]巢穴

> 同一個場景，六種風格各寫一段。

### 簡潔（Concise）

洞穴入口很窄。裡面很暗，有臭味。你們聽到前方有動靜。

### 小說（Literary）

洞穴的入口像一張被撕裂的嘴，邊緣的岩石參差不齊，上面掛著枯黃的藤蔓。一股混合了腐肉和濕毛皮的氣味從黑暗中湧出，幾乎讓你想退後一步。你的火把照亮了最初幾公尺的通道——牆壁上有爪子留下的刮痕，地面散落著碎骨。更深處，有什麼東西在移動，發出細碎的、像是指甲刮地板的聲音。

### 電影（Cinematic）

鏡頭從隊伍身後緩緩推進，穿過藤蔓遮蔽的洞口。光線驟暗。畫面切到火把的特寫——火焰在氣流中搖曳，映出牆壁上密密麻麻的爪痕。鏡頭下搖到地面：碎骨。再推進——黑暗的盡頭，兩點微弱的黃光閃了一下。然後消失了。

### 搞笑（Comedic）

洞穴入口飄出一股令人感動的惡臭，大概是[怪物]們的家常菜。你們走進去的時候，地上的碎骨嘎吱作響——要嘛是陷阱，要嘛是上一組冒險者。裡面傳來窸窸窣窣的聲音。可能是[怪物]，也可能是一隻非常大的老鼠。不管是哪個，都不太想見你們。

### 黑暗（Gritty）

洞口的岩壁上有深褐色的汙漬，從高處一直延伸到地面。不需要是追蹤者也知道那是什麼。空氣又濕又重，每吸一口都帶著一股甜膩的腐臭——那是屍體放了幾天之後的味道。你的火把照亮了地面：碎骨、毛皮、還有一隻靴子。裡面還有腳。更深處傳來一聲低沉的嘶吼，然後是什麼東西被拖行的聲音。

### 言情（Romance）

洞穴的入口被藤蔓半掩著，像是刻意遮擋的帷幕。潮濕的氣息從深處湧出，帶著一種原始的、近乎野性的味道。你走進去的時候，手臂不小心碰到了身旁同伴的手背——對方沒有移開。火光搖曳中，牆壁上的陰影像是在低語。更深處傳來輕柔的聲響，像是嘆息，又像是呼喚。

## 混搭提示 — 敘事 × 難度

> 敘事風格和難度風格可以自由組合。以下是一些有趣的搭配。

| 組合 | 體驗描述 |
|------|---------|
| 簡潔 × 傳統 | 純粹的桌遊體驗，快速、乾淨、規則優先 |
| 小說 × 效果 | 最「標準」的 TTRPG 體驗，故事和骰子都有戲劇性 |
| 電影 × 高手 | 硬核動作電影，每場戰鬥都是生死關頭 |
| 搞笑 × 菜雞 | 歡樂派對，適合朋友聚會輕鬆玩 |
| 黑暗 × 高手 | 末日求生，每個選擇都有代價，不適合心臟不好的人 |
| 言情 × 效果 | 硬核挑戰但[怪物]會嬌喘，戲劇張力和情感張力雙管齊下 |
| 搞笑 × 高手 | 表面搞笑但真的會死，笑著笑著就哭了 |

## 風格切換

遊戲中任何時候，玩家可說「切換敘事 XX」來調整：

- DM 更新 `campaign.md` 的敘事風格設定
- 立即套用新風格
- 戰鬥中途切換也可以，下一段描述就生效

> 預設風格：小說（Literary）— 中等長度、最通用，新手不會覺得太短或太長。

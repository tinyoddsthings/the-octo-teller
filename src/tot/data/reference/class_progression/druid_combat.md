# 德魯伊（Druid）戰鬥 SOP

> 本文件是德魯伊的**戰鬥決策框架**。
> 具體法術/能力從角色卡讀取，套用到以下流程。
> 面板格式見 [combat_flow.md](../rules/combat_flow.md) C 節

## A. 職業戰鬥定位

| 項目 | 說明 |
|------|------|
| 戰鬥角色 | 控場/治療/多功能（Versatile Controller & Healer）|
| 主砲 | 專注控場法術（糾纏術（Entangle）、月光術（Moonbeam）等）|
| 核心機制 | **準備施法者**：長休可更換全部法術列表 + **野性形態（Wild Shape）** 雙模式作戰 |
| 設計哲學 | 每日法術依預期場景準備；控場 > 治療 > 直傷。野性形態是額外生命值（HP）池 + 偵查工具 |

### 德魯伊 vs 牧師（兩大感知（WIS）準備施法者）

| | 德魯伊 | 牧師 |
|---|--------|------|
| 護甲 | 輕甲 + 盾牌（守護者可中甲）| 中甲 + 盾牌（部分子職可重甲）|
| 生存機制 | 野性形態額外生命值（HP）池 | 較高防禦等級（AC）+ 治療法術 |
| 控場能力 | **極強**（糾纏、荊棘叢生、月光） | 中等（靈體守衛、束縛） |
| 治療能力 | 中等（療傷、治療之言） | **極強**（專職治療者） |
| 召喚能力 | **強**（召喚動物群、召喚精類） | 弱 |
| 長休靈活度 | 可完全更換準備法術 | 同上 |
| at-will 傷害 | 低（戲法無修正值加成，Lv.7 前） | 低 |

**DM 心法：** 德魯伊的回合需要判斷：施法者模式（控場/治療）vs 野性形態模式（肉盾/近戰）。
多數情況下，先放專注控場法術 → 維持施法者模式 → 形勢不利再切野性形態當肉盾。

## B. 回合決策流程

### B-1. 主流程

```
回合開始
│
├─ 0. 回合開始觸發效果
│     ├─ 有專注法術嗎？→ 標記在面板狀態欄
│     ├─ 專注法術有回合開始觸發嗎？
│     │   └─ 如月光術（Moonbeam）：移動光柱 → 目標豁免
│     └─ 正在野性形態中嗎？→ 標記形態 + 剩餘臨時生命值（HP）
│
├─ 1. 模式判斷（施法者 vs 野性形態）
│     ├─ 目前是施法者模式？
│     │   ├─ 需要控場/治療/輸出 → 留在施法者模式
│     │   ├─ 生命值（HP）低 + 法術告急 → 附贈變身（吸收傷害）
│     │   └─ 需要近戰纏鬥/擋前排 → 附贈變身
│     │
│     └─ 目前是野性形態中？
│         ├─ 臨時生命值（HP）還多 + 無需施法 → 繼續野獸攻擊
│         ├─ 臨時生命值（HP）快歸零 → 讓形態自然結束或附贈解除
│         ├─ 隊友需要治療/控場 → 附贈解除 → 施法
│         └─ ⚠️ 正在維持專注法術 → 野獸形態中受傷也要做專注檢定！
│
├─ 2. 附贈動作（有就先決定，不佔主動作）
│     ├─ 野性形態（Wild Shape）：變身/解除變身
│     ├─ 治療之言（Healing Word）：1 環附贈、遠程治療
│     ├─ 野性重生（Wild Resurgence, Lv.5+）：
│     │   ├─ 野性形態次數 → 1 環欄位（每日 1 次，免費）
│     │   └─ 法術欄位 → 恢復 1 次野性形態（附贈）
│     ├─ 子職業附贈能力？（海之憤怒、星辰射手等）
│     └─ 都不需要 → 跳過
│
├─ 3. 主動作（按類型優先順序）
│     ├─ 🔮 有專注控場法術要開？→ 見 B-2
│     │   └─ 糾纏術、月光術、召喚動物群 等
│     ├─ 💚 隊友需要治療？→ 療傷術（主動作版本更強）
│     ├─ ✨ 戲法攻擊（施法者模式）
│     │   └─ 產焰術（Produce Flame）/ 星光微火（Starry Wisp）/ 荊棘鞭（Thorn Whip）
│     ├─ 🐻 野獸攻擊（野性形態中）
│     │   └─ 使用野獸的攻擊動作（算魔法攻擊）
│     ├─ 🩸 有免費種族能力可用？→ 優先使用（不花欄位）
│     └─ 衝刺/脫離/躲避（通用動作）
│
├─ 4. 移動
│     ├─ 施法者模式：保持安全距離（9-18m），避免觸發借機攻擊（AoO）
│     └─ 野性形態：依據野獸形態的速度和角色定位（肉盾前排/偵查側翼）
│
└─ 回合結束
```

### B-2. 「該不該花法術欄位？」

德魯伊是準備施法者，欄位數量中等，需要在控場、治療、召喚之間分配：

```
要花法術欄位嗎？
│
├─ Q1：這個法術的效用持續幾回合？
│   ├─ 專注控場 3+ 回合 → ✅ 高性價比
│   ├─ 一次性治療但隊友快倒 → ✅ 必要
│   └─ 只是一次性低傷害 → ❌ 戲法就好
│
├─ Q2：控場 vs 治療哪個更優先？
│   ├─ 控住敵人 = 預防傷害 → 通常控場優先
│   ├─ 隊友已經倒地（0 HP）→ 治療之言拉起（附贈）
│   └─ 隊友低血但還能戰鬥 → 控場 > 治療
│
├─ Q3：還剩幾個欄位？
│   ├─ 充裕 → 可以花
│   ├─ 剩 2-3 → 保留 1 個給治療之言（Healing Word）
│   └─ 只剩 1 → 保命/拉人專用
│
├─ Q4：專注衝突？
│   ├─ 新法術需要專注 且 正在維持更好的專注 → 不切換
│   └─ 新法術效果 > 現有專注 → 切換
│
└─ Q5：野性形態次數是否該用？
    ├─ 需要額外生命值（HP）池撐過接下來的打擊 → 變身
    ├─ 已有強力專注法術在維持中 → 變身當肉盾 + 維持專注
    └─ 法術欄位和野性形態都吃緊 → 考慮野性重生（Lv.5+）轉換
```

## C. 面板生成模板

DM 輪到德魯伊行動時，從角色卡讀取具體能力，按以下模板生成面板。
（面板分區子表結構定義見 [combat_flow.md](../rules/combat_flow.md) C 節）

### C-0. 狀態表補充：專注 + 野性形態追蹤

狀態表（第一段）的 📌 狀態欄中，需要標示：

| 情境 | 📌 狀態欄標示 |
|------|-------------|
| 正在維持專注法術 | `{法術名}專注中`（如：`糾纏術專注中`）|
| 正在野性形態中 | `野性形態：{形態名}（臨時 HP {X}/{Y}）` |
| 專注 + 野性形態同時 | 兩者都標示（**核心組合**）|
| 無專注、無變身 | 不標示 |

> 德魯伊只能同時維持 **1 個** 專注法術。
> 野性形態中**可以維持專注**，但**不能施法**（Lv.18 前）。
> 法術表中標 `✅⚠️換` 的法術 = 施放時會自動結束現有專注。
> 標 `❌免` 的法術 = 不需專注，可與現有專注並存。

### C-1. 動作表（分區子表設計）

統一編號橫跨所有子表，玩家直接喊號選擇。
每區只顯示該類型需要的欄位，避免空白。
**重要：** 若正在野性形態中，隱藏施法選項（Lv.18 前），只顯示野獸攻擊 + 通用動作。

---

**🔮 法術區（消耗欄位）** — ⚠️ 野性形態中不可用
來源：角色卡「準備法術」中施法時間為「動作」的法術。
**欄位分環級顯示：**

> 1 環：{目前}/{最大} ║ 2 環：{目前}/{最大} ║ 3 環：{目前}/{最大} ║ …

| # | 名稱 | 環級 | 命中/難度（DC）| 傷害/效果 | 消耗欄位 | 專注 | 備註 |
|---|------|------|---------------|----------|---------|------|------|
| N | {spell} | {X}環 | {+hit / 難度（DC）} | {effect} | {X}環×1 | ✅⚠️換 / ❌免 | {升環效果} |

欄位說明：
- **環級**：德魯伊法術有不同環級，欄位消耗對應法術環級（可用更高環欄位施放以獲得升環效果）。
- **消耗欄位**：消耗的欄位環級 × 數量。如 `2環×1`。
- **專注**：
  - `✅⚠️換` = 需要專注，施放時會取消目前的專注法術
  - `❌免` = 不需專注，可與現有專注並存
  - `🔄儀式` = 可用儀式施放（10 分鐘，不消耗欄位，僅限戰鬥外）

## D. 法術分類決策

德魯伊的法術按戰鬥用途分為以下類型。
DM 從角色卡讀取具體準備法術，歸類後套用決策：

### D-1. 專注控場型（如糾纏術、月光術、荊棘叢生）

**特徵：** 需要專注、持續多回合限制敵人行動或造成區域傷害

| 判斷 | 決策 |
|------|------|
| 多個敵人聚集在一起 | ✅ 糾纏術/荊棘叢生（Spike Growth）覆蓋大範圍 |
| 需要持續每回合施壓 | ✅ 月光術（Moonbeam）每回合造成傷害 + 可移動 |
| 預期 3+ 回合戰鬥 | ✅ 專注法術性價比高 |
| 戰鬥只剩 1-2 回合 | ❌ 不值得建立專注，戲法收尾 |
| 已有更好的專注在維持中 | 權衡：現有專注的剩餘回合價值 vs 新法術效果 |

### D-2. 治療型（如療傷術、治療之言、群體療傷術）

**特徵：** 恢復隊友生命值（HP），緊急拉起倒地隊友

| 判斷 | 決策 |
|------|------|
| 隊友倒地（0 HP）| ✅ **最高優先** — 治療之言（附贈，遠程）拉起 |
| 多人低血 | ✅ 活力靈氣（Aura of Vitality, 3 環）或群體療傷術（5 環）|
| 隊友低血但還能戰鬥 | ❌ 控場比治療更高效（預防 > 治療）|
| 戰鬥外治療 | 用低環療傷術/神莓術（Goodberry）節約高環欄位 |

### D-3. 召喚型（如召喚動物群、召喚精類、召喚野獸）

**特徵：** 需要專注、持續多回合、提供額外行動經濟

| 判斷 | 決策 |
|------|------|
| 預期長時間戰鬥（5+ 回合）| ✅ 召喚物的總行動經濟非常划算 |
| 需要前排肉盾 | ✅ 召喚物可吸收傷害 |
| 場地狹窄、多友軍 | ⚠️ 召喚物可能阻擋隊友或被範圍傷害（AoE）清光 |
| 敵人有大量範圍傷害（AoE）| ❌ 召喚物集體被清，浪費高環欄位 |

### D-4. 區域傷害型（如雷鳴波、呼喚閃電、冰風暴）

**特徵：** 一次性範圍傷害，通常不需專注

| 判斷 | 決策 |
|------|------|
| 敵人密集但隊友不在範圍內 | ✅ 高效輸出 |
| 已有好的專注在維持中且不想切換 | ✅ 選不需專注的區域傷害（如冰風暴 ❌免）|
| 能同時命中 3+ 目標 | ✅ 效率超過單體法術 |
| 只有 1-2 個目標 | ❌ 用戲法或單體法術更高效 |

### D-5. 防禦/輔助型（如樹皮術、能量防護、行動自由）

**特徵：** 增強自身或隊友防禦力

| 判斷 | 決策 |
|------|------|
| 預期被集中攻擊 | ✅ 先開防禦再做其他事 |
| 需要對抗特定傷害類型 | ✅ 能量防護（Protection from Energy）針對性強 |
| 沒有受威脅 | ❌ 浪費欄位和專注 |

## E. 荒野變身（Wild Shape）決策

> 德魯伊的第二核心機制。不同於法術，這是形態切換 + 額外生命值（HP）池。

### E-1. 何時變身

```
要使用野性形態嗎？
│
├─ 場景 1：開場前先放專注控場 → 然後變身
│   └─ ✅ 最佳組合！先放糾纏術/月光術 → 附贈變身 → 野獸形態維持專注
│       結果：控場效果持續 + 你有額外生命值（HP）池 + 野獸攻擊
│
├─ 場景 2：生命值（HP）低，法術快用光
│   └─ ✅ 野性形態 = 免費生命值（HP）池，讓你撐到戰鬥結束
│
├─ 場景 3：需要前排肉盾
│   └─ ✅ 變身大型野獸，臨時生命值（HP）吸收傷害，保護後排
│
├─ 場景 4：法術欄位充足且需要持續施法
│   └─ ❌ 留在施法者模式更靈活（治療、控場、切換法術）
│
├─ 場景 5：偵查/探索（非戰鬥）
│   └─ ✅ 小型動物（蜘蛛、老鼠）偵查；鳥類飛行偵查（Lv.8+ 可飛行）
│
└─ 場景 6：野性夥伴（Wild Companion）
    └─ 消耗 1 次野性形態 → 施放尋獲魔寵（Find Familiar），不消耗法術欄位
        適用：需要偵查但不想自己冒險
```

### E-2. 專注法術 + 野性形態組合

**核心戰術：** 這是德魯伊最強的戰鬥模式。

```
最佳行動序列：
│
├─ 第 1 回合：主動作施放專注控場法術（如糾纏術 1 環）
│              附贈動作變身野性形態
│
├─ 第 2+ 回合：野獸形態攻擊（主動作）
│               維持專注法術（免費，只需通過受傷時的專注檢定）
│               如需移動法術效果（如月光術），可在回合開始/結束時移動
│
└─ 注意事項：
    ├─ 野獸形態受傷 → 做體魄（CON）專注檢定
    │   └─ 難度（DC）= 10 或傷害的一半（取較高者）
    │   └─ 用野獸的體魄（CON）豁免
    ├─ 臨時生命值（HP）歸零 → 恢復原形態 → 專注不自動結束
    │   └─ 但溢出傷害打到本體 → 需要另一次專注檢定
    └─ 不能在野獸形態中施放新法術（Lv.18 前）
```

### E-3. 形態選擇原則

| 目的 | 推薦形態類型 | 理由 |
|------|-------------|------|
| 肉盾/近戰 | 大型野獸（熊、巨蠍等）| 高生命值（HP）+ 多重攻擊 |
| 偵查 | 小型（蜘蛛、老鼠）/ 飛行（鷹，Lv.8+）| 不引人注意 / 空中視角 |
| 游泳/水下 | 水生（鯊魚、章魚等）| 游泳速度 |
| 逃跑 | 高速動物（馬、鷹）| 最大移動速度 |

### E-4. 臨時生命值（HP）規則

| 等級 | 基礎臨時生命值（HP）| 月亮結社臨時生命值（HP）|
|------|------------------|--------------------|
| 公式 | = 德魯伊等級 | = 3 × 德魯伊等級 |
| Lv.2 | 2 | 6 |
| Lv.4 | 4 | 12 |
| Lv.6 | 6 | 18 |
| Lv.8 | 8 | 24 |
| Lv.10 | 10 | 30 |

> 2024 PHB 野性形態改為「獲得臨時生命值（HP）」而非「替換為野獸生命值（HP）」。
> 臨時生命值（HP）歸零 → 自動解除變身，溢出傷害轉到本體。

## F. 子職業戰術差異

德魯伊的子職業深刻改變戰鬥風格：

### 月亮結社（Circle of the Moon）— 戰鬥變身型
- **核心改變：** 野性形態臨時生命值（HP）× 3、最大 CR = 等級/3
- 野獸形態中可附贈施放療傷術（Cure Wounds）自療
- **戰鬥風格：** 變身為主，施法為輔 → 近戰肉盾 + 持續生存
- **DM 注意：** 此子職業的野性形態是主要戰鬥手段，不是備用方案

### 大地結社（Circle of the Land）— 純施法型
- **核心改變：** 自然恢復（短休回欄位，Lv.6+）+ 額外環境法術
- 環境法術永遠準備、不佔準備數量
- **戰鬥風格：** 純施法者，野性形態僅用於生存/偵查
- **DM 注意：** 法術欄位更充裕，可以更大膽使用法術

### 海洋結社（Circle of the Sea）— 元素戰鬥型
- **核心改變：** 海之憤怒（附贈動作 1d6 冰冷/閃電 + 推 1.5m，WIS 次/長休）
- Lv.10 室外飛行；閃電/雷鳴抗性
- **戰鬥風格：** 附贈輸出 + 控場法術 + 元素傷害
- **DM 注意：** 海之憤怒不需專注也不消耗欄位，每回合穩定附贈傷害

### 星辰結社（Circle of the Stars）— 星辰形態型
- **核心改變：** 星辰形態（消耗野性形態次數但不變成野獸）
  - 射手座（Archer）：附贈 1d8 + WIS 光輝攻擊
  - 聖杯座（Chalice）：治療法術額外 +1d8 + WIS
  - 巨龍座（Dragon）：專注檢定最低 10
- 免費指引之箭（Guiding Bolt）= 熟練加值次/長休
- **戰鬥風格：** 施法者 + 星辰形態加成，不需要野獸形態
- **DM 注意：** 星辰形態中**可以正常施法**，因為你沒有變成野獸

## G. 防禦層次（通用框架）

被攻擊時，按以下順序檢查可用防禦：

```
1. 基礎防禦等級（AC）
   ├─ 輕甲 + 盾牌（11 + DEX + 2）= 通常 15-16
   ├─ 守護者（Warden）：中甲 + 盾牌 → 可達 17-18
   └─ 野性形態中：使用野獸的防禦等級（AC）
   ↓ 被命中？
2. 野性形態 = 額外生命值（HP）池（最強防禦層）
   ├─ 臨時生命值（HP）先消耗
   ├─ 歸零 → 恢復本體，溢出傷害轉到本體
   └─ 月亮結社：3 × 等級的臨時生命值（HP）= 非常厚
   ↓ 無野性形態 or 已解除？
3. 有預先施放的防禦法術？
   ├─ 樹皮術（Barkskin）→ 防禦等級（AC）下限 17
   ├─ 能量防護（Protection from Energy）→ 特定傷害抗性
   └─ 被動生效，不消耗動作
   ↓ 持續受傷？
4. 治療能力自救
   ├─ 治療之言（Healing Word）→ 附贈自療
   ├─ 療傷術（Cure Wounds）→ 主動作自療
   └─ 月亮結社：野獸形態中附贈施放療傷術
   ↓ 危險！
5. 逃脫手段
   ├─ 變身小型/飛行野獸逃離
   ├─ 脫離（Disengage）+ 移動
   └─ 法術位移（如樹徑 Tree Stride, 5 環）
```

**DM 操作：** 德魯伊的防禦核心是野性形態的額外生命值（HP）池。
多數德魯伊防禦等級（AC）中等（14-17），但配合野性形態可以非常耐打。

## H. 等級區間變化

| 等級 | 關鍵變化 | 對決策流程的影響 |
|------|----------|-----------------|
| **Lv.1** | 純施法者，無野性形態 | 戲法 + 1 環法術。慎用欄位 |
| **Lv.2** | **野性形態開放**（2 次/長休） | 雙模式上線。專注 + 變身組合可用 |
| **Lv.3** | **子職業**、2 環法術 | 月光術（Moonbeam）上線。子職業定義戰鬥風格 |
| **Lv.5** | 3 環法術、**野性重生** | 召喚動物群（Conjure Animals）上線。法術/形態可互轉 |
| **Lv.6** | 3 次野性形態、子職業特性 | 更多變身次數。大地結社短休回欄位 |
| **Lv.7** | 4 環法術、**元素之怒** | 戲法傷害提升（強效施法 + WIS）或近戰加 1d8（原初打擊） |
| **Lv.8** | **野獸 CR 上限 1、可飛行** | 飛行偵查上線。更強的野獸形態 |
| **Lv.9** | 5 環法術 | 群體療傷術（Mass Cure Wounds）、召喚元素生物等強力法術 |
| **Lv.11** | 6 環法術 | 治癒術（Heal）、陽光術（Sunbeam）等戰略級法術 |
| **Lv.15** | **強化元素之怒** | 原初打擊 → 2d8 / 強效施法 → 額外給臨時生命值（HP）|
| **Lv.18** | **野獸施法（Beast Spells）** | 野性形態中可施法 → 遊戲規則改變，真正的全能戰鬥者 |
| **Lv.20** | **大德魯伊（Archdruid）** | 野性形態無限使用 → 永遠有額外生命值（HP）池 |

**德魯伊等級區間核心策略：**
- **Lv.1-2：** 純施法者，戲法 + 控場法術。Lv.2 野性形態上線後開始雙模式
- **Lv.3-4：** 子職業定義風格。月亮結社轉向近戰，其他結社強化施法
- **Lv.5-8：** 黃金時期。3 環控場/召喚 + 野性重生資源管理 + 元素之怒加成
- **Lv.9-14：** 高階法術開放，召喚物更強。野性形態逐漸從戰鬥主力退化為工具
- **Lv.15-20：** 強化元素之怒 + Lv.18 野獸施法 = 野性形態重新成為強力選項

## I. 子職業速查

DM 根據角色卡的子職業類型，參考對應戰術提示：

### 月亮結社（Circle of the Moon）
- 野性形態臨時生命值（HP）= 3 × 等級 → 變身是核心戰鬥手段
- 野獸形態中附贈施放療傷術（Cure Wounds）自療
- Lv.14 月形態：可施法 + 2d10 光輝額外傷害
- **優先序：** 變身 → 近戰攻擊 → 自療 → 法術為輔

### 大地結社（Circle of the Land）
- 環境法術永遠準備（不佔準備數量）→ 更多法術選擇
- 自然恢復（Lv.6）：短休回法術欄位 → 欄位更充裕
- Lv.10 免疫中毒、抗毒素傷害
- **優先序：** 控場法術 → 戲法 → 野性形態僅生存/偵查用

### 海洋結社（Circle of the Sea）
- 海之憤怒：附贈動作 1d6（→2d6→3d6）+ 推人，WIS 次/長休
- Lv.10 室外飛行 + 閃電/雷鳴抗性
- Lv.14 海洋之賜：團隊冰冷/閃電/雷鳴抗性 + 游泳 + 水下呼吸
- **優先序：** 控場法術 + 每回合附贈海之憤怒 → 穩定雙輸出

### 星辰結社（Circle of the Stars）
- 星辰形態（消耗野性形態次數，但**不變成野獸**→ 可施法）
- 免費指引之箭（Guiding Bolt）= 熟練加值次/長休
- Lv.6 宇宙預兆（反應 ±1d6）
- Lv.10 星座效果升級（2d8）+ 可每回合切換星座
- **射手座：** 附贈攻擊（輸出型）
- **聖杯座：** 治療加成（輔助型）
- **巨龍座：** 專注保底 10（控場型 — 幾乎不會斷專注）
- **優先序：** 星辰形態 + 施法 → 視星座選擇調整戰術

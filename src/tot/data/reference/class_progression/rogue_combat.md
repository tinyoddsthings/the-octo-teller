# 盜賊（Rogue）戰鬥 SOP

> 本文件是盜賊的**戰鬥決策框架**。
> 具體武器/能力從角色卡讀取，套用到以下流程。
> 面板格式見 [combat_flow.md](../rules/combat_flow.md) C 節

---

## A. 職業戰鬥定位

| 項目 | 說明 |
|------|------|
| 戰鬥角色 | 單發高爆（Single-Hit Burst DPR）|
| 主砲 | 武器攻擊 + 偷襲（Sneak Attack）— 每回合 1 次，必須觸發 |
| 核心機制 | **偷襲**：一擊附加大量 d6 骰，觸發條件為優勢或盟友鄰接 |
| 設計哲學 | 每回合只攻擊 1 次但傷害集中；靈巧動作（Cunning Action）提供無限機動力和生存力 |

### 盜賊 vs 其他近戰/物理職業

| | 盜賊 | 戰士 |
|---|--------|----------|
| 攻擊次數 | 1 次（全等級）| 多次（Lv.5+ = 2 次，Lv.11+ = 3 次）|
| 單次傷害 | 高（偷襲加成）| 中等（每次穩定）|
| 觸發條件 | 需要優勢或盟友鄰接 | 無條件 |
| 生存手段 | 靈巧動作（Cunning Action）+ 靈敏閃避（Uncanny Dodge）+ 閃避（Evasion）| 高生命值（HP）+ 重甲 + 二次振作（Second Wind）|
| 資源消耗 | 幾乎無限（偷襲/靈巧動作不消耗資源）| 動作如潮（Action Surge）/ 二次振作有次數限制 |

**DM 心法：** 盜賊的回合核心就是「觸發偷襲」+「用靈巧動作安全撤退或取得優勢」。
偷襲沒觸發 = 傷害腰斬，每回合都要幫盜賊確認觸發條件。

## B. 回合決策流程

### B-1. 主流程

```
回合開始
│
├─ 0. 回合開始觸發效果
│     └─ 有狡猾打擊（Cunning Strike）的持續效果嗎？（Lv.5+）
│
├─ 1. 偷襲觸發檢查（最重要！先確認再決定怎麼打）
│     ├─ 有攻擊優勢嗎？
│     │   ├─ 上回合躲藏成功（隱形中）→ ✅ 有優勢
│     │   ├─ 目標被定身/昏迷/目盲 → ✅ 有優勢
│     │   ├─ 穩定瞄準（Steady Aim）可用？（本回合未移動）→ 附贈動作獲得優勢
│     │   └─ 都沒有 → 看條件 (b)
│     ├─ 目標 1.5m 內有未失能的盟友嗎？
│     │   ├─ 有 → ✅ 可偷襲（即使沒優勢，但不能有劣勢）
│     │   └─ 沒有 → ❌ 無法偷襲，考慮重新定位
│     └─ ⚠️ 有劣勢嗎？→ 劣勢會抵消優勢 → 偷襲也失效！
│
├─ 2. 主動作
│     ├─ ⚔️ 預設：攻擊（靈巧/遠程武器）+ 偷襲傷害
│     │   → 近戰：短劍/匕首/細劍等靈巧（Finesse）武器
│     │   → 遠程：短弓/手弩/匕首投擲
│     │   → Lv.5+：觸發偷襲時可選用狡猾打擊（Cunning Strike）效果
│     ├─ 🩸 有免費種族/限次能力？→ 可在攻擊前後使用
│     └─ ❓ 無法觸發偷襲？
│         ├─ 先用附贈動作躲藏取得優勢 → 再攻擊
│         ├─ 移動到有盟友鄰接目標的位置 → 再攻擊
│         └─ 都不行 → 仍然攻擊（沒偷襲總比不打好）
│
├─ 3. 附贈動作（靈巧動作 Cunning Action — Lv.2+）
│     └─ 見 B-2 決策
│
├─ 4. 移動
│     ├─ 近戰盜賊：打完 → 靈巧脫離 → 撤退到安全位置
│     ├─ 遠程盜賊：保持距離（短弓 24m/96m）
│     └─ ⚠️ 用了穩定瞄準（Steady Aim）→ 本回合速度 = 0，無法移動！
│
└─ 回合結束
```

### B-2. 附贈動作決策（靈巧動作 Cunning Action）

盜賊每回合都有附贈動作可用（Lv.2+），這是核心生存和觸發工具：

```
附贈動作要做什麼？
│
├─ Q1：攻擊前還是攻擊後？
│   ├─ 攻擊前：需要優勢觸發偷襲
│   │   ├─ 穩定瞄準（Steady Aim）→ 本回合未移動？→ ✅ 優勢但速度歸零
│   │   └─ 躲藏（Hide）→ 有掩蔽可躲嗎？→ ✅ 隱匿成功則優勢
│   └─ 攻擊後：確保生存
│       ├─ 在敵人觸及範圍內？→ 脫離（Disengage）→ 安全撤退
│       ├─ 需要更遠距離逃跑？→ 衝刺（Dash）
│       └─ 為下回合準備優勢？→ 躲藏（Hide）
│
├─ Q2：穩定瞄準（Steady Aim）vs 躲藏（Hide）vs 脫離（Disengage）
│   ├─ 穩定瞄準：保證優勢但不能移動 → 遠程站定輸出時最佳
│   ├─ 躲藏：需要掩蔽且通過檢定 → 成功率高的情境才用
│   ├─ 脫離：打完安全撤退 → 近戰盜賊標準操作
│   └─ 衝刺：緊急逃脫/追擊 → 不常用但偶爾必要
│
├─ Q3：子職業附贈動作？
│   ├─ 盜賊（Thief）→ 快手（Fast Hands）：使用道具/巧手
│   ├─ 靈刃（Soulknife）→ 第二把靈能刀刃附贈攻擊（1d4）
│   ├─ 秘法騙徒（Arcane Trickster）→ 法師之手控制
│   └─ 刺客（Assassin）→ Lv.9 移動瞄準（Steady Aim 不減速度）
│
└─ 都不需要 → 躲藏準備下回合優勢（盜賊永遠有事做）
```

## D. 偷襲決策

偷襲（Sneak Attack）是盜賊 100% 的傷害核心。**每回合必須觸發。**

### D-1. 偷襲觸發條件

| 條件 | 說明 | 常見觸發方式 |
|------|------|-------------|
| **(a) 攻擊有優勢** | 任何來源的優勢皆可 | 躲藏成功、穩定瞄準、目標被定身/目盲/昏迷、盟友幫助（Help）|
| **(b) 盟友鄰接** | 目標 1.5m 內有未失能的盟友 + 盜賊無劣勢 | 前排隊友站在目標旁邊 |
| **武器限制** | 必須使用靈巧（Finesse）或遠程（Ranged）武器 | 短劍、細劍、匕首、短弓、手弩 |
| **⚠️ 劣勢否決** | 有劣勢時，即使滿足 (a) 或 (b) 也**不能偷襲** | 中毒、目盲、近距離射遠程、被遮蔽目標 |
| **次數** | 每**回合**1 次（不是每輪）| 自己回合 + 借機攻擊各可觸發 1 次 |

### D-2. 偷襲觸發優先順序

```
能偷襲嗎？
│
├─ 1. 目標旁有盟友嗎？（最簡單・最常見）
│     └─ ✅ 有 → 直接攻擊，偷襲自動觸發（只要沒劣勢）
│
├─ 2. 沒有盟友鄰接 → 需要取得優勢
│     ├─ 上回合躲藏成功？→ ✅ 隱形 = 優勢
│     ├─ 穩定瞄準可用？（未移動）→ ✅ 附贈 = 優勢（但速度歸零）
│     ├─ 目標有狀態？（倒地近戰/目盲/定身/昏迷）→ ✅ 優勢
│     └─ 都沒有 → 本回合用附贈躲藏再攻擊，或移動到有盟友的位置
│
├─ 3. 完全無法觸發？
│     └─ 仍然攻擊（武器基礎傷害 > 不打），但考慮用準備（Ready）
│         → 設條件「盟友移動到目標旁邊時我攻擊」→ 用反應觸發偷襲
│
└─ 💡 借機攻擊提醒
      └─ 若本回合已用偷襲 → 借機攻擊不能再偷襲
      └─ 若本回合沒用偷襲 → 借機攻擊**可以偷襲！**（每回合限制，不是每輪）
```

### D-3. 狡猾打擊（Cunning Strike）效果選擇（Lv.5+）

觸發偷襲時，可從偷襲骰中扣除骰數來附加效果。
Lv.11+ 可同時用 2 個效果。

**豁免 難度（DC）= 8 + 敏捷（DEX）修正 + 熟練加值**

| 等級 | 效果 | 代價 | 目標豁免 | 適用情境 |
|------|------|------|---------|---------|
| Lv.5 | 毒素（Poison）| 1d6 | 體魄（CON）| 長期削弱（中毒 = 攻擊+檢定劣勢），需帶毒藥工具包 |
| Lv.5 | 絆摔（Trip）| 1d6 | 敏捷（DEX）| 擊倒 → 近戰隊友下攻有優勢，目標大型以下 |
| Lv.5 | 撤離（Withdraw）| 1d6 | — | 攻擊後移動半速不引發借機攻擊（AoO），免費安全撤退 |
| Lv.14 | 恍惚（Daze）| 2d6 | 體魄（CON）| 目標下回合只能移動/動作/附贈三選一 |
| Lv.14 | 擊暈（Knock Out）| 6d6 | 體魄（CON）| 昏迷（超高代價，需 7d6+ 偷襲骰才實用）|
| Lv.14 | 遮蔽（Obscure）| 3d6 | 敏捷（DEX）| 目盲直到目標下回合結束 |

**選擇建議：**
```
偷襲骰夠多嗎？（扣完代價後至少剩 2-3d6 傷害）
├─ 是 → 看戰場需求
│     ├─ 需要安全撤退？→ 撤離（Withdraw）— 不需豁免，100% 生效
│     ├─ 隊友近戰想打目標？→ 絆摔（Trip）— 倒地 = 隊友優勢
│     ├─ 長期戰鬥？→ 毒素（Poison）— 持續多回合削弱
│     └─ Lv.14+ 高偷襲骰？→ 恍惚/遮蔽 — 強力控場
└─ 否（偷襲骰太少）→ 全打傷害，不用狡猾打擊
```

## E. 靈巧動作決策（Cunning Action）

靈巧動作（Cunning Action）是盜賊 Lv.2 起的核心附贈動作，提供無限次的衝刺（Dash）、脫離（Disengage）、躲藏（Hide）。

### E-1. 三選一決策樹

```
本回合附贈要做什麼？
│
├─ 情境 A：近戰攻擊後，仍在敵人觸及範圍
│   ├─ 1. 脫離（Disengage）→ 安全走開，不吃借機攻擊（AoO）
│   │   → ✅ 最常用，近戰盜賊的標準操作
│   ├─ 2. 衝刺（Dash）→ 跑更遠但會吃借機攻擊（AoO）
│   │   → ❌ 除非需要極限距離逃脫（接受挨一下）
│   └─ 3. 躲藏（Hide）→ 附近有掩蔽就躲
│       → ⚠️ 仍在敵人旁邊通常沒掩蔽，不推薦
│
├─ 情境 B：遠程攻擊（不在敵人旁邊）
│   ├─ 1. 躲藏（Hide）→ 為下回合攻擊取得優勢
│   │   → ✅ 有掩蔽時最佳（柱子、矮牆、暗處）
│   ├─ 2. 穩定瞄準（Steady Aim）→ 站定取得優勢
│   │   → ✅ 無掩蔽但安全站位時（省去找掩蔽的步驟）
│   └─ 3. 衝刺（Dash）→ 拉開距離
│       → 有人在靠近你時
│
├─ 情境 C：需要追趕/逃跑
│   ├─ 1. 衝刺（Dash）→ 附贈 + 主動作都衝刺 = 3 倍速度
│   │   → ✅ 極限追擊或全力逃跑
│   └─ 2. 脫離（Disengage）+ 正常移動
│       → 不需極限距離但要避免借機攻擊（AoO）
│
└─ 情境 D：回合開始，需要優勢觸發偷襲
    ├─ 1. 躲藏（Hide）→ 先躲再打（需掩蔽）
    ├─ 2. 穩定瞄準（Steady Aim）→ 先瞄再打（不能移動）
    └─ 3. 都不行 → 移動到有盟友的位置再攻擊
```

### E-2. 核心原則

| 規則 | 說明 |
|------|------|
| **攻擊後脫離** | 近戰盜賊的標準循環：移近 → 攻擊 → 附贈脫離 → 走開 |
| **躲藏 = 投資** | 本回合犧牲附贈動作 → 下回合攻擊有優勢 → 偷襲保證觸發 |
| **穩定瞄準 = 懶人優勢** | 不用掩蔽、不用骰檢定，但代價是不能移動（遠程射手專用）|
| **衝刺少用** | 只在需要極限機動時使用，大多數時候脫離或躲藏更有價值 |
| **永遠有事做** | 盜賊的附贈動作**從不浪費**，每回合都該用靈巧動作 |

## F. 子職業戰術差異

### 1. 盜賊（Thief）
- **快手（Fast Hands）：** 附贈動作使用物品/巧手 → 戰鬥中附贈喝藥水、附贈用卷軸
- **二樓工作（Second-Story Work）：** 攀爬速度 = 移動速度 → 立體機動、高處射擊
- **超凡隱匿（Supreme Sneak，Lv.9）：** 隱匿攻擊不解除隱形 → 持續隱形輸出
- **使用魔法物品（Use Magic Device，Lv.13）：** 調諧上限 4、充能省電、用任何法術卷軸
- **盜賊反射（Thief's Reflexes，Lv.17）：** 第一輪行動兩次（先攻 和 先攻-10）→ 雙倍偷襲
- **戰鬥風格：** 萬能工具人型，附贈動作靈活度最高

### 2. 刺客（Assassin）
- **暗殺（Assassinate，Lv.3）：** 先攻擲骰優勢 + 第一輪對未行動目標攻擊優勢 + 額外等級傷害
- **刺客工具（Assassin's Tools，Lv.3）：** 偽裝工具包 + 毒藥工具包熟練
- **移動瞄準（Roving Aim，Lv.9）：** 穩定瞄準不減速度 → 可移動 + 優勢，**極強**
- **淬毒武器（Envenom Weapons，Lv.13）：** 毒素狡猾打擊額外 2d6 傷害，無視毒素抗性
- **致命一擊（Death Strike，Lv.17）：** 第一輪偷襲命中 → 體魄豁免失敗 → 傷害翻倍
- **戰鬥風格：** 開局爆發型，第一輪傷害極高；之後退化為普通盜賊

### 3. 秘法騙徒（Arcane Trickster）
- **施法能力（Spellcasting，Lv.3）：** 有法師法術（智力施法）→ 法術欄位有限但實用
- **法師之手詭計（Mage Hand Legerdemain，Lv.3）：** 隱形之手，附贈動作控制
- **魔法伏擊（Magical Ambush，Lv.9）：** 隱形中施法 → 目標豁免劣勢 → 控場更強
- **竊法者（Spell Thief，Lv.17）：** 反應偷取敵人法術
- **戰鬥風格：** 混合型，有法術輔助但核心仍是偷襲；面板需加 🔮 法術欄位
- **⚠️ 面板特殊：** 需要在狀態表加 🔮 法術欄位欄，動作表加法術區

### 4. 靈刃（Soulknife）
- **靈能刀刃（Psychic Blades，Lv.3）：** 空手凝聚武器（1d6 心靈傷害，靈巧/投擲 60/120）
  - 附贈動作第二刀（1d4 心靈傷害）→ 相當於雙持但更強
  - 精通：纏擾（Vex）→ 命中後下次攻擊同目標有優勢
- **靈能力量（Psionic Power，Lv.3）：** 靈能骰用於增幅檢定、心靈感應
- **導引打擊（Homing Strikes，Lv.9）：** 未命中可加靈能骰 → 減少偷襲落空
- **靈能傳送（Psychic Teleportation，Lv.9）：** 附贈動作短程傳送
- **靈能面紗（Psychic Veil，Lv.13）：** 魔法動作隱形 1 小時
- **撕裂心智（Rend Mind，Lv.17）：** 偷襲命中 → 目標震懾
- **戰鬥風格：** 最穩定的偷襲觸發（Vex 連招 + 導引打擊），心靈傷害無視多數抗性

## G. 防禦層次

盜賊靠的不是硬扛，而是**不被打到**。

```
1. 基礎防禦等級（AC）（輕甲 + 敏捷（DEX）修正）
   → 皮甲 11 + DEX，鑲嵌皮甲 12 + DEX
   → 典型 AC 14-16（敏捷 16-20）
   ↓ 被攻擊？

2. 位置防禦 — 根本不在敵人攻擊範圍內
   ├─ 靈巧動作 — 脫離（Disengage）→ 每回合安全撤退
   ├─ 靈巧動作 — 衝刺（Dash）→ 極限拉距
   └─ 靈巧動作 — 躲藏（Hide）→ 攻擊者不知道你在哪
   ↓ 仍被攻擊？

3. 靈敏閃避（Uncanny Dodge）（Lv.5+，反應動作）
   → 被你看得見的攻擊者命中 → 傷害**減半**
   → ⚠️ 消耗反應（不能再借機攻擊或其他反應）
   → ⚠️ 只對**單次攻擊**有效，多段攻擊只擋第一下
   ↓ 範圍效果（AoE）？

4. 閃避（Evasion）（Lv.7+，被動）
   → 敏捷（DEX）豁免取半傷的效果：
   →   成功 → 完全不受傷害（原本半傷 → 0）
   →   失敗 → 只受半傷（原本全傷 → 半傷）
   → 對火球術（Fireball）等範圍法術極強
   ↓ 仍然危險？

5. 脫困手段
   ├─ 靈巧動作 — 衝刺（下回合逃跑）
   ├─ 狡猾打擊 — 撤離（Lv.5+，攻擊後移動半速不引發 AoO）
   └─ 靈能傳送（Soulknife Lv.9，附贈動作短程傳送）
```

**DM 操作：** 盜賊防禦等級（AC）偏中等，但有多層減傷和逃脫手段。
靈敏閃避是每輪最重要的防禦反應——遇到大傷害時優先使用。

## H. 等級區間變化

| 等級 | 偷襲骰 | 關鍵變化 | 對決策流程的影響 |
|------|--------|----------|-----------------|
| **Lv.1** | 1d6 | 偷襲、專精（Expertise）、武器精通（Weapon Mastery）| 偷襲傷害有限，但觸發習慣從第一天養成 |
| **Lv.2** | 1d6 | **靈巧動作（Cunning Action）** | 核心機動力上線：附贈衝刺/脫離/躲藏 |
| **Lv.3** | 2d6 | **子職業 + 穩定瞄準（Steady Aim）** | 戰鬥風格分化；多一種觸發優勢手段 |
| **Lv.4** | 2d6 | 屬性值提升（ASI）| 敏捷（DEX）提升 → 命中/傷害/防禦等級（AC）全面提升 |
| **Lv.5** | **3d6** | **狡猾打擊（Cunning Strike）+ 靈敏閃避（Uncanny Dodge）** | 里程碑！偷襲可附加效果；反應防禦上線 |
| **Lv.7** | 4d6 | **閃避（Evasion）+ 可靠才能（Reliable Talent）** | 範圍法術免疫 + 技能檢定下限保證 |
| **Lv.9** | 5d6 | 子職特性 | 子職業第二波強化（移動瞄準/導引打擊/超凡隱匿/魔法伏擊）|
| **Lv.11** | 6d6 | **進階狡猾打擊（Improved Cunning Strike）** | 可同時用 2 個狡猾打擊效果 |
| **Lv.13** | 7d6 | 子職特性 | 子職業第三波（使用魔法物品/靈能面紗/淬毒武器/多才詭術）|
| **Lv.14** | 7d6 | **詭詐打擊（Devious Strikes）** | 新增恍惚/擊暈/遮蔽選項 |
| **Lv.15** | 8d6 | 狡猾心智（Slippery Mind）| 感知（WIS）+魅力（CHA）豁免熟練 → 抗控場 |
| **Lv.17** | 9d6 | 子職特性 | 子職業巔峰（盜賊反射/致命一擊/竊法者/撕裂心智）|
| **Lv.18** | 9d6 | 飄忽（Elusive）| 沒有攻擊擲骰能對你有優勢 |
| **Lv.20** | 10d6 | 幸運一擊（Stroke of Luck）| 失敗視為 20（短休恢復）|

**偷襲骰數 = 傷害基準線：**
- Lv.1-2：1d6 → 偷襲是加分但不算爆炸
- Lv.3-4：2d6 → 偷襲開始有感
- Lv.5-8：3-4d6 → 偷襲是主要傷害來源，狡猾打擊提供控場選項
- Lv.9-12：5-6d6 → 偷襲傷害可觀，可承受扣骰做狡猾打擊
- Lv.13+：7d6+ → 偷襲傷害極高，狡猾打擊幾乎每回合都值得用

## I. 子職業速查

DM 根據角色卡的子職業類型，參考對應戰術提示：

### 盜賊（Thief）
- 附贈動作最靈活（靈巧動作 + 快手）→ 附贈喝藥水/用卷軸
- 攀爬速度提供立體機動 → 高處射擊 + 躲藏
- Lv.13 用任何法術卷軸 → 戰略級爆發力
- Lv.17 第一輪雙回合 → 兩次偷襲

### 刺客（Assassin）
- 第一輪爆發最強 → 先攻優勢 + 攻擊優勢 + 額外等級傷害
- Lv.9 移動瞄準 = 穩定瞄準不減速度 → 移動 + 優勢每回合
- 毒藥工具包熟練 → 搭配狡猾打擊毒素效果更好
- 後續回合退化為普通盜賊 → 需要短戰鬥最大化開局

### 秘法騙徒（Arcane Trickster）
- 有法術但欄位極少 → 法術用於控場/輔助，不用於傷害
- 推薦法術：迷霧術（Fog Cloud，破敵人視線）、易容術（Disguise Self）、魅惑人類（Charm Person）
- 隱形中施法 → 目標豁免劣勢（Lv.9 魔法伏擊）
- 法師之手偵查/偷竊 → 不用靠近危險區域

### 靈刃（Soulknife）
- 雙刀流（主手 1d6 + 附贈 1d4）→ 偷襲觸發機會最穩定
- 纏擾（Vex）精通：命中後下次攻擊同目標優勢 → 自產優勢
- 靈能骰加命中（導引打擊）→ 偷襲幾乎不落空
- 心靈傷害無視多數物理抗性 → 對不死/構裝體仍有效
- Lv.9 靈能傳送提供戰術位移 → 不需靈巧脫離

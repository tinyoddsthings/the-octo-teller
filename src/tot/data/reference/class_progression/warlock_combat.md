# 魔導士（Warlock）戰鬥 SOP

> 本文件是魔導士的**戰鬥決策框架**。
> 具體法術/能力從角色卡讀取，套用到以下流程。
> 面板格式見 [combat_flow.md](../rules/combat_flow.md) C 節

---

## A. 職業戰鬥定位

| 項目 | 說明 |
|------|------|
| 戰鬥角色 | 遠程持續輸出（Sustained Ranged DPR）|
| 主砲 | 魔能轟擊（Eldritch Blast）— 戲法，無限使用 |
| 核心機制 | **契約魔法**：欄位少但全為最高環級，**短休全回** |
| 設計哲學 | 魔能轟擊（EB）是免費穩定輸出，法術是改變戰局的王牌 |

### 魔導士 vs 其他施法者

| | 魔導士 | 法師/牧師 |
|---|--------|----------|
| 欄位數量 | 極少（Lv.4 = 2 個）| 多（Lv.4 = 7+ 個）|
| 欄位恢復 | **短休** | 長休 |
| at-will 傷害 | 高（EB + 苦痛加成）| 低（戲法無修正值加成）|
| 決策壓力 | 每次施法都是重大抉擇 | 可以「試用」較低環法術 |

**DM 心法：** 魔導士的回合通常很快——80% 的時間就是 EB。
只有在法術能明確改變戰局時才值得花欄位。

## B. 回合決策流程

### B-1. 主流程

```
回合開始
│
├─ 0. 回合開始觸發效果
│     └─ 有專注法術嗎？→ 標記在面板狀態欄
│
├─ 1. 附贈動作（有就先決定，不佔主動作）
│     ├─ 有專注法術且目標已死？
│     │   └─ 該法術可移標記嗎？（如脆弱詛咒 Hex）→ 附贈移標記
│     ├─ 角色卡有附贈施放的法術？
│     │   └─ 見 B-2 判斷是否值得花欄位
│     ├─ 有非法術的附贈能力？（如魔法靈巧）
│     │   └─ 適用條件滿足 → 使用
│     └─ 都不需要 → 跳過
│
├─ 2. 主動作（按類型優先順序）
│     ├─ ⚔️ 預設：魔能轟擊（EB）
│     │   → 穩定輸出，不消耗資源，永遠是安全選擇
│     ├─ 🔮 值得花欄位嗎？→ 見 B-2
│     │   ├─ 控場法術（改變戰局）
│     │   ├─ 防禦法術（保命）
│     │   └─ 增強法術（提升後續回合）
│     ├─ 🩸 有免費種族能力可用？→ 優先使用（不花欄位）
│     └─ ✨ 其他戲法（輔助/偵測用途）
│
├─ 3. 移動
│     └─ 遠程職業：保持安全距離（理想 9-18m）
│         避免進入敵人觸及範圍（觸發借機攻擊（AoO）風險）
│
└─ 回合結束
```

### B-2. 「該不該花法術欄位？」

魔導士的欄位極少，每次使用都是戰略決策：

```
要花法術欄位嗎？
│
├─ Q1：這場戰鬥後能短休嗎？
│   ├─ 能 → 放心用，短休全回
│   └─ 不確定 → 繼續判斷
│
├─ Q2：這個法術能改變戰局嗎？
│   ├─ 控場法術讓 BOSS 停手 → 值得
│   ├─ 增益法術加 3+ 回合的持續效果 → 值得
│   └─ 只是多一點傷害 → 不值得，EB 就好
│
├─ Q3：還剩幾個欄位？
│   ├─ 滿 → 可以花 1 個
│   ├─ 剩 1 → 保命優先（逃脫/防禦法術）
│   └─ 0 → 純 EB 或用魔法靈巧恢復
│
└─ Q4：專注衝突？
    ├─ 新法術需要專注 且 正在維持更好的專注 → 不切換
    └─ 新法術效果 > 現有專注 → 切換
```

## C. 面板生成模板

DM 輪到魔導士行動時，從角色卡讀取具體能力，按以下模板生成面板。
（面板分區子表結構定義見 [combat_flow.md](../rules/combat_flow.md) C 節）

### C-0. 狀態表補充：專注追蹤

狀態表（第一段）的 📌 狀態欄中，需要標示專注資訊：

| 情境 | 📌 狀態欄標示 |
|------|-------------|
| 正在維持專注法術 | `{法術名}專注中`（如：`脆弱詛咒專注中`）|
| 無專注 | 不標示（留給其他狀態）|

> 魔導士只能同時維持 **1 個** 專注法術。
> 法術表中標 `✅⚠️換` 的法術 = 施放時會自動結束現有專注。
> 標 `❌免` 的法術 = 不需專注，可與現有專注並存。

### C-1. 動作表（分區子表設計）

統一編號橫跨所有子表，玩家直接喊號選擇。
每區只顯示該類型需要的欄位，避免空白。

---

**⚔️ 戲法區（無限・免費）**
來源：角色卡「戲法」。攻擊戲法排前，輔助排後。

| # | 名稱 | 命中/難度（DC）| 傷害/效果 | 備註 |
|---|------|---------------|----------|------|
| 1 | {主砲 — 通常魔能轟擊（EB）} | +{法術攻擊} | {damage} | {祈喚加成} |
| 2 | {其他攻擊戲法} | ... | ... | |
| N | {輔助戲法} | — | {effect} | |

> 戲法無消耗、無次數限制、無專注（除非特別標註）。不需要 🔮/專注/次數欄位。

---

**🔮 法術區（消耗欄位）** — 欄位：{目前}/{最大}（{X}環）
來源：角色卡「已知法術」中施法時間為「動作」的法術。

| # | 名稱 | 命中/難度（DC）| 傷害/效果 | 消耗欄位 | 專注 | 備註 |
|---|------|---------------|----------|---|------|------|
| N | {spell} | {+hit / 難度（DC）} | {effect} | 1 | ✅⚠️換 / ❌免 | |

欄位說明：
- **消耗欄位**：消耗幾個欄位。**魔導士特殊：** 契約魔法一律消耗 1 個欄位，
  且全部以最高環級施放（Lv.4 = 全部 2 環）。所以此欄一律填 `1`。
  保留此欄是為了視覺提醒「這會消耗珍貴欄位」+ 跨職業通用性。
- **專注**：
  - `✅⚠️換` = 需要專注，施放時會取消目前的專注法術
  - `❌免` = 不需專注，可與現有專注並存
  - `🔄儀式` = 可用儀式施放（10 分鐘，不消耗欄位，僅限戰鬥外）

---

**🩸 種族/限次能力**
來源：角色卡「種族特性」及其他限次能力。

| # | 名稱 | 命中/難度（DC）| 傷害/效果 | 剩餘 | 恢復 | 備註 |
|---|------|---------------|----------|------|------|------|
| N | {ability} | {+hit / 難度（DC）} | {effect} | ✅{X}/{Y} | 長休/短休 | |

欄位說明：
- **剩餘**：`✅1/1` = 可用、`❌0/1` = 已用完
- **恢復**：`長休` 或 `短休`（標明何時回復）

### C-4. 魔寵表（契約之鎖限定）

無契約之鎖的魔導士沒有魔寵，跳過此段。
從角色卡「魔寵」欄位讀取具體種類和數值。

**🐙 魔寵行動**（共用主人先攻，在主人後行動）

| 🐙 魔寵行為 | 效果 | 消耗（魔寵）| 消耗（主人）| ⚠️ 注意 |
|-------------|------|-----------|-----------|----------|
| **Help（幫助）** | 指定敵人 → 主人下攻該目標有**優勢** | 動作 | **無** | {是否破隱形看魔寵種類} |
| **Dash（衝刺）** | 移動距離加倍 | 動作 | **無** | |
| **Disengage（脫離）** | 移動不觸發借機攻擊（AoO）| 動作 | **無** | |
| **Hide（躲藏）** | 隱匿檢定 | 動作 | **無** | |
| **{魔寵攻擊}** | {從角色卡讀取攻擊數據} | **反應（Reaction）** | ❌ **主人的攻擊動作（Attack Action）** | 主人不能同回合施法 |
| **傳遞接觸法術** | 透過魔寵位置施放接觸法術 | **反應（Reaction）** | 主人的施法動作 | 需要有接觸法術 |
| **感知共享** | 透過魔寵的感官感知 | — | ❌ **主人的動作（Action）** | 不能攻擊/施法 |

> **魔寵 stats：** 從角色卡讀取（防禦等級（AC）/ 生命值（HP）/ 速度 / 特殊能力）
> **核心原則：** Help 給優勢 > 攻擊（代價太高，除非有鎖鏈主宰之投資（Investment of the Chain Master）祈喚）
> **死亡恢復：** 1 小時儀式重新召喚，戰鬥中無法恢復

## D. 法術分類決策

魔導士的法術按戰鬥用途分為以下類型。
DM 從角色卡讀取具體法術，歸類後套用決策：

### D-1. 持續增益型（如 Hex）

**特徵：** 附贈施放、需要專注、持續加成每回合傷害

| 判斷 | 決策 |
|------|------|
| 戰鬥剛開始，預期 ≥3 回合 | ✅ 第一回合附贈開啟 → EB |
| 戰鬥快結束（1-2 回合）| ❌ 不值得花欄位，直接 EB |
| 目標死了但戰鬥繼續 | 附贈移標記到新目標（免費）|
| 需要換更好的專注法術 | 施放新法術 → 舊的自動結束 |

### D-2. 控場型（如 Suggestion、Hold Person）

**特徵：** 需要豁免、通常需要專注、可改變整場戰局

| 判斷 | 決策 |
|------|------|
| 能一招結束戰鬥 / 消除最大威脅 | ✅ 值得花欄位 + 放棄現有專注 |
| 目標豁免加值很高，大概率失敗 | ❌ 太冒險，EB 更穩 |
| 已有更好的專注在維持中 | 權衡：現有專注的剩餘價值 vs 控場效果 |

### D-3. 防禦型（如 Mirror Image、Armor of Agathys）

**特徵：** 通常不需要專注、提供生存力

| 判斷 | 決策 |
|------|------|
| 預期被集火 / 已被近戰包圍 | ✅ 生存比傷害重要 |
| 不需專注的防禦法術 | 優先選擇（可與增益專注並存）|
| 安全站位、沒有威脅 | ❌ 浪費欄位 |

### D-4. 位移型（如 Misty Step）

**特徵：** 附贈施放、不佔主動作、緊急脫困

| 判斷 | 決策 |
|------|------|
| 被近戰包圍且無法正常脫離 | ✅ 附贈傳送 → 主動作 EB |
| 需要到達無法步行到的位置 | ✅ 戰術位移 |
| 只是移動更遠一點 | ❌ 正常移動就好，省欄位 |

## E. 魔寵戰術（契約之鎖限定）

> 無契約之鎖的魔導士跳過本節。
> 動作經濟詳細規則見 [combat_flow.md](../rules/combat_flow.md) C 節第五段。

### E-1. 魔寵回合決策

```
魔寵的回合（主人後、同先攻）
│
├─ 有隱形能力且正在隱形？
│   └─ Help 給主人攻擊優勢（多數魔寵 Help 不破隱形）
│
├─ 失去隱形？
│   └─ Hide 恢復隱形
│
├─ 主人不需要優勢？
│   ├─ Help 給其他隊友
│   └─ Dash 重新定位
│
└─ 讓魔寵攻擊？
    ├─ 有鎖鏈主宰之投資（Investment）祈喚？→ 用主人附贈動作命令 → 值得
    └─ 沒有？→ 消耗主人整個動作（Action）→ 幾乎不值得
```

### E-2. 核心原則

| 規則 | 說明 |
|------|------|
| **Help 優先** | 不消耗主人資源、通常不破隱形、給攻擊優勢 |
| **攻擊看祈喚** | 有鎖鏈主宰之投資（Investment）→ 附贈命令（划算）；沒有 → 犧牲整個動作（虧本）|
| **生命值（HP）很低** | 魔寵生命值（HP）通常很低，避開範圍效果（AoE）|
| **死亡恢復** | 魔寵死亡後可重新召喚（1 小時儀式），戰鬥中無法恢復 |

## F. 契約差異

魔導士的契約（Pact）影響戰鬥風格：

### 契約之鎖（Pact of the Chain）
- 有魔寵 → 見 E 節
- 戰鬥優勢：免費 Help 給優勢、偵查、傳遞接觸法術
- 戰鬥劣勢：沒有額外攻擊或額外法術

### 契約之刃（Pact of the Blade）
- 可召喚武器 → 近戰 / 投擲攻擊取代 EB
- 搭配「渴血之刃」祈喚 → 額外攻擊
- 戰鬥風格完全不同：近戰戰士型魔導士

### 契約之書（Pact of the Tome）
- 額外戲法（從任何職業列表）→ 更多 at-will 選項
- 搭配儀式施法 → 非戰鬥效用大增
- 戰鬥風格：更像傳統施法者

## G. 防禦層次（通用框架）

被攻擊時，按以下順序檢查可用防禦：

```
1. 基礎防禦等級（AC）（護甲 + 敏捷（DEX）+ 裝備加成）
   ↓ 被命中？
2. 反應防禦？（護盾術 Shield 等）
   → 從角色卡/道具檢查可用反應
   ↓ 仍被打中？
3. 有預先施放的防禦法術？（鏡影術等）
   → 被動生效，不消耗動作
   ↓ 持續受傷？
4. 有暫時生命值（HP）能力？（阿加西斯之甲等）
   → 下回合附贈動作啟動
   ↓ 危險！
5. 有緊急位移？（迷蹤步等）
   → 下回合附贈動作逃離
```

**DM 操作：** 從角色卡的「防禦層次」欄位按順序檢查。
多數魔導士防禦等級（AC）偏低（12-16），反應防禦和位移是核心生存手段。

## H. 等級區間變化

| 等級 | 關鍵變化 | 對決策流程的影響 |
|------|----------|-----------------|
| **Lv.1-4** | 欄位 1-2 個、EB 1 束 | EB 為主。欄位極珍貴，每次施法慎重 |
| **Lv.5** | **EB 升 2 束** | 傷害翻倍。EB 更強 → 法術的機會成本更高 |
| **Lv.7** | 4 環法術 | 更強的控場/防禦選項 |
| **Lv.9** | 5 環法術、聯繫宗主 | 戰略級法術開放 |
| **Lv.11** | **3 欄位** + 秘法奧義（6 環）| 欄位壓力稍微緩解。秘法奧義獨立於欄位 |
| **Lv.17** | **4 欄位** + 9 環秘法奧義 | 接近傳統施法者的靈活度 |

**EB 束數 = 決策基準線：**
- Lv.1-4：1 束 → 法術容易超越 EB 傷害 → 值得用
- Lv.5-10：2 束 → 法術需要更好的理由才值得放棄 EB
- Lv.11-16：3 束 → EB 傷害很高，只有控場/防禦才值得花欄位
- Lv.17+：4 束 → EB 成為最強 at-will，法術純粹用於戰略目的

## I. 宗主子職業速查

DM 根據角色卡的宗主類型，參考對應戰術提示：

### 邪魔宗主（Fiend）
- 擊殺敵人 → 獲得暫時生命值（HP）（鼓勵搶尾刀）
- 天生適合消耗戰，越打越硬

### 遠古邪神（Great Old One）
- 心靈感應溝通 → 無聲戰術協調
- 心靈法術擅長偵測和控制

### 妖精宗主（Archfey）
- 傳送 + 魅惑/恐懼組合
- 擅長進出戰場、控制敵人行為

### 天界宗主（Celestial）
- 有治療能力（療癒之光）→ 半輔助定位
- 額外光耀/火焰傷害法術

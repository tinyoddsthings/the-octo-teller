# 吟遊詩人（Bard）戰鬥 SOP

> 本文件是吟遊詩人的**戰鬥決策框架**。
> 具體法術/能力從角色卡讀取，套用到以下流程。
> 面板格式見 [combat_flow.md](../rules/combat_flow.md) C 節

---

## A. 職業戰鬥定位

| 項目 | 說明 |
|------|------|
| 戰鬥角色 | 支援/控場施法者（Support / Controller）|
| 主砲 | 戲法（如惡毒嘲諷 Vicious Mockery）— 傷害低但附帶減益 |
| 核心機制 | **吟遊鼓舞（Bardic Inspiration）**：附贈動作給盟友額外骰；全施法者法術位，長休恢復 |
| 設計哲學 | 靈感骰 + 法術雙管齊下，讓全隊更強；自身傷害不高，價值在於提升團隊上限 |

### 吟遊詩人 vs 其他支援施法者

| | 吟遊詩人 | 牧師（Cleric）| 法師（Wizard）|
|---|--------|-------------|-------------|
| 法術位數量 | 多（全施法者）| 多（全施法者）| 多（全施法者）|
| 法術位恢復 | **長休** | 長休 | 長休（+秘法恢復）|
| 獨特資源 | 吟遊鼓舞骰（附贈動作、Lv.5+ 短休回）| 引導神力（Channel Divinity）| 秘法恢復（Arcane Recovery）|
| at-will 輸出 | 低（戲法附減益但傷害低）| 中（神聖火焰 Sacred Flame）| 中（火焰矢 Fire Bolt）|
| 支援強度 | **極高**（靈感骰 + 治療 + Buff）| 高（治療 + Buff + 引導）| 低（偏控場/輸出）|
| 法術彈性 | **Lv.10 魔法秘密偷學他職法術** | 綁定領域法術 | 法術書無限擴充 |

**DM 心法：** 吟遊詩人的回合核心是「靈感骰給誰 + 法術用什麼」。
多數回合是**附贈給靈感 + 主動作施法/戲法**的組合。傷害不是首要目標，讓隊友發揮更強才是。

## B. 回合決策流程

### B-1. 主流程

```
回合開始
│
├─ 0. 回合開始觸發效果
│     └─ 有專注法術嗎？→ 標記在面板狀態欄
│
├─ 1. 附贈動作（有就先決定，不佔主動作）
│     ├─ 有吟遊鼓舞剩餘？
│     │   └─ 見 E 節判斷是否給、給誰
│     ├─ 有專注法術且目標已死？
│     │   └─ 該法術可移標記嗎？→ 附贈移標記
│     ├─ 角色卡有附贈施放的法術？（如治療之言 Healing Word）
│     │   └─ 見 B-2 判斷是否值得花法術位
│     │   ⚠️ 附贈施放法術的回合，主動作只能用戲法（施法規則）
│     ├─ 子職附贈能力？（靈感披風、威嚴披風等）
│     │   └─ 適用條件滿足 → 使用
│     └─ 都不需要 → 跳過
│
├─ 2. 主動作（按優先順序）
│     ├─ 🔮 需要控場/Buff 嗎？→ 見 D 節法術分類
│     │   ├─ 控場法術（改變戰局）
│     │   ├─ 增益法術（提升隊友）
│     │   └─ 治療法術（保命急救）
│     ├─ ⚔️ 不需要法術 → 戲法攻擊
│     │   → 惡毒嘲諷（Vicious Mockery）附帶攻擊劣勢，是支援型戲法首選
│     ├─ 🩸 有免費種族能力可用？→ 優先使用（不花法術位）
│     └─ ✨ 其他戲法（輔助/偵測用途）
│
├─ 3. 移動
│     └─ 支援職業：中距離站位（9-15m），靠近隊友但避開前線
│         確保靈感骰 18m射程覆蓋全隊
│         避免進入敵人觸及範圍（觸發借機攻擊（AoO）風險）
│
└─ 回合結束
```

### B-2. 「法術位 vs 吟遊鼓舞：花哪個？」

吟遊詩人擁有兩種核心資源，每回合都需權衡：

```
資源分配決策
│
├─ Q1：場上有隊友需要靈感嗎？
│   ├─ 有人即將進行關鍵擲骰（BOSS 攻擊、重要豁免）
│   │   └─ ✅ 附贈給靈感（免費、高效益）
│   ├─ 全隊狀況穩定、沒有人手上有未使用的靈感骰
│   │   └─ ✅ 給最可能用到的人
│   └─ 所有隊友都已持有靈感骰
│       └─ 附贈動作空出 → 可用附贈法術或子職能力
│
├─ Q2：要花法術位嗎？
│   ├─ 控場法術能消除最大威脅？→ ✅ 值得
│   ├─ 治療法術能救起倒地隊友？→ ✅ 值得（治療之言附贈施放更佳）
│   ├─ Buff 法術能持續 3+ 回合受益？→ ✅ 值得
│   ├─ 只是多一點傷害？→ ❌ 戲法就好
│   └─ 剩餘法術位不多？→ 保留給治療之言（Healing Word）急救
│
├─ Q3：還剩幾個法術位？
│   ├─ 多 → 可以積極使用
│   ├─ 剩 1-2 → 保留至少 1 個給治療之言（Healing Word）
│   └─ 0 → 純戲法 + 靈感骰
│
├─ Q4：吟遊鼓舞剩幾次？
│   ├─ Lv.5+：短休回全部 → 若能短休就放心用
│   ├─ Lv.5+：也可消耗法術位恢復 1 次（靈感之泉 Font of Inspiration）
│   ├─ Lv.1-4：長休才回 → 每次給都要想清楚
│   └─ 剩 0 次 → 附贈動作空出，可用附贈法術
│
└─ Q5：專注衝突？
    ├─ 新法術需要專注 且 正在維持更好的專注 → 不切換
    └─ 新法術效果 > 現有專注 → 切換
```

### C-0. 狀態表補充：專注追蹤

狀態表（第一段）的 📌 狀態欄中，需要標示專注資訊：

| 情境 | 📌 狀態欄標示 |
|------|-------------|
| 正在維持專注法術 | `{法術名}專注中`（如：`專注祝福術中`）|
| 無專注 | 不標示（留給其他狀態）|

> 吟遊詩人只能同時維持 **1 個** 專注法術。
> 法術表中標 `✅⚠️換` 的法術 = 施放時會自動結束現有專注。
> 標 `❌免` 的法術 = 不需專注，可與現有專注並存。

## D. 法術分類決策

吟遊詩人的法術按戰鬥用途分為以下類型。
DM 從角色卡讀取具體法術，歸類後套用決策：

### D-1. 治療型（如 Healing Word、Cure Wounds）

**特徵：** 恢復隊友生命值（HP），治療之言為附贈動作、治療創傷為動作

| 判斷 | 決策 |
|------|------|
| 隊友倒地（0 生命值（HP））| ✅ **最高優先 — 治療之言（附贈）立即拉起** |
| 隊友生命值（HP）偏低但未倒 | ⚠️ 視情況 — 可能等下一輪或靠其他治療 |
| 全隊生命值（HP）健康 | ❌ 不浪費法術位在預防性治療 |
| 治療之言 vs 治療創傷 | 治療之言（附贈 + 遠程）遠優於治療創傷（動作 + 觸碰）|

**核心原則：** 治療之言是吟遊詩人最高效的救場手段 — 附贈施放，不佔主動作，能遠程拉起倒地隊友。**永遠保留至少 1 個法術位給治療之言。**

### D-2. 增益型（如 Bless、Heroism、Enhance Ability）

**特徵：** 強化隊友能力，通常需要專注

| 判斷 | 決策 |
|------|------|
| 戰鬥剛開始，預期 ≥3 回合 | ✅ 第一回合施放 Buff（如祝福術 Bless 加 3 人攻擊/豁免）|
| 戰鬥快結束（1-2 回合）| ❌ 不值得花法術位 + 專注 |
| 已有更好的專注在維持中 | 權衡：現有專注的剩餘價值 vs 新 Buff |
| 多人受益 vs 單人受益 | 優先選擇多人受益的增益法術 |

### D-3. 控場型（如 Hold Person、Hypnotic Pattern、Fear）

**特徵：** 需要豁免、通常需要專注、可改變整場戰局

| 判斷 | 決策 |
|------|------|
| 能一招控住多個敵人 / 消除最大威脅 | ✅ 值得花法術位 + 放棄現有專注 |
| 目標豁免加值很高，大概率失敗 | ❌ 太冒險，改用增益或戲法 |
| 已有更好的專注在維持中 | 權衡：現有專注的剩餘價值 vs 控場效果 |
| 會波及隊友的範圍控場（如催眠圖紋 Hypnotic Pattern）| ⚠️ 確認隊友站位再使用 |

### D-4. 傷害型（如 Dissonant Whispers、Shatter）

**特徵：** 直接造成傷害，部分附帶額外效果

| 判斷 | 決策 |
|------|------|
| 不諧之語（Dissonant Whispers）迫使目標移動 → 觸發隊友借機攻擊（AoO）| ✅ 傷害 + 戰術價值高 |
| 碎裂術（Shatter）能命中多個敵人密集區 | ✅ AoE 效率高 |
| 只打單體且無附加效果 | ❌ 吟遊詩人不是輸出主力，省法術位 |

### D-5. 位移/脫困型（如 Dimension Door、Misty Step（若魔法秘密偷學））

**特徵：** 緊急脫困或戰術位移

| 判斷 | 決策 |
|------|------|
| 被近戰包圍且無法正常脫離 | ✅ 用位移法術 + 剩餘動作施法/戲法 |
| 需要到達無法步行到的位置 | ✅ 戰術位移 |
| 只是移動更遠一點 | ❌ 正常移動就好，省法術位 |

## E. 吟遊詩人靈感決策

> 吟遊鼓舞（Bardic Inspiration）是吟遊詩人最核心的職業機制。
> 正確使用靈感骰的價值 **遠超** 多數法術。

### E-1. 給不給？

```
要給吟遊鼓舞嗎？
│
├─ Q1：有隊友手上沒有靈感骰嗎？
│   ├─ 有 → 繼續判斷
│   └─ 全隊都持有 → 不給（一人一顆上限）→ 附贈動作做別的
│
├─ Q2：附贈動作有更好的用途嗎？
│   ├─ 需要附贈施放法術（如治療之言救人）→ 法術優先
│   ├─ 子職附贈能力更好？（靈感披風 AoE 臨時 HP 等）→ 子職優先
│   └─ 都不需要 → ✅ 給靈感
│
├─ Q3：靈感剩餘次數？
│   ├─ Lv.5+（靈感之泉）且能短休 → 放心給
│   ├─ Lv.5+：法術位可補充靈感（靈感之泉） → 必要時轉換
│   ├─ 剩餘充足（≥ 一半）→ 可以給
│   └─ 剩 1 次 → 保留給關鍵時刻（倒地隊友豁免、BOSS 攻擊）
│
└─ Q4：戰鬥強度？
    ├─ 小型遭遇、低威脅 → 節省靈感
    └─ BOSS 戰、高威脅 → 每回合都給
```

### E-2. 給誰？

```
靈感給誰？（優先順序）
│
├─ 1. 即將進行關鍵擲骰的人
│     ├─ 近戰打手即將攻擊 BOSS → 攻擊擲骰加骰
│     ├─ 隊友即將做重要豁免 → 豁免加骰
│     └─ 關鍵技能檢定 → 檢定加骰
│
├─ 2. 最可能用到的人
│     ├─ 前線近戰（攻擊頻率高、被攻擊頻率高）→ 攻擊/勇氣學院防禦
│     ├─ 手上有殺招的人（如聖武士的神聖一擊確認命中）
│     └─ 即將面對危險的人
│
├─ 3. 沒有靈感骰的人
│     └─ 在全隊都安全的情況下，確保每個人手上有 1 顆
│
└─ 4. 不該給的情況
      ├─ 已經持有靈感骰的人 → 不能疊加
      ├─ 遠離戰場、不太會用到的人
      └─ 自己（可以，但通常給隊友效益更高；知識學院 Lv.14 無雙技藝除外）
```

### E-3. 靈感骰大小對照

| 吟遊詩人等級 | 鼓舞骰 | 平均加值 | 相當於 |
|------------|--------|---------|-------|
| 1-4 | d6 | +3.5 | 約 +3 攻擊/豁免 |
| 5-9 | d8 | +4.5 | 約 +4 攻擊/豁免 |
| 10-14 | d10 | +5.5 | 約 +5 攻擊/豁免 |
| 15-20 | d12 | +6.5 | 約 +6 攻擊/豁免 |

> **2024 PHB 關鍵機制：** 靈感骰是在 D20 檢定**失敗後**才擲、加到結果上，有機會將失敗轉為成功。這意味著靈感骰**不會浪費在已經成功的擲骰上**。

## F. 子職業戰術差異

吟遊詩人的子職業大幅影響戰鬥風格：

### 舞蹈學院（College of Dance）
- **無甲防禦**（10 + 敏捷（DEX）+ 魅力（CHA））→ 不穿護甲反而更強
- 消耗靈感時附帶**徒手打擊**（敏捷（DEX）based、傷害 = 鼓舞骰 + 敏捷（DEX）修正）
- Lv.6：激勵之舞 → 反應動作消耗靈感移動 + 帶盟友移動（不觸發借機攻擊（AoO））
- Lv.6：同步腳步 → 先攻時消耗靈感加全隊先攻
- **戰鬥風格：** 近戰支援混合，靈感消耗 = 攻擊 + 支援同時發生

### 魅惑學院（College of Glamour）
- **靈感披風（Mantle of Inspiration）**：附贈消耗靈感 → 多名盟友獲臨時生命值（HP）（骰值 ×2）+ 免費移動
- Lv.6：威嚴披風 → 附贈施放命令術（Command）不消耗法術位，魅惑目標自動失敗
- **戰鬥風格：** 大範圍臨時生命值（HP）+ 位置調整，團隊整體韌性大增

### 知識學院（College of Lore）
- **辛辣嘲諷（Cutting Words）**：反應動作消耗靈感 → 扣敵人傷害/攻擊/檢定
- 3 個額外技能熟練
- Lv.6：魔法發現 → 提前偷學 2 個他職法術
- Lv.14：無雙技藝 → 自己的檢定/攻擊失敗時用靈感加骰（失敗不消耗！）
- **戰鬥風格：** 最純正的支援/控場，靈感骰攻防兩用

### 勇氣學院（College of Valor）
- **戰鬥訓練**：中甲 + 盾牌 + 軍用武器 → 防禦等級（AC）大幅提升
- **戰鬥鼓舞**：持有靈感的盟友可用於**加傷害**或**加防禦等級（AC）**
- Lv.6：額外攻擊（Extra Attack）→ 可做武器 + 戲法組合
- Lv.14：戰鬥魔法 → 施法後附贈動作武器攻擊
- **戰鬥風格：** 半戰士半施法者，可站前線

## G. 防禦層次（通用框架）

被攻擊時，按以下順序檢查可用防禦：

```
1. 基礎防禦等級（AC）（輕甲 + 敏捷（DEX）；勇氣學院可中甲 + 盾牌）
   → 一般吟遊詩人 AC 偏低（12-15），勇氣學院可到 18-19
   ↓ 被命中？
2. 反應防禦？
   → 辛辣嘲諷（Cutting Words）— 知識學院限定：反應減少傷害/命中
   → 護盾術（Shield）— 若有魔法秘密偷學
   → 從角色卡/道具檢查可用反應
   ↓ 仍被打中？
3. 有預先施放的防禦法術？（鏡影術 Mirror Image 等）
   → 被動生效，不消耗動作
   ↓ 持續受傷？
4. 有暫時生命值（HP）能力？
   → 魅惑學院靈感披風提供的臨時 HP
   → 英勇術（Heroism）每回合補充臨時 HP
   ↓ 危險！
5. 有緊急位移/脫困？
   → 正常移動 + 脫離（Disengage）
   → 位移法術（如任意門 Dimension Door）
   → 舞蹈學院激勵之舞（反應移動不觸發 AoO）
```

**DM 操作：** 從角色卡的「防禦層次」欄位按順序檢查。
一般吟遊詩人防禦等級（AC）偏低，依賴**法術防禦 + 位置管理**生存。
勇氣學院例外——中甲 + 盾牌讓他們可以站前線。

## H. 等級區間變化

| 等級 | 關鍵變化 | 對決策流程的影響 |
|------|----------|-----------------|
| **Lv.1-2** | 鼓舞 d6、2 個戲法、1 環法術位 | 靈感珍貴（長休才回）。以戲法 + 靈感為主 |
| **Lv.3** | 子職業開放、2 環法術 | 子職大幅改變戰鬥風格。見 F 節 |
| **Lv.4** | 屬性值提升、第 3 戲法 | 魅力（CHA）提升 → 法術難度（DC）+1、靈感次數可能 +1 |
| **Lv.5** | **靈感之泉**、鼓舞 d8、3 環法術 | **轉捩點：靈感短休回**。可以每場戰鬥慷慨使用靈感。法術位可換靈感 |
| **Lv.6** | 子職特性 | 知識：魔法發現（偷學 2 法術）；勇氣：額外攻擊；魅惑：威嚴披風；舞蹈：激勵之舞 + 同步腳步 |
| **Lv.7** | 反魅咒、4 環法術 | 反應動作多一個選擇（對抗魅惑/恐懼）。更強的控場法術開放 |
| **Lv.9** | 第二次專精、5 環法術 | 技能更強。5 環法術（如高等回復術 Greater Restoration）是戰略資源 |
| **Lv.10** | **魔法秘密**、鼓舞 d10 | **里程碑：可從牧師/德魯伊/法師列表學法術**。大幅擴展可用工具 |
| **Lv.14** | 子職最終特性 | 知識：無雙技藝（自用靈感不浪費）；勇氣：戰鬥魔法；魅惑：不破威嚴；舞蹈：領導閃避 |
| **Lv.18** | 卓越鼓舞、6 環法術位 | 先攻時靈感 < 2 自動補到 2 → 靈感永不枯竭 |

**關鍵里程碑：**
- **Lv.5（靈感之泉）** = 從「靈感要省著用」變成「每場戰鬥都能慷慨使用」
- **Lv.10（魔法秘密）** = 從「吟遊詩人法術限定」變成「跨職業法術隨你選」

DM 根據角色卡的子職類型，參考對應戰術提示：

### 舞蹈學院（College of Dance）

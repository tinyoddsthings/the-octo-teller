# 武僧（Monk）戰鬥 SOP

> 本文件是武僧的**戰鬥決策框架**。
> 具體數值從角色卡讀取，套用到以下流程。
> 面板格式見 [combat_flow.md](../rules/combat_flow.md) C 節

---

## A. 職業戰鬥定位

| 項目 | 說明 |
|------|------|
| 戰鬥角色 | 近戰機動打手（Skirmisher / Mobile Striker）|
| 主砲 | 徒手打擊（Unarmed Strike）+ 武僧武器 — 多段攻擊、附贈動作追打 |
| 核心機制 | **集氣點（Focus Points）**：等於武僧等級，**短休全回** |
| 設計哲學 | 免費附贈徒手打擊是穩定底線，集氣點換爆發力或生存力 |

### 武僧 vs 其他近戰

| | 武僧 | 戰士 | 盜賊 |
|---|------|------|------|
| 攻擊次數 | 多（動作 2 次 + 附贈 1~3 次）| 多（動作 2-4 次 + Action Surge）| 少（1 次但偷襲骰大）|
| 單擊傷害 | 低（武術骰 d6~d12）| 高（武器骰 + 戰鬥風格）| 極高（偷襲加成）|
| 防禦等級（AC）| 中（10 + 敏捷（DEX）+ 感知（WIS），無甲）| 高（重甲 + 盾）| 中（輕甲 + 敏捷）|
| 機動性 | **極高**（無甲移動 +10~9m，疾風步免費衝刺）| 低 | 高（靈巧動作脫離/衝刺）|
| 資源恢復 | **短休全回** | 短休回部分 | 無消耗資源 |
| 控場 | 震懾拳（Stunning Strike）| 無內建 | 無內建 |

**DM 心法：** 武僧的回合攻擊多但傷害分散——先決定附贈動作（免費打 or 花集氣點），再依序解析多段攻擊。震懾拳是改變戰局的關鍵資源。

## B. 回合決策流程

### B-1. 主流程

```
回合開始
│
├─ 0. 回合開始觸發效果
│     ├─ Lv.10+ 自我修復（Self-Restoration）：移除 1 個魅惑/恐懼/中毒
│     └─ 元素調諧啟動中？→ 打擊觸及 +3m、可轉元素傷害
│
├─ 1. 移動（武僧通常先接近再打）
│     └─ 速度 = 基礎 9m + 無甲移動加成
│         ├─ 需要極遠位移？→ 疾風步（免費衝刺）或花 1 集氣點（脫離+衝刺）
│         └─ 正常接近 → 省下附贈動作打人
│
├─ 2. 主動作：攻擊（Attack Action）
│     ├─ Lv.1-4：1 次攻擊
│     └─ Lv.5+：2 次攻擊（額外攻擊）
│     每次命中時 →
│       ├─ 震懾拳？→ 見 B-2（每回合限 1 次，Lv.5+）
│       └─ 子職業追加？（傷害之手 / 元素打擊推拉 / 開掌技巧）
│
├─ 3. 附贈動作（武僧核心 — 每回合必有）
│     ├─ 免費：附贈徒手打擊（Bonus Unarmed Strike）— 1 次
│     ├─ 1 集氣點：疾風連擊（Flurry of Blows）— 2 次（Lv.10+ = 3 次）
│     ├─ 0 集氣點：耐心防禦 — 脫離（Disengage）
│     ├─ 1 集氣點：耐心防禦 — 脫離 + 閃避（Dodge）
│     ├─ 0 集氣點：疾風步 — 衝刺（Dash）
│     └─ 1 集氣點：疾風步 — 脫離 + 衝刺 + 跳躍加倍
│     → 見 B-2 決定花不花集氣點
│
└─ 回合結束
```

### B-2. 「附贈動作怎麼選？」+ 集氣點決策

武僧每回合**必有附贈動作**——關鍵是花不花集氣點、花在哪。

```
附贈動作決策
│
├─ Q1：需要逃跑/重新定位嗎？
│   ├─ 被多個敵人包圍、生命值（HP）低 → 耐心防禦（花 1 = 脫離+閃避）
│   ├─ 需要衝到遠處目標 → 疾風步（免費衝刺 or 花 1 脫離+衝刺）
│   └─ 不需要 → 繼續判斷
│
├─ Q2：攻擊命中了嗎？想用震懾拳嗎？（Lv.5+）
│   ├─ 命中高價值目標（施法者/BOSS）→ 花 1 震懾拳
│   ├─ 目標體魄（CON）豁免極高 → 不花，打白工機率大
│   └─ 已經用過本回合的震懾拳 → 不能再用
│
├─ Q3：要加打嗎？
│   ├─ 集氣點充足 or 短休在望 → 疾風連擊（1 集氣點 → 2 次攻擊）
│   ├─ 集氣點緊張 → 免費附贈徒手打擊（1 次）
│   └─ Lv.10+：疾風連擊升級為 3 次打擊，更值得花
│
└─ Q4：集氣點剩多少？
    ├─ 充足（≥ 等級一半）→ 積極使用
    ├─ 偏低（2-3 點）→ 保留給震懾拳和保命
    ├─ 只剩 1 點 → 保留給耐心防禦（保命）或關鍵震懾拳
    └─ 0 點 → 免費附贈打擊 + 免費疾風步/耐心防禦
```

**集氣點優先度排序（一般情境）：**

| 優先 | 用途 | 理由 |
|:----:|------|------|
| 1 | 震懾拳（Stunning Strike）| 控場 > 傷害，能改變整場戰鬥 |
| 2 | 耐心防禦（Patient Defense）花 1 | 生存是第一要務，閃避極強 |
| 3 | 疾風連擊（Flurry of Blows）| 穩定增傷，但免費附贈打擊已夠用 |
| 4 | 疾風步（Step of the Wind）花 1 | 少用，除非需要脫離+衝刺組合 |
| 5 | 偏斬攻擊反射（Deflect Attacks redirect）| 情境性，將傷害減至 0 才能反射 |

## D. 集氣點決策

武僧的集氣點是唯一的消耗資源，短休全回。決策邏輯：

### D-1. 震懾拳（Stunning Strike）— Lv.5+

**特徵：** 命中後花 1 集氣點，目標體魄（CON）豁免，失敗=震懾（Stunned），成功=速度減半+下次攻擊優勢

| 判斷 | 決策 |
|------|------|
| 目標是施法者/關鍵敵人 | ✅ 值得花，震懾 = 失去整回合 + 被攻擊優勢 |
| 目標體魄（CON）豁免加值極高（如巨人、龍）| ❌ 大概率浪費，用疾風連擊多打幾下 |
| 目標即使成功也有效（速度減半+優勢）| ✅ 2024 版成功也有回報，風險較低 |
| 集氣點只剩 1-2 點 | 看情況：用在關鍵目標，否則保命 |
| 戰鬥快結束（1-2 回合）| ❌ 不值得，直接輸出 |

### D-2. 疾風連擊（Flurry of Blows）vs 免費附贈打擊

**特徵：** 花 1 集氣點 → 從 1 次附贈打擊升級為 2 次（Lv.10+ = 3 次）

| 判斷 | 決策 |
|------|------|
| 集氣點充足 + 可以短休 | ✅ 花 1 升級，總傷害提升明顯 |
| 集氣點緊張，需要保留震懾拳 | ❌ 免費打 1 次就好 |
| 開掌武者用疾風連擊 | ✅ 每次命中可附加推撞/絆摔/迷亂，控場價值高 |
| 慈悲武者用疾風連擊 | ✅ 可替代 1 次打擊為免費治療之手 |
| Lv.10+（3 次打擊）| ✅ 價值極高，積極使用 |

### D-3. 耐心防禦（Patient Defense）

**特徵：** 免費=脫離；花 1=脫離+閃避（Lv.10+ 額外臨時生命值）

| 判斷 | 決策 |
|------|------|
| 被多敵包圍，需要安全撤退 | 免費版脫離通常夠用 |
| 預期下回合會被集火 | ✅ 花 1 開閃避，攻擊你的全部劣勢 |
| 生命值（HP）偏低但不想逃 | ✅ 花 1，站著打但有閃避保護 |
| 安全站位，無威脅 | ❌ 不需要，拿去打人 |

### D-4. 疾風步（Step of the Wind）

**特徵：** 免費=衝刺；花 1=脫離+衝刺+跳躍加倍

| 判斷 | 決策 |
|------|------|
| 目標在正常移動範圍外 | 免費衝刺通常夠（速度已含無甲移動加成）|
| 需要衝過敵陣到後排目標 | ✅ 花 1 = 脫離+衝刺，不吃借機攻擊穿過去 |
| 純粹需要跑更遠 | 免費版就好 |
| 需要帶隊友跑（Lv.10+）| ✅ 花 1，帶走瀕死隊友 |

## E. 武僧武器與徒手打擊

### E-1. 武術骰進程

| 等級 | 武術骰 | 徒手/武僧武器傷害 |
|:----:|:------:|:----------------:|
| 1-4 | d6 | d6 + 敏捷（DEX）|
| 5-10 | d8 | d8 + 敏捷（DEX）|
| 11-16 | d10 | d10 + 敏捷（DEX）|
| 17-20 | d12 | d12 + 敏捷（DEX）|

> **武術骰 vs 武器骰：** 徒手打擊和武僧武器的傷害擲武術骰。若武器本身骰面更大（如巨棒 1d8 > Lv.1-4 的 d6），可使用武器骰。隨等級提升武術骰會超越多數武器骰。

### E-2. 推薦武器選擇

| 武器 | 傷害 | 屬性 | 適用情境 |
|------|------|------|----------|
| 短劍（Shortsword）| 1d6 穿刺 | 巧手/輕型 | 低等級主力，有巧手屬性 |
| 長矛（Spear）| 1d6 穿刺 | 投擲/靈巧 | 可投擲的靈巧武器 |
| 長棍（Quarterstaff）| 1d6 鈍擊 | 靈巧 | 經典武僧武器，靈巧屬性 |
| 彎刀（Scimitar）| 1d6 揮砍 | 巧手/輕型 | 與短劍類似 |
| 匕首（Dagger）| 1d4 穿刺 | 巧手/輕型/投擲 | 備用投擲武器 |

> **Lv.5+ 後徒手打擊 ≥ 多數武僧武器傷害（d8）**，武器的主要價值在屬性（投擲、巧手）和傷害類型。
> **Lv.6 強力打擊（Empowered Strikes）** 讓徒手打擊可選力場傷害（Force），繞過多數抗性。

### E-3. 每回合攻擊次數總表

| 等級 | 動作攻擊 | 附贈（免費）| 附贈（疾風連擊）| 最大總攻擊次數 |
|:----:|:--------:|:----------:|:--------------:|:-------------:|
| 1 | 1 | 1 | — | 2 |
| 2-4 | 1 | 1 | 2 | 3 |
| 5-9 | 2 | 1 | 2 | 4 |
| 10+ | 2 | 1 | 3 | 5 |

## F. 子職業戰術差異

武僧的子職業（Lv.3 選擇）顯著影響戰鬥風格。
DM 從角色卡讀取子職業，參考對應段落：

### 開掌武者（Warrior of the Open Hand）
- **定位：** 純粹近戰控場 + 輸出
- **核心連招：** 疾風連擊 → 每次命中附加開掌技巧（迷亂/推撞/絆摔）
- **戰術重點：**
  - 迷亂（Addle）：目標無法借機攻擊 → 武僧可安全打完跑開
  - 絆摔（Topple）：目標倒地 → 隊友近戰攻擊全部優勢
  - 推撞（Push）：推離 4.5m → 推下懸崖/推出區域效果
- **Lv.6 周身圓滿：** 附贈動作自我治療（{武術骰+WIS}），{WIS}次/長休
- **Lv.11 疾步：** 任何附贈動作後可加免費疾風步（衝刺），機動性爆表
- **Lv.17 震顫掌：** 4 集氣點 → 延遲引爆 10d12 力場傷害

### 慈悲武者（Warrior of Mercy）
- **定位：** 近戰打手 + 副治療
- **核心連招：** 疾風連擊 → 替代 1 次打擊為免費治療之手（不花集氣點）
- **戰術重點：**
  - 傷害之手（1 集氣點）：命中附加 {武術骰+WIS} 黯蝕傷害
  - 治療之手（1 集氣點，疾風連擊中免費）：觸碰恢復 {武術骰+WIS} 生命值
  - Lv.6+ 傷害之手附加中毒狀態、治療之手可解除狀態
- **附贈動作衝突：** 疾風連擊（攻擊+治療混搭）vs 直接附贈治療之手（1 集氣點）
- **Lv.11 進化：** 疾風連擊中每次打擊都可替換為治療 or 免費傷害之手，次數={WIS}/長休

### 暗影武者（Warrior of Shadow）
- **定位：** 潛行偵查 + 暗殺突擊
- **核心連招：** 黑暗術（1 集氣點）→ 自己能看見 → 攻擊有優勢（目標目盲）
- **戰術重點：**
  - 黑暗術：可移動黑暗區域，製造永久優勢
  - 暗視 18m：微光中正常看
  - Lv.6 影步：微光/黑暗中附贈傳送 18m + 下次近戰攻擊優勢
- **附贈動作衝突：** 影步（傳送+優勢）vs 疾風連擊（多段攻擊），取決於定位
- **Lv.17 暗影斗篷：** 隱形 + 免費疾風連擊（不花集氣點），極強爆發

### 元素武者（Warrior of the Elements）
- **定位：** 中距離戰士 + 範圍傷害
- **核心連招：** 元素調諧（1 集氣點，10 分鐘）→ 觸及 +3m + 元素傷害 + 推拉 3m
- **戰術重點：**
  - 元素調諧持續 10 分鐘，跨遭遇使用
  - 推拉 3m可配合地形/隊友 AoE
  - Lv.6 元素爆發（2 集氣點）：36m內 6m球體 AoE，{3×武術骰}傷害
- **附贈動作衝突：** 元素爆發用魔法動作（Action），不是附贈 → 搶佔主動作
- **Lv.11 元素步伐：** 調諧中獲得飛行和游泳速度

## G. 防禦層次

武僧防禦以**高機動性**和**傷害減免**為主，而非厚甲高防禦等級：

```
1. 防禦等級（AC） = 10 + 敏捷（DEX）+ 感知（WIS）（無甲防禦）
   → 典型值：10 + 4(DEX) + 3(WIS) = 17，隨屬性成長提升
   ↓ 被命中？

2. 偏斬攻擊（Deflect Attacks）— 反應
   → 減少傷害 1d10 + 敏捷修正 + 武僧等級
   → Lv.3：僅限鈍擊/穿刺/揮砍
   → Lv.13+：任意傷害類型
   → 傷害減至 0 → 可花 1 集氣點反射回去
   ↓ 仍受傷？

3. 耐心防禦（Patient Defense）— 附贈動作（下回合準備）
   → 免費版：脫離（安全撤退）
   → 花 1 集氣點：脫離 + 閃避（攻擊你劣勢 + 敏捷豁免優勢）
   → Lv.10+：花 1 額外獲得 2×武術骰 臨時生命值
   ↓ 仍處於危險？

4. 機動性脫離 — 移動
   → 速度 30 + 無甲移動加成（+10~9m）
   → 疾風步（免費衝刺 or 花 1 脫離+衝刺）
   → 武僧移動速度通常是全隊最快，打完就跑
   ↓ 受到法術 AoE？

5. 迴避（Evasion）— Lv.7 被動
   → 敏捷豁免半傷的效果 → 成功=0 傷害、失敗=半傷
   → 對火球術（Fireball）等 AoE 極強

6. 其他被動防禦
   → Lv.10 自我修復：每回合結束移除魅惑/恐懼/中毒
   → Lv.14 修煉倖存者：全豁免熟練 + 花 1 集氣點重擲失敗的豁免
   → Lv.18 超凡防禦：花 3 集氣點 → 1 分鐘除力場外全抗性
```

**DM 操作：** 武僧的防禦是「打了就跑 + 反應減傷」模式。
提醒玩家：偏斬攻擊是**反應**（每輪 1 次），耐心防禦是**附贈動作**（代替攻擊）。

## H. 等級區間變化

| 等級 | 關鍵變化 | 對決策流程的影響 |
|------|----------|-----------------|
| **Lv.1** | 武術、無甲防禦、附贈徒手打擊 | 每回合 2 次攻擊（1 動作 + 1 附贈），無集氣點 |
| **Lv.2** | 集氣點（2 點）、疾風連擊/耐心防禦/疾風步、無甲移動+10、異常代謝 | 附贈動作選擇出現，每戰先攻時回血+回集氣（1 次/長休）|
| **Lv.3** | 偏斬攻擊、子職業 | 反應防禦上線，子職業改變戰鬥風格 |
| **Lv.4** | ASI、慢墜 | 提升敏捷/感知 → AC 和命中同步提升 |
| **Lv.5** | **額外攻擊 + 震懾拳** | **巨大轉折：** 攻擊 2→4 次（動作2+附贈2）、控場上線、武術骰升 d8 |
| **Lv.6** | 強力打擊、子職特性 | 徒手可選力場傷害（繞過抗性），子職業第二波能力 |
| **Lv.7** | 迴避（Evasion）| AoE 法術幾乎無威脅，生存力大增 |
| **Lv.9** | 特技移動 | 可在垂直面和水面移動，定位更靈活 |
| **Lv.10** | **精進集氣 + 自我修復** | 疾風連擊升為 3 次打擊、耐心防禦加臨時HP、免疫常見狀態 |
| **Lv.11** | 子職特性、武術骰升 d10 | 子職業核心能力進化 |
| **Lv.14** | 修煉倖存者 | 全豁免熟練 + 重擲，幾乎不可能被控 |
| **Lv.17** | 子職特性、武術骰升 d12 | 子職業終極能力 |
| **Lv.18** | 超凡防禦 | 3 集氣點 → 全抗性（除力場），坦度飆升 |
| **Lv.20** | 身心合一 | DEX、WIS 各 +4（上限 25），AC 和命中再跳一級 |

**武僧等級里程碑：**
- **Lv.5 是轉折點：** 額外攻擊 + 震懾拳讓武僧從「打手」變成「控場打手」
- **Lv.10 是第二波飆升：** 疾風連擊 3 次打擊、精進集氣全面強化
- **集氣點隨等級線性成長：** 高等級武僧資源壓力逐漸降低

## I. 子職業速查

DM 根據角色卡的子職業，快速查閱戰鬥重點：

### 開掌武者（Warrior of the Open Hand）
- 疾風連擊命中附加推撞/絆摔/迷亂 → 打完控場
- Lv.6 附贈自我治療 → 續航力
- Lv.11 任何附贈動作後加免費疾風步 → 機動性爆表
- Lv.17 震顫掌 → 4 集氣點延遲引爆 10d12 力場傷害

### 慈悲武者（Warrior of Mercy）
- 疾風連擊中 1 次打擊可替換為免費治療 → 攻守兼備
- 傷害之手 1 集氣點加黯蝕傷害 → 每回合穩定增傷
- Lv.6 傷害之手附加中毒、治療可解狀態 → 半治療者
- Lv.11 疾風連擊中免費傷害之手+多次替換治療 → 攻防一體
- Lv.17 復活（5 集氣點）→ 緊急救援

### 暗影武者（Warrior of Shadow）
- 黑暗術（1 集氣點）自己能看 → 製造永久攻擊優勢
- 暗視 18m → 黑暗環境自在行動
- Lv.6 影步 → 附贈傳送 18m + 攻擊優勢
- Lv.11 強化影步 → 移除光線限制 + 傳送後立即打擊
- Lv.17 暗影斗篷 → 隱形 + 免費疾風連擊，極限爆發

### 元素武者（Warrior of the Elements）
- 元素調諧（1 集氣點 10 分鐘）→ 觸及+3m、元素傷害、推拉
- 元素操控戲法 → 非戰鬥效用
- Lv.6 元素爆發 → 2 集氣點 36m AoE（{3×武術骰}）
- Lv.11 元素步伐 → 飛行+游泳速度
- Lv.17 元素化身 → 傷害抗性、毀滅步伐、強力打擊

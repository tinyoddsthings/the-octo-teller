# 聖騎士（Paladin）戰鬥 SOP

> 本文件是聖騎士的**戰鬥決策框架**。
> 具體法術/能力從角色卡讀取，套用到以下流程。
> 面板格式見 [combat_flow.md](../rules/combat_flow.md) C 節

---

## A. 職業戰鬥定位

| 項目 | 說明 |
|------|------|
| 戰鬥角色 | 近戰爆發輸出 + 團隊守護（Melee Nova + Party Protector）|
| 主砲 | 武器攻擊 + 神聖懲擊（Divine Smite）— 命中後決定是否追加 |
| 核心機制 | **半施法者（Half-Caster）**：法術欄位少，大量用於懲擊爆發；聖療術（Lay on Hands）獨立治療池 |
| 設計哲學 | 武器攻擊是免費穩定輸出，懲擊是爆發王牌，法術欄位在「懲擊 vs 施法」間拉鋸 |

### 聖騎士 vs 戰士（Fighter）

| | 聖騎士 | 戰士 |
|---|--------|------|
| 攻擊次數 | 2 次（Lv.5+）| 2→3→4 次 |
| 爆發傷害 | **極高**（懲擊疊加，暴擊翻倍）| 穩定但無爆發機制 |
| 持續力 | 受法術欄位限制 | 動作如潮（Action Surge）短休回 |
| 團隊貢獻 | **守護靈光（Aura of Protection）**、治療、法術 | 純傷害 |
| 防禦力 | 重甲 + 盾牌 + 自我治療 | 重甲 + 盾牌 + 再起（Second Wind）|

**DM 心法：** 聖騎士的回合核心是「打 → 命中了嗎？ → 要懲擊嗎？」。
每次命中都是一次「要不要花欄位」的微型決策。Lv.6 後守護靈光讓全隊豁免暴漲，站位極其重要。

## B. 回合決策流程

### B-1. 主流程

```
回合開始
│
├─ 0. 回合開始觸發效果
│     ├─ 有專注法術嗎？→ 標記在面板狀態欄
│     └─ 光輝打擊（Radiant Strikes，Lv.11+）→ 每次近戰命中自動 +1d8 光輝
│
├─ 1. 附贈動作（有就先決定，不佔主動作）
│     ├─ 聖療術（Lay on Hands）→ 觸碰治療，見 E 節判斷
│     ├─ 角色卡有附贈施放的法術？（如灼熱神擊 Searing Smite）
│     │   └─ 見 B-2 判斷是否值得花欄位
│     ├─ 引導神力需要附贈動作的？（如神聖感知 Divine Sense）
│     │   └─ 見 E 節
│     └─ 都不需要 → 跳過
│
├─ 2. 主動作 — 攻擊動作（Attack Action）
│     ├─ ⚔️ 預設：武器攻擊（Lv.5+ = 兩次）
│     │   → 每次命中後 → 懲擊決策（見 D 節）
│     │   → 穩定輸出，不消耗資源，永遠是安全選擇
│     ├─ 🔮 施法取代攻擊？→ 見 B-2
│     │   ├─ 控場法術（人類定身術 Hold Person 等）
│     │   ├─ 增益法術（祝福術 Bless、信仰之盾 Shield of Faith）
│     │   └─ 治療法術（療傷術 Cure Wounds — 但聖療術通常更好）
│     ├─ ⚡ 引導神力需要動作/魔法動作的？→ 見 E 節
│     │   └─ 如神聖武器（Sacred Weapon）搭配攻擊動作使用
│     ├─ 🩸 有免費種族能力可用？→ 優先使用（不花欄位）
│     └─ 以上皆否 → 武器攻擊
│
├─ 3. 移動
│     └─ 近戰職業：站到最需要的位置
│         ├─ 優先考慮守護靈光（Aura of Protection，Lv.6+）覆蓋隊友
│         ├─ 鎖定高威脅目標（可觸發借機攻擊（AoO）阻止逃跑）
│         └─ 避免離隊太遠導致靈光覆蓋不到
│
└─ 回合結束
```

### B-2. 「法術欄位花在哪？」——懲擊 vs 施法的拉鋸

聖騎士的法術欄位同時是懲擊的燃料和施法的資源，每次使用都在兩者間抉擇：

```
要花法術欄位嗎？
│
├─ Q1：用在懲擊還是法術？
│   ├─ 剛暴擊（Critical Hit）了！→ 幾乎永遠懲擊（骰子翻倍）
│   ├─ 需要控場/增益改變戰局 → 法術（如人類定身術讓全隊暴擊）
│   ├─ 只是想多一點傷害 → 攢著，下次暴擊再懲擊
│   └─ BOSS 快死了差一點 → 懲擊收頭
│
├─ Q2：還剩幾個欄位？
│   ├─ 滿 → 可以積極使用
│   ├─ 剩 1-2 → 保留 1 個應急（治療/關鍵懲擊）
│   └─ 0 → 純武器攻擊 + 聖療術撐場
│
├─ Q3：今天還有幾場戰鬥？
│   ├─ 最後一場 → 全力輸出，欄位用光
│   ├─ 還有 2+ 場 → 節約使用
│   └─ 不確定 → 保守策略
│
└─ Q4：專注衝突？
    ├─ 新法術需要專注 且 正在維持更好的專注 → 不切換
    └─ 新法術效果 > 現有專注 → 切換
```

## C. 面板生成模板

DM 輪到聖騎士行動時，從角色卡讀取具體能力，按以下模板生成面板。
（面板分區子表結構定義見 [combat_flow.md](../rules/combat_flow.md) C 節）

### C-0. 狀態表補充：專注追蹤 + 靈光標示

狀態表（第一段）的 📌 狀態欄中，需要標示：

| 情境 | 📌 狀態欄標示 |
|------|-------------|
| 正在維持專注法術 | `{法術名}專注中`（如：`祝福術專注中`）|
| 無專注 | 不標示（留給其他狀態）|
| 守護靈光生效中（Lv.6+）| `守護靈光 3m（+{CHA mod} 豁免）` |

> 聖騎士只能同時維持 **1 個** 專注法術。
> 法術表中標 `✅⚠️換` 的法術 = 施放時會自動結束現有專注。
> 標 `❌免` 的法術 = 不需專注，可與現有專注並存。

## D. 神聖懲擊決策

聖騎士最核心的戰鬥決策：何時懲擊、用幾環。

### D-1. 要不要懲擊？

```
命中了！要懲擊嗎？
│
├─ 這次是暴擊（Critical Hit）嗎？
│   ├─ 是！→ 幾乎永遠懲擊！（所有懲擊骰翻倍）
│   │   └─ 用最高環欄位 → 最大化爆發
│   └─ 否 → 繼續判斷
│
├─ 目標是不死生物（Undead）或邪魔（Fiend）？
│   ├─ 是 → 額外 +1d8，懲擊效益更高
│   └─ 否 → 繼續判斷
│
├─ 這一擊能擊殺目標嗎？
│   ├─ 是 → 懲擊收頭（避免浪費在已死目標上）
│   └─ 不確定 → 繼續判斷
│
├─ 還剩幾個欄位？
│   ├─ 充裕（≥3）→ 可以懲擊
│   ├─ 普通（2）→ 只在暴擊或關鍵時刻懲擊
│   ├─ 剩 1 → 保留，除非能結束戰鬥
│   └─ 0 → 不能懲擊（但檢查免費懲擊 1/長休）
│
├─ 今天還有幾場戰鬥？
│   ├─ 最後一場 → 積極懲擊
│   └─ 還有 → 節約
│
└─ 有更好的法術該用欄位施放嗎？
    ├─ 是（如控場法術能影響更多敵人）→ 省下欄位
    └─ 否 → 懲擊
```

### D-2. 用幾環的欄位？

| 情境 | 建議環級 | 傷害 |
|------|---------|------|
| 普通命中，想省資源 | 1 環 | 2d8 光輝 |
| 命中不死/邪魔 | 1 環 | 3d8 光輝（額外 +1d8）|
| 暴擊！ | 最高可用環級 | 全部骰子翻倍 |
| 收頭（差一點傷害）| 1 環 | 2d8 通常就夠 |
| 大型 BOSS 戰全力輸出 | 2-3 環 | 3d8–4d8（暴擊翻倍 6d8–8d8）|

### D-3. 懲擊傷害速查

| 欄位環級 | 基本傷害 | vs 不死/邪魔 | 暴擊（基本）| 暴擊（vs 不死/邪魔）|
|---------|---------|-------------|-----------|-------------------|
| 1 環 | 2d8 | 3d8 | 4d8 | 6d8 |
| 2 環 | 3d8 | 4d8 | 6d8 | 8d8 |
| 3 環 | 4d8 | 5d8 | 8d8 | 10d8 |
| 4 環 | 5d8 | 6d8 | 10d8 | 12d8 |
| 5 環 | 6d8 | 7d8 | 12d8 | 14d8 |

> **2024 PHB 注意：** 神聖懲擊（Divine Smite）現在是一個**法術**（附贈動作施放），不再是 2014 版的純職業特性。
> 施法時間為「附贈動作，命中後立即使用」。每次長休可**免費施展一次**（Lv.2 聖騎士懲擊特性）。

## E. 聖療術與引導神力

### E-1. 聖療術（Lay on Hands）管理

```
需要使用聖療術嗎？
│
├─ 隊友倒地（0 HP）？
│   ├─ 附贈動作觸碰 → 花 1 HP 就能拉起來
│   └─ 比療傷術（Cure Wounds）更划算（不花欄位）
│
├─ 自己血量危險（≤25% HP）？
│   ├─ 附贈動作自我治療 → 不影響主動作攻擊
│   └─ 比喝藥水好（藥水消耗動作）
│
├─ 隊友中毒（Poisoned）？
│   └─ 花 5 HP 解毒（不回復生命值（HP））
│
├─ Lv.14+ 隊友有嚴重狀態？
│   └─ 每個狀態花 5 HP 移除（目盲/魅惑/耳聾/恐懼/麻痺/昏迷）
│
└─ 池子管理
    ├─ 戰鬥中：小額多次（拉人 > 大量治療）
    ├─ 戰鬥間：補到安全血線
    └─ 保留至少 5-10 HP 應急
```

**治療池量對照：**

| 聖騎士等級 | 池總量 | 拉人次數（1 HP）| 解毒次數 |
|:---:|:---:|:---:|:---:|
| 1 | 5 HP | 5 次 | 1 次 |
| 3 | 15 HP | 15 次 | 3 次 |
| 5 | 25 HP | 25 次 | 5 次 |
| 10 | 50 HP | 50 次 | 10 次 |

### E-2. 引導神力（Channel Divinity）管理

| 項目 | 說明 |
|------|------|
| 使用次數 | Lv.3–9：2 次 → Lv.10+：3 次 |
| 恢復 | 短休回 1 次，長休全回 |
| 通用選項 | 神聖感知（Divine Sense）— 附贈偵測天界/邪魔/不死 |
| Lv.9+ | 驅敵術（Abjure Foes）— 魔法動作，恐懼多個敵人 |
| 子職選項 | 見 I 節各誓約 |

```
要用引導神力嗎？
│
├─ 能短休嗎？
│   ├─ 能 → 放心用（短休回 1 次）
│   └─ 不確定 → 保守使用
│
├─ 子職引導神力能大幅改變戰局？
│   ├─ 仇敵誓言（Vow of Enmity）→ 對 BOSS 優勢 1 分鐘 → 值得
│   ├─ 神聖武器（Sacred Weapon）→ +CHA 命中 10 分鐘 → 值得
│   ├─ 自然之怒（Nature's Wrath）→ 束縛多個敵人 → 值得
│   └─ 效果不明顯 → 省下
│
└─ 神聖感知有偵測需求？
    └─ 懷疑有不死/邪魔/天界偽裝 → 使用
```

## F. 子職戰術差異

聖騎士的誓約（Oath）決定引導神力選項和永久準備法術，影響戰鬥風格：

### 忠誠誓約（Oath of Devotion）
- **神聖武器（Sacred Weapon）**：攻擊骰 +魅力修正（CHA mod），10 分鐘
- 戰鬥風格：穩定命中型，搭配高魅力（CHA）命中率極高
- 永久法術亮點：防護善惡（Protection from Evil and Good）、信仰之盾（Shield of Faith）

### 榮耀誓約（Oath of Glory）
- **激勵懲擊（Inspiring Smite）**：懲擊後分配臨時生命值（HP）給隊友
- **無雙運動員（Peerless Athlete）**：運動/特技優勢 + 跳躍加成
- 戰鬥風格：進攻型團輔，懲擊同時強化全隊
- 永久法術亮點：指引之箭（Guiding Bolt）、加速術（Haste）

### 上古誓約（Oath of the Ancients）
- **自然之怒（Nature's Wrath）**：束縛（Restrained）多個敵人，力量豁免
- 戰鬥風格：控場 + 抗魔，Lv.7 防護靈光（Aura of Warding）給抗性
- 永久法術亮點：迷蹤步（Misty Step）、月光術（Moonbeam）
- Lv.15 **不朽哨兵（Undying Sentinel）**：0 HP 時不倒

### 復仇誓約（Oath of Vengeance）
- **仇敵誓言（Vow of Enmity）**：對單一目標攻擊優勢 1 分鐘
- 戰鬥風格：**單體爆發最強**，優勢 = 更高暴擊率 = 更多懲擊翻倍
- 永久法術亮點：獵人印記（Hunter's Mark）、迷蹤步（Misty Step）、加速術（Haste）
- Lv.7 **不懈復仇者（Relentless Avenger）**：借機攻擊命中後追擊

## G. 防禦層次

被攻擊時，按以下順序檢查可用防禦：

```
1. 基礎防禦等級（AC）（重甲 + 盾牌 + 戰鬥風格）
   → 聖騎士典型 AC：鏈甲 16 + 盾牌 +2 = 18（防禦風格 +1 = 19）
   → 板甲 18 + 盾牌 +2 = 20（防禦風格 +1 = 21）
   ↓ 被命中？
2. 有反應防禦嗎？（子職特性如榮耀防禦、反應法術）
   → 從角色卡/道具檢查可用反應
   ↓ 仍被打中？
3. 生命值（HP）緩衝
   → 聖騎士 d10 生命骰，生命值（HP）不低
   → 有臨時生命值（HP）嗎？（英雄氣概 Heroism 等）
   ↓ 血量下降？
4. 聖療術（Lay on Hands）自我治療
   → 附贈動作回血，不影響攻擊
   ↓ 危險！
5. 需要豁免的效果？
   → 守護靈光（Aura of Protection，Lv.6+）：所有豁免 +魅力修正（CHA mod）
   → 勇氣靈光（Aura of Courage，Lv.10+）：免疫恐懼（Frightened）
   → 這是聖騎士最強的防禦特性！
```

### 防禦等級（AC）速查

| 裝備組合 | 防禦等級（AC） | 備註 |
|---------|------|------|
| 鏈甲（Chain Mail）| 16 | 起始裝備 |
| 鏈甲 + 盾牌（Shield）| 18 | 起始裝備全配 |
| 板甲（Plate）+ 盾牌 | 20 | 中期目標 |
| 板甲 + 盾牌 + 防禦風格（Defense）| 21 | 接近頂配 |
| 以上 + 信仰之盾（Shield of Faith）| +2 | 專注法術 |

**DM 操作：** 聖騎士通常是隊伍中防禦等級（AC）最高的角色之一。
搭配守護靈光後，**全隊豁免都大幅提升**——這是聖騎士對團隊最大的隱性貢獻。

## H. 等級區間變化

| 等級 | 關鍵變化 | 對決策流程的影響 |
|------|----------|-----------------|
| **Lv.1** | 聖療術、施法（2 個 1 環欄位）、武器精通 | 2024 版 Lv.1 即可施法。欄位極少，聖療術是主要治療手段 |
| **Lv.2** | 戰鬥風格、聖騎士懲擊（Divine Smite 法術 + 免費 1/長休）| 懲擊開放！每次命中多一個決策點。免費懲擊優先用在暴擊上 |
| **Lv.3** | 引導神力 ×2、子職 | 子職引導神力上線，開啟第二套戰術（見 F 節各誓約）|
| **Lv.5** | **額外攻擊**、忠誠戰馬、2 環欄位 | **傷害翻倍**。每回合 2 次命中 = 2 次懲擊機會。2 環懲擊 = 3d8 |
| **Lv.6** | **守護靈光（Aura of Protection）** | **職業定義特性！** 全隊豁免 +魅力修正（CHA mod）。站位變成戰略核心 |
| **Lv.9** | 驅敵術、3 環欄位 | 3 環懲擊 = 4d8。驅敵術提供群體恐懼控場 |
| **Lv.10** | 勇氣靈光、引導神力 ×3 | 靈光內全隊免疫恐懼。引導神力多 1 次 |
| **Lv.11** | **光輝打擊（Radiant Strikes）** | 每次近戰命中自動 +1d8 光輝傷害。穩定 DPR 大幅提升，不消耗資源 |
| **Lv.14** | 復原之觸 | 聖療術可解除更多狀態，戰場急救能力大增 |
| **Lv.18** | 靈光擴展| 站位壓力大減，靈光覆蓋幾乎整個戰場 |

**等級區間總結：**
- **Lv.1-4（新手期）**：欄位極少，懲擊要非常謹慎。聖療術是核心續航
- **Lv.5-6（成長期）**：額外攻擊 + 守護靈光 = 聖騎士完全體雛形。DPR 和團隊貢獻同時起飛
- **Lv.7-10（成熟期）**：子職靈光上線，引導神力充裕，開始有餘裕懲擊
- **Lv.11-14（巔峰期）**：光輝打擊讓每次攻擊免費 +1d8，懲擊變成錦上添花
- **Lv.15-20（傳奇期）**：子職頂級特性 + 5 環法術 + 9m靈光 = 近戰神話

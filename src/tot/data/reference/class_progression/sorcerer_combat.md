# 術士（Sorcerer）戰鬥 SOP

> 本文件是術士的**戰鬥決策框架**。
> 具體法術/能力從角色卡讀取，套用到以下流程。
> 來源：2024 PHB pp.138-150

---

## A. 職業戰鬥定位

| 項目 | 說明 |
|------|------|
| 戰鬥角色 | 遠程爆發輸出 + 超魔法強化（Burst Ranged DPR + Metamagic Enhancement）|
| 主砲 | 攻擊戲法（如火焰箭 Fire Bolt、術法爆裂 Sorcerous Burst）— 無限使用 |
| 核心機制 | **超魔法（Metamagic）+ 魔力點（Sorcery Points）**：用 SP 強化法術效果，改變施法方式 |
| 設計哲學 | 法術已知少但每個都能被超魔法強化。「少而精、變化多」|

### 術士 vs 法師（Wizard）

| | 術士（Sorcerer） | 法師（Wizard） |
|---|--------|----------|
| 法術已知 | **少**（Lv.5 = 9 個準備）| **多**（可抄寫擴充、每日自由替換）|
| 法術位 | 完整施法者（同法師）| 完整施法者 |
| 靈活度來源 | **超魔法**改變法術運作方式 | **法術數量**覆蓋更多情境 |
| 資源管理 | 法術位 + 魔力點雙資源 | 法術位 + 奧術恢復（Arcane Recovery）|
| 施法屬性 | 魅力（CHA）| 智力（INT）|
| 防禦等級（AC）| 無護甲（龍族子職 = 10+DEX+CHA）| 無護甲（依賴法師護甲 Mage Armor）|
| 獨特優勢 | 孿生控場、瞬發 + 戲法連發、微妙無法被反制 | 法術書無限擴充、儀式施法 |

**DM 心法：** 術士的回合決策分兩層——**施什麼法術** + **要不要用超魔法強化**。
超魔法是術士的靈魂，選對超魔法 + 法術的組合 = 超越法師的爆發力。

## B. 回合決策流程

### B-1. 主流程

```
回合開始
│
├─ 0. 回合開始觸發效果
│     ├─ 有專注法術嗎？→ 標記在面板狀態欄
│     └─ 天生法術（Innate Sorcery）是否啟動中？
│         → DC+1、法術攻擊優勢、可用 2 個超魔法（Lv.7+）
│
├─ 1. 附贈動作（有就先決定，不佔主動作）
│     ├─ 天生法術（Innate Sorcery）未啟動且有使用次數？
│     │   └─ 預期本場戰鬥施法 ≥2 回合 → 附贈啟動
│     ├─ 魔力之泉（Font of Magic）：需要創建法術位？
│     │   └─ 附贈動作：SP → 法術位（見 B-2 轉換表）
│     ├─ 角色卡有附贈施放的法術？
│     │   └─ 見 B-2 判斷是否值得
│     └─ 都不需要 → 跳過
│
├─ 2. 主動作（按優先順序）
│     ├─ 🔮 值得花法術位嗎？→ 見 B-2
│     │   ├─ 控場法術（改變戰局）→ 考慮超魔法強化
│     │   │   └─ 強化（Heightened）→ 目標豁免劣勢
│     │   │   └─ 謹慎（Careful）→ 保護隊友
│     │   │   └─ 孿生（Twinned）→ 單目標法術打兩個
│     │   ├─ 爆發法術（AoE / 高傷害）→ 考慮超魔法強化
│     │   │   └─ 強效（Empowered）→ 重擲低傷害骰
│     │   │   └─ 瞬發（Quickened）→ 附贈施放 → 主動作再打戲法
│     │   ├─ 防禦法術（保命）
│     │   └─ 增強法術（提升後續回合）
│     ├─ ⚔️ 預設：攻擊戲法
│     │   → 穩定輸出，不消耗資源
│     │   → 天生法術啟動中 = 攻擊有優勢
│     ├─ 🩸 有免費種族/限次能力？→ 優先使用（不花法術位）
│     └─ ✨ 其他戲法（輔助/偵測用途）
│
├─ 3. 移動
│     └─ 遠程職業：保持安全距離（理想 9-18m）
│         避免進入敵人觸及範圍（觸發借機攻擊（AoO）風險）
│         無護甲 → 站位比多數職業更重要
│
└─ 回合結束
```

### B-2. 「法術位 + 魔力點消耗決策」

術士有**雙資源**系統：法術位施法、魔力點（SP）強化。兩者可互轉。

```
要花法術位嗎？
│
├─ Q1：這個法術能改變戰局嗎？
│   ├─ AoE 清場 / 控場讓 BOSS 停手 → 值得
│   ├─ 只是多一點傷害 → 不值得，戲法就好
│   └─ 搭配超魔法能讓效果翻倍？→ 值得（如孿生定身、強化火球）
│
├─ Q2：要搭配超魔法嗎？
│   ├─ SP 充足（≥ 一半）→ 可以用
│   ├─ SP 快見底 → 只用便宜的（1 SP：微妙、強效、謹慎）
│   └─ SP = 0 → 是否值得犧牲法術位轉 SP？
│
├─ Q3：還剩多少法術位 + SP？
│   ├─ 法術位滿 + SP 滿 → 積極施法 + 超魔法
│   ├─ 法術位過半 / SP 過半 → 正常節奏
│   ├─ 法術位 ≤ 2 + SP ≤ 3 → 保守，留護盾術（Shield）防禦用
│   └─ 幾乎空了 → 純戲法 + 微妙省 SP 打控場
│
└─ Q4：專注衝突？
    ├─ 新法術需要專注 且 正在維持更好的專注 → 不切換
    └─ 新法術效果 > 現有專注 → 切換
```

#### 魔力之泉（Font of Magic）轉換表

| 方向 | 操作 | 動作消耗 | 說明 |
|------|------|---------|------|
| 法術位 → SP | 消耗法術位 | **無需動作** | 獲得 = 法術位環級的 SP |
| SP → 法術位 | 消耗 SP | **附贈動作** | 花費見下表，最高 5 環 |

| 創建法術位環級 | SP 花費 | 最低術士等級 |
|-------------|--------|-----------|
| 1 環 | 2 SP | Lv.2 |
| 2 環 | 3 SP | Lv.3 |
| 3 環 | 5 SP | Lv.5 |
| 4 環 | 6 SP | Lv.7 |
| 5 環 | 7 SP | Lv.9 |

> **注意：** 法術位 → SP 是 1:1 轉換（1 環位 = 1 SP），但 SP → 法術位是溢價（1 環位 = 2 SP）。轉換方向有「匯率差」，通常法術位→SP 更划算。

## C. 面板生成模板

DM 輪到術士行動時，從角色卡讀取具體能力，按以下模板生成面板。

### C-0. 狀態表補充：專注 + 天生法術追蹤

狀態表（第一段）的 📌 狀態欄中，需要標示：

| 情境 | 📌 狀態欄標示 |
|------|-------------|
| 正在維持專注法術 | `{法術名}專注中`（如：`加速術專注中`）|
| 天生法術啟動中 | `天生法術啟動（DC+1、攻擊優勢）`，Lv.7+加註 `可用 2 個超魔法` |
| 無專注、無啟動 | 不標示（留給其他狀態）|

> 術士只能同時維持 **1 個** 專注法術。
> 法術表中標 `✅⚠️換` 的法術 = 施放時會自動結束現有專注。
> 標 `❌免` 的法術 = 不需專注，可與現有專注並存。

## D. 法術分類決策

術士已知法術較少，每個法術都該有明確戰術用途。
DM 從角色卡讀取具體法術，歸類後套用決策：

### D-1. 爆發傷害型（如 Fireball、Lightning Bolt）

**特徵：** 高傷害、通常 AoE、不需專注

| 判斷 | 決策 |
|------|------|
| 3+ 個敵人密集 | ✅ AoE 值得 → 考慮謹慎（Careful）保護隊友 |
| 隊友在範圍內 | 謹慎法術（Careful）→ 隊友自動成功豁免 |
| 想最大化傷害 | 強效法術（Empowered）→ 重擲低傷害骰 |
| 想同回合再打一輪 | 瞬發法術（Quickened）→ 附贈施放 → 主動作戲法 |
| 只有 1 個目標 | ❌ AoE 浪費，考慮單目標法術或戲法 |

### D-2. 控場型（如 Hold Person、Hypnotic Pattern、Slow）

**特徵：** 需要豁免、通常需要專注、可改變整場戰局

| 判斷 | 決策 |
|------|------|
| 目標豁免弱 | ✅ 直接放 |
| 目標豁免強 | 強化法術（Heightened）→ 豁免劣勢 → 大幅提升成功率 |
| 單目標但想控兩個 | 孿生法術（Twinned）→ 如人類定身術控 2 人 |
| 對方有施法者可能反制 | 微妙法術（Subtle）→ 無法被反制法術（Counterspell）中斷 |
| 已有更好的專注在維持中 | 權衡：現有專注的剩餘價值 vs 控場效果 |

### D-3. 防禦/增益型（如 Shield、Mage Armor、Haste）

**特徵：** 提供生存力或強化隊友

| 判斷 | 決策 |
|------|------|
| 護盾術（Shield）| 反應施放，幾乎永遠值得（AC+5）|
| 加速術（Haste）| 隊友近戰輸出 → 孿生（Twinned）加速兩人 = 超強 |
| 法師護甲（Mage Armor）| 戰前施放，不佔戰鬥動作 |
| 飛行術（Fly）| 脫離地面威脅 → 考慮孿生帶隊友飛 |

### D-4. 位移型（如 Misty Step、Dimension Door）

**特徵：** 附贈或動作施放、緊急脫困/戰術移動

| 判斷 | 決策 |
|------|------|
| 被近戰包圍且無法正常脫離 | ✅ 附贈傳送 → 主動作施法/戲法 |
| 需要到達無法步行到的位置 | ✅ 戰術位移 |
| 只是移動更遠一點 | ❌ 正常移動就好，省法術位 |

## E. 超魔法決策

超魔法（Metamagic）是術士的**核心差異化能力**。
以下是每個超魔法選項的最佳使用時機：

### E-1. 超魔法速查表

| 超魔法 | SP | 最佳搭配法術 | 核心用途 | 優先度 |
|--------|-----|------------|---------|--------|
| 孿生法術（Twinned） | 1 | 加速術（Haste）、人類定身術（Hold Person）、變形術（Polymorph）、飛行術（Fly） | 單目標法術 → 2 個目標 | ★★★ |
| 瞬發法術（Quickened） | 2 | 火球術（Fireball）、控場法術 | 附贈施法 → 主動作戲法 | ★★★ |
| 微妙法術（Subtle） | 1 | 反制法術（Counterspell）、暗示術（Suggestion）、魅惑術 | 無成分施法 → 無法被反制/察覺 | ★★★ |
| 強效法術（Empowered） | 1 | 火球術（Fireball）、寒冰錐（Cone of Cold）等多骰法術 | 重擲低傷害骰 | ★★ |
| 強化法術（Heightened） | 2 | 人類定身術、怪物定身術、放逐術（Banishment）、催眠圖紋 | 目標豁免劣勢 | ★★ |
| 謹慎法術（Careful） | 1 | 火球術（Fireball）、催眠圖紋（Hypnotic Pattern）、恐懼術（Fear） | 隊友自動成功 AoE 豁免 | ★★ |
| 追蹤法術（Seeking） | 1 | 灼熱射線（Scorching Ray）、繽紛寶珠（Chromatic Orb） | 攻擊未中 → 重擲 | ★ |
| 遠程法術（Distant） | 1 | 觸碰法術、射程受限法術 | 射程加倍 / 觸碰→9m | ★ |
| 延時法術（Extended） | 1 | 法師護甲（Mage Armor）、增強能力（Enhance Ability） | 持續時間加倍 + 專注優勢 | ★ |
| 變質法術（Transmuted） | 1 | 火球術 → 改閃電/寒冷等 | 對應敵人弱點 / 繞過抗性 | ★ |

### E-2. 超魔法決策流程

```
施法時——要用超魔法嗎？
│
├─ Q1：這法術能被超魔法大幅強化嗎？
│   ├─ 單目標控場/增益 → 孿生（Twinned）= 效果翻倍
│   ├─ 高價值法術但敵方豁免強 → 強化（Heightened）
│   ├─ AoE 但隊友在範圍內 → 謹慎（Careful）
│   ├─ 想同回合再打一輪 → 瞬發（Quickened）
│   ├─ 對方有施法者 → 微妙（Subtle）防反制
│   └─ 傷害法術骰子多 → 強效（Empowered）
│
├─ Q2：SP 夠嗎？
│   ├─ SP 充足 → 放心用
│   ├─ SP 緊張 → 優先 1 SP 選項（微妙、強效、謹慎）
│   └─ SP = 0 → 考慮犧牲低環法術位轉 SP
│
├─ Q3：天生法術（Innate Sorcery）啟動中？（Lv.7+）
│   ├─ 是 → 可同時用 2 個超魔法！
│   │   └─ 經典組合：強化 + 孿生、瞬發 + 強效
│   └─ 否 → 只能用 1 個（強效/追蹤例外，可疊加）
│
└─ Q4：Lv.20 奧術神化（Arcane Apotheosis）？
    └─ 天生法術啟動中 → 每回合 1 個超魔法免費（0 SP）
```

### E-3. 經典超魔法組合

| 組合名 | 操作 | SP 花費 | 效果 |
|--------|------|---------|------|
| **孿生加速** | 孿生（Twinned）+ 加速術（Haste） | 1 SP + 3 環位 | 2 個近戰隊友同時加速 |
| **瞬發爆發** | 瞬發（Quickened）+ 火球術 → 主動作火焰箭 | 2 SP + 3 環位 | 一回合雙輸出（AoE + 單目標）|
| **必中控場** | 強化（Heightened）+ 人類定身術 | 2 SP + 2 環位 | 豁免劣勢 → 定身成功率大增 |
| **無聲反制** | 微妙（Subtle）+ 反制法術（Counterspell） | 1 SP + 3 環位 | 反制無法被反反制（無成分 → 對方不知道你在施法）|
| **安全火球** | 謹慎（Careful）+ 火球術 | 1 SP + 3 環位 | 隊友自動通過豁免，不受傷 |
| **雙超魔爆發** | 瞬發 + 強效（Lv.7+ 天生法術中）| 3 SP + 法術位 | 附贈施法 + 重擲低傷害骰 |

## F. 子職業戰術差異

術士的起源子職（Sorcerer Subclass / Origin）影響戰鬥風格和額外資源：

### 龍族術法（Draconic Sorcery）— 「最硬的術士」

| 特性 | 戰鬥影響 |
|------|---------|
| 龍族韌性（Lv.3） | 額外 HP（+1/等級+3）、AC = 10+DEX+CHA（無需法師護甲）|
| 龍族法術（Lv.3） | 繽紛寶珠、龍之吐息、飛行術等永遠準備 |
| 元素親和（Lv.6） | 選定元素傷害抗性 + 魅力修正加到同類型法術傷害 |
| 龍翼（Lv.14） | 18m飛行，附贈啟動 |
| 龍族夥伴（Lv.18） | 免費召喚巨龍、可移除專注 |

**戰術重點：**
- 省下法師護甲（Mage Armor）法術位 → 其他法術更多彈性
- Lv.6 後同元素法術加 CHA 修正 → 火焰術士 + 火球術傷害更高
- 搭配變質法術（Transmuted）→ 所有傷害法術都能觸發元素親和加成

### 狂野魔法術法（Wild Magic Sorcery）— 「賭博型術士」

| 特性 | 戰鬥影響 |
|------|---------|
| 狂野魔法湧動（Lv.3） | 用法術位施術士法術後，擲 d20 = 20 → 狂野湧動表 |
| 混沌之潮（Lv.3） | 1 次 D20 檢定優勢（長休/用法術位施法後恢復 → 自動觸發湧動表）|
| 扭轉命運（Lv.6） | 反應，1 SP，擲 1d4 加減他人 d20 結果 |
| 控制混沌（Lv.14） | 湧動表擲兩次選一 |
| 馴化湧動（Lv.18） | 直接選擇湧動表效果（長休恢復）|

**戰術重點：**
- 混沌之潮（Tides of Chaos）給自己攻擊/豁免優勢 → 下次施法自動觸發湧動
- 扭轉命運（Bend Luck）= 極強反應能力，1 SP 影響任何 d20
- 湧動表有正面也有負面，DM 注意追蹤觸發條件
- **主動觸發策略：** 用混沌之潮 → 施法觸發湧動 → Lv.14+ 選有利結果

### 異怪術法（Aberrant Sorcery）— 「靈能術士」

| 特性 | 戰鬥影響 |
|------|---------|
| 靈能法術（Lv.3） | 哈達之臂、不諧之語、偵測思想等永遠準備 |
| 心靈對話（Lv.3） | 附贈，心靈感應交流（CHA 修正哩數）|
| 靈能術法（Lv.6） | 靈能法術可用 SP 施放（取代法術位）→ 自動無成分（= 免費微妙）|
| 心靈防禦（Lv.6） | 心靈傷害抗性 + 魅惑/恐懼豁免優勢 |
| 形體啟示（Lv.14） | SP 換飛行/游泳/視隱/穿行（10 分鐘）|
| 扭曲內爆（Lv.18） | 36m傳送 + 9m AoE 力場傷害 |

**戰術重點：**
- Lv.6 核心突破：靈能法術用 SP 施放 = 省法術位 + 自動微妙（無成分）
- 不諧之語、哈達之臂等用 SP 施放 = 無法被反制、不消耗法術位
- 心靈對話 = 無聲戰術溝通（對玩家隱藏資訊）
- SP 消耗更快（法術 + 超魔法 + 靈能術法都吃 SP）→ 資源管理更緊張

### 鐘律術法（Clockwork Sorcery）— 「秩序術士 / 半輔助」

| 特性 | 戰鬥影響 |
|------|---------|
| 鐘律法術（Lv.3） | 救助術（Aid）、防護善惡、次級復原術等永遠準備 |
| 恢復平衡（Lv.3） | 反應：取消 18m內目標的優勢或劣勢（CHA 次/長休）|
| 律法堡壘（Lv.6） | 魔法動作：花 1-5 SP 創建護盾（d8 骰），減傷直到長休 |
| 秩序之醉（Lv.14） | 附贈：D20 擲 9 以下視為 10 + 敵方攻擊不能有優勢 |
| 鐘律巡行（Lv.18） | 魔法動作：9m立方體內治療 100 HP + 修復 + 解除 6 環以下法術 |

**戰術重點：**
- 恢復平衡 = 超強反應能力，取消敵人優勢（或取消隊友劣勢）
- 律法堡壘 = 提前給隊友減傷護盾，不需專注，可與增益法術疊加
- 鐘律法術列表偏輔助/防禦 → 與術士本身的攻擊法術互補
- 最接近「輔助型術士」的子職，但仍有完整的爆發輸出能力

## G. 防禦層次

被攻擊時，按以下順序檢查可用防禦：

```
1. 基礎防禦等級（AC）
   → 無護甲（10+DEX）或 龍族（10+DEX+CHA）或 法師護甲（13+DEX）
   ↓ 被命中？
2. 反應防禦？
   → 護盾術（Shield）：AC+5 至下回合（1 環位）
   → 吸收元素（Absorb Elements）：元素傷害減半（如有準備）
   → 恢復平衡（鐘律子職）：取消敵人攻擊優勢
   ↓ 仍被打中？
3. 有預先施放的防禦法術？
   → 鏡影術（Mirror Image）— 被動生效，不消耗動作
   → 模糊術（Blur）— 攻擊者劣勢（專注）
   → 律法堡壘護盾（鐘律子職）— 消耗護盾骰減傷
   ↓ 持續受傷？
4. 血量見底 → 緊急脫離
   → 迷蹤步（Misty Step）— 附贈傳送 9m
   → 任意門（Dimension Door）— 150m傳送
   → 龍翼（龍族 Lv.14）— 飛行脫離
   → 形體啟示：熒光飛行（異怪 Lv.14）— 飛行脫離
```

**DM 操作：** 從角色卡的「防禦層次」欄位按順序檢查。
術士是**最脆弱的施法者之一**（D6 生命骰、無護甲），護盾術和站位是核心生存手段。
龍族術法的自然護甲 + 額外 HP 是例外——比其他術士硬很多。

## H. 等級區間變化

| 等級 | 關鍵變化 | 對決策流程的影響 |
|------|----------|-----------------|
| **Lv.1** | 天生法術（Innate Sorcery）2 次/長休 | 施法攻擊有優勢 + DC+1。戲法為主 |
| **Lv.2** | 超魔法 2 個 + 魔力點 2 SP | 核心上線。開始法術+超魔法組合。SP 極少，精打細算 |
| **Lv.3** | 子職特性 + 2 環法術 | 子職額外法術。資源稍多（3 SP） |
| **Lv.4** | ASI + 第 5 個戲法 | 提升 CHA → 提升 DC 和攻擊 |
| **Lv.5** | **3 環法術** + 術法復甦 | 火球/閃電束/加速術開放！短休可回 SP。傷害/控場大幅提升 |
| **Lv.7** | **術法化身** + 4 環法術 | SP 花 2 啟動天生法術 + 一次施法可用 2 個超魔法！雙超魔組合上線 |
| **Lv.9** | 5 環法術 + 9 SP | 怪物定身/寒冰錐/石牆術。SP 足夠支撐多次超魔法 |
| **Lv.10** | **超魔法 +2（共 4 個）** | 超魔法覆蓋更多情境 |
| **Lv.11** | 6 環法術 | 解離/連鎖閃電/群體暗示。戰略級法術 |
| **Lv.17** | **超魔法 +2（共 6 個）**、9 環法術 | 祈願術（Wish）。全部超魔法幾乎可覆蓋所有情境 |
| **Lv.20** | **奧術神化** | 天生法術啟動中每回合 1 個免費超魔法！SP 壓力大減 |

### SP 管理里程碑

| 等級 | SP 總量 | 術法復甦 | SP 經濟感受 |
|------|---------|---------|-----------|
| Lv.2-4 | 2-4 | 無 | 極緊張。每個 SP 都是重大抉擇 |
| Lv.5-6 | 5-6 | 短休回 2-3 | 稍緩。一場戰鬥可用 2-3 次超魔法 |
| Lv.7-9 | 7-9 | 短休回 3-4 | 舒適。可支撐每場 3-4 次超魔法 |
| Lv.10-16 | 10-16 | 短休回 5-8 | 充裕。可積極使用超魔法，偶爾轉換法術位 |
| Lv.17-20 | 17-20 | 短休回 8-10 | 豐富。超魔法幾乎每次施法都可搭配 |

## I. 子職業速查

DM 根據角色卡的子職類型，參考對應戰術提示：

### 龍族術法（Draconic Sorcery）
- 最硬的術士：額外 HP + 天然 AC（10+DEX+CHA）
- 省下法師護甲法術位 → 多一個法術選擇
- Lv.6 元素親和：同元素法術 + CHA 修正傷害 → 搭配變質法術全觸發
- Lv.14 龍翼：飛行 → 站位安全性大增

### 狂野魔法術法（Wild Magic Sorcery）
- 混沌之潮 → 免費優勢 → 施法觸發湧動表（DM 追蹤）
- 扭轉命運 = 1 SP 反應，±1d4 影響任何 d20
- Lv.14+ 湧動表擲兩次選一 → 風險大減
- DM 需熟悉狂野魔法湧動表（見 reference/class_progression/sorcerer.md）

### 異怪術法（Aberrant Sorcery）
- Lv.6 靈能術法：靈能法術用 SP 施放 = 省法術位 + 自動微妙
- SP 消耗比其他子職更快（超魔法 + 靈能術法都吃 SP）
- 心靈對話 = 無聲戰術溝通
- Lv.14 飛行/視隱 = SP 換機動力

### 鐘律術法（Clockwork Sorcery）
- 恢復平衡 = 反應取消優勢/劣勢（CHA 次/長休）→ 對 DM：敵方優勢被取消影響很大
- 律法堡壘 = SP 換 d8 護盾給隊友，不需專注 → 戰前 buff
- 法術列表偏輔助防禦 → 最接近「團隊型術士」
- Lv.14 秩序之醉：D20 ≤ 9 視為 10 + 敵方攻擊無優勢 → 1 分鐘無敵模式

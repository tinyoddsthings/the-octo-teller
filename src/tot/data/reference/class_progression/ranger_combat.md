# 遊俠（Ranger）戰鬥 SOP

> 本文件是遊俠的**戰鬥決策框架**。
> 具體法術/能力從角色卡讀取，套用到以下流程。
> 面板格式見 [combat_flow.md](../rules/combat_flow.md) C 節

---

## A. 職業戰鬥定位

| 項目 | 說明 |
|------|------|
| 戰鬥角色 | 半施法武裝打手（Martial Half-Caster）— 遠程或雙武器 |
| 主砲 | 武器攻擊（Weapon Attacks）— 長弓（Longbow）/ 雙武器（Dual Wield）|
| 核心機制 | **獵人印記（Hunter's Mark）** 加成 + 半施法者（Half-Caster）法術增幅 |
| 設計哲學 | 武器攻擊是免費穩定輸出，法術是增幅傷害與控場的調味料 |

### 遊俠 vs 其他武裝職業

| | 遊俠 | 戰士（Fighter）| 盜賊（Rogue）|
|---|--------|----------|----------|
| 法術欄位 | 有（半施法者，Lv.1 起）| 無（除弈劍客）| 無（除詭術師）|
| 欄位恢復 | **長休（Long Rest）** | — | — |
| at-will 傷害 | 高（武器攻擊 + 獵人印記）| 高（多次攻擊）| 中（偷襲攻擊，1 次/回合）|
| 決策壓力 | 專注法術選擇 + 附贈動作競爭 | 低（持續攻擊）| 中（定位取巧）|
| 探索效用 | **極高**（自然/追蹤/生存）| 低 | 中（技能多）|

**DM 心法：** 遊俠的回合核心是「武器攻擊」。
獵人印記（Hunter's Mark）在大多數戰鬥開局附贈施放後，
後續回合就是穩定的攻擊 + 印記傷害。法術欄位留給真正需要的時刻。

## B. 回合決策流程

### B-1. 主流程

```
回合開始
│
├─ 0. 回合開始觸發效果
│     └─ 有專注法術嗎？→ 標記在面板狀態欄
│
├─ 1. 附贈動作（有就先決定，不佔主動作）
│     ├─ 還沒上獵人印記？
│     │   └─ 戰鬥預期 ≥2 回合 → 附贈施放獵人印記
│     ├─ 獵人印記目標已死？
│     │   └─ 附贈移標記到新目標（免費，不消耗欄位）
│     ├─ 有其他附贈施放的法術？（如纏繞打擊）
│     │   └─ 見 B-2 判斷是否值得花欄位 + 專注衝突
│     ├─ 馴獸師：命令野獸行動（附贈）
│     ├─ 自然面紗（Lv.14）：附贈隱形
│     ├─ 雙武器戰鬥：附贈額外攻擊
│     │   └─ ⚠️ 與獵人印記施放/移標記衝突（同回合只能選一）
│     └─ 都不需要 → 跳過
│
├─ 2. 主動作（攻擊動作 = 預設最佳選擇）
│     ├─ ⚔️ 預設：攻擊動作（Attack Action）
│     │   ├─ Lv.5+：兩次武器攻擊
│     │   ├─ 每次命中 → 獵人印記加 1d6
│     │   └─ 子職加成（巨像殺手 +1d8 / 群戰突破額外攻擊 / 恐懼打擊 +2d6 等）
│     ├─ 🔮 值得花欄位施法嗎？→ 見 B-2
│     │   ├─ 範圍控場（糾纏術、荊棘叢生）
│     │   ├─ 治療（療傷術、復生術）
│     │   └─ 召喚（召喚野獸、群獸召喚）
│     ├─ 🩸 有免費種族能力可用？→ 優先使用（不花欄位）
│     └─ ✨ 戲法（Lv.2+：指引術等，非攻擊用途）
│
├─ 3. 移動
│     ├─ 弓手：保持安全距離（理想 9-18m / 30-18m）
│     ├─ 雙武器：進入 1.5m近戰範圍
│     └─ Lv.6+ 漫遊：速度 +3m，善用攀爬/游泳速度
│
└─ 回合結束
```

### B-2. 「該不該花法術欄位？」

遊俠是半施法者，欄位比全施法者少得多。法術增幅武器但不取代武器。

```
要花法術欄位嗎？
│
├─ Q1：獵人印記能免費施放嗎？
│   ├─ 宿敵（Favored Enemy）次數還有 → 免費，不消耗欄位
│   └─ 免費次數用完 → 需要消耗欄位
│
├─ Q2：這個法術比「攻擊 + 獵人印記」好多少？
│   ├─ 範圍控場能影響 3+ 個敵人 → 值得
│   ├─ 召喚法術能持續多回合 → 值得
│   ├─ 只是單體傷害 → 不值得，武器攻擊更穩
│   └─ 治療：隊友瀕死 → 必須花
│
├─ Q3：還剩幾個欄位？
│   ├─ 充裕（≥3）→ 可以使用
│   ├─ 偏少（1-2）→ 留給治療/緊急控場
│   └─ 0 → 純武器攻擊
│
└─ Q4：專注衝突？
    ├─ 新法術需要專注 且 獵人印記正在運作 → 放棄印記值得嗎？
    │   ├─ 新法術影響整場戰局（糾纏術、荊棘叢生）→ 值得切換
    │   └─ 獵人印記持續加傷害更好 → 不切換
    └─ 新法術不需專注 → 無衝突，安全施放
```

## C. 面板生成模板

DM 輪到遊俠行動時，從角色卡讀取具體能力，按以下模板生成面板。

### C-0. 狀態表補充：專注追蹤

狀態表（第一段）的 📌 狀態欄中，需要標示專注資訊：

| 情境 | 📌 狀態欄標示 |
|------|-------------|
| 正在維持專注法術 | `{法術名}專注中`（如：`獵人印記專注中 → {目標名}`）|
| 無專注 | 不標示（留給其他狀態）|

> 遊俠只能同時維持 **1 個** 專注法術。
> 大多數遊俠戰鬥法術都需要專注 → 選擇專注法術是核心決策。
> 法術表中標 `✅⚠️換` 的法術 = 施放時會自動結束現有專注。
> 標 `❌免` 的法術 = 不需專注，可與現有專注並存。

### C-1. 動作表（分區子表設計）

統一編號橫跨所有子表，玩家直接喊號選擇。
每區只顯示該類型需要的欄位，避免空白。

來源：角色卡「武器」。主武器排前，副手排後。

| # | 名稱 | 命中 | 傷害 | 射程/觸及 | 武器精通 | 備註 |
|---|------|------|------|----------|---------|------|
| 1 | {主武器 — 如長弓} | +{DEX + 熟練 + 戰鬥風格} | {damage} + 獵人印記 1d6 | {range} | {mastery} | {戰鬥風格加值} |
| 2 | {近戰武器 — 如短劍} | +{mod + 熟練} | {damage} | 1.5m | {mastery} | |
| 3 | {雙武器副手} | +{mod + 熟練} | {damage} | 1.5m | {mastery} | 附贈動作攻擊 |

欄位說明：
- **武器精通（Weapon Mastery）**：角色選擇的 2 種武器可使用精通屬性（如 Slow、Vex、Nick 等）
- **獵人印記加傷**：每次命中 +1d6（Lv.20 升為 +1d10），寫在主武器備註即可
- **戰鬥風格加值**：箭術（Archery）= 命中 +2；決鬥（Dueling）= 傷害 +2 等

> Lv.5+ 攻擊動作可攻擊 2 次。額外攻擊（Extra Attack）標示在武器區標題。

**🔮 法術區（消耗欄位）** — 欄位：{目前}/{最大}（{X}環）
來源：角色卡「已準備法術」中施法時間為「動作」的法術。

| # | 名稱 | 命中/難度（DC）| 傷害/效果 | 消耗欄位 | 專注 | 備註 |
|---|------|---------------|----------|---|------|------|
| N | {spell} | {+hit / DC} | {effect} | {環級} | ✅⚠️換 / ❌免 | |

欄位說明：
- **消耗欄位**：遊俠為半施法者，消耗欄位數 = 法術環級。
  欄位少且長休才恢復，每次使用都需慎重。
- **專注**：
  - `✅⚠️換` = 需要專注，施放時會取消目前的專注法術（通常是獵人印記）
  - `❌免` = 不需專注，可與現有專注並存
  - `🔄儀式` = 可用儀式施放（10 分鐘，不消耗欄位，僅限戰鬥外）

## D. 法術分類決策

遊俠的法術按戰鬥用途分為以下類型。
DM 從角色卡讀取具體法術，歸類後套用決策：

### D-1. 專注增益型（核心：獵人印記）

**特徵：** 附贈施放、需要專注、持續加成每回合傷害

| 判斷 | 決策 |
|------|------|
| 戰鬥剛開始，預期 ≥2 回合 | ✅ 第一回合附贈施放獵人印記 → 攻擊動作 |
| 戰鬥快結束（1 回合）| ❌ 不值得，直接攻擊 |
| 目標死了但戰鬥繼續 | 附贈移標記到新目標（免費）|
| 纏繞打擊（Ensnaring Strike）| 附贈施放 → 下次命中觸發束縛 → 但取代獵人印記專注 |

### D-2. 範圍控場型（如糾纏術、荊棘叢生）

**特徵：** 需要專注、影響地形/多目標、會取代獵人印記

| 判斷 | 決策 |
|------|------|
| 敵人密集（3+ 個在範圍內）| ✅ 值得放棄獵人印記 |
| 需要阻止敵人推進 | ✅ 荊棘叢生（Spike Growth）製造危險地形 |
| 只有 1-2 個敵人 | ❌ 獵人印記的持續加傷更好 |
| 已有其他隊友在控場 | ❌ 不需重複控場，維持印記 |

### D-3. 治療型（如療傷術、次級復原術、復生術）

**特徵：** 不需專注（療傷術、復生術）、緊急救援

| 判斷 | 決策 |
|------|------|
| 隊友瀕死（0 生命值（HP））| ✅ 必須治療，人命 > 傷害 |
| 隊友中毒/疾病 | ✅ 次級復原術（Lesser Restoration）|
| 隊友只是輕傷 | ❌ 留給專職治療者 |
| 復生術（Revivify）救人 | ✅ 絕對值得 3 環欄位 |

### D-4. 召喚型（如召喚野獸、群獸召喚）

**特徵：** 需要專注、持續多回合、額外行動經濟

| 判斷 | 決策 |
|------|------|
| 長期戰鬥（5+ 回合預期）| ✅ 召喚物的總輸出超過獵人印記 |
| 需要前排肉盾 | ✅ 召喚野獸擋住敵人 |
| 短期戰鬥或已有專注 | ❌ 獵人印記更穩 |

### D-5. 無專注傷害型（如荊棘箭雨、閃電箭、齊射召喚）

**特徵：** 不需專注、一次性傷害、可與獵人印記並存

| 判斷 | 決策 |
|------|------|
| 敵人群聚在範圍內 | ✅ 範圍傷害補充武器攻擊 |
| 可與獵人印記並存 | ✅ 不衝突，額外傷害 |
| 欄位充裕 | ✅ 可以自由使用 |
| 欄位緊張 | ❌ 留給治療/控場 |

## E. 獵人印記與專注管理

> 獵人印記（Hunter's Mark）是遊俠的招牌法術，正確使用是遊俠輸出的核心。

### E-1. 獵人印記基礎

| 項目 | 內容 |
|------|------|
| 施法時間 | 附贈動作（Bonus Action）|
| 持續時間 | 專注（Concentration），最長 1 小時（高環升級可延長）|
| 效果 | 標記目標；你每次以武器命中該目標 +1d6 傷害 |
| 移動標記 | 目標降至 0 生命值（HP）→ 附贈動作移到新目標（免費，不消耗欄位）|
| 免費施放 | 宿敵（Favored Enemy）提供免費次數（Lv.1 = 2 次 → Lv.17 = 6 次），長休恢復 |

### E-2. 獵人印記決策樹

```
獵人印記決策
│
├─ 戰鬥開始，還沒上印記
│   ├─ 預期 ≥2 回合？→ ✅ 第一回合附贈施放
│   ├─ 只剩 1 個弱敵？→ ❌ 直接攻擊解決
│   └─ 有更好的附贈要用？（如雙武器首回合）→ 權衡
│
├─ 印記目標死亡
│   ├─ 還有其他敵人？→ ✅ 附贈移標記（免費）
│   └─ 戰鬥結束？→ 印記自然消散
│
├─ 要切換到其他專注法術？
│   ├─ 糾纏術/荊棘叢生影響 3+ 敵人 → ✅ 放棄印記，控場更重要
│   ├─ 召喚野獸在長戰鬥中 → ✅ 召喚物的總價值 > 印記
│   └─ 單純想換法術但印記還在賺傷害 → ❌ 維持印記
│
├─ 印記被打斷（專注檢定失敗）
│   ├─ 免費次數還有？→ 下回合附贈重新施放（免費）
│   └─ 免費次數用完？→ 消耗欄位重新施放 或 改用其他專注法術
│
└─ Lv.13 不懈獵人（Relentless Hunter）
    └─ 受傷不會中斷獵人印記專注 → 不再需要專注檢定（僅限印記）
```

### E-3. 專注管理原則

| 原則 | 說明 |
|------|------|
| **印記是預設專注** | 除非有明確更好的選擇，否則維持獵人印記 |
| **命中次數 = 印記價值** | 額外攻擊（Lv.5）後每回合 2 次命中 = 2d6 額外傷害 → 印記更值得 |
| **免費次數優先** | 宿敵提供的免費次數不消耗欄位 → 優先用免費次數 |
| **專注保護投資** | 堅韌體質（Constitution）豁免加值影響專注維持 → 考慮戰場堅韌專長 |
| **Lv.13 里程碑** | 不懈獵人讓印記幾乎不會被打斷 → 印記成為永久增益 |
| **Lv.17 精準獵人** | 對印記目標攻擊有優勢 → 印記的價值再次大幅提升 |

## F. 子職業戰術差異

遊俠的子職影響戰鬥風格和額外能力：

### 獵人（Hunter）
- **巨像殺手（Colossus Slayer）：** 目標缺血時每回合 +1d8 → 對付 BOSS 型敵人
- **群戰突破（Horde Breaker）：** 額外攻擊鄰近的不同敵人 → 多敵人戰鬥
- 兩個選項可短休/長休後切換 → 根據預期戰鬥調整
- Lv.7 防禦策略：突圍（避 AoO）或多重攻擊防禦（連續被攻擊劣勢）
- Lv.11 卓越獵人之獵物：印記傷害擴散到 9m內另一個敵人
- Lv.15 卓越獵人防禦：反應獲得傷害抗性

### 馴獸師（Beast Master）
- 核心：**原始夥伴（Primal Companion）** 提供額外行動經濟
- 野獸預設閃避 → 需附贈動作命令才能攻擊
- Lv.5 可犧牲 1 次攻擊命令野獸 → 附贈動作留給獵人印記
- **附贈動作更擁擠**：命令野獸 vs 獵人印記 vs 雙武器攻擊
- 詳細野獸戰術見 C-4 節

### 幽暗潛獵者（Gloom Stalker）
- **恐懼伏擊者：** 首回合速度 +3m + 恐懼打擊（+2d6 心靈傷害，感知修正次數/長休）
- **先攻加值：** 加感知修正 → 經常先手
- **暗影視覺：** 黑暗中對黑暗視覺生物隱形 → 地下城/夜戰極強
- 戰鬥風格：**爆發型** — 第一回合傾盡全力
- Lv.11 潛獵者連擊：恐懼打擊升 2d8 + 突襲打擊/群體恐懼
- Lv.15 暗影閃避：反應讓攻擊劣勢 + 傳送 9m

### 精靈漫遊者（Fey Wanderer）
- **恐怖打擊：** 每回合 1 次武器命中 +1d4 心靈傷害（Lv.11 升 1d6）→ 被動加傷
- **超凡魅力：** 魅力檢定加感知修正 → 社交場景強
- 子職法術：魅惑人類、迷蹤步（Misty Step）→ 控制 + 機動性
- Lv.7 迷惑扭轉：反應強迫敵人魅惑/恐懼 → 反控場能力
- Lv.11 精類援軍：召喚精類免材料成分 + 1 次免費/免專注 → 額外行動經濟
- Lv.15 迷蹤漫遊者：迷蹤步免費多次 + 可帶一人 → 極強機動性

## G. 防禦層次（通用框架）

被攻擊時，按以下順序檢查可用防禦：

```
1. 基礎防禦等級（AC）（中甲 + 敏捷（DEX）+ 盾牌？）
   → 遊俠可穿中甲（最高半身甲 AC 15 + DEX(max 2) = 17）
   → 弓手不持盾；劍盾型可 +2 盾牌
   → 戰鬥風格防禦（Defense）= AC 再 +1
   ↓ 被命中？
2. 有反應防禦？
   → 幽暗潛獵者 Lv.15 暗影閃避：攻擊劣勢 + 傳送
   → 獵人 Lv.15 卓越獵人防禦：傷害抗性
   ↓ 仍被打中？
3. 有臨時生命值（HP）？
   → 不倦（Lv.10）：魔法動作獲得 1d8 + 感知修正臨時 HP
   ↓ 持續受傷？
4. 有位移能力？
   → 漫遊（Lv.6）：速度 +3m，拉開距離
   → 自然面紗（Lv.14）：附贈隱形 → 下回合脫離
   → 迷蹤步（精靈漫遊者子職法術）：附贈傳送 9m
   ↓ 危險！
5. 有治療法術？
   → 療傷術、神莓術 → 下回合自我治療
```

**DM 操作：** 從角色卡的「防禦層次」欄位按順序檢查。
遊俠防禦等級（AC）中等（14-17），以距離和機動性為主要生存手段。
弓手遊俠應優先保持距離，避免進入近戰。

## H. 等級區間變化

| 等級 | 關鍵變化 | 對決策流程的影響 |
|------|----------|-----------------|
| **Lv.1** | 獵人印記免費 2 次、武器精通 2 種 | 印記 + 攻擊為核心循環。欄位極少（2 個 1 環）|
| **Lv.2** | 戰鬥風格、戲法 2 個、靈巧探險者 | 箭術 +2 命中 → 弓手傷害穩定。戲法（指引術等）增加效用 |
| **Lv.3** | **子職** + 3 環欄位 | 戰鬥風格定型。獵人加傷 / 馴獸師加夥伴 / 幽暗爆發 / 精靈被動加傷 |
| **Lv.5** | **額外攻擊** + 2 環法術 + 印記免費 3 次 | 傷害翻倍。每回合 2 次攻擊 × 印記 = 2d6 額外 → 印記價值大增 |
| **Lv.6** | 漫遊：速度 +10、攀爬/游泳 | 機動性大增。弓手更容易保持距離 |
| **Lv.9** | 3 環法術 + 專精 | 群獸召喚、復生術等強力法術開放。專注決策更複雜 |
| **Lv.13** | **不懈獵人：印記免專注檢定** + 4 環法術 | 印記穩定性大幅提升，不再怕被打斷 |
| **Lv.14** | 自然面紗：附贈隱形 | 生存/戰術選項大增 |
| **Lv.17** | **精準獵人：對印記目標攻擊優勢** + 5 環法術 | 印記目標 = 必中目標，傷害巔峰 |
| **Lv.20** | 仇敵殺手：印記傷害升 d10 | 每次命中 +1d10，武器攻擊的終極形態 |

**獵人印記價值成長曲線：**
- Lv.1-4：1 次攻擊 × 1d6 → 印記平均 +3.5/回合
- Lv.5-12：2 次攻擊 × 1d6 → 印記平均 +7/回合 → 核心增益
- Lv.13+：不怕被打斷 → 印記成為穩定永久增益
- Lv.17+：攻擊優勢 + Lv.20 d10 → 印記價值巔峰

### 獵人（Hunter）

- Lv.11 免費/免專注召喚精類 → 與印記不衝突的額外行動經濟

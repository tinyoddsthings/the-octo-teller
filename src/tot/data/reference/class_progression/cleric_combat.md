# 牧師（Cleric）戰鬥 SOP

> 本文件是牧師的**戰鬥決策框架**。
> 具體法術/能力從角色卡讀取，套用到以下流程。
> 面板格式見 [combat_flow.md](../rules/combat_flow.md) C 節

---

## A. 職業戰鬥定位

| 項目 | 說明 |
|------|------|
| 戰鬥角色 | 支援核心 + 中距離傷害（Support Core / Mid-Range DPR）|
| 主砲 | 神聖火焰（Sacred Flame）— 戲法，無限使用；靈體守衛（Spirit Guardians）— 持續範圍傷害 |
| 核心機制 | **準備施法**：每日長休可更換整個法術列表，**多環級法術欄位**，長休全回 |
| 設計哲學 | 治療是安全網而非主輸出；祝福/控場提升全隊效率；靈體守衛 + 靈體武器是傷害巔峰組合 |

### 牧師 vs 其他施法者/治療者

| | 牧師 | 魔導士 | 德魯伊 |
|---|------|--------|--------|
| 欄位數量 | 多（Lv.4 = 7 個，分環級）| 極少（Lv.4 = 2 個）| 多（與牧師相同）|
| 欄位恢復 | **長休** | 短休 | 長休 |
| at-will 傷害 | 中（戲法無修正加成，Lv.7 前）| 高（魔能轟擊 + 祈喚加成）| 中 |
| 治療能力 | **最強**（多種治療法術 + 領域加成）| 無（天界宗主例外）| 有但法術選擇較少 |
| 護甲 | 中/重甲 + 盾牌 = 高防禦等級（AC）| 輕甲 = 低 | 中甲 + 盾牌（非金屬）|
| 決策壓力 | 每回合在治療/傷害/增益間抉擇 | 幾乎每回合 EB | 在施法/變形間抉擇 |

**DM 心法：** 牧師的回合需要最多判斷——先評估全隊生命值（HP）狀態，再決定治療、增益還是輸出。
關鍵原則：**不要過度治療**。預防傷害（祝福術、靈體守衛殺敵）> 事後治療。

## B. 回合決策流程

### B-1. 主流程

```
回合開始
│
├─ 0. 回合開始觸發效果
│     ├─ 有專注法術嗎？→ 標記在面板狀態欄
│     └─ 靈體守衛（Spirit Guardians）生效？
│         → 剛進入或在範圍內開始回合的敵人：感知（WIS）豁免
│
├─ 1. 緊急評估：有人瀕死嗎？
│     ├─ 有隊友 0 生命值（HP）→ 最優先救人
│     │   ├─ 治療之言（Healing Word）→ 附贈動作遠程救起（最佳選擇）
│     │   └─ 療傷術（Cure Wounds）→ 動作觸碰治療（佔主動作）
│     └─ 無人瀕死 → 進入一般決策
│
├─ 2. 主動作（按優先順序）
│     ├─ 🔮 有高價值法術可施放嗎？→ 見 B-2
│     │   ├─ 專注法術尚未開啟？
│     │   │   ├─ 靈體守衛（Spirit Guardians）→ 多敵近戰時最強
│     │   │   ├─ 祝福術（Bless）→ 全隊增益
│     │   │   └─ 其他專注法術
│     │   ├─ 非專注法術（可與現有專注並存）
│     │   │   └─ 指引之箭（Guiding Bolt）、造成傷害（Inflict Wounds）等
│     │   └─ 治療法術（隊友生命值 < 50%）
│     ├─ ⚡ 引導神力可用？→ 見 E 節
│     │   ├─ 神聖火花（Divine Spark）→ 治療或傷害（魔法動作）
│     │   ├─ 領域引導能力 → 依子職業決定
│     │   └─ 驅散不死生物（Turn Undead）→ 面對不死時
│     ├─ ✨ 預設：戲法攻擊
│     │   ├─ 神聖火焰（Sacred Flame）→ 敏捷（DEX）豁免，忽略掩蔽
│     │   ├─ 亡者喪鐘（Toll the Dead）→ 感知（WIS）豁免，已受傷目標d12
│     │   └─ 光輝之語（Word of Radiance）→ 多敵鄰接時
│     └─ 🩸 有免費種族能力可用？→ 優先使用
│
├─ 3. 附贈動作（有就用，不佔主動作）
│     ├─ 治療之言（Healing Word）→ 最常見的附贈法術
│     │   └─ 救瀕死隊友 or 補血（不與主動作法術同回合——
│     │       ⚠️ 同回合施放附贈法術時，主動作只能用戲法）
│     ├─ 靈體武器（Spiritual Weapon）→ 附贈攻擊（2024 PHB 需專注）
│     │   └─ 已召喚 → 附贈攻擊（免費）
│     └─ 領域附贈能力？→ 依子職業
│
├─ 4. 移動
│     ├─ 有靈體守衛？→ 主動貼近敵人（讓更多敵人進入範圍）
│     ├─ 無靈體守衛？→ 保持中距離（9-12m），前排後方
│     └─ 穿重甲 + 盾牌 = 高防禦等級（AC），不怕借機攻擊但速度可能慢
│
└─ 回合結束
```

### B-2. 「該用哪個環級的法術欄位？」

牧師欄位多且分環級，資源管理是長期戰略：

```
要花法術欄位嗎？用幾環？
│
├─ Q1：戰鬥預期還有幾場？
│   ├─ 今天最後一場 → 放心用高環
│   ├─ 還有 1-2 場 → 保留 1 個最高環和回生術（Revivify）欄位
│   └─ 不確定 → 保守使用，優先低環
│
├─ Q2：法術升環有明顯收益嗎？
│   ├─ 療傷術（Cure Wounds）：每升 1 環 +1d8 治療 → 值得
│   ├─ 靈體守衛（Spirit Guardians）：每升 1 環 +1d8 傷害/目標 → 非常值得
│   ├─ 祝福術（Bless）：升環無收益 → 用 1 環就好
│   └─ 命令術（Command）：每升 1 環 +1 目標 → 多敵時值得
│
├─ Q3：專注衝突？
│   ├─ 已有更好的專注（如靈體守衛）→ 不要換成祝福術
│   ├─ 新法術效果 > 現有專注 → 切換
│   └─ 非專注法術（指引之箭、療傷術）→ 無衝突，可自由使用
│
├─ Q4：回生術（Revivify）保底？
│   └─ 永遠保留至少 1 個 3 環欄位 + 300GP 鑽石
│       → 除非確定今天不會再戰鬥
│
└─ Q5：治療 vs 輸出的機會成本？
    ├─ 治療 1d8+WIS 生命值 ≈ 阻止敵人一次攻擊
    │   → 如果殺死敵人能阻止更多攻擊 → 攻擊更划算
    ├─ 隊友 > 50% 生命值 → 不急著治療，輸出/增益更好
    └─ 隊友 < 25% 生命值 → 治療優先（避免倒地）
```

## C. 面板生成模板

DM 輪到牧師行動時，從角色卡讀取具體能力，按以下模板生成面板。
（面板分區子表結構定義見 [combat_flow.md](../rules/combat_flow.md) C 節）

### C-0. 狀態表補充：專注追蹤 + 多環欄位

狀態表（第一段）中，需要標示專注資訊和多環級欄位：

| 情境 | 📌 狀態欄標示 |
|------|-------------|
| 正在維持專注法術 | `{法術名}專注中`（如：`靈體守衛專注中`）|
| 無專注 | 不標示（留給其他狀態）|

> 牧師只能同時維持 **1 個** 專注法術。
> 法術表中標 `✅⚠️換` 的法術 = 施放時會自動結束現有專注。
> 標 `❌免` 的法術 = 不需專注，可與現有專注並存。

**法術欄位顯示格式：** 多環級分開列出。

| 範例 | 說明 |
|------|------|
| `3×1環 + 2×2環` | Lv.3 牧師（4個1環 + 2個2環，已用1個1環）|
| `4×1環 + 3×2環 + 2×3環` | Lv.5 牧師（滿欄位）|

### C-1. 動作表（分區子表設計）

統一編號橫跨所有子表，玩家直接喊號選擇。
每區只顯示該類型需要的欄位，避免空白。

來源：角色卡「戲法」。攻擊戲法排前，輔助排後。

| # | 名稱 | 命中/難度（DC）| 傷害/效果 | 備註 |
|---|------|---------------|----------|------|
| 1 | {攻擊戲法 — 通常神聖火焰（Sacred Flame）} | 難度（DC）{法術DC} 敏捷（DEX）| {damage} 光輝 | 忽略掩蔽 |
| 2 | {其他攻擊戲法 — 如亡者喪鐘（Toll the Dead）} | 難度（DC）{法術DC} 感知（WIS）| {damage} 死靈 | 已受傷 d12 |

**🔮 法術區（消耗欄位）** — 欄位：{X}×1環 + {Y}×2環 + {Z}×3環 ...
來源：角色卡「準備法術」中施法時間為「動作」的法術。

| # | 名稱 | 命中/難度（DC）| 傷害/效果 | 消耗欄位 | 專注 | 備註 |
|---|------|---------------|----------|---|------|------|
| N | {spell} | {+hit / 難度（DC）} | {射程}・{effect} | {最低環級}+ | ✅⚠️換 / ❌免 | {領域加成} |

欄位說明：
- **消耗欄位**：標示該法術的**最低環級**。牧師可以用更高環級欄位施放以獲得增強效果（升環施法 Upcasting）。標示格式 `{環級}+`（如 `1+` 表示最低 1 環，可用更高環）。
- **專注**：
  - `✅⚠️換` = 需要專注，施放時會取消目前的專注法術
  - `❌免` = 不需專注，可與現有專注並存
  - `🔄儀式` = 可用儀式施放（10 分鐘，不消耗欄位，僅限戰鬥外）
- **領域加成**：如生命領域的生命門徒（Disciple of Life）額外治療 = 2 + 欄位等級

## D. 法術分類決策

牧師的法術按戰鬥用途分為以下類型。
DM 從角色卡讀取具體準備法術，歸類後套用決策：

### D-1. 治療型（如 Cure Wounds、Healing Word）

**特徵：** 恢復隊友生命值，牧師核心角色

| 判斷 | 決策 |
|------|------|
| 隊友 0 生命值 → 瀕死中 | ✅ **最高優先**。治療之言（附贈）→ 主動作還能做別的 |
| 隊友 < 25% 生命值 | ✅ 治療避免倒地，但先考慮能否一回合殺死攻擊者 |
| 隊友 50-75% 生命值 | ❌ 通常不治。祝福術/攻擊更划算 |
| 隊友 > 75% 生命值 | ❌ 絕對不治。浪費欄位 |
| 戰鬥外治療 | 用低環治療 or 祈禱治療（Prayer of Healing）效率最高 |

> **黃金法則：** 把敵人殺死 = 阻止所有未來傷害。治療只是阻止 1 回合傷害。
> 除非隊友即將倒地，否則優先消滅威脅。

### D-2. 增益型（如 Bless、Shield of Faith）

**特徵：** 提升全隊效率，通常需要專注，效果隨回合數累積

| 判斷 | 決策 |
|------|------|
| 戰鬥剛開始，預期 ≥ 3 回合 | ✅ 祝福術第一回合開啟 → 全隊 +1d4 命中/豁免 |
| 只剩 1-2 敵人 | ❌ 戰鬥快結束，不值得花欄位 + 專注 |
| 已有靈體守衛在專注 | ❌ 不要為了祝福術放棄靈體守衛 |
| 前排重甲隊友被集火 | 信仰之盾 +2 防禦等級 → 值得（尤其低環欄位）|

### D-3. 傷害型（如 Guiding Bolt、Inflict Wounds、Spirit Guardians）

**特徵：** 直接造成傷害，部分有附加效果

| 判斷 | 決策 |
|------|------|
| 多敵近戰、預期 ≥ 2 回合 | ✅ 靈體守衛 → 每敵每回合 3d8（最高效傷害法術）|
| 單體高優先目標 | ✅ 指引之箭（4d6 + 下次攻擊優勢）或造成傷害（3d10 近戰）|
| 用戲法就能解決 | ❌ 不花欄位，神聖火焰/亡者喪鐘 |
| 靈體守衛已在運作 | 用戲法/引導神力輸出，不需再花攻擊法術欄位 |

### D-4. 控場型（如 Command、Hold Person、Banishment）

**特徵：** 限制敵人行動，改變戰局

| 判斷 | 決策 |
|------|------|
| BOSS 單體 + 感知（WIS）豁免不高 | ✅ 束縛人類（Hold Person）→ 麻痺 = 自動暴擊 |
| 多個中等敵人 | ✅ 命令術（Command）升環 → 同時影響多目標 |
| 外位面生物 | ✅ 放逐術（Banishment）→ 直接移除戰場 |
| 目標感知豁免很高 | ❌ 太冒險，不如用確定性傷害/增益 |

### D-5. 防禦型（如 Shield of Faith、Sanctuary、Death Ward）

**特徵：** 保護自己或隊友

| 判斷 | 決策 |
|------|------|
| 自己維持靈體守衛且被集火 | ✅ 庇護術（Sanctuary）保護自己 + 維持專注 |
| 預期高傷害 BOSS 戰 | ✅ 戰前施放防死結界（Death Ward）→ 不消耗戰鬥動作 |
| 前排隊友防禦等級（AC）不足 | 信仰之盾 +2 防禦等級 → 1 環划算 |
| 安全站位、沒有威脅 | ❌ 不需防禦法術 |

## E. 引導神力（Channel Divinity）

### E-1. 引導神力決策

引導神力是**獨立於法術欄位的資源**，短休恢復。使用時機與法術不同：

```
使用引導神力嗎？
│
├─ Q1：短休前還需要幾次？
│   ├─ 剩 2 次且預期只剩 1 場戰鬥 → 放心用
│   ├─ 剩 1 次 → 保留給高價值時機
│   └─ 剩 0 次 → 等短休
│
├─ Q2：用哪個引導能力？
│   ├─ 需要治療但不想花法術欄位？→ 神聖火花（治療模式）
│   ├─ 需要傷害但目標體魄（CON）豁免不高？→ 神聖火花（傷害模式）
│   ├─ 面對不死生物群？→ 驅散不死生物
│   └─ 領域能力觸發條件滿足？→ 使用領域引導
│
└─ Q3：是否與法術行動衝突？
    ├─ 引導神力 = 魔法動作（佔主動作）
    │   → 不能同回合再施法（除戲法附贈）
    ├─ 但不受「附贈法術限制」影響
    │   → 可以附贈治療之言 + 主動作引導神力 ✅
    └─ 某些領域引導（如詭術分身）是附贈動作
        → 主動作仍可施法 ✅
```

### E-2. 神聖火花（Divine Spark）使用時機

| 情境 | 模式 | 說明 |
|------|------|------|
| 隊友中傷、不想花法術欄位 | 治療 | {Xd8+WIS} 自動治療，不消耗法術欄位 |
| 敵人體魄（CON）低 | 傷害 | {Xd8+WIS} 傷害，成功豁免減半 |
| 法術欄位緊張 | 治療/傷害 | 替代法術欄位的免費資源 |
| 短休前最後一戰 | 隨意 | 反正短休回 1 次 |

> **成長曲線：** Lv.2 = 1d8+WIS、Lv.7 = 2d8+WIS、Lv.13 = 3d8+WIS、Lv.18 = 4d8+WIS

### E-3. 驅散不死生物（Turn Undead）使用時機

| 情境 | 決策 |
|------|------|
| 面對 3+ 不死生物 | ✅ 一次性控制大量不死 |
| Lv.5+：面對低階不死 | ✅ 灼燒不死（Sear Undead）額外造成 {WIS mod}d8 傷害 |
| 面對單一強力不死 BOSS | ❓ 感知豁免可能高 → 風險評估 |
| 無不死生物 | ❌ 無效 |

## F. 領域差異

牧師的子職業（領域 Domain）深刻影響戰鬥風格。DM 從角色卡讀取領域後，參考對應調整：

### 生命領域（Life Domain）— 治療專精

| 調整項目 | 說明 |
|----------|------|
| **護甲** | 重甲訓練（基礎職業選守護者即可）→ 防禦等級（AC）高 |
| **治療加成** | 生命門徒（Disciple of Life）：法術治療 +2+環級 → 大幅提升每個治療法術效率 |
| **自我治療** | Lv.6 受祝治療者（Blessed Healer）：治療他人時自己也回 2+環級 生命值 |
| **引導神力** | 保全生命（Preserve Life）：5×等級 生命值分配給血量過半的隊友 |
| **決策偏移** | 更傾向治療（因為加成讓治療效率遠超其他領域）|
| **Lv.17** | 至高治療（Supreme Healing）：治療骰全取最大值 |

### 光明領域（Light Domain）— 火焰與範圍傷害

| 調整項目 | 說明 |
|----------|------|
| **護甲** | 中甲 + 盾牌（無重甲，除非選守護者）|
| **領域法術** | 火球術（Fireball）、灼熱射線（Scorching Ray）→ 攻擊性法術大增 |
| **反應** | 守護閃光（Warding Flare）：反應使攻擊劣勢，{WIS mod} 次/長休 |
| **引導神力** | 黎明光輝（Radiance of the Dawn）：9m 範圍 2d10+等級 光輝傷害 + 驅散魔法黑暗 |
| **決策偏移** | 更偏進攻型施法者，火球 + 靈體守衛 = 範圍傷害專家 |
| **Lv.6** | 守護閃光改短休恢復 + 給臨時生命值 |

### 詭術領域（Trickery Domain）— 隱匿與操控

| 調整項目 | 說明 |
|----------|------|
| **護甲** | 中甲 + 盾牌 |
| **領域法術** | 隱形術（Invisibility）、無痕行蹤（Pass without Trace）→ 非戰鬥效用極高 |
| **引導神力** | 召喚分身（Invoke Duplicity）：**附贈動作**施放 → 不佔主動作。可從幻象位置施法 + 近戰攻擊優勢 |
| **Lv.6** | 詭術者的換位：附贈移動分身時與幻象交換位置 → 戰術傳送 |
| **決策偏移** | 更靈活、更多戰術選項、但正面戰力稍弱 |

### 戰爭領域（War Domain）— 武器與強化攻擊

| 調整項目 | 說明 |
|----------|------|
| **護甲** | 重甲 + 盾牌（選守護者）→ 最高防禦等級 |
| **領域法術** | 靈體武器、信仰之盾、靈體守衛 → 全部自動準備（免費）|
| **引導能力** | 引導打擊（Guided Strike）：攻擊未中 +10 加值；戰祭司（War Priest）：附贈動作武器攻擊（{WIS mod} 次/短休）|
| **Lv.6** | 戰神祝福：引導神力施放信仰之盾/靈體武器 → 不消耗法術欄位 + 不需專注 |
| **決策偏移** | 更偏近戰戰士型。武器攻擊 + 神聖打擊 + 靈體守衛 = 前排輸出 |

## G. 防禦層次（通用框架）

被攻擊時，按以下順序檢查可用防禦：

```
1. 基礎防禦等級（AC）
   （中甲/重甲 + 盾牌 + 可能的信仰之盾 → 通常 18-21）
   ↓ 被命中？
2. 反應防禦？
   ├─ 守護閃光（Warding Flare）→ 光明領域：使攻擊劣勢
   ├─ 引導打擊（反向）→ 戰爭領域：盟友未命中 +10
   └─ 從道具/多職業檢查其他反應
   ↓ 仍被打中？
3. 有預先施放的防禦法術？
   ├─ 信仰之盾（Shield of Faith）→ 防禦等級 +2
   ├─ 防死結界（Death Ward）→ 首次降至 0 HP 時改為 1 HP
   └─ 其他增益效果
   ↓ 持續受傷？
4. 自我治療能力？
   ├─ 治療之言（Healing Word）→ 附贈動作自補
   ├─ 神聖火花（Divine Spark）→ 治療模式自補
   └─ 生命領域：受祝治療者 → 治療他人時順便自補
   ↓ 危險！
5. 專注保護
   ├─ 受傷時體魄（CON）豁免維持專注 → 難度（DC）= max(10, 傷害/2)
   ├─ 戰爭施法者（War Caster）專長（如有）→ 專注豁免優勢
   └─ 體魄（CON）豁免不佳 → 考慮預防性躲避（Dodge）
```

**DM 操作：** 牧師通常有高防禦等級（AC）（18-21），生存力遠優於魔導士。
核心風險是**專注法術被打斷**（靈體守衛等），要提醒玩家專注檢定。

## H. 等級區間變化

| 等級 | 關鍵變化 | 對決策流程的影響 |
|------|----------|-----------------|
| **Lv.1-2** | 戲法 3 個、1 環 2-3 個欄位、Lv.2 引導神力 | 戲法為主。祝福術或療傷術是主要法術。引導神力（神聖火花）提供免費治療/傷害 |
| **Lv.3** | 子職業、2 環法術解鎖、領域法術 | **靈體武器**登場 → 附贈動作經濟大幅提升。領域引導能力開放 |
| **Lv.5** | 3 環法術、灼燒不死生物 | **靈體守衛**登場 → 牧師傷害巔峰。回生術（Revivify）保底。戰鬥風格定型 |
| **Lv.7** | 4 環法術、神佑打擊 | 放逐術（Banishment）開放。神聖打擊/強效施法 → 戲法/武器傷害加成 |
| **Lv.9** | 5 環法術 | 群體療傷術（Mass Cure Wounds）、焰擊術（Flame Strike）→ 範圍治療/傷害 |
| **Lv.10** | 神聖干涉（Divine Intervention）| 1次/長休免費施放 5 環以下法術 → 緊急王牌 |
| **Lv.14** | 強化神佑打擊 | 神聖打擊 2d8 或強效施法 + 臨時生命值 → 持續戰力提升 |
| **Lv.17+** | 子職業頂點、8-9 環法術 | 至高治療/光明之冠/戰鬥化身 → 子職業決定戰鬥風格上限 |

**牧師等級核心里程碑：**
- **Lv.3 前：** 純支援型——祝福術 + 治療 + 戲法
- **Lv.3-4：** 靈體武器 → 附贈動作經濟改善
- **Lv.5+：** 靈體守衛定型 → 前排範圍傷害 + 支援混合體
- **Lv.10+：** 神聖干涉 = 每天一次免費王牌

## I. 子職業速查

DM 根據角色卡的領域類型，參考對應戰術提示：

### 生命領域（Life Domain）
- 治療法術額外恢復 2+環級 生命值 → 每次治療都比其他領域多
- 保全生命（引導神力）：5×等級 的治療池分配給血量過半（Bloodied）的隊友
- Lv.6 治療他人時自己回血 → 鼓勵積極治療
- Lv.17 治療骰全取最大值 → 治療量翻倍
- **定位：** 團隊生命線，危急時刻的支柱

### 光明領域（Light Domain）
- 守護閃光（反應）：使攻擊劣勢，WIS mod 次/長休
- 黎明光輝（引導神力）：9m 範圍 2d10+等級 光輝傷害
- 領域法術含火球術、灼熱射線 → 攻擊型牧師
- Lv.17 光明之冠：敵人對火焰/光輝法術豁免劣勢
- **定位：** 範圍傷害施法者兼備防禦反應

### 詭術領域（Trickery Domain）
- 召喚分身（引導神力，附贈動作）：從幻象位置施法 + 近戰優勢
- Lv.6 換位：與幻象交換位置 → 戰術傳送
- 領域法術含隱形術、無痕行蹤 → 潛行與控制
- Lv.17 幻象結束時治療 + 盟友也獲攻擊優勢
- **定位：** 靈活的戰術支援，適合需要創意解法的場景

### 戰爭領域（War Domain）
- 引導打擊：攻擊未中 +10 → 關鍵一擊保證命中
- 戰祭司（附贈動作）：附贈武器攻擊，WIS mod 次/短休
- Lv.6 戰神祝福：引導神力施放信仰之盾/靈體武器 → 不消耗欄位 + 不需專注
- Lv.17 戰鬥化身：鈍擊/穿刺/揮砍抗性 → 物理傷害減半
- **定位：** 前排戰士型牧師，高防禦等級 + 武器輸出 + 法術支援

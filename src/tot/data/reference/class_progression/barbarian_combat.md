# 野蠻人（Barbarian）戰鬥 SOP

> 本文件是野蠻人的**戰鬥決策框架**。
> 具體武器/能力從角色卡讀取，套用到以下流程。
> 面板格式見 [combat_flow.md](../rules/combat_flow.md) C 節

---

## A. 職業戰鬥定位

| 項目 | 說明 |
|------|------|
| 戰鬥角色 | 近戰持續輸出 + 坦克（Melee Sustained DPR + Tank）|
| 主砲 | 武器攻擊（Weapon Attack）— 巨斧（Greataxe）/ 巨劍（Greatsword）等大型武器 |
| 核心機制 | **狂暴（Rage）**：附贈動作（Bonus Action）啟動，物理抗性 + 傷害加成 + 力量優勢 |
| 設計哲學 | 狂暴中衝入敵陣吸收傷害、輸出穩定近戰傷害，是隊伍的前線支柱 |

### 野蠻人 vs 戰士（Fighter）

| | 野蠻人 | 戰士 |
|---|--------|----------|
| 防禦方式 | 狂暴抗性（物理半傷）+ 高生命值（HP）（d12）| 高防禦等級（AC）+ 護甲 |
| 輸出模式 | 狂暴傷害加成 + 魯莽攻擊（Reckless Attack）優勢 | 動作如潮（Action Surge）爆發 + 戰鬥風格 |
| 資源節奏 | 狂暴次數有限（短休回 1、長休全回）| 動作如潮短休回、第二風短休回 |
| 決策壓力 | 何時開狂暴、是否魯莽攻擊（承受風險換傷害）| 何時動作如潮、戰術機動 |

**DM 心法：** 野蠻人的回合節奏明快——確認狂暴狀態 → 魯莽攻擊 → 砍人。
核心決策只有兩個：**要不要開狂暴** 和 **要不要魯莽攻擊**。

## B. 回合決策流程

### B-1. 主流程

```
回合開始
│
├─ 0. 回合開始觸發效果
│     ├─ 狂暴（Rage）中嗎？→ 標記在面板狀態欄
│     ├─ 子職回合開始效果？（如世界樹→臨時生命值（HP）分配）
│     └─ 其他持續效果檢查
│
├─ 1. 附贈動作（Bonus Action）（有就先決定）
│     ├─ 還沒開狂暴？→ 見 B-2 判斷是否啟動
│     │   └─ 啟動狂暴 → 附贈動作消耗
│     │       └─ Lv.7+ 本能猛撲（Instinctive Pounce）：
│     │           可移動至多一半速度（不觸發借機攻擊）
│     ├─ 已在狂暴中，需要維持？
│     │   └─ 本回合預計不會攻擊或迫使豁免？→ 附贈動作延續狂暴
│     ├─ 子職附贈動作？
│     │   ├─ 狂戰士 Lv.14：威嚇存在（Intimidating Presence）
│     │   ├─ 野性之心（鷹）：附贈衝刺（Dash）
│     │   ├─ 世界樹 Lv.14：沿樹而行（傳送 18m）
│     │   └─ 狂信者 Lv.10：熱誠存在（Zealous Presence）
│     └─ 都不需要 → 跳過
│
├─ 2. 主動作（Attack Action）
│     ├─ ⚔️ 預設：武器攻擊
│     │   └─ 使用魯莽攻擊（Reckless Attack）嗎？→ 見 E 節
│     │       ├─ 是 → 本回合所有力量（STR）攻擊有優勢
│     │       │   └─ Lv.9+ 殘暴打擊（Brutal Strike）？
│     │       │       → 放棄一次優勢換 +1d10 傷害 + 特殊效果
│     │       └─ 否 → 正常攻擊
│     ├─ Lv.5+ 額外攻擊（Extra Attack）→ 2 次攻擊
│     ├─ 🩸 有免費種族能力可用？→ 優先使用
│     └─ 🔧 其他動作（擒抱/推擠——狂暴中力量檢定有優勢）
│
├─ 3. 移動
│     └─ 近戰職業：衝入敵群、站穩前線
│         ├─ 優先走向最大威脅或最脆弱的隊友身邊
│         ├─ Lv.5+ 疾走（Fast Movement）：速度 +3m
│         └─ 擋在法師/遠程隊友前面，迫使敵人打你（你有抗性）
│
└─ 回合結束
      └─ ⚠️ 狂暴維持檢查：本回合是否有攻擊/迫使豁免/用附贈延續？
          ├─ 有 → 狂暴繼續到下一回合結束
          └─ 沒有 → 狂暴結束！
```

### B-2. 「該不該開狂暴？」

狂暴次數有限，每次啟動都是資源決策：

```
要開狂暴嗎？
│
├─ Q1：戰鬥有多危險？
│   ├─ 高威脅（Boss / 大量敵人 / 被集火）→ 立刻開
│   ├─ 中威脅 → 繼續判斷
│   └─ 低威脅（雜魚清掃）→ 可以不開，省資源
│
├─ Q2：還剩幾次狂暴？
│   ├─ 滿（或接近滿）→ 放心開
│   ├─ 剩 1 次 → 只在高威脅戰鬥開
│   └─ 0 次 → 無法狂暴，純武器攻擊
│
├─ Q3：戰鬥後能休息嗎？
│   ├─ 能長休 → 放心全用
│   ├─ 能短休 → 短休恢復 1 次，但仍需節約
│   └─ 連續戰鬥 → 嚴格保留
│
├─ Q4：預期戰鬥持續幾回合？
│   ├─ ≥3 回合 → 狂暴的抗性和傷害加成值得
│   ├─ 1-2 回合 → 可能不值得（但如果很危險仍然開）
│   └─ 不確定 → 保險起見開啟
│
└─ Q5：需要施法嗎？（多職 / 種族法術）
    ├─ 是 → 不能開狂暴（狂暴中無法施法或專注）
    └─ 否 → 正常判斷
```

### C-0. 狀態表補充：狂暴追蹤

狀態表（第一段）的 📌 狀態欄中，需要標示狂暴資訊：

| 情境 | 📌 狀態欄標示 |
|------|-------------|
| 正在狂暴 | `🔴 狂暴中（第 X 回合）`（追蹤回合數，最多 10 分鐘）|
| 狂暴 + 魯莽攻擊 | `🔴 狂暴中` `⚠️ 魯莽（被攻擊有優勢 至下回合開始）` |
| 未狂暴 | 不標示 |

> 狂暴中的固定效果（不需每回合重新宣告）：
> - 力量（STR）武器/徒手打擊傷害 + 狂暴傷害加值
> - 鈍擊（Bludgeoning）、穿刺（Piercing）、揮砍（Slashing）抗性
> - 力量檢定和豁免有優勢
> - 無法施法或專注

## D. 狂暴決策

狂暴（Rage）是野蠻人的核心機制，所有戰鬥決策都圍繞它展開。

### D-1. 狂暴效果總覽

| 效果 | 說明 |
|------|------|
| 傷害加成 | 力量武器/徒手命中時 +{狂暴傷害}（見等級表）|
| 物理抗性 | 鈍擊、穿刺、揮砍傷害減半 |
| 力量優勢 | 力量檢定和豁免有優勢（擒抱/推擠超強）|
| 無法施法 | 無法施放或專注任何法術 |
| 武器精通替換 | 可將 1 種武器精通替換為推擠/絆倒/緩速 |

### D-2. 狂暴維持規則（2024 PHB）

```
狂暴持續到你回合結束。
要延續到下一回合結束，必須在本回合做到以下之一：
│
├─ (a) 對敵人發動攻擊
├─ (b) 迫使敵人進行豁免
└─ (c) 以附贈動作延續狂暴
│
狂暴最多持續 10 分鐘（100 回合）。
│
立即結束：
├─ 失去意識（Unconscious）
├─ 獲得失能（Incapacitated）狀態
└─ 你主動結束（無需動作）
```

> **DM 注意：** 2024 版狂暴維持比舊版寬鬆——可以用附贈動作延續，不一定非要攻擊或受傷。但如果三個條件都沒達成，狂暴在回合結束時終止。

### D-3. 狂暴資源管理

| 等級 | 狂暴次數/長休 | 狂暴傷害 | 策略建議 |
|------|:------------:|:--------:|----------|
| 1-2 | 2 | +2 | 極度珍貴。只在必要戰鬥開啟 |
| 3-5 | 3 | +2 | 每次短休回 1 次，節奏稍寬 |
| 6-8 | 4 | +2 | 可應付大多數冒險日 |
| 9-11 | 4 | +3 | 傷害提升，同時有殘暴打擊 |
| 12-15 | 5 | +3 | Lv.15 持續狂暴：啟動時回復 1 次 |
| 16-17 | 5→6 | +4 | 資源充裕 |
| 18-20 | 6 | +4 | 幾乎可以每場戰鬥狂暴 |

## E. 魯莽攻擊決策

魯莽攻擊（Reckless Attack）是每回合的風險/報酬判斷，不消耗任何資源。

### E-1. 魯莽攻擊機制

- **啟動時機：** 你回合的第一次力量攻擊時宣告
- **好處：** 該回合所有力量攻擊擲骰獲得優勢
- **代價：** 所有針對你的攻擊擲骰獲得優勢，直到你的下一回合開始

### E-2. 該不該魯莽攻擊？

```
魯莽攻擊決策
│
├─ 狂暴中嗎？
│   ├─ 是 → 有物理抗性，代價較低 → 傾向使用
│   └─ 否 → 代價很高（沒有抗性抵銷被打痛）→ 慎重
│
├─ 本回合命中有多重要？
│   ├─ 必須命中（收尾/觸發特殊效果）→ ✅ 使用
│   ├─ 正常輸出回合 → 看以下風險
│   └─ 無所謂（敵人快死了）→ ❌ 不需要
│
├─ 面臨多少攻擊源？
│   ├─ 1-2 個近戰敵人 → 風險可控 ✅
│   ├─ 3+ 個敵人包圍 → 高風險 ⚠️
│   └─ 有大量遠程攻擊者 → 高風險 ⚠️
│
├─ 目前生命值如何？
│   ├─ 高（>70%）→ 承受得起 ✅
│   ├─ 中（30-70%）→ 看隊伍治療資源
│   └─ 低（<30%）→ ❌ 太危險
│
└─ Lv.9+ 殘暴打擊考量
    ├─ 想用殘暴打擊？→ 必須先宣告魯莽 ✅
    └─ 不需要殘暴打擊的額外效果 → 正常判斷
```

### E-3. 魯莽攻擊 + 狂暴聯動效益

| 情境 | 效益分析 |
|------|----------|
| 狂暴 + 魯莽 | **黃金組合**：優勢攻擊 + 抗性抵銷被打的代價 |
| 僅魯莽（無狂暴）| 高風險：優勢攻擊但被打痛無折扣 |
| 僅狂暴（無魯莽）| 穩健：抗性 + 傷害加成，但命中率正常 |
| 都沒有 | 純肉搏，適合低威脅戰鬥 |

> **經驗法則：** 狂暴中幾乎永遠魯莽攻擊——抗性讓代價大幅降低，優勢讓暴擊機率翻倍（5% → 9.75%）。

## F. 子職業戰術差異

DM 根據角色卡的子職，參考對應的戰鬥風格調整：

### 狂戰士之路（Path of the Berserker）— 純輸出暴力機器

| 等級 | 特性 | 戰術影響 |
|------|------|----------|
| Lv.3 | 狂亂（Frenzy）| 魯莽攻擊時第一個命中額外 + 狂暴傷害加值 |
| Lv.6 | 心無旁騖之怒 | 狂暴中免疫魅惑和恐懼 → 衝鋒無懼 |
| Lv.10 | 反擊（Retaliation）| **反應**：1.5m內被命中 → 近戰反擊 → 鼓勵站在敵群中 |
| Lv.14 | 威嚇存在 | **附贈動作**恐懼 9m內目標 → 控場能力 |

**DM 要點：** 狂戰士就是砍。魯莽攻擊的狂亂加成讓第一擊更痛，反擊讓被打也有回報。

### 野性之心之路（Path of the Wild Heart）— 靈活適應型

| 等級 | 特性 | 戰術影響 |
|------|------|----------|
| Lv.3 | 野性之怒 | **每次狂暴選一個**：熊（全抗性）/ 鷹（附贈衝刺）/ 狼（給盟友優勢）|
| Lv.6 | 野性面貌 | 非戰鬥效用（黑暗視覺/攀爬/游泳）|
| Lv.10 | 自然之語 | 非戰鬥（儀式施法）|
| Lv.14 | 野性之力 | 新增：獵鷹（飛行）/ 獅（敵人離開受傷）/ 公羊（命中推 1.5m）|

**DM 要點：** 每次狂暴前問玩家選哪個動物。熊是坦克首選（幾乎全抗性），狼是團隊首選（盟友優勢），鷹是機動首選。

### 世界樹之路（Path of the World Tree）— 支援型坦克

| 等級 | 特性 | 戰術影響 |
|------|------|----------|
| Lv.3 | 生命之樹活力 | 啟動狂暴獲得臨時生命值 + 每回合給盟友臨時生命值 |
| Lv.6 | 樹之枝幹 | **反應**：盟友 +{狂暴傷害} 防禦等級或豁免 + 武器觸及 +3m |
| Lv.10 | 猛擊之根 | 推擠/拉拽時可讓目標倒地 + 推拉距離 +1.5m |
| Lv.14 | 沿樹而行 | **附贈動作**傳送 18m + 每次狂暴一次帶 6 人傳送 45m |

**DM 要點：** 世界樹是團隊型野蠻人。每回合開始記得分配臨時生命值。Lv.6 的反應保護盟友非常強力。

### 狂信者之路（Path of the Zealot）— 神聖傷害 + 不死身

| 等級 | 特性 | 戰術影響 |
|------|------|----------|
| Lv.3 | 神聖狂怒（Divine Fury）| 狂暴中每回合第一擊 +{Xd6 + 半等級} 黯蝕/光輝傷害 |

## G. 防禦層次

被攻擊時，按以下順序檢查可用防禦：

```
1. 基礎防禦等級（AC）
   ├─ 穿甲：中甲 + 敏捷（DEX）（上限 +2）+ 盾牌（如有）
   └─ 無甲：野性防禦（Unarmored Defense）= 10 + DEX mod + CON mod + 盾牌（如有）
   ↓ 被命中？
2. 狂暴抗性（被動，不消耗動作）
   → 鈍擊、穿刺、揮砍傷害減半
   → 野性之心（熊）：幾乎所有傷害類型減半
   ↓ 持續受傷？
3. 高生命值（HP）池（d12 生命骰）
   → 野蠻人 HP 天生最高，能硬吃傷害
   → 世界樹：啟動狂暴時額外獲得臨時 HP
   ↓ 需要豁免？
4. 危險感知（Danger Sense）
   → 敏捷豁免有優勢（能看見威脅時）
   → 火球術（Fireball）等範圍傷害的保命利器
   ↓ HP 歸零？
5. 不屈狂暴（Relentless Rage，Lv.11+）
   → 體魄豁免 DC 10（每次成功 +5）→ HP 降至 1 而非 0
   → 狂信者 Lv.14：花 1 次狂暴 → HP = 野蠻人等級
```

**DM 操作：** 野蠻人的防禦不是「不被打中」，而是「被打了也不怕」。
狂暴抗性 + 高 HP + 危險感知 = 前線最耐打的職業。

## H. 等級區間變化

| 等級 | 關鍵變化 | 對決策流程的影響 |
|------|----------|-----------------|
| **Lv.1-2** | 狂暴 ×2、魯莽攻擊（Lv.2）、危險感知（Lv.2）| 核心循環建立：狂暴 → 魯莽 → 砍。狂暴次數少，慎用 |
| **Lv.3-4** | 子職特性、狂暴 ×3 | 子職開始區分戰鬥風格。狂暴稍微寬裕 |
| **Lv.5** | **額外攻擊 + 疾走** | 傷害翻倍（2 次攻擊 + 狂暴傷害）。機動性大增 |
| **Lv.7** | 野性本能 + 本能猛撲 | 先攻優勢 → 更常先手。狂暴同時衝鋒定位 |
| **Lv.9** | **殘暴打擊 +1d10** + 狂暴傷害 +3 | 新決策層：魯莽時是否用殘暴打擊？傷害質量提升 |
| **Lv.11** | **不屈狂暴** | 生存力大躍進。HP 歸零不等於倒地 |
| **Lv.13** | 強化殘暴打擊（新效果選項）| 殘暴打擊新增傷口撕裂/體力削減，控場能力增加 |
| **Lv.15** | **持續狂暴** | 狂暴不再需要每回合維持。啟動時恢復 1 次使用次數 |
| **Lv.17** | 殘暴打擊 2d10 + 雙效果 | 殘暴打擊成為核心輸出手段，每次命中可選兩個效果 |
| **Lv.20** | 原始鬥士（STR/CON +4）| 屬性巔峰，攻擊/傷害/AC/HP 全面提升 |

**核心里程碑：**
- **Lv.5（額外攻擊）：** 戰鬥力分水嶺，DPR 翻倍
- **Lv.9（殘暴打擊）：** 新增決策層，輸出上限提升
- **Lv.11（不屈狂暴）：** 坦度質變，極難被擊倒
- **Lv.15（持續狂暴）：** 資源管理大幅簡化

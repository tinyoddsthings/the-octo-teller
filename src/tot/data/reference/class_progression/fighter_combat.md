# 戰士（Fighter）戰鬥 SOP

> 本文件是戰士的**戰鬥決策框架**。
> 具體武器/能力從角色卡讀取，套用到以下流程。
> 面板格式見 [combat_flow.md](../rules/combat_flow.md) C 節

---

## A. 職業戰鬥定位

| 項目 | 說明 |
|------|------|
| 戰鬥角色 | 近戰持續輸出 / 前排扛線（Sustained Melee DPR / Frontline Tank）|
| 主砲 | 武器攻擊（Weapon Attacks）— 額外攻擊（Extra Attack）提供多段輸出 |
| 核心機制 | **武器精通（Weapon Mastery）** + **戰術衝刺（Action Surge）**：精通屬性控場、爆發回合翻倍動作 |
| 設計哲學 | 武器攻擊是免費穩定輸出，戰術衝刺是扭轉戰局的爆發王牌 |

### 戰士 vs 其他近戰職業

| | 戰士 | 野蠻人 | 聖武士 |
|---|--------|--------|--------|
| 攻擊次數 | 最多（Lv.5=2、Lv.11=3、Lv.20=4）| 固定 2（Lv.5+）| 固定 2（Lv.5+）|
| 防禦等級（AC）| 高（重甲+盾=20+）| 中（無甲防禦）| 高（重甲+盾）|
| 爆發手段 | 戰術衝刺（短休回）| 魯莽攻擊（無限・換劣勢）| 神聖打擊（消耗欄位）|
| 資源壓力 | 低：核心輸出不消耗資源 | 極低：狂暴有限但攻擊免費 | 高：打擊消耗法術欄位 |
| 能力值提升（ASI）| 最多（6 次 + 史詩恩賜）| 標準（4 次）| 標準（4 次）|

**DM 心法：** 戰士的回合通常很直觀——攻擊 + 武器精通效果。
複雜度來自「何時按下戰術衝刺」和「選擇哪個精通屬性」。

## B. 回合決策流程

### B-1. 主流程

```
回合開始
│
├─ 0. 回合開始觸發效果
│     └─ 有持續狀態嗎？（恐懼、中毒等）→ 標記在面板狀態欄
│     └─ 鬥士（Champion）Lv.10+：無英勇靈感？→ 給自己一個
│
├─ 1. 附贈動作（有就先決定，不佔主動作）
│     ├─ 受傷了嗎？
│     │   └─ 第二風（Second Wind）還有次數？
│     │       ├─ 血量不足 50% → ✅ 使用（回復 1d10+戰士等級）
│     │       ├─ Lv.5+：附帶戰術轉移（Tactical Shift），可移動半速不觸發借機攻擊
│     │       └─ 血量充足 → 留著
│     ├─ 有子職業附贈動作？（如戰鬥大師的閃避步法、虛招攻擊、突進攻擊、鼓舞）
│     │   └─ 適用條件滿足 → 使用
│     ├─ 靈能戰士 Lv.7+：需要飛行？→ 靈能飛躍
│     ├─ 靈能戰士 Lv.15：需要掩護隊友？→ 力場壁壘
│     └─ 都不需要 → 跳過
│
├─ 2. 主動作（按類型優先順序）
│     ├─ ⚔️ 預設：攻擊動作（Attack Action）
│     │   → 主武器攻擊 × 攻擊次數（Lv.1-4=1、Lv.5-10=2、Lv.11-19=3、Lv.20=4）
│     │   → 每次命中觸發武器精通屬性
│     │   → 戰鬥大師：每次命中/觸發時可疊加 1 個戰技
│     │   → 靈能戰士：每回合 1 次命中可加靈能打擊
│     ├─ 💥 需要爆發嗎？→ 見 B-2 戰術衝刺判斷
│     │   └─ 戰術衝刺 → 本回合額外 1 個動作（不含魔法動作）
│     ├─ 🔮 奧法騎士限定：以法術取代攻擊？
│     │   └─ Lv.7+：攻擊動作中將 1 次攻擊替換為法師戲法
│     │   └─ Lv.18+：將 2 次攻擊替換為 1 個 1-2 環法術
│     ├─ 🩸 有免費種族能力可用？→ 優先使用（不花資源）
│     └─ 通用動作（衝刺/脫離/躲避 — 戰士很少需要）
│
├─ 3. 移動
│     └─ 近戰職業：靠近目標、佔據要道
│         前排扛線：站在施法者和敵人之間
│         多個敵人：利用武器精通的劈斬（Cleave）/ 橫掃攻擊站位
│         使用第二風時（Lv.5+）：附帶戰術轉移半速不觸發借機攻擊
│
└─ 回合結束
```

### B-2. 「該不該按戰術衝刺（Action Surge）？」

戰術衝刺是短休回復的爆發資源，每回合只能用一次：

```
要用戰術衝刺嗎？
│
├─ Q1：這場戰鬥後能短休嗎？
│   ├─ 能 → 放心用，短休回復
│   └─ 不確定 → 繼續判斷
│
├─ Q2：這回合額外一個動作能改變戰局嗎？
│   ├─ 能集中火力擊殺關鍵目標 → ✅ 值得
│   ├─ BOSS 瀕死，多一輪攻擊可收頭 → ✅ 值得
│   ├─ 奧法騎士：額外動作施放關鍵法術 → ✅ 值得
│   └─ 只是多打一點傷害，戰況穩定 → ❌ 留著
│
├─ Q3：隊友有生命危險嗎？
│   ├─ 需要額外動作救人（衝過去、擊殺威脅目標）→ ✅ 值得
│   └─ 局勢安全 → ❌ 留著
│
└─ Q4：這是最終/最難的戰鬥嗎？
    ├─ 是 → 第一或第二回合用，最大化效益
    └─ 否 → 保守使用
```

### B-3. 「該不該用第二風（Second Wind）？」

第二風是附贈動作，有多次使用次數（Lv.4 = 3 次），短休回復 1 次、長休全回：

```
要用第二風嗎？
│
├─ Q1：目前血量多少？
│   ├─ ≤ 50%（血量不足 Bloodied）→ ✅ 立刻用
│   ├─ 50-75% → 看情境
│   └─ > 75% → ❌ 不急
│
├─ Q2：Lv.5+ 需要戰術轉移嗎？
│   ├─ 被包圍需要脫離 → ✅ 即使血量高也可以用（移動半速不觸發借機攻擊）
│   └─ 站位安全 → 純看血量決定
│
├─ Q3：次數壓力？
│   ├─ 次數充足（2+）→ 放心用
│   ├─ 剩 1 次 → 留給真正危險的時刻
│   └─ 記住：戰術心智（Tactical Mind）也消耗第二風次數
│
└─ Q4：戰術心智 vs 回血？
    ├─ 能力檢定很重要且失敗了 → 可以賭一把（失敗不消耗次數）
    └─ 血量很低 → 回血優先
```

## D. 武器與攻擊決策

戰士的核心輸出圍繞武器攻擊和精通屬性選擇。
DM 從角色卡讀取具體武器，歸類後套用決策：

### D-1. 武器精通（Weapon Mastery）選擇

**原則：** 戰士可同時使用 3-4 種武器的精通屬性（依等級），每次長休可替換 1 種。

| 精通屬性 | 適用情境 | 優先度 |
|---------|---------|--------|
| 擦傷（Graze）| 巨劍/長柄刀 — miss 仍造成傷害 | 穩定輸出首選 |
| 迫擊（Vex）| 刺劍/短劍 — 命中後下次攻擊有優勢 | 靈巧系首選、連擊利器 |
| 絆倒（Topple）| 戰斧/巨錘/長棍 — 擊倒敵人 | 控場、配合隊友近戰 |
| 劈斬（Cleave）| 巨斧/戟 — 命中後順劈第二目標 | 多敵人時 |
| 削弱（Sap）| 連枷/長劍/晨星 — 敵人下次攻擊劣勢 | 坦克/減傷 |
| 推擊（Push）| 戰鎬/長槍 — 推開 3m | 控制站位、推下懸崖 |
| 快刺（Nick）| 彎刀/匕首 — 雙持攻擊納入攻擊動作 | 雙武器戰鬥 |
| 減速（Slow）| 長弓/標槍 — 減速 3m | 遠程風箏、阻止逃跑 |

### D-2. 戰鬥風格（Fighting Style）影響

| 戰鬥風格 | 對戰鬥決策的影響 |
|---------|-----------------|
| 防禦（Defense）| AC+1，純坦，無需改變攻擊策略 |
| 決鬥（Dueling）| 單手武器+2 傷害，搭配盾牌最佳 → 一手劍盾流 |
| 大武器戰鬥（GWF）| 傷害骰 1/2 視為 3，搭配巨劍/巨斧 → 雙手大武器流 |
| 箭術（Archery）| 遠程+2 命中，搭配長弓 → 遠程輸出流 |
| 雙武器戰鬥（TWF）| 副手攻擊加修正值，搭配快刺武器 → 多段攻擊流 |
| 守護（Protection）| 反應給鄰近友軍攻擊者劣勢（需盾牌）→ 保護型坦克 |
| 截擊（Interception）| 反應減傷 1d10+熟練（需武器/盾）→ 減傷型坦克 |

### D-3. 戰術大師（Tactical Master）— Lv.9+

```
攻擊命中，原武器精通屬性不適用？
│
├─ 想控制敵人站位？→ 替換為推擊（Push）— 推離 3m
├─ 想削弱敵人輸出？→ 替換為削弱（Sap）— 下次攻擊劣勢
├─ 想阻止敵人逃跑/靠近？→ 替換為減速（Slow）— 速度 -3m
└─ 原精通屬性更好？→ 不替換
```

> 戰術大師讓每把武器都能靈活切換控場效果，大幅提升戰士的戰術彈性。

## E. 戰鬥大師戰技（Battle Master Maneuvers）

> 非戰鬥大師子職業跳過本節。
> 戰技消耗優勢骰（Superiority Dice），短休/長休全回。每次攻擊限用 1 個戰技。

### E-1. 戰技分類與決策

**攻擊增強型（命中時觸發）** — 不改變動作經濟，純加值

| 戰技 | 額外效果 | 適用時機 |
|------|---------|---------|
| 分心打擊（Distracting Strike）| 隊友對該目標下次攻擊有優勢 | 隊友還沒行動、盜賊需要偷襲觸發 |
| 激怒攻擊（Goading Attack）| 目標攻擊你以外有劣勢 | 坦克嘲諷、保護後排 |
| 威嚇攻擊（Menacing Attack）| 目標恐懼 | 控場、阻止靠近 |
| 繳械攻擊（Disarming Attack）| 目標掉落武器 | 對付武器依賴的敵人 |
| 推撞攻擊（Pushing Attack）| 推離 4.5m | 推下懸崖、推出範圍 |
| 絆摔攻擊（Trip Attack）| 目標俯臥 | 配合近戰隊友、阻止移動 |
| 戰術移位攻擊（Maneuvering Attack）| 友軍用反應移動半速 | 拉友軍脫困 |
| 橫掃攻擊（Sweeping Attack）| 順劈鄰近第二目標 | 群體戰鬥 |

**精準補救型（未命中時觸發）**

| 戰技 | 效果 | 適用時機 |
|------|------|---------|
| 精準攻擊（Precision Attack）| 攻擊骰加優勢骰 | 差一點命中時（最常用戰技之一）|

**反應型（非自己回合觸發）**

| 戰技 | 觸發 | 效果 | 適用時機 |
|------|------|------|---------|
| 格擋（Parry）| 被近戰傷害 | 減傷 = 優勢骰 + 力量/敏捷修正 | 被集火、大傷害 |
| 還擊（Riposte）| 敵人近戰未命中你 | 反應攻擊 + 優勢骰傷害 | 高防禦等級戰士特別有效 |

**附贈動作型（消耗附贈動作）**

| 戰技 | 效果 | 適用時機 |
|------|------|---------|
| 虛招攻擊（Feinting Attack）| 對 1.5m目標下次攻擊有優勢 + 命中加優勢骰 | 需要保證命中 |
| 閃避步法（Evasive Footwork）| 脫離 + AC 加優勢骰 | 脫離包圍 |
| 突進攻擊（Lunging Attack）| 衝刺 + 直線 1.5m後近戰命中加優勢骰 | 衝鋒開場 |
| 鼓舞（Rally）| 友軍獲臨時生命值 = 優勢骰 + 戰士等級/2 | 友軍血量低 |

### E-2. 戰技決策流程

```
有優勢骰可用嗎？
│
├─ 攻擊骰結果出來了
│   ├─ 命中 → 選擇攻擊增強型戰技
│   │   ├─ 隊友需要優勢？→ 分心打擊
│   │   ├─ 需要控場？→ 絆摔/推撞/威嚇/繳械
│   │   ├─ 需要嘲諷？→ 激怒攻擊
│   │   ├─ 友軍需要脫困？→ 戰術移位攻擊
│   │   ├─ 旁邊有第二敵人？→ 橫掃攻擊
│   │   └─ 純傷害？→ 任意戰技（都加優勢骰傷害）
│   └─ 未命中，差距在 d8/d10/d12 範圍內？
│       └─ 精準攻擊 → 可能翻盤命中
│
├─ 不是自己回合
│   ├─ 被近戰打中且傷害高？→ 格擋
│   └─ 敵人近戰打你 miss？→ 還擊（額外攻擊機會）
│
├─ 需要附贈動作戰技？
│   ├─ 第二風不急用？→ 可用附贈戰技
│   └─ 第二風更重要？→ 放棄附贈戰技
│
└─ 優勢骰數量壓力？
    ├─ 充足（3+）→ 積極使用
    ├─ 剩 1-2 → 保留給精準攻擊和格擋/還擊
    └─ Lv.15 堅韌不拔：每回合 1 次免費 d8 戰技
```

## G. 防禦層次

被攻擊時，按以下順序檢查可用防禦：

```
1. 基礎防禦等級（AC）（重甲/中甲 + 盾牌 + 戰鬥風格加成）
   → 戰士 AC 通常很高（18-21）
   ↓ 被命中？
2. 反應防禦？
   → 戰鬥風格反應：截擊（減傷 1d10+熟練）/ 守護（攻擊者劣勢）
   → 戰鬥大師：格擋（減傷 = 優勢骰 + 力量/敏捷修正）
   → 奧法騎士：護盾術（AC+5 直到下回合）
   → 靈能戰士：防護力場（減傷 = 靈能骰 + 智力修正）
   ↓ 仍被打中？
3. 有第二風次數嗎？
   → 下回合附贈動作回血（1d10+戰士等級）
   → Lv.5+：附帶戰術轉移脫離危險位置
   ↓ 持續受傷？
4. 不屈（Indomitable）— Lv.9+
   → 豁免失敗時重擲（加戰士等級），對抗控場法術
   ↓ 危險！
5. 戰術衝刺 → 額外動作脫離/衝刺/躲避
   → 最後手段的緊急撤退
```

**DM 操作：** 戰士的防禦核心是高防禦等級和生命值。
相比魔導士依賴位移和反應法術，戰士靠硬扛 + 第二風回血。
重甲+盾牌的戰士防禦等級（AC）20 可以無視大量低命中攻擊。

## H. 等級區間變化

| 等級 | 關鍵變化 | 對決策流程的影響 |
|------|----------|-----------------|
| **Lv.1** | 戰鬥風格、第二風（2次）、武器精通（3種）| 單次攻擊 + 精通屬性。決策簡單：攻擊 → 精通效果 |
| **Lv.2** | **戰術衝刺**（1次/短休）、戰術心智 | 爆發回合開放。何時用是核心決策 |
| **Lv.3** | **子職業** | 戰鬥大師得戰技、鬥士擴暴擊、奧法騎士得法術、靈能戰士得靈能 |
| **Lv.4** | 能力值提升、第二風增至 3 次 | 命中/傷害/防禦等級提升。第二風更多次 = 更耐打 |
| **Lv.5** | **額外攻擊**（2次）、武器精通（4種）、戰術轉移 | **最大跳躍點：** 傷害翻倍。戰術衝刺 = 1 回合 4 次攻擊。第二風附帶免費脫離 |
| **Lv.6** | 能力值提升 | 繼續強化數值 |
| **Lv.7** | 子職特性 | 戰鬥大師+2 戰技+第 5 骰、鬥士額外風格、奧法騎士戰爭魔法、靈能戰士念動精通 |
| **Lv.9** | **不屈**（1次）、**戰術大師** | 反豁免保命開放。武器精通可臨時替換為推擊/削弱/減速 |
| **Lv.10** | 子職特性、第二風增至 4 次 | 戰鬥大師優勢骰升 d10、鬥士英勇戰士、靈能戰士心靈抗性 |
| **Lv.11** | **雙額外攻擊（3次）** | 傷害再次大幅提升。戰術衝刺 = 1 回合 6 次攻擊 |
| **Lv.13** | 不屈（2次）、研習攻擊 | Miss 後下次攻擊有優勢 → 穩定度再提升 |
| **Lv.17** | 戰術衝刺（2次/短休）、不屈（3次）| 爆發次數翻倍，保命能力大幅強化 |
| **Lv.20** | **三額外攻擊（4次）** | 1 回合 4 次攻擊（衝刺 = 8 次），D&D 最高攻擊頻率 |

**攻擊次數 = 決策基準線：**
- Lv.1-4：1 次攻擊 → 每次命中都重要，武器精通效果影響大
- Lv.5-10：2 次攻擊 → 穩定輸出，精通效果每回合可觸發 2 次
- Lv.11-16：3 次攻擊 → 高頻輸出，戰技/靈能可更靈活分配
- Lv.17+：3-4 次攻擊 + 2 次衝刺 → 爆發輪可達 6-8 次攻擊

## I. 子職業速查

DM 根據角色卡的子職業類型，參考對應戰術提示：

### 鬥士（Champion）
- **精進致命：** 19-20 暴擊（Lv.15 升至 18-20）→ 鼓勵多段攻擊最大化暴擊機率
- **卓越運動員：** 先攻和運動檢定優勢 → 通常先手行動
- 暴擊後可移動半速不觸發借機攻擊 → 暴擊後重新定位
- **Lv.10 英勇戰士：** 每回合開始無英勇靈感則自動獲得 → 每回合保底有靈感可用
- **DM 心法：** 最簡單的子職業，幾乎不需額外決策。攻擊就對了，暴擊自然來

### 戰鬥大師（Battle Master）
- **優勢骰 + 戰技：** 短休全回復。戰技選擇是最大決策點
- **核心戰技推薦：** 精準攻擊（救命中）、格擋/還擊（反應增值）、絆摔攻擊（控場）
- **知彼知己（Lv.7）：** 附贈辨識敵人弱點 → 戰鬥開場第一件事
- **Lv.15 堅韌不拔：** 每回合 1 次免費 d8 戰技 → 資源壓力大幅降低
- **DM 心法：** 最戰術型子職業。需要為玩家提供戰技觸發時機的判斷，詳見 E 節

### 奧法騎士（Eldritch Knight）
- **施法能力：** 智力施法、法師法術列表、1/3 施法者進度
- **戰爭契約：** 武器不可繳械，附贈召回 → 永遠不會失去武器
- **Lv.7 戰爭魔法：** 攻擊動作中 1 次攻擊替換為戲法 → 攻擊 + 戲法 combo
- **Lv.10 奧法打擊：** 武器命中 → 目標對你法術豁免有劣勢 → 先打再放控場
- **Lv.15 奧法衝鋒：** 戰術衝刺時傳送 9m → 極強位移
- **DM 心法：** 半施法者，法術欄位有限。核心仍是武器攻擊，法術是增強手段。護盾術是最佳法術（反應 +5 AC）

### 靈能戰士（Psi Warrior）
- **靈能骰：** 短休回 1、長休全回。三種用法：打擊加傷、減傷防護、念動移物
- **靈能打擊：** 每回合 1 次命中加力場傷害 → 穩定額外 DPR
- **防護力場：** 反應減傷 → 保命/保隊友的核心能力
- **Lv.7 念動推力：** 靈能打擊附加擊倒/推開 → 控場能力大增
- **Lv.10 守護心靈：** 心靈傷害抗性 + 消除魅惑/恐懼 → 反控場
- **DM 心法：** 資源管理型。靈能骰在打擊和防護間分配是核心決策。短休只回 1 顆要謹慎

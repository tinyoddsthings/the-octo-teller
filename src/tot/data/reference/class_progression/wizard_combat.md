# 法師（Wizard）戰鬥 SOP

> 本文件是法師的**戰鬥決策框架**。
> 具體法術/能力從角色卡讀取，套用到以下流程。
> 面板格式見 [combat_flow.md](../rules/combat_flow.md) C 節

---

## A. 職業戰鬥定位

| 項目 | 說明 |
|------|------|
| 戰鬥角色 | 遠程控場/爆發（Ranged Control / Burst AoE）|
| 主砲 | 傷害型戲法（Fire Bolt / Ray of Frost / Toll the Dead）— 無限使用 |
| 核心機制 | **準備施法者**：法術書知道大量法術，每日準備智力（INT）修正 + 法師等級個；多環級法術位，**長休恢復** + 奧術恢復（Arcane Recovery）短休補充 |
| 設計哲學 | 全遊戲最大法術列表 → 正確的法術準備 = 勝利一半；戲法是保底輸出，法術位用來控場/爆發/防禦 |

### 法師 vs 術士（Sorcerer）

| | 法師（Wizard，智力（INT）） | 術士（Sorcerer，魅力（CHA）） |
|---|--------|----------|
| 法術知識量 | **極大**（法術書持續擴充）| 固定（已知法術數量有限）|
| 每日彈性 | 高（長休換準備列表）| 低（已知固定，不可換）|
| 法術位管理 | 標準多環級法術位 + 奧術恢復 | 標準法術位 + 魔力點（Sorcery Points）轉換 |
| 戰場即興 | 儀式施法（Ritual）不耗欄位 | 超魔（Metamagic）改變法術效果 |
| 生存能力 | **極脆弱**（d6、無護甲）靠法術撐 | 脆弱（d6、無護甲）但體質（CON）豁免熟練 |
| 決策特色 | 「我有沒有準備對的法術？」 | 「要不要花魔力點強化這次施放？」 |

**DM 心法：** 法師的回合看似選擇多，但有明確優先級——
先看場上是否需要控場/解圍 → 不需要就用戲法穩輸出。
法術位是**長期資源**，不像魔導士短休全回，需要統籌整天的戰鬥。

## B. 回合決策流程

### B-1. 主流程

```
回合開始
│
├─ 0. 回合開始觸發效果
│     ├─ 有專注法術嗎？→ 標記在面板狀態欄
│     └─ 有持續性法術效果？（如焰刃術移動、動畫物件指揮）→ 處理
│
├─ 1. 場況評估（法師特有：選項多，需要快速判斷）
│     ├─ 是否有高價值控場機會？
│     │   └─ 多個敵人聚集 → 範圍法術（AoE）
│     │   └─ 單一高威脅目標 → 束縛/放逐
│     ├─ 是否有緊急防禦需求？
│     │   └─ 自己/隊友快死 → 防禦優先
│     └─ 以上皆否 → 戲法穩輸出
│
├─ 2. 附贈動作（有就先決定，不佔主動作）
│     ├─ 有附贈施放的法術？（如迷蹤步 Misty Step）
│     │   └─ 見 B-2 判斷是否值得花欄位
│     ├─ 有持續法術需要附贈動作操控？
│     │   └─ （如焰刃術 Flaming Sphere 移動、遙控之手 Bigby's Hand）
│     └─ 都不需要 → 跳過
│
├─ 3. 主動作（按優先順序）
│     ├─ 🔮 需要控場嗎？
│     │   → 蛛網術（Web）/ 催眠圖紋（Hypnotic Pattern）/
│     │     束縛人類（Hold Person）/ 緩慢術（Slow）
│     │   → 見 B-2 判斷欄位消耗
│     ├─ 🔮 需要爆發傷害嗎？
│     │   → 火球術（Fireball）/ 閃電箭（Lightning Bolt）/
│     │     灼熱射線（Scorching Ray）
│     ├─ 🔮 需要防禦/輔助？
│     │   → 反制法術（Counterspell）已在反應 / 加速術（Haste）/
│     │     飛行術（Fly）/ 高等隱形術（Greater Invisibility）
│     ├─ 🩸 有免費種族能力可用？→ 優先使用（不花欄位）
│     └─ ✨ 預設：傷害型戲法
│         → 火焰箭（Fire Bolt）/ 亡者喪鐘（Toll the Dead）/
│           寒冰射線（Ray of Frost）
│
├─ 4. 移動
│     └─ **極端脆弱職業：保持最大安全距離**
│         ├─ 遠程戲法射程 36m（120ft）→ 盡量站到極限射程
│         ├─ 利用掩體（Cover）→ +2/+5 防禦等級（AC）
│         ├─ 避免進入任何敵人觸及範圍（借機攻擊（AoO）風險）
│         └─ 若被近身 → 下回合迷蹤步（Misty Step）脫離
│
└─ 回合結束
    └─ 有回合結束觸發效果？→ 處理
```

### B-2. 「法術位花費決策」

法師擁有多環級法術位，管理比魔導士更複雜：

```
要花法術位嗎？
│
├─ Q1：這是今天第幾場戰鬥？
│   ├─ 第 1 場 → 可適度使用，但保留高環位
│   ├─ 中段 → 優先用低環位，高環位留給 BOSS 戰
│   └─ 最後一場（確定）→ 全力輸出
│
├─ Q2：有用過奧術恢復（Arcane Recovery）嗎？
│   ├─ 沒用 → 可以多花一些低環位（短休能補回）
│   └─ 已用 → 省著花
│
├─ Q3：用哪一環的法術位？
│   ├─ 能用低環位達成目的 → 用低環位
│   │   （例：護盾術 1 環就夠、法師護甲 1 環）
│   ├─ 升環效果顯著 → 考慮升環
│   │   （例：魔法飛彈 → 每升一環 +1 箭）
│   └─ 5 環以上 → 通常是戰略級法術，確認真的需要
│
├─ Q4：這個法術能改變戰局嗎？
│   ├─ 範圍控場影響 3+ 個敵人 → 值得
│   ├─ 單體控制讓 BOSS 停手 → 值得
│   ├─ 只是比戲法多一點傷害 → 不值得
│   └─ 防禦保命 → 生命無價，值得
│
└─ Q5：專注衝突？
    ├─ 新法術需要專注 且 正在維持更好的專注 → 不切換
    ├─ 現有專注效果已遞減（目標死亡等）→ 切換
    └─ 新法術效果 > 現有專注 → 切換
```

## C. 面板生成模板

DM 輪到法師行動時，從角色卡讀取具體能力，按以下模板生成面板。
（面板分區子表結構定義見 [combat_flow.md](../rules/combat_flow.md) C 節）

### C-0. 狀態表補充：專注追蹤

狀態表（第一段）的 📌 狀態欄中，需要標示專注資訊：

| 情境 | 📌 狀態欄標示 |
|------|-------------|
| 正在維持專注法術 | `{法術名}專注中`（如：`蛛網術專注中`）|
| 無專注 | 不標示（留給其他狀態）|
| 法師護甲生效中 | `法師護甲（防禦等級（AC）= 13+敏捷（DEX））`（非專注，不衝突）|

> 法師只能同時維持 **1 個** 專注法術。
> 法術表中標 `✅⚠️換` 的法術 = 施放時會自動結束現有專注。
> 標 `❌免` 的法術 = 不需專注，可與現有專注並存。

### C-1. 動作表（分區子表設計）

統一編號橫跨所有子表，玩家直接喊號選擇。
每區只顯示該類型需要的欄位，避免空白。

| # | 名稱 | 命中/難度（DC）| 傷害/效果 | 最低環位 | 專注 | 備註 |
|---|------|---------------|----------|---------|------|------|
| N | {spell} | {+hit / 難度（DC）} | {effect} | {X}環 | ✅⚠️換 / ❌免 | {升環效果} |

欄位說明：
- **最低環位**：此法術最低使用的法術位環數。法師可以用更高環位施放以增強效果。

- **專注**：
  - `✅⚠️換` = 需要專注，施放時會取消目前的專注法術
  - `❌免` = 不需專注，可與現有專注並存

### C-3. 反應表

**🛡️ 反應（極度重要 — 法師核心生存手段）**

| 類型 | 來源 | 填入規則 |
|------|------|----------|
| 護盾術（Shield）| 1 環，+5 防禦等級（AC）至下回合開始 | **最高優先級反應** — 被命中立刻考慮 |
| 吸收元素（Absorb Elements）| 1 環，抗性該傷害類型至下回合 | 受到元素傷害時使用 |
| 反制法術（Counterspell）| 3 環，取消敵方法術 | 敵人施法時使用，見 D-5 |
| 借機攻擊（AoO）| 固定 | 所有角色都有（但法師很少使用）|

> **反應每回合只有 1 次** — 護盾術、吸收元素、反制法術三選一。
> 優先級通常：反制法術（擋掉一整個法術）> 護盾術（擋一次攻擊）> 吸收元素。
> 但若自己快死，護盾術保命優先。

### C-4. 奧術恢復追蹤

**📦 奧術恢復（Arcane Recovery）** — ✅{1}/{1}（短休）

| 項目 | 數值 |
|------|------|
| 可恢復環數總和 | ⌈法師等級 ÷ 2⌉（無條件進位）|
| 單一法術位上限 | 不得高於 5 環 |
| 使用時機 | 短休結束時 |
| 恢復頻率 | 1 次/長休 |

> **DM 提示：** 短休時提醒法師玩家是否要使用奧術恢復。
> 最佳策略通常是恢復已消耗的最高環位（不超過 5 環）。

## D. 法術分類決策

法師擁有全遊戲最多的法術選擇。DM 從角色卡讀取已準備法術，歸類後套用決策：

### D-1. 範圍傷害型（Blast / AoE）

**代表法術：** 火球術（Fireball）、閃電箭（Lightning Bolt）、冰風暴（Ice Storm）、圓錐寒冰（Cone of Cold）

| 判斷 | 決策 |
|------|------|
| 3+ 個敵人聚集，不波及隊友 | ✅ 黃金時機，法師最強的場景 |
| 敵人散開，只打到 1-2 個 | ❌ 效率太低，用戲法或單體法術 |
| 隊友在爆炸範圍內 | ⚠️ 除非塑能師（Evoker）有雕塑法術，否則避免 |
| 敵人有該元素抗性/免疫 | ❌ 換傷害類型或用控場 |

### D-2. 控場型（Control）

**代表法術：** 蛛網術（Web）、催眠圖紋（Hypnotic Pattern）、束縛人類（Hold Person）、緩慢術（Slow）、力場之牆（Wall of Force）

| 判斷 | 決策 |
|------|------|
| 能同時影響多個敵人 | ✅ 法師控場的最大價值 |
| 目標豁免弱項對應法術的豁免類型 | ✅ 命中率高，值得 |
| 目標豁免加值極高 | ❌ 太冒險，換其他法術或戲法 |
| 已有控場專注在維持中 | 權衡：現有控場的剩餘價值 vs 新控場效果 |
| BOSS 戰、只有單一高威脅目標 | ✅ 束縛/放逐讓 BOSS 停手 = 改變戰局 |

### D-3. 輔助/增益型（Buff）

**代表法術：** 加速術（Haste）、飛行術（Fly）、高等隱形術（Greater Invisibility）、法師護甲（Mage Armor）

| 判斷 | 決策 |
|------|------|
| 戰鬥前有準備時間 | ✅ 法師護甲（8 小時）戰前必開 |
| 加速術給近戰主力 | ✅ 如果預期長戰鬥（3+ 回合）且無更好專注 |
| 高等隱形術給刺客型隊友 | ✅ 持續攻擊優勢 + 防禦劣勢 |
| 戰鬥快結束（1-2 回合）| ❌ 增益來不及回本 |

### D-4. 防禦/位移型（Defense / Mobility）

**代表法術：** 護盾術（Shield）、吸收元素（Absorb Elements）、迷蹤步（Misty Step）、鏡影術（Mirror Image）

| 判斷 | 決策 |
|------|------|
| 被攻擊命中但 +5 防禦等級（AC）能閃過 | ✅ 護盾術（反應，1 環）|
| 受到大量元素傷害 | ✅ 吸收元素（反應，1 環）|
| 被近戰包圍無法正常脫離 | ✅ 迷蹤步（附贈，2 環）→ 主動作戲法 |
| 預期被集火 | ✅ 鏡影術（動作，2 環，❌免，不佔專注）|
| 安全站位、沒有威脅 | ❌ 省法術位 |

### D-5. 反制型（Counter）

**代表法術：** 反制法術（Counterspell）、解除魔法（Dispel Magic）

| 判斷 | 決策 |
|------|------|
| 敵方施放高威脅法術（Fireball / Hold Person / 治療等）| ✅ 反制法術（反應，3 環）|
| 敵方施放低威脅法術 | ❌ 別浪費 3 環位 + 反應 |
| 不確定敵方法術環數 | 3 環施放 → 自動擋 3 環以下；更高需智力（INT）檢定 |
| 有友方法術效果要解除 | ✅ 解除魔法（動作，3 環）|

> **反制法術 vs 護盾術的反應衝突：** 同一回合只能用一個。
> 通常反制法術價值更高（擋掉整個法術 > 擋一次攻擊），除非自己快死。

## E. 法術書管理

法師獨有的核心機制。DM 需理解以下規則：

### E-1. 準備法術 vs 法術書

```
法術書（Spellbook）
│
├─ 「知道」的法術 — 所有記錄在法術書中的法術
│   ├─ 起始：6 個 1 環法術
│   ├─ 每次升級：+2 個（有對應法術位環數的法師法術）
│   └─ 抄錄：從卷軸/他人法術書抄入（每環 2 小時 + 50 GP）
│
└─ 「準備」的法術 — 每日從法術書中選擇
    ├─ 數量：見特性表（Lv.1 = 4, Lv.4 = 7, Lv.5 = 9...）
    ├─ 限制：只能準備有對應法術位環數的法術
    ├─ 更換：長休後可全部重選
    └─ Lv.5 記憶法術（Memorize Spell）：短休額外準備 1 個（不計上限）
```

### E-2. 儀式施法（Ritual Casting）

| 規則 | 說明 |
|------|------|
| 條件 | 法術書中有該法術 + 法術有儀式標籤 |
| **不需要準備** | 只要在法術書中就能儀式施放 |
| 時間 | 正常施法時間 + 10 分鐘 |
| 消耗 | **不消耗法術位** |
| 戰鬥中 | ❌ 不適用（太慢）|
| 戰鬥外常用 | 偵測魔法（Detect Magic）、鑑定術（Identify）、尋獲魔寵（Find Familiar）、警報術（Alarm）|

> **DM 提示：** 法師的儀式施法是巨大的資源節省。
> 短休/長休/探索時提醒玩家：「要不要儀式施放偵測魔法/鑑定術？」

### E-3. 每日準備建議框架

角色卡應標示法師的每日準備列表。以下是準備的思考框架：

```
每日準備法術配置（{智力（INT）修正 + 法師等級} 個）
│
├─ 必備防禦（通常 2-3 個）
│   ├─ 護盾術（Shield）— 1 環，反應 +5 防禦等級（AC）
│   ├─ 吸收元素（Absorb Elements）— 1 環，反應元素抗性
│   └─ 迷蹤步（Misty Step）— 2 環，附贈緊急脫困
│
├─ 核心控場/傷害（通常 3-4 個）
│   ├─ 至少 1 個範圍控場（如蛛網術 / 催眠圖紋）
│   ├─ 至少 1 個範圍傷害（如火球術 / 閃電箭）
│   └─ 備選：單體控場（束縛人類 / 放逐術）
│
├─ 反制（通常 1 個）
│   └─ 反制法術（Counterspell）— 3 環，反應取消敵方法術
│
├─ 增益/輔助（通常 1-2 個）
│   └─ 依任務需求：飛行術 / 加速術 / 隱形術
│
└─ 剩餘位置 → 探索/社交用途
    └─ 不需準備儀式法術（可直接從法術書儀式施放）
```

## F. 子職業戰術差異

法師的子職業在 Lv.3 選定，決定專長方向：

### 防護師（Abjurer）— 耐久型法師

```
核心機制：奧術結界（Arcane Ward）
│
├─ 啟動條件：施放使用法術位的防護學派法術
│   └─ 常見觸發：護盾術、法師護甲、吸收元素
│
├─ 結界生命值（HP）= 法師等級 ×2 + 智力（INT）修正
│   └─ 例：Lv.4 法師，智力（INT）+3 → 結界 11 點生命值（HP）
│
├─ 補充：每次施放防護法術 → 恢復 = 法術環數 ×2
│
└─ 戰術影響：
    ├─ 護盾術不只加防禦等級（AC），還補結界
    ├─ 比其他法師多一層 HP 緩衝
    ├─ Lv.6：反應保護隊友（投射結界）→ 半坦定位
    └─ Lv.14：法術傷害抗性 + 法術豁免優勢 → 最硬法師
```

### 預言師（Diviner）— 命運操控型

```
核心機制：預兆（Portent）
│
├─ 長休後擲 2 顆 d20 → 記錄結果（Lv.14 升為 3 顆）
│
├─ 使用：替代任何你能看到的生物的 d20 擲骰
│   ├─ 擲骰前或擲骰後均可（但在 DM 宣布結果前）
│   └─ 攻擊、豁免、屬性檢定皆可
│
└─ 戰術影響：
    ├─ 高骰（15+）→ 保給自己或隊友的關鍵攻擊/豁免
    ├─ 低骰（5-）→ 用在敵人的豁免上 → 確保控場命中
    ├─ 中間骰 → 視情況，通常不急著用
    ├─ Lv.6：施放 2 環以上預言法術 → 恢復 1 個低環位
    └─ **DM 特別注意：** 每次戰鬥開始提醒玩家預兆骰的數值
```

### 塑能師（Evoker）— 爆破型

```
核心機制：
│
├─ Lv.3 強效戲法（Potent Cantrip）
│   └─ 敵人豁免成功仍受半傷 → 戲法傷害更穩定
│
├─ Lv.6 雕塑法術（Sculpt Spells）
│   └─ AoE 法術可保護 1+法術環數 個盟友 → 自動通過豁免且不受傷
│   └─ **核心優勢：** 可以對著近戰隊友和敵人一起丟火球
│
├─ Lv.10 強化塑能（Empowered Evocation）
│   └─ 塑能法術 +智力（INT）修正 到一次傷害擲骰
│
└─ 戰術影響：
    ├─ 是唯一可以安心在隊友旁邊放 AoE 的法師
    ├─ 火球術/閃電箭的使用頻率和時機遠比其他子職業靈活
    ├─ 戲法也有保底傷害 → 最強的「零消耗」回合
    └─ Lv.14 超載（Overchannel）→ 偶爾一發最大傷害
```

### 幻術師（Illusionist）— 欺瞞型

```
核心機制：
│
├─ Lv.3 強化幻術（Improved Illusions）
│   ├─ 弱效幻象（Minor Illusion）同時有聲音 + 視覺
│   └─ 幻象法術可改用附贈動作施放 → 主動作可攻擊/施法
│
├─ Lv.6 幻影生物（Phantasmal Creatures）
│   └─ 召喚妖精（Summon Fey）不需專注 → 可與其他專注法術並存
│
├─ Lv.10 幻影自我（Illusory Self）
│   └─ 反應：被命中時自動閃避（1 次/短休或花 2 環位）
│
└─ 戰術影響：
    ├─ 創意導向：DM 需要配合幻術的創造性運用
    ├─ 附贈施放幻象 → 節省動作經濟
    ├─ 不需專注的召喚物 → 額外行動經濟
    └─ Lv.14 幻象成真 → 將幻象物件實體化（極高創意天花板）
```

## G. 防禦層次

法師是全遊戲最脆弱的職業之一（d6 生命骰、無護甲熟練）。防禦完全依賴法術和站位：

```
0. 戰前防禦（不佔戰鬥動作）
   └─ 法師護甲（Mage Armor）→ 防禦等級（AC）= 13 + 敏捷（DEX）修正（8 小時）
      └─ 無護甲時基礎 AC = 10 + 敏捷（DEX）→ 法師護甲是必備
   ↓
1. 站位防禦（最重要・零消耗）
   ├─ 保持最遠距離（戲法射程 36m / 120ft）
   ├─ 利用掩體（半掩體 +2 AC、四分之三掩體 +5 AC）
   └─ 站在前排戰士身後
   ↓ 仍被攻擊？
2. 反應防禦：護盾術（Shield）
   └─ +5 防禦等級（AC）至下回合開始（1 環法術位）
   └─ 例：法師護甲 AC 15 + 護盾術 = AC 20
   ↓ 受到元素傷害？
3. 反應防禦：吸收元素（Absorb Elements）
   └─ 獲得該元素傷害抗性（減半傷害）+ 下次近戰額外傷害
   ↓ 敵人施法？
4. 反應防禦：反制法術（Counterspell）
   └─ 取消敵方法術（3 環位；高環需智力（INT）檢定）
   ↓ 被近身？
5. 緊急位移：迷蹤步（Misty Step）
   └─ 附贈動作傳送 9m（30ft）→ 主動作仍可用
   ↓ 預期持續被集火？
6. 鏡影術（Mirror Image）— 不需專注
   └─ 3 個分身，被攻擊時有機率打到分身而非本體
   ↓ 真正危險？
7. 逃生方案
   ├─ 任意門（Dimension Door）→ 150m 傳送（4 環）
   ├─ 隱形術（Invisibility）→ 脫離視線
   └─ 氣態形體（Gaseous Form）→ 穿越縫隙逃脫
```

**DM 操作要點：**
- 法師基礎防禦等級（AC）很低（通常 13-16 含法師護甲）
- 生命值（HP）全職業最低（d6 生命骰）
- **護盾術是最常使用的法術** — 幾乎每場戰鬥都會用
- 被近戰攻擊到 = 非常危險，需要立刻考慮脫困

## H. 等級區間變化

| 等級 | 關鍵變化 | 對決策流程的影響 |
|------|----------|-----------------|
| **Lv.1-2** | 1 環法術 ×2-3 位、戲法 3 個 | 非常脆弱。護盾術 + 法師護甲是生存關鍵。戲法為主，法術位保護用 |
| **Lv.3-4** | 2 環法術開放、子職業獲得 | 迷蹤步加入 → 生存力提升。蛛網術 = 強力控場。開始有攻防選擇 |
| **Lv.5-6** | **3 環法術 = 巨大躍升** | 火球術 / 反制法術 / 飛行術 / 催眠圖紋 → 法師戰場影響力爆增。記憶法術（短休 +1 準備）增加彈性。子職業 Lv.6 特性上線 |
| **Lv.7-8** | 4 環法術 | 放逐術 / 變形術 / 高等隱形術 / 力場護盾。開始有「一招結束戰鬥」的能力 |
| **Lv.9-10** | **5 環法術** | 力場之牆 / 動畫物件 / 遙控之手。戰略級控場。子職業 Lv.10 特性 |
| **Lv.11-12** | 6 環法術 | 鎖鏈閃電 / 解離術 / 質量暗示術。單一法術可改變整場戰鬥 |
| **Lv.13-16** | 7-8 環法術 | 指印牢籠 / 傳送術 / 迷宮術。可以直接移除敵人。戰鬥內外皆有壓倒性影響 |
| **Lv.17-20** | 9 環法術 + 法術精通 + 招牌法術 | 許願術 / 流星火雨。法術精通（1+2 環免費）+ 招牌法術（兩個 3 環免費）→ 資源壓力大減 |

**法師等級里程碑：**
- **Lv.1：** 護盾術 + 法師護甲 = 生存基礎
- **Lv.5：** 火球術 = 法師的「成年禮」，AoE 傷害大幅提升
- **Lv.9：** 力場之牆 = 法師最強控場之一，無豁免的物理隔離
- **Lv.18：** 法術精通 → 護盾術/迷蹤步免費施放 = 永久性防禦提升

## I. 子職業速查

DM 根據角色卡的子職業類型，參考對應戰術提示：

### 防護師（Abjurer）
- 奧術結界 = 額外生命值（HP）層 → 最硬的法師
- 施放防護法術同時補結界 → 護盾術一石二鳥
- Lv.6 可反應保護隊友 → 半坦定位
- Lv.14 法術傷害抗性 + 法術豁免優勢 → 反法師戰專家

### 預言師（Diviner）
- **預兆骰** = 每長休 2 次（Lv.14 升 3 次）改寫命運
- 高骰用在關鍵攻擊/豁免，低骰用在敵人豁免 → 確保控場命中
- Lv.6 施放預言法術恢復低環位 → 資源管理優勢
- DM 每場戰鬥開始需提醒：「你的預兆骰是 {X} 和 {Y}」

### 塑能師（Evoker）
- **雕塑法術**（Lv.6）= 隊友在 AoE 中免傷 → 可以安心炸
- 強效戲法 = 戲法保底半傷 → at-will 輸出更穩
- 強化塑能 = 塑能法術 +智力（INT）修正傷害 → 數字更高
- 超載（Lv.14）= 一發最大傷害，但有自傷代價

### 幻術師（Illusionist）
- 幻象法術改用附贈動作施放 → 動作經濟優勢
- 召喚妖精不需專注（Lv.6）→ 可與其他專注並存
- 幻影自我（Lv.10）= 反應閃避攻擊 → 額外生存力
- 高度依賴創意 → DM 需要靈活裁定幻術效果

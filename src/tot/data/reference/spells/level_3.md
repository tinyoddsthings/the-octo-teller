# 法術詳情 — 3 環（3rd Level）

> 中央法術庫。各職業共用，按環級分檔。

#### 反制法術（Counterspell）

| 項目 | 內容 |
|------|------|
| 環級 | 3 環 |
| 學派 | 防護（Abjuration） |
| 施法時間 | 1 反應（當你看見 18m內的生物施展法術時） |
| 射程 | 18m |
| 成分 | S |
| 持續時間 | 瞬間 |

**效果：** 你試圖打斷一個正在施法的生物。若目標法術為 3 環以下，自動被反制。若目標法術為更高環級，你須進行屬性檢定（使用你的施法屬性），DC 為 10 + 目標法術環級，成功才能反制。

**升環：** 以等於或高於目標法術環級的環位施展時，自動反制。

#### 解除魔法（Dispel Magic）

| 項目 | 內容 |
|------|------|
| 環級 | 3 環 |
| 學派 | 防護（Abjuration） |
| 施法時間 | 1 動作 |
| 射程 | 36m |
| 成分 | V, S |
| 持續時間 | 瞬間 |

**效果：** 選擇射程內一個生物、物品或魔法效果。目標上 3 環以下的法術自動結束。更高環級的法術須進行屬性檢定（使用施法屬性），DC 為 10 + 法術環級，成功則結束該法術。

**升環：** 以等於或高於目標法術環級的環位施展時，自動解除。

#### 魔法陣（Magic Circle）

| 項目 | 內容 |
|------|------|
| 環級 | 3 環 |
| 學派 | 防護（Abjuration） |
| 施法時間 | 1 分鐘 |
| 射程 | 3m |
| 成分 | V, S, M（聖水或粉狀鐵和銀，價值至少 100 gp） |
| 持續時間 | 1 小時 |

**效果：** 你劃出一個 3m半徑、3m高的圓柱能量場。選擇以下一種或多種生物類型：天界生物、元素生物、妖精、邪魔、不死生物。被選中類型的生物無法自願進入圓柱、對圓柱內的生物攻擊具有劣勢、且無法魅惑/恐嚇/附身圓柱內的目標。你也可反轉法術，將選定生物困在圓柱內。

**升環：** 每高一環，持續時間增加 1 小時。

**升環：** 以 6 環位施展時，法術不需專注且持續到被解除為止。

#### 操縱屍體（Animate Dead）

| 項目 | 內容 |
|------|------|
| 環級 | 3 環 |
| 學派 | 死靈（Necromancy） |
| 施法時間 | 1 分鐘 |
| 射程 | 3m |
| 成分 | V, S, M（一滴血、一片肉和一撮骨灰） |
| 持續時間 | 瞬間 |

**效果：** 你選擇射程內一堆骸骨或一具中型或小型人形生物屍體。該目標成為不死生物：骸骨變為骷髏（Skeleton），屍體變為殭屍（Zombie）。在你的每個回合中，你可用附贈動作以心靈命令你用此法術創造的、且在 18m內的任何生物（可同時命令多個，對每個發出相同命令）。你決定生物下一回合的動作和移動方向，或發出一般命令（如守護某房間或走廊）。若你不發出命令，生物會採取閃避動作並僅移動以避免危險。該生物受你控制 24 小時，之後停止服從你的命令。要維持對該生物的控制，你必須在 24 小時到期前再次對其施展此法術。這種使用方式可重新控制你用此法術操縱的最多 4 個不死生物，而非操縱新的屍骸。

**升環：** 每高一環，額外操縱或重新控制 2 個不死生物。每個生物必須來自不同的屍體或骸骨堆。

| 射程 | 36m |
| 成分 | V, S, M（一把扇子和一根羽毛） |
| 持續時間 | C 1 分鐘 |

**效果：** 在射程內你選擇的地面上升起一道強風牆，最長 15m、高 4.5m、厚 0.3m。你可將牆壁塑造成任何形狀，但必須形成一條連續的地面路徑。當牆壁出現時，區域內的每個生物須進行力量豁免，失敗時受到 4d8 鈍擊傷害，成功則傷害減半。強風阻止霧氣、煙霧和其他氣體通過。小型或更小的飛行生物或物體無法穿越牆壁。被帶入牆壁的鬆散輕質材料會被吹上天。箭矢、弩箭和其他普通投射物若射向牆壁後方的目標會被偏轉向上，自動未命中。巨人投擲的巨石、攻城器械的彈藥等類似投射物不受影響。氣態形體的生物無法穿越牆壁。

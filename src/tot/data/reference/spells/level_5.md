# 法術詳情 — 5 環（5th Level）

#### 異界探詢（Contact Other Plane）

| 項目 | 內容 |
|------|------|
| 環級 | 5 環 |
| 學派 | 預言（Divination） |
| 施法時間 | 1 分鐘（R） |
| 射程 | 自身 |
| 成分 | V |
| 持續時間 | 1 分鐘 |

**效果：** 你與一個半神、死者之靈或其他異界存在進行心靈接觸。須進行 DC 15 的智力豁免，失敗時受到 6d6 心靈傷害並失去行為能力（incapacitated），直到完成長休。成功後，你可提出最多 5 個是非題，該存在會給予簡短而真實的回答。

#### 夢境術（Dream）

| 項目 | 內容 |
|------|------|
| 環級 | 5 環 |
| 學派 | 幻術（Illusion） |
| 施法時間 | 1 分鐘 |
| 射程 | 特殊 |
| 成分 | V, S, M（一把沙子、一滴墨水和一根睡著的信使的羽毛） |
| 持續時間 | 8 小時 |

**效果：** 你指定一個你認識的、與你同位面的目標。你（或自願的碰觸生物作為信使）進入目標的夢境，可塑造夢境環境並傳遞訊息。目標醒來後完全記得夢境內容。若信使將夢境化為噩夢，目標須進行感知豁免，失敗時受到 3d6 心靈傷害且無法從該休息中獲益。

#### 怪物定身術（Hold Monster）

| 項目 | 內容 |
|------|------|
| 環級 | 5 環 |
| 學派 | 惑控（Enchantment） |
| 施法時間 | 1 動作 |
| 射程 | 27m |
| 成分 | V, S, M（一小塊鐵） |
| 持續時間 | C 1 分鐘 |

**效果：** 目標須進行感知豁免。失敗時被麻痺（paralyzed）。目標在每回合結束時可重新豁免，成功則法術結束。與人類定身術不同，此法術可影響任何生物類型。

**升環：** 每高一環，可多影響 1 個目標。

> ⚠️ 此為 2024 PHB 新增法術，dm20 資料庫無此條目，已依 PDF 核對。

| 施法時間 | 1 小時 |

| 持續時間 | 24 小時 |

**效果：** 你嘗試束縛射程內一個天界生物、元素生物、妖精或邪魔（目標通常須先被魔法陣等法術約束）。目標須進行魅力豁免，失敗時被束縛並必須服從你的命令。被束縛的生物會盡力服從指令。

**升環：** 6 環 = 10 天；7 環 = 30 天；8 環 = 180 天；9 環 = 1 年又 1 天。

**效果：** 你嘗試改造目標的記憶。目標須進行感知豁免（若已被你或同伴魅惑，則自動失敗）。失敗時，目標被魅惑且失去行為能力。你可以重塑目標過去 24 小時內最多 10 分鐘的記憶——消除、修改或植入新記憶。被修改的記憶中若有不合理之處，目標會自行合理化填補。

#### 活化物品（Animate Objects）

| 項目 | 內容 |
|------|------|
| 環級 | 5 環 |
| 學派 | 變化（Transmutation） |
| 施法時間 | 1 動作 |
| 射程 | 36m |
| 成分 | V, S |
| 持續時間 | C 1 分鐘 |

**效果：** 你選擇射程內數量至多等同於你施法屬性調整值的非魔法、未被穿戴或持握、未固定於表面的非巨型（Gargantuan）物品。中型以下的物品計為 1 件；大型計為 2 件；超大型計為 3 件。每件物品長出肢體並成為構裝體（Construct），使用活化物品數據塊（Animated Object stat block），受你控制直到法術結束或其 HP 降至 0。它們在戰鬥中共用你的先攻序並緊接你之後行動。你可用附贈動作下令 150m內的活化物品（可同時對多個下達相同命令）。未收到命令的物品執行閃避動作。HP 降至 0 時恢復原物品形態，溢出傷害轉移至物品。

**升環：** 每高一環，物品的猛擊（Slam）傷害增加 1d4（中型以下）、1d6（大型）或 1d12（超大型）。

| 項目 | 內容 |

| 環級 | 5 環 |
| 學派 | 變化（Transmutation） |

| 持續時間 | C 1 分鐘 |

**效果：** 你在射程內創造一隻大型的半透明發光之手，持續期間存在。手的 AC 為 20，HP 等於你的 HP 上限，力量 26（+8），敏捷 10（+0）。手不佔據空間。你可用附贈動作指揮巨手移動最多 18m並執行以下之一：**緊握拳（Clenched Fist）**——進行近戰法術攻擊，命中時造成 4d8 力場傷害；**強制之手（Forceful Hand）**——嘗試推動一個巨型以下的生物 1.5m加上你的施法屬性調整值乘以 1.5m的距離；**緊攫之手（Grasping Hand）**——嘗試擒抱（Grapple）一個巨型以下的生物（手的力量用於檢定），被擒抱的目標每回合受到 4d6+你的施法屬性調整值力場傷害；**阻擋之手（Interposing Hand）**——擋在你和你選擇的一個生物之間，你對該生物具有半掩護（Half Cover），該生物無法穿過手的空間。

**升環：** 每高一環，緊握拳傷害增加 2d8，緊攫之手傷害增加 2d6。

**效果：** 你獲得進入樹木並在同種樹木間穿越的能力。兩棵樹都必須是活的且至少與你一樣大。你用 1.5m移動速度進入一棵樹，立刻得知 150m內所有同種樹木的位置，並可作為進入動作的一部分傳送到其中一棵樹中或直接走出你所在的樹。你出現在目的地樹 1.5m內你選擇的位置，花費另外 1.5m移動速度。若你沒有剩餘移動速度，則出現在你進入的樹 1.5m內。每回合只能使用一次此傳送能力，且每回合結束時必須在樹外。

#### 力場之牆（Wall of Force）

| 項目 | 內容 |
|------|------|
| 環級 | 5 環 |
| 學派 | 塑能（Evocation） |
| 施法時間 | 1 動作 |
| 射程 | 36m |
| 成分 | V, S, M（一片玻璃碎片） |
| 持續時間 | C 10 分鐘 |

**效果：** 一道隱形的力場之牆出現在射程內你選擇的位置。牆可為水平、垂直或傾斜。你可將其塑造為半球形圓頂或球體（半徑最多 3m），或是由十個 3m乘 3m的面板組成的平面（每塊面板必須與另一塊相鄰）。無論形式，牆厚 1/4 吋。若牆出現時切過一個生物的空間，該生物被推到牆的一側（由你選擇）。任何物理實體皆無法穿過牆。牆免疫所有傷害，且無法被解除魔法（Dispel Magic）消除，但瓦解術（Disintegrate）可立即摧毀它。牆也延伸至靈界位面（Ethereal Plane），阻止靈界穿越。

#### 石牆術（Wall of Stone）

| 項目 | 內容 |
|------|------|
| 環級 | 5 環 |
| 學派 | 塑能（Evocation） |
| 施法時間 | 1 動作 |
| 射程 | 36m |
| 成分 | V, S, M（一小塊花崗岩方塊） |
| 持續時間 | C 10 分鐘 |

**效果：** 你在射程內創造一道非魔法的實心石牆。牆厚 6 吋，由十個 3m乘 3m的面板組成（每塊必須與另一塊相鄰）。你也可以製作 3m乘 6m、厚 3 吋的面板。若牆出現時切過一個生物的空間，該生物被推到牆的一側（由你選擇）。若生物會被牆四面圍困（或被牆與其他固體表面圍困），可進行敏捷豁免，成功時用反應移動至多其速度以逃離。牆可塑造為任何形狀，但必須與現有石頭合併並受其支撐。每個面板 AC 15，每吋厚度 30 HP，免疫毒素和心靈傷害。若你在法術完整持續期間維持專注，牆成為永久存在且無法被解除魔法。

**效果：** 你散發出令人敬畏的高貴氣場，影響以你為中心 3m散發區域（Emanation）內你選擇的每個生物。選擇效果：**魅惑**——目標須進行感知豁免，失敗時被魅惑（Charmed），視你為友善；**恐懼**——目標須進行感知豁免，失敗時被恐嚇（Frightened）。受影響的生物在每回合結束時可重新進行感知豁免以結束效果。

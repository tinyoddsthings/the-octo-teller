# 法術詳情 — 4 環（4th Level）

> 中央法術庫。各職業共用，按環級分檔。
> 全部條目已驗證 ✅

#### 放逐術（Banishment）

| 項目 | 內容 |
|------|------|
| 環級 | 4 環 |
| 學派 | 防護（Abjuration） |
| 施法時間 | 1 動作 |
| 射程 | 18m |
| 成分 | V, S, M（一件目標厭惡的物品） |
| 持續時間 | C 1 分鐘 |

**效果：** 目標須進行魅力豁免。失敗時被放逐。若目標原生於你所在的位面，它被送往一個無害的半位面，法術結束時返回。若目標並非原生於當前位面，它被遣回自己的位面；若法術持續了完整的持續時間，目標不會返回。

**升環：** 每高一環，可多放逐 1 個目標。

#### 枯萎術（Blight）

| 項目 | 內容 |
|------|------|
| 環級 | 4 環 |
| 學派 | 死靈（Necromancy） |
| 施法時間 | 1 動作 |
| 射程 | 9m |
| 成分 | V, S |
| 持續時間 | 瞬間 |

**效果：** 黯蝕能量從目標體內抽取水分和活力。目標須進行體質豁免，失敗時受到 8d8 黯蝕傷害，成功則傷害減半。植物型生物的豁免具有劣勢，且受到最大傷害。非魔法的普通植物（非生物）則直接枯萎死亡。

**升環：** 每高一環，傷害增加 1d8。

|------|------|

| 射程 | 27m |
| 成分 | V, S, M（三個堅果殼） |
| 持續時間 | C 1 分鐘 |

**效果：** 以射程內一點為中心的 3m半徑球形內，每個生物須進行感知豁免。失敗的生物在每回合開始時擲 d10 決定行為：1 = 隨機方向移動全部移動力；2～6 = 不移動、不行動；7～8 = 用近戰攻擊隨機鄰近生物（無目標則不行動）；9～10 = 正常行動。目標在每回合結束時可重新豁免。

**升環：** 每高一環，半徑增加 1.5m。

#### 控水術（Control Water）

| 項目 | 內容 |
|------|------|
| 環級 | 4 環 |
| 學派 | 變化（Transmutation） |
| 施法時間 | 1 動作 |
| 射程 | 90m |
| 成分 | V, S, M（水與塵土的混合物） |
| 持續時間 | C 10 分鐘 |

**效果：** 直至法術結束前，你可控制射程內最多 30m立方區域的水體，選擇以下效果之一，並可在後續回合以魔法動作（Magic action）重複同一效果或切換：**洪流**——靜水水位上升最多 6m；若在大型水體中則產生 6m高的浪潮衝向另一側，巨型或更小的載具被浪潮攜帶，被衝擊的載具有 25% 機率翻覆。水位持續升高直至法術結束或你切換效果；若產生浪潮，浪潮在你的下一回合開始時重複。**分水**——在水體中分出壕溝，被分開的水在兩側形成水牆。壕溝持續至法術結束或切換效果，之後水在下一輪緩慢回填。**改道**——令區域內的流水朝你選擇的方向移動，即使需越過障礙或逆流而上；離開法術區域後恢復正常流向。**渦流**——在區域中心產生旋渦（區域至少 15m見方、7.5m深）。旋渦底寬 1.5m、頂寬最多 15m、高 7.5m。水中且在旋渦 7.5m內的生物被拉近 3m。生物首次在回合中進入旋渦或在其中結束回合時須進行力量豁免，失敗受到 2d8 鈍擊傷害，成功則傷害減半。游離旋渦需以動作進行力量（運動）檢定對抗你的法術豁免 DC。

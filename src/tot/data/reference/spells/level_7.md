# 法術詳情 — 7 環（7th Level）

#### 靈體步行（Etherealness）

| 項目 | 內容 |
|------|------|
| 環級 | 7 環 |
| 學派 | 咒法（Conjuration） |
| 施法時間 | 1 動作 |
| 射程 | 自身 |
| 成分 | V, S |
| 持續時間 | 8 小時 |

**效果：** 你踏入靈界（Ethereal Plane）。你能看見原來位面的灰色影像，視距 18m。你可在任何方向移動。只有靈界的生物和效果能影響你。法術結束時你返回原來位面你離開的位置（若已被佔據則返回最近的未佔據空間）。

**升環：** 每高一環，可多帶 1 個自願的觸碰生物。

#### 指印牢籠（Forcecage）

| 項目 | 內容 |
|------|------|
| 環級 | 7 環 |
| 學派 | 塑能（Evocation） |
| 施法時間 | 1 動作 |
| 射程 | 30m |
| 成分 | V, S, M（紅寶石粉末，價值至少 1,500 gp） |
| 持續時間 | 1 小時 |

**效果：** 在射程內創造一個不可移動的力場牢籠。選擇形態：**柵欄型**——3m立方的柵欄（間距 0.5 吋）；**箱型**——3m立方的密封方塊。沒有任何東西能穿過牢籠。試圖從牢籠內傳送或跨位面旅行的生物須進行魅力豁免（對抗你的法術豁免 DC），失敗則傳送失敗。解除魔法（Dispel Magic）無法影響此法術。

#### 召喚天使（Conjure Celestial）

| 項目 | 內容 |
|------|------|
| 環級 | 7 環 |
| 學派 | 咒法（Conjuration） |
| 施法時間 | 1 動作 |
| 射程 | 27m |
| 成分 | V, S |
| 持續時間 | C 10 分鐘 |

**效果：** 你召喚來自上層位面的靈體，在射程內一點顯現為一道光柱——3m半徑、12m高的圓柱形區域。對圓柱內你能看見的每個生物，選擇以下光芒之一：目標回復等同於 4d12 + 你的施法屬性調整值的生命值；或目標須進行敏捷豁免，失敗時受到 6d12 光耀（radiant）傷害，成功減半。法術持續期間，圓柱發出明亮光照。你在自己的回合移動時，也可移動圓柱最多 9m。每當圓柱移入你能看見的生物所在空間，或每當你能看見的生物進入圓柱或在其中結束回合時，你可對其施加上述光芒之一。每個生物每回合只受此法術影響一次。

**升環：** 每高一環，治療量和傷害各增加 6d12。

#### 延遲爆裂火球（Delayed Blast Fireball）

| 項目 | 內容 |
|------|------|
| 環級 | 7 環 |
| 學派 | 塑能（Evocation） |
| 施法時間 | 1 動作 |
| 射程 | 45m |
| 成分 | V, S, M（一顆蝙蝠糞球和硫磺） |
| 持續時間 | C 1 分鐘 |

**效果：** 你射出一道黃色光束，在射程內你選擇的一點凝結為一顆發光珠子並持續存在。法術結束時珠子爆炸，以該點為中心 6m半徑球形內的所有生物須進行敏捷豁免，失敗時受到累積的火焰傷害，成功減半。法術基礎傷害為 12d6，每當你的回合結束且法術未結束時，傷害增加 1d6。若有生物在法術結束前觸碰珠子，該生物須進行敏捷豁免；失敗則法術結束、珠子爆炸；成功則該生物可將珠子投擲最遠 12m。若投出的珠子進入生物的空間或撞到固體物體，法術結束、珠子爆炸。爆炸會點燃範圍內未被穿戴或攜帶的可燃物體。

**升環：** 每高一環，基礎傷害增加 12d6。

#### 海市蜃樓（Mirage Arcane）

| 項目 | 內容 |
|------|------|
| 環級 | 7 環 |
| 學派 | 幻術（Illusion） |
| 施法時間 | 10 分鐘 |
| 射程 | 視線所及 |
| 成分 | V, S |
| 持續時間 | 10 天 |

**效果：** 你使最多 1 平方英里的地形在視覺、聽覺、嗅覺甚至觸覺上看似另一種地形。空地或道路可被偽裝成沼澤、丘陵、峽谷等崎嶇或不可通行的地形；池塘可看似草地；懸崖可看似緩坡；亂石溝可看似寬闊平路。同樣，你可改變建築物的外觀或在沒有建築的地方添加建築。此法術無法偽裝、隱藏或新增生物。幻象包含聽覺、視覺、觸覺和嗅覺元素，因此可以將平坦地面變為困難地形（或反之）或以其他方式阻礙移動。任何從法術範圍中移出的幻象地形碎片（如石頭或樹枝）立即消失。擁有真知術（Truesight）的生物能看穿幻象至地形的真實形態，但幻象的其他所有元素仍然存在，因此該生物雖知道幻象存在，仍可與幻象進行物理互動。

#### 復活術（Resurrection）

| 項目 | 內容 |
|------|------|
| 環級 | 7 環 |
| 學派 | 死靈（Necromancy） |
| 施法時間 | 1 小時 |
| 射程 | 觸碰 |
| 成分 | V, S, M（一顆價值至少 1,000 gp 的鑽石，消耗） |
| 持續時間 | 瞬間 |

**效果：** 你觸碰死亡不超過 1 世紀、非老死且死亡時非不死生物（Undead）的生物，使其復活並完全恢復所有生命值。此法術亦會中和死亡時影響目標的毒素、關閉所有致命傷口並修復所有缺失的身體部位。復活後的目標在 D20 檢定上受到 -4 減值；每完成一次長休息，此減值降低 1，直到歸零。若目標死亡超過 365 天，施法者在完成長休息前無法再施展任何法術，且所有 D20 檢定具有劣勢。

#### 模擬生命（Simulacrum）

| 項目 | 內容 |
|------|------|
| 環級 | 7 環 |
| 學派 | 幻術（Illusion） |
| 施法時間 | 12 小時 |
| 射程 | 觸碰 |
| 成分 | V, S, M（價值至少 1,500 gp 的紅寶石粉末，消耗） |
| 持續時間 | 永久（直到被解除） |

**效果：** 你為在施法全程都處於你 3m內的一隻野獸或類人生物創造模擬體。施法結束時你觸碰目標和一堆與目標同等大小的冰雪，冰雪變為模擬體（構裝體 Construct）。模擬體使用原始生物施法時的遊戲數據，但生命值上限減半且無法施展此法術。模擬體對你和你指定的生物友善（Friendly），服從你的命令，在戰鬥中於你的回合行動。模擬體無法升級，也無法進行短休息或長休息。若模擬體受到傷害，唯一恢復方式是在你進行長休息時修復它——你花費每恢復 1 HP 價值 100 gp 的材料，修復期間模擬體必須在你 1.5m內。模擬體持續存在直到 HP 降至 0，屆時化為雪並融化消失。若你再次施展此法術，之前創造的模擬體立即被摧毀。

#### 符記（Symbol）

| 項目 | 內容 |
|------|------|
| 環級 | 7 環 |
| 學派 | 防護（Abjuration） |
| 施法時間 | 1 分鐘 |
| 射程 | 觸碰 |
| 成分 | V, S, M（鑽石粉末，價值至少 1,000 gp，消耗） |
| 持續時間 | 永久（直到觸發或解除） |

**效果：** 你在一個表面（如地板或牆壁）或可關閉的物品（如書或箱子）上刻下一個有害的符文，直徑不超過 3m。若刻在物品上，該物品必須留在原處；若被移動超過施法位置 3m，符文破裂且法術結束。符文幾乎不可見，須通過感知（Perception）檢定（DC 等於你的法術豁免 DC）才能察覺。施法時設定觸發條件和效果。觸發後符文發光，填滿 18m半徑球形的微光，持續 10 分鐘後法術結束。符文啟動時在球形內的生物受到效果影響，首次在回合中進入球形或在其中結束回合的生物亦然（每回合僅觸發一次）。效果選項：**死亡**——體質豁免，失敗 10d10 黯蝕傷害，成功減半；**不和**——感知豁免，失敗則與他人爭論 1 分鐘（期間攻擊擲骰和屬性檢定具有劣勢）；**恐懼**——感知豁免，失敗被恐嚇（frightened）1 分鐘，每回合必須移動至少 9m遠離符文；**疼痛**——體質豁免，失敗被失能（incapacitated）1 分鐘；**沉眠**——感知豁免，失敗陷入昏迷（unconscious）10 分鐘（受到傷害或被搖醒則清醒）；**暈眩**——感知豁免，失敗被擊暈（stunned）1 分鐘。

#### 傳送術（Teleport）

| 項目 | 內容 |
|------|------|
| 環級 | 7 環 |
| 學派 | 咒法（Conjuration） |
| 施法時間 | 1 動作 |
| 射程 | 3m |
| 成分 | V |
| 持續時間 | 瞬間 |

**效果：** 你瞬間將自己和射程內你能看見的最多 8 個自願生物（或一個你能看見的大型或更小的物品，且該物品不能被不自願的生物持有或攜帶）傳送至同一位面內你選擇的目的地。目的地必須是你已知的地點。DM 擲 1d100 並查閱「傳送結果表」。熟悉度分為：**永久圓陣**——你知道其符文序列的永久傳送陣；**連結物品**——你持有來自目的地且在過去 6 個月內取得的物品；**非常熟悉**——多次到訪或仔細研究的地點；**偶然見過**——見過超過一次但不太熟悉；**僅見一次或僅憑描述**——只見過一次或透過他人描述得知的地點；**虛假目的地**——不存在的地方。**失誤（Mishap）**：每個傳送的生物或物品受到 3d10 力場傷害，DM 重骰決定去向。**相似地點**：出現在視覺或主題相似的其他地點。**偏離目標**：出現在距目的地 2d12 英里遠的隨機方向。**精準到達**：抵達預定目的地。

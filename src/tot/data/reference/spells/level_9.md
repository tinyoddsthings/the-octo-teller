# 法術詳情 — 9 環（9th Level）

> 中央法術庫。各職業共用，按環級分檔。
> 全部條目已驗證 ✅

---

#### 星界投射（Astral Projection）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 死靈（Necromancy） |
| 施法時間 | 1 小時 |
| 射程 | 3m |
| 成分 | V, S, M（每人一顆價值至少 1,000 gp 的紫鋯石和一根價值至少 100 gp 的銀棒，消耗） |
| 持續時間 | 特殊 |

**效果：** 你和最多 8 個自願的生物投射出星界體。你的肉體留在原位處於假死狀態，一條銀線連接星界體和肉體。你可在星界（Astral Plane）中旅行，並透過星界進入其他位面。若銀線被切斷（極罕見事件）或肉體上的法術被解除，你將返回肉體。

#### 先見之明（Foresight）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 預言（Divination） |
| 施法時間 | 1 分鐘 |
| 射程 | 觸碰 |
| 成分 | V, S, M（一根蜂鳥的羽毛） |
| 持續時間 | 8 小時 |

**效果：** 你觸碰的生物在法術持續期間獲得預知未來的能力。目標不會被突襲（surprised），攻擊擲骰、屬性檢定和豁免擲骰具有優勢，且其他生物對目標的攻擊擲骰具有劣勢。此法術不需專注。

#### 次元門（Gate）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 咒法（Conjuration） |
| 施法時間 | 1 動作 |
| 射程 | 18m |
| 成分 | V, S, M（一顆價值至少 5,000 gp 的鑽石） |
| 持續時間 | C 1 分鐘 |

**效果：** 你在射程內打開一道通往其他位面特定位置的傳送門。傳送門為 5～6m直徑的圓形開口，朝向你選擇的方向。任何生物或物品通過傳送門即到達另一側。你也可以說出一個特定存在的名字——若該存在不在你的位面，它會被吸引通過傳送門來到你面前。

#### 禁錮術（Imprisonment）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 防護（Abjuration） |
| 施法時間 | 1 分鐘 |
| 射程 | 9m |
| 成分 | V, S, M（視禁錮形式而定，每種至少 500 gp） |
| 持續時間 | 直到被解除 |

**效果：** 你對目標施加強大的禁錮。目標須進行感知豁免，失敗時被禁錮。選擇以下禁錮形式之一：**埋葬**——目標被埋入地下深處並處於假死狀態；**鎖鏈**——目標被沉重的鎖鏈束縛在異次元空間中；**對沖牢籠**——目標被限制在一個小型封閉區域；**微型禁錮**——目標被縮小並封入寶石或類似物品內；**沉眠**——目標陷入永久睡眠，無法被喚醒。解除魔法（Dispel Magic）無法解除此法術，但許願術（Wish）可以。

#### 致命密語（Power Word Kill）

| 項目 | 內容 |

#### 真正變形（True Polymorph）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 變化（Transmutation） |
| 施法時間 | 1 動作 |
| 射程 | 9m |
| 成分 | V, S, M（一滴水銀、一團阿拉伯膠和一縷煙） |
| 持續時間 | C 1 小時 |

**效果：** 你將一個生物變形為另一個生物或物品，或將一個物品變形為一個生物。**生物→生物：** 新形態的挑戰等級（CR）必須 ≤ 目標的等級或 CR。目標獲得新形態的數據塊。**生物→物品：** 目標連同裝備變為非魔法物品。**物品→生物：** 物品變為 CR ≤ 9 的生物。不自願的目標可進行感知豁免避免。若專注維持完整的 1 小時，變形成為永久（直到被解除）。

#### 怪影殺手（Weird）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 幻術（Illusion） |
| 施法時間 | 1 動作 |
| 射程 | 36m |
| 成分 | V, S |
| 持續時間 | C 1 分鐘 |

**效果：** 以射程內一點為中心的 9m半徑球形內，你從每個生物最深層的恐懼中召喚幻象。範圍內每個生物須進行感知豁免，失敗時被恐嚇（frightened）。被恐嚇的生物在每回合開始時受到 4d10 心靈傷害。受影響的生物在每回合結束時可重新進行感知豁免，成功則對其效果結束。

#### 群體治癒（Mass Heal）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 防護（Abjuration） |
| 施法時間 | 1 動作 |
| 射程 | 18m |
| 成分 | V, S |
| 持續時間 | 瞬間 |

**效果：** 一股治癒能量從你流向周圍的生物。你恢復共計最多 700 點生命值，由你分配給射程內你能看見的任意數量生物。被此法術治癒的生物還會移除目盲（blinded）、耳聾（deafened）和中毒（poisoned）狀態。

#### 隕石群（Meteor Swarm）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 塑能（Evocation） |
| 施法時間 | 1 動作 |
| 射程 | 1 英里 |
| 成分 | V, S |
| 持續時間 | 瞬間 |

**效果：** 閃亮的火球從天而降，墜落在射程內你能看見的 4 個不同地點。以每個地點為中心的 12m半徑球形內，每個生物須進行敏捷豁免，失敗時受到 20d6 火焰傷害 + 20d6 鈍擊傷害，成功減半。若生物處於多個球形範圍內，只受影響一次。範圍內未被穿戴或攜帶的非魔法物品也會受到此傷害，可燃物品著火。

#### 治癒之語（Power Word Heal）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 惑控（Enchantment） |
| 施法時間 | 1 動作 |
| 射程 | 18m |
| 成分 | V, S |
| 持續時間 | 瞬間 |

**效果：** 一波治癒能量沖刷射程內你能看見的一個生物。目標恢復全部生命值。此外，若目標具有以下狀態，則該狀態結束：魅惑（charmed）、恐嚇（frightened）、麻痺（paralyzed）、中毒（poisoned）或擊暈（stunned）。若目標具有倒地（prone）狀態，目標可用反應站起。

#### 虹光之牆（Prismatic Wall）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 防護（Abjuration） |
| 施法時間 | 1 動作 |
| 射程 | 18m |
| 成分 | V, S |
| 持續時間 | 10 分鐘 |

**效果：** 在射程內一點形成一面閃爍七彩光芒的不透明垂直牆壁——最長 27m、高 9m、厚 1 吋；或形成直徑最大 9m的球形。若牆壁位於生物佔據的空間，法術立即無效結束。牆壁在 30m內發出明亮光照，額外 30m為微光。你及你在施法時指定的生物可無害地通過或靠近牆壁。其他能看見牆壁的生物若移動至距牆壁 6m內或在該距離內開始回合，須進行體質豁免，失敗則被目盲（blinded）1 分鐘。牆壁由七層顏色組成，生物伸入或穿越牆壁時逐層受到效果（每層須進行敏捷豁免，見「虹光層表 Prismatic Layers」）。牆壁（AC 10）可按紅→紫的順序逐層摧毀，每層有特定的摧毀方式。反魔法力場對牆壁無效，解除魔法只能影響紫色層。

#### 變身術（Shapechange）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 變化（Transmutation） |
| 施法時間 | 1 動作 |
| 射程 | 自身 |
| 成分 | V, S, M（一個價值至少 1,500 gp 的翡翠玉環） |
| 持續時間 | C 1 小時 |

**效果：** 你在持續時間內變形為另一個生物，或用魔法動作（Magic action）變形為不同的合格形態。新形態必須是你曾見過的、CR 不高於你等級或 CR 的生物，且不能是構裝體（Construct）或不死生物（Undead）。施法時你獲得等同於首次變形形態 HP 的臨時生命值（Temporary HP），法術結束時未用完的臨時 HP 消失。你的遊戲數據被所選形態的數據取代，但保留你的生物類型、陣營、人格、智力/感知/魅力值、生命值、生命骰、熟練項和溝通能力。若你具有施法特性（Spellcasting），亦保留。變形時你決定裝備是掉落地面還是隨形態變化大小。

#### 風暴復仇（Storm of Vengeance）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 咒法（Conjuration） |
| 施法時間 | 1 動作 |
| 射程 | 1 英里 |
| 成分 | V, S |
| 持續時間 | C 1 分鐘 |

**效果：** 在射程內一點形成翻騰的暴風雲，擴展至 90m半徑。暴風雲出現時，其下方每個生物須進行體質豁免，失敗受 2d6 雷鳴傷害並在持續時間內獲得耳聾（deafened）狀態。之後你每個回合開始時產生不同效果：**第 2 回合**——酸雨，雲下方每個生物和物品受 4d6 強酸傷害；**第 3 回合**——你從雲中召喚 6 道閃電，打擊雲下方 6 個不同的生物或物品，每個目標須進行敏捷豁免，失敗受 10d6 閃電傷害，成功減半；**第 4 回合**——冰雹，雲下方每個生物受 2d6 鈍擊傷害；**第 5-10 回合**——陣風和凍雨，雲下方每個生物受 1d6 寒冷傷害。法術結束前，該區域為困難地形和重度遮蔽，區域內無法進行武器遠程攻擊，且有強風吹過。

#### 時間停止（Time Stop）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 變化（Transmutation） |
| 施法時間 | 1 動作 |
| 射程 | 自身 |
| 成分 | V |
| 持續時間 | 瞬間 |

**效果：** 你短暫停止所有其他生物的時間流動。你連續獲得 1d4 + 1 個回合，期間可正常使用動作和移動。若你在這段期間使用的動作或創造的效果影響到你以外的生物，或影響到他人穿戴或攜帶的物品，法術立即結束。此外，若你移動至距施法地點超過 1,0m的地方，法術也立即結束。

#### 真正復活（True Resurrection）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 死靈（Necromancy） |
| 施法時間 | 1 小時 |
| 射程 | 觸碰 |
| 成分 | V, S, M（價值至少 25,000 gp 的鑽石，消耗） |
| 持續時間 | 瞬間 |

**效果：** 你觸碰一個死亡不超過 200 年、因任何原因死亡（老死除外）的生物，使其復活並完全恢復所有生命值。此法術關閉所有傷口、中和毒素、治癒所有魔法傳染病並解除死亡時影響目標的所有詛咒。法術可替換受損或缺失的器官和肢體。若目標曾為不死生物（Undead），會被恢復為其非不死形態。若目標的原始身體已不存在，你必須說出生物的名字，該生物會出現在你身邊 3m內的未佔據空間。

#### 祈願術（Wish）

| 項目 | 內容 |
|------|------|
| 環級 | 9 環 |
| 學派 | 咒法（Conjuration） |
| 施法時間 | 1 動作 |
| 射程 | 自身 |
| 成分 | V |
| 持續時間 | 瞬間 |

**效果：** 祈願術是凡人所能施展的最強法術。最基本的用途是複製任何 8 環或以下的法術（無需滿足該法術的任何要求，包括高價材料成分）。你也可選擇以下效果之一：創造一個價值不超過 25,000 gp 的非魔法物品（任何維度不超過 90m，出現在你能看見的地面未佔據空間）；讓你和你能看見的最多 20 個生物恢復全部 HP 並解除高等復原術（Greater Restoration）所列的所有效果；賦予你能看見的最多 10 個生物對你選擇的一種傷害類型的永久抗性；賦予你能看見的最多 10 個生物對單一法術或魔法效果的免疫（持續 8 小時）；將你的一個專長替換為另一個你有資格獲得的專長；強制重骰最近 1 輪內的任何一次 d20 擲骰（可指定優勢或劣勢，並選擇使用重骰或原始結果）。你也可許願上述以外的事，但 DM 有極大裁量權——願望越大，出錯的可能性越高。**壓力代價：** 若你以複製法術以外的方式使用祈願術，之後每次施法都會受到每法術等級 1d10 黯蝕傷害（無法減免），力量降至 3 持續 2d4 天（每天休息可減少 2 天），且有 33% 機率你永遠無法再施展祈願術。

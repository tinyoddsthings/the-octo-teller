# 法術詳情 — 8 環（8th Level）

> 中央法術庫。各職業共用，按環級分檔。

#### 困惑大法（Befuddlement）

| 項目 | 內容 |
|------|------|
| 環級 | 8 環 |
| 學派 | 惑控（Enchantment） |
| 施法時間 | 1 動作 |
| 射程 | 45m |
| 成分 | V, S, M（一把黏土、水晶、玻璃或礦物球） |
| 持續時間 | 瞬間 |

**效果：** 你用壓倒性的心靈能量攻擊目標的智力。目標須進行智力豁免，失敗時其智力和魅力變為 1。目標無法施法、啟動魔法物品、理解語言或進行有邏輯的溝通。但目標能辨識同伴、跟隨並保護他們。目標每 30 天可重新進行智力豁免。高等復原術（Greater Restoration）、療癒術（Heal）或許願術（Wish）可終結此效果。

#### 半位面（Demiplane）

| 項目 | 內容 |
|------|------|
| 環級 | 8 環 |
| 學派 | 咒法（Conjuration） |
| 施法時間 | 1 動作 |
| 射程 | 18m |
| 成分 | S |
| 持續時間 | 1 小時 |

**效果：** 你在射程內一個你能看見的平面上創造一扇暗影之門。門開啟後通往一個 9m × 9m × 9m的空半位面。每次施展此法術時，你可創造新的半位面或連接到之前創造的半位面。門關閉後，仍在半位面內的生物被困在其中，直到門再次打開。

#### 支配怪物（Dominate Monster）

| 項目 | 內容 |
|------|------|
| 環級 | 8 環 |
| 學派 | 惑控（Enchantment） |
| 施法時間 | 1 動作 |
| 射程 | 18m |
| 成分 | V, S |
| 持續時間 | C 1 小時 |

**效果：** 射程內一個生物須進行感知豁免。失敗時被魅惑。你與目標建立心靈感應連結，可用動作精確控制其行動。目標每次受到傷害時可重新進行感知豁免。與支配人類不同，此法術可影響任何生物類型。

**升環：** 9 環 = C 8 小時。

**效果：** 你說出一個強大的密語。若目標生物當前生命值為 150 或以下，則被震懾（stunned）。被震懾的目標在每回合結束時可進行體質豁免，成功則效果結束。

#### 動物形態（Animal Shapes）

| 項目 | 內容 |
|------|------|
| 環級 | 8 環 |
| 學派 | 變化（Transmutation） |
| 施法時間 | 1 動作 |
| 射程 | 9m |
| 成分 | V, S |
| 持續時間 | 24 小時 |

**效果：** 你選擇射程內你能看見的任意數量自願生物。每個目標變形為你選擇的一種大型或更小的野獸形態，挑戰等級必須 ≤ 4（每個目標可選不同形態）。在之後的回合中，你可用魔法動作（Magic action）再次變換所有目標的形態。目標的遊戲數據被所選野獸的數據取代，但保留其生物類型、生命值、生命骰、陣營、溝通能力以及智力、感知和魅力值。目標的行動受限於野獸形態的體型，無法施法。裝備融入新形態且無法使用。目標獲得等同於首次變形所選野獸 HP 的臨時生命值（Temporary HP），法術結束時未用完的臨時 HP 消失。變形持續至法術結束或目標以附贈動作結束變形。

#### 反魔法力場（Antimagic Field）

| 項目 | 內容 |
|------|------|
| 環級 | 8 環 |
| 學派 | 防護（Abjuration） |
| 施法時間 | 1 動作 |
| 射程 | 自身（3m放射） |
| 成分 | V, S, M（鐵屑） |
| 持續時間 | C 1 小時 |

**效果：** 一道反魔法光環以 3m放射（Emanation）圍繞你。光環內任何人都無法施法、進行魔法動作（Magic action）或產生其他魔法效果，且上述行為也無法針對或影響光環內的事物。魔法物品的魔法屬性在光環內或對光環內的事物不起作用。由法術或其他魔法創造的效果區域無法延伸入光環，且光環內無人能傳送進出或使用位面旅行。傳送門在光環中暫時關閉。持續中的法術（神器或神祇施放的除外）在區域內被壓制——效果暫停但壓制期間仍計入持續時間。解除魔法（Dispel Magic）對此光環無效，且不同反魔法力場的光環不會互相抵消。

#### 反感/同感（Antipathy/Sympathy）

| 項目 | 內容 |
|------|------|
| 環級 | 8 環 |
| 學派 | 惑控（Enchantment） |
| 施法時間 | 1 小時 |
| 射程 | 18m |
| 成分 | V, S, M（醋和蜂蜜的混合物） |
| 持續時間 | 10 天 |

**效果：** 施法時選擇**反感**或**同感**，並指定一個巨型或更小的生物或物品為目標，再指定一種生物類型（如紅龍、哥布林、吸血鬼等）。指定類型的生物在進入目標 36m範圍時須進行感知豁免。**反感**——豁免失敗的生物獲得恐嚇（frightened）狀態，必須用移動遠離目標，沿最安全路線行動。**同感**——豁免失敗的生物獲得魅惑（charmed）狀態，必須用移動盡可能接近目標，沿最安全路線行動；若在目標 1.5m內，該生物無法自願移開；若被目標傷害，魅惑的生物可重新進行感知豁免以結束效果。**效果結束：** 若恐嚇或魅惑的生物在其回合結束時距離目標超過 36m，可進行感知豁免；成功則效果結束。成功豁免的生物免疫此效果 1 分鐘，之後可再次受到影響。

#### 複製術（Clone）

| 項目 | 內容 |
|------|------|
| 環級 | 8 環 |
| 學派 | 死靈（Necromancy） |
| 施法時間 | 1 小時 |
| 射程 | 觸碰 |
| 成分 | V, S, M（一顆價值至少 1,000 gp 的鑽石（消耗），以及一個可密封的容器（價值至少 2,000 gp 且足以容納被複製的生物）） |
| 持續時間 | 瞬間 |

**效果：** 你觸碰一個生物或其至少 1 立方吋的肉體。在法術所用的容器中形成該生物的惰性複製體，經 120 天完成培育；你可選擇複製體的年齡與原始生物相同或更年輕。複製體在容器未受干擾的情況下可無限期保持惰性。若原始生物在複製體完成培育後死亡，且其靈魂自由並願意返回，靈魂將轉移至複製體。複製體在外觀上與原始生物完全相同，擁有相同的人格、記憶和能力，但不包含原始生物的裝備。原始生物的遺骸（若有）變為惰性，無法被復活，因為其靈魂已轉移至別處。

#### 控制天氣（Control Weather）

| 項目 | 內容 |
|------|------|
| 環級 | 8 環 |
| 學派 | 變化（Transmutation） |
| 施法時間 | 10 分鐘 |
| 射程 | 自身（5 英里半徑） |
| 成分 | V, S, M（燃燒的薰香） |
| 持續時間 | C 8 小時 |

**效果：** 你必須在戶外施法，若你進入室內則法術提前結束。施法時你改變當前天氣狀況（由 DM 決定基準）。你可改變降水、氣溫和風力。新條件需 1d4 × 10 分鐘逐漸生效。條件生效後，你可再次更改。法術結束後天氣逐漸恢復正常。改變天氣時，在以下表格中找到當前狀態並將其階段調升或調降一級。改變風力時也可改變風向。（見降水表、氣溫表、風力表）

#### 地震術（Earthquake）

| 項目 | 內容 |
|------|------|
| 環級 | 8 環 |
| 學派 | 變化（Transmutation） |
| 施法時間 | 1 動作 |
| 射程 | 150m |
| 成分 | V, S, M（一塊碎裂的岩石） |
| 持續時間 | C 1 分鐘 |

**效果：** 你選擇射程內你能看見的一點。以該點為中心的 30m半徑圓形地面產生劇烈震動，地面變為困難地形。施法時及之後你每回合結束時，站在地面的每個生物須進行敏捷豁免，失敗則被擊倒（prone）且專注被打斷。你還可造成以下額外效果：**地裂**——施法回合結束時產生共 1d6 道裂縫，由你決定位置（不能在建築物下），每道裂縫 1d10 × 3m深、3m寬，從區域一端延伸至另一端；處於裂縫空間的生物須進行敏捷豁免，失敗則墜入，成功則隨裂縫邊緣移動。**建築物**——震動對區域內接觸地面的建築物造成 50 鈍擊傷害（施法時及你每回合結束時）。建築物 HP 降至 0 時倒塌，距倒塌建築物半身高距離內的生物須進行敏捷豁免，失敗受 12d6 鈍擊傷害、被擊倒（prone）並被瓦礫埋住（需 DC 20 力量〔運動〕檢定以動作脫出），成功則傷害減半。

| 射程 | 45m |
| 成分 | V, S |

**效果：** 在射程內一點召喚一片由餘燼和煙霧組成的旋轉雲霧，填滿以該點為中心的 6m半徑球形。雲霧所在區域為重度遮蔽（Heavily Obscured）。雲霧持續至法術結束或被強風（如陣風術 Gust of Wind）驅散。雲霧出現時，範圍內每個生物須進行敏捷豁免，失敗時受到 10d8 火焰傷害，成功減半。當球形移入生物所在空間，或生物進入球形或在其中結束回合時，亦須進行此豁免（每個生物每回合只豁免一次）。你每回合開始時，雲霧朝你選擇的方向移動 3m。

| 射程 | 18m |

#### 日炎爆裂（Sunburst）

| 項目 | 內容 |
|------|------|
| 環級 | 8 環 |
| 學派 | 塑能（Evocation） |
| 施法時間 | 1 動作 |
| 射程 | 45m |
| 成分 | V, S, M（一塊太陽石） |
| 持續時間 | 瞬間 |

**效果：** 以射程內你選擇的一點為中心，18m半徑球形內爆發出耀眼的陽光。範圍內每個生物須進行體質豁免，失敗時受到 12d6 光耀（radiant）傷害並獲得目盲（blinded）狀態 1 分鐘；成功則傷害減半且不受目盲。被此法術致盲的生物在每回合結束時可重新進行體質豁免，成功則目盲結束。此法術可驅散其範圍內任何法術所創造的黑暗（Darkness）。

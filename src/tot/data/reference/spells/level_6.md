#### 奧術之門（Arcane Gate）

| 項目 | 內容 |
|------|------|
| 環級 | 6 環 |
| 學派 | 咒法（Conjuration） |
| 施法時間 | 1 動作 |
| 射程 | 150m |
| 成分 | V, S |
| 持續時間 | C 10 分鐘 |

**效果：** 你在射程內創造兩個相連的傳送門，各為 3m直徑的光環，懸浮在你能看見的平面上。任何生物或物品進入其中一個門就從另一個門出現。你可指定門為單向或雙向。

#### 創造不死生物（Create Undead）

| 項目 | 內容 |
|------|------|
| 環級 | 6 環 |
| 學派 | 死靈（Necromancy） |
| 施法時間 | 1 分鐘 |
| 射程 | 3m |
| 成分 | V, S, M（裝有墓土的陶罐、裝有鹹水的陶罐、每具屍體一顆價值 150 gp 的黑瑪瑙） |
| 持續時間 | 瞬間 |

**效果：** 只能在夜間施展。你將最多 3 具中型或小型類人屍體變為食屍鬼（ghoul），服從你的口頭命令。每 24 小時你必須重新施展此法術以維持控制，否則它們變為不受控。

**升環：** 7 環 = 4 食屍鬼或 2 屍妖（ghast）/幽鬼（wight）；8 環 = 5 食屍鬼或 3 屍妖/幽鬼或 2 木乃伊；9 環 = 6 食屍鬼或 4 屍妖/幽鬼或 3 木乃伊。

| 成分 | V, S, M（三根銀別針） |

**效果：** 你向射程內一個你能看見的目標發射一道閃電。然後三道閃電從該目標彈射到最多 3 個你選擇的其他目標，每個必須在第一個目標 9m內。目標可以是生物或物品，且每個目標只能被一道閃電命中。每個目標須進行敏捷豁免，失敗時受到 10d8 閃電（Lightning）傷害，成功則傷害減半。

**升環：** 每高一環，額外增加 1 道彈射閃電至另一個目標。

#### 解離術（Disintegrate）

| 項目 | 內容 |
|------|------|
| 環級 | 6 環 |
| 學派 | 變化（Transmutation） |
| 施法時間 | 1 動作 |
| 射程 | 18m |
| 成分 | V, S, M（磁石粉末和一撮塵土） |
| 持續時間 | 瞬間 |

**效果：** 你向射程內一個你能看見的目標發射一道綠色射線。目標可以是生物、非魔法物品或魔法力場的造物（如力場之牆 Wall of Force 所造的牆）。被指定的生物須進行敏捷豁免，失敗時受到 10d6+40 力場（Force）傷害。若此傷害使其降至 0 HP，目標連同其非魔法裝備一起化為灰塵，只能透過真實復活術（True Resurrection）或祈願術（Wish）復活。此法術自動瓦解一個大型以下的非魔法物品或魔法力場造物；若目標為超大型以上，則瓦解其 3m方格部分。

**升環：** 每高一環，傷害增加 10d6+40。

#### 禁制術（Forbiddance）

| 項目 | 內容 |
|------|------|
| 環級 | 6 環 |
| 學派 | 防護（Abjuration） |
| 施法時間 | 10 分鐘（R） |
| 射程 | 觸碰 |
| 成分 | V, S, M（價值至少 1,000 gp 的紅寶石粉末） |
| 持續時間 | 1 天 |

**效果：** 你為一個區域（最多 3,600 平方m地板面積，高度最多 9m）設置抵禦魔法旅行的結界。在持續期間，生物無法傳送進入該區域，也無法使用傳送門（如法術門 Gate 創造的傳送門）進入。法術同時阻止經由星界位面、靈界位面、妖精荒野、影暗荒域或位面轉移（Plane Shift）進入。此外，你可選擇一種或多種生物類型（異常體、天界、元素、妖精、邪魔或不死），被選類型的生物首次在一回合中進入區域或在其中結束回合時，受到 5d10 光輝或黯蝕傷害（施法時選擇）。你可設定密語，說出密語進入的生物不受傷害。連續 30 天在同一位置施展則效果永久化（直到被解除），最後一次施展時材料被消耗。

#### 法術無效結界（Globe of Invulnerability）

| 項目 | 內容 |
|------|------|
| 環級 | 6 環 |
| 學派 | 防護（Abjuration） |
| 施法時間 | 1 動作 |
| 射程 | 自身 |
| 成分 | V, S, M（一顆玻璃珠） |
| 持續時間 | C 1 分鐘 |

**效果：** 你周身出現一個不動的半透明屏障，形成 3m的散發區域（Emanation），持續期間存在。任何從屏障外施展的 5 環以下法術無法影響屏障內的任何事物。此類法術可以指定屏障內的生物和物品為目標，但對它們沒有效果。同樣地，屏障內的區域不受此類法術的效果區域影響。

**升環：** 每高一環，可阻擋的法術環數增加 1。

#### 移土術（Move Earth）

| 學派 | 塑能（Evocation） |

| 射程 | 36m |

# 法術詳情 — 1 環（1st Level）

> 中央法術庫。各職業共用，按環級分檔。

#### 阿加西斯之甲（Armor of Agathys）

| 項目 | 內容 |
|------|------|
| 環級 | 1 環 |
| 學派 | 防護（Abjuration） |
| 施法時間 | 1 動作 |
| 射程 | 自身 |
| 成分 | V, S, M（一杯水） |
| 持續時間 | 1 小時 |

**效果：** 你獲得 5 點暫時生命值。在法術持續期間，若你仍擁有此法術的暫時生命值，當一個生物以近戰攻擊命中你時，該生物受到 5 點寒冷傷害。

**升環：** 每高一環，暫時生命值和寒冷傷害各增加 5。

#### 冷凍之觸（Chill Touch）

| 項目 | 內容 |
|------|------|
| 環級 | 戲法 |
| 學派 | 死靈（Necromancy） |
| 施法時間 | 1 動作 |
| 射程 | 觸碰 |
| 成分 | V, S |
| 持續時間 | 瞬間 |

**效果：** 對一個你能觸及的生物進行近戰法術攻擊。命中時造成 1d10 黯蝕傷害，且目標在你下一回合開始前無法恢復生命值。若目標為不死生物，還會在同一期間內對你的攻擊擲骰具有劣勢。

**升級：** 5 級時 2d10、11 級時 3d10、17 級時 4d10。

#### 奇蹟術（Thaumaturgy）

| 項目 | 內容 |
|------|------|
| 環級 | 戲法 |
| 學派 | 變化（Transmutation） |
| 施法時間 | 1 動作 |
| 射程 | 9m |
| 成分 | V |
| 持續時間 | 最多 1 分鐘 |

**效果：** 你在射程內顯現一個輕微的神蹟。多次施放時最多可同時維持 3 個 1 分鐘效果。從以下選擇其一：**變目（Altered Eyes）：** 在 1 分鐘內改變你雙眼的外觀。**震響之聲（Booming Voice）：** 你的聲音在 1 分鐘內最多放大至三倍音量，在持續時間內你的魅力（威嚇）檢定具有優勢。**火焰戲法（Fire Play）：** 使火焰在 1 分鐘內搖曳、變亮、變暗或改變顏色。**隱形之手（Invisible Hand）：** 瞬間使一扇未上鎖的門窗彈開或猛然關上。**幻音（Phantom Sound）：** 在射程內你選擇的一點製造一個瞬間的聲響，如雷鳴、烏鴉叫聲或不祥的低語。**顫動（Tremors）：** 使地面產生 1 分鐘無害的輕微震顫。

**升級：** 無升級效果。

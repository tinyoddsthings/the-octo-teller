# 法術詳情 — 2 環（2nd Level）

> 中央法術庫。各職業共用，按環級分檔。

#### 匕雲術（Cloud of Daggers）

| 項目 | 內容 |
|------|------|
| 環級 | 2 環 |
| 學派 | 咒法（Conjuration） |
| 施法時間 | 1 動作 |
| 射程 | 18m |
| 成分 | V, S, M（一片玻璃碎片） |
| 持續時間 | C 1 分鐘 |

**效果：** 你在射程內一點周圍 1.5m立方範圍內充滿旋轉的匕首。一個生物在首次進入此區域或在其中開始回合時，受到 4d4 揮砍傷害。

**升環：** 每高一環，傷害增加 2d4。

| 施法時間 | 1 動作 |
| 射程 | 36m |
| 成分 | V, S |
| 持續時間 | C 1 分鐘 |

**效果：** 射程內一個生物須進行感知豁免。失敗時被魅惑，頭上出現一頂扭曲的鐵冠。被魅惑的生物在每個回合開始前，你必須用附贈動作指定一個目標——該生物須用近戰攻擊攻擊指定目標（若近戰範圍內無指定目標，則可自由行動）。目標在每回合結束時可重新進行感知豁免，成功則法術結束。

## 1. 武器（Weapons）

### 武器屬性說明

| 屬性 | 英文 | 效果 |
|------|------|------|
| 靈巧 | Finesse | 攻擊/傷害可用力量（STR）或敏捷（DEX），由玩家選 |
| 輕型 | Light | 可用於雙武器戰鬥（附贈動作攻擊第二把輕型武器） |
| 重型 | Heavy | 小型（Small）生物使用時攻擊劣勢 |
| 雙手 | Two-Handed | 必須雙手持用 |
| 多用 | Versatile | 單手/雙手皆可，雙手傷害較高（括號內數值） |
| 投擲 | Thrown | 可投擲進行遠程攻擊，用力量（STR）修正（靈巧武器也可用 DEX） |
| 觸及 | Reach | 觸及範圍增加 1.5m，變為 3m |
| 裝填 | Loading | 每次攻擊動作只能射 1 發（即使有額外攻擊） |
| 彈藥 | Ammunition | 需要彈藥，用敏捷（DEX）修正。射程格式：正常/劣勢 |
| 特殊 | Special | 有特殊規則（見備註） |

### 武器精通屬性（Weapon Mastery Properties）

> 來源：2024 PHB p.213。每種武器都有一個精通屬性，但角色必須擁有「武器精通（Weapon Mastery）」職業特性才能啟用。

| 精通屬性 | 效果 |
|---------|------|
| **劈斬（Cleave）** | 命中後可對第一目標 1.5m內的第二個生物進行近戰攻擊，命中造成武器傷害但不加能力修正（負數仍算）。每回合 1 次 |
| **擦傷（Graze）** | 攻擊骰未命中時，仍造成等於攻擊能力修正的傷害（同武器傷害類型） |
| **快刺（Nick）** | 輕型武器的額外攻擊改為攻擊動作的一部分（而非附贈動作）。每回合 1 次 |
| **推擊（Push）** | 命中大型或更小的目標時，可將其直線推離你至多 3m |
| **削弱（Sap）** | 命中時，目標在你下回合開始前的下一次攻擊骰有**劣勢** |
| **減速（Slow）** | 命中並造成傷害時，目標速度在你下回合開始前減少 3m。不疊加 |
| **絆倒（Topple）** | 命中時可強迫目標進行體魄豁免（DC = 8 + 攻擊能力修正 + 熟練加值），失敗則**俯臥** |
| **迫擊（Vex）** | 命中並造成傷害時，你對該目標的下一次攻擊骰有**優勢**（你下回合結束前） |

**擁有武器精通的職業：**
- 野蠻人（Barbarian）— Lv.1 起，2 種近戰武器（限 Melee）
- 戰士（Fighter）— Lv.1 起，3 種武器
- 聖騎士（Paladin）— Lv.1 起，2 種武器
- 遊俠（Ranger）— Lv.1 起，2 種武器
- 盜賊（Rogue）— Lv.1 起，2 種武器

### 特殊武器規則

- **騎槍（Lance）：** 1.5m 內攻擊劣勢。非騎乘時需雙手。騎乘時可單手使用
- **網（Net）：** 命中時大型以下目標被束縛（Restrained）。力量（STR）DC 10 或揮砍 AC 10 可掙脫。每次只能攻擊 1 個目標

### 武器選擇速查

| 定位 | 推薦武器 | 原因 |
|------|----------|------|
| STR 近戰（單手+盾） | 長劍 / 戰斧 / 戰錘 | 1d8 多用，持盾 AC +2 |
| STR 近戰（雙手爆發） | 巨劍（2d6）/ 巨斧（1d12） | 最高近戰傷害。巨劍平均高，巨斧暴擊爽 |
| DEX 近戰 | 刺劍（1d8）| DEX 近戰最高傷害 |
| 雙武器 | 短劍 + 短劍 / 彎刀 | 靈巧 + 輕型，附贈打第二下 |
| 遠程（DEX） | 長弓（1d8・180m）| 傷害和射程最佳 |
| 簡易遠程 | 輕弩（1d8） | 不需軍用武器熟練 |

## 2. 護甲（Armor）

### 防禦等級（AC）計算

| 類別 | AC 公式 | 敏捷（DEX）加值 |
|------|---------|----------------|
| 無甲 | 10 + DEX 修正 | 無上限 |
| 輕甲 | 護甲基礎 + DEX 修正 | 無上限 |
| 中甲 | 護甲基礎 + DEX 修正（最高 +2） | 上限 +2 |
| 重甲 | 護甲基礎（固定值） | 不加 DEX |
| 盾牌 | 任何護甲 AC **+2** | — |

| 裝備 | AC 加值 | 價格 | 重量 | 備註 |

### 護甲選擇速查

| DEX 修正 | 最佳選擇 | 最高 AC |
|----------|----------|---------|
| +0 或更低 | 重甲（全身鎧） | 18（+盾 = 20） |
| +1 | 重甲（鍛鍊甲/全身鎧） | 17-18（+盾 = 19-20） |
| +2 | 中甲（半身鎧）或重甲 | 17-18（+盾 = 19-20） |
| +3 | 中甲（半身鎧）= 輕甲（鑲釘皮甲） | 17 = 15（中甲贏） |
| +4 | 輕甲（鑲釘皮甲）| 16（+盾 = 18） |
| +5 | 輕甲（鑲釘皮甲）| 17（+盾 = 19） |

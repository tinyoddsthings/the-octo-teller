# 創角問卷：現實→D&D 角色塑造（生成指南）

## 使用說明

- 本指南用於創角步驟 11b，DM 根據下方主題清單**即時生成**題目和選項
- **不存固定題目**——每次創角 DM 重新出題，保持新鮮感
- 一次問一題，等玩家回答後再問下一題
- 每題 6 個選項：1-4 為 DM 生成的預設答案，5 = 跳過，6 = 自己描述
- 跳過的題目 DM 根據已答題目 + 角色卡既有資訊（種族/職業/背景）自動生成，**不告知玩家影響**
- 問卷不出現 D&D 術語，全用日常情境提問
- 主題名稱和情境方向是 DM 內部參考，**不給玩家看**
- 若玩家在 11a 選「有人設 → 引導問題」，用相同框架但把「你」改成「你的角色」

---

## 出題規則

1. **題目格式**：DM 根據每個主題的「情境方向」自由生成一道情境題，題目描述一個具體的日常場景
2. **選項要求**：
   - 1-4 為 DM 生成的選項，必須有明顯差異性（不能都偏同一方向）
   - 5 = 跳過（固定）
   - 6 = 自己描述（固定）
3. **情境原則**：用日常生活場景（購物、通勤、聚餐、工作等），禁止出現 D&D 術語或奇幻設定
4. **節奏**：一次一題，等回答再問下一題
5. **題數**：全選預設答案（1-4）時每個主題一題；善用自述（選項 6）可大幅減少題數。DM 可視情況合併相近主題或微調數量，不硬性規定
6. **引導模式**（有人設）：把「你」改成「你的角色」，情境改為角色可能遇到的日常場景
7. **自述跳題**：玩家選「自己描述」（選項 6）時，DM 分析回答內容，判斷涵蓋了哪些主題，將已涵蓋的主題標記為「已收集」，後續跳過不再出題。例：玩家在「整體氣質」題自述了外觀全貌 → DM 可同時標記「體型」「穿衣風格」「標誌特徵」「髮型」為已收集，直接跳到下一個未收集的主題區塊
8. **進度顯示**：每題回答後顯示「`（了解進度：X%）`」。計算方式 = 已收集主題數 / 主題清單總數 × 100%。預設答案（1-4）= +1 主題；跳過（5）= +0%（已處理但不計入進度，DM 稍後自動推測填入）；自述（6）= +N 主題（DM 判斷涵蓋數）。進度達 100% 時所有主題已收集，DM 主動結束問卷不再出題
9. **開場提示**：DM 在問第一題之前，先向玩家說明問卷機制，大意如下（DM 可用自己的語氣改寫）：
   > 接下來我會問你 22 個面向的問題，幫我了解你這個人。每題有 4 個選項可以選，也可以跳過或自己描述。如果你在某一題的自述涵蓋了多個面向——比如一口氣描述了你的整體外觀——我就會把相關的面向一起算進去，後面那些題就不用再問了。所以善用自述的話，題數會少很多。

## DM 轉換映射表

| 角色卡欄位 | 主題來源 | 轉換邏輯 |
|-----------|---------|---------|
| **外觀** | 整體氣質、體型、穿衣風格、標誌特徵、髮型 | 保留體型/氣質/風格，加入種族特徵（如提夫林：角、尾巴、異色膚/瞳） |
| **性格** | 日常節奏、社交傾向、決策模式、紀律感、衝突應對、團隊角色 | 綜合日常節奏 + 社交 + 決策 + 衝突反應，寫 2~3 句性格描述 |
| **理念** | 秩序觀、核心價值 | 規則態度 = 秩序軸，改變世界 = 善惡軸，合成 1 句核心信念 |
| **關係** | 待人態度、介入意願、核心連結 | 待人方式 + 重視的連結 → 1 句對角色最重要的人/事/信條 |
| **缺點** | 出包模式、壓力反應、恐懼核心 | 出包模式 + 壓力反應 + 恐懼 → 1 句 D&D 化的角色缺陷 |
| **陣營** | 秩序觀、核心價值、待人態度、介入意願 | 秩序觀 = 秩序↔混沌軸，核心價值 + 待人 + 介入意願 = 善↔惡軸，合成九宮格陣營 |
| **其他特徵** | 日常怪癖、隱藏技能 + 自由描述中提煉 | 怪癖 + 隱藏技能 → D&D 化的角色小特質，寫 1~3 條 |
| **背景故事** | 核心追求 + 全部已答主題 + 角色卡全部已填欄位（種族、職業、起源、語言、專長、法術、魔寵、裝備等） | 綜合角色卡所有已確定設定 + 問卷結果，串聯成 3~5 句：「現實→D&D世界的對應經歷→踏上冒險的理由」 |

### 轉換原則

- 轉換結果用角色卡的中文格式書寫
- 種族特徵必須融入外觀描述（不能寫出現實人類外觀）
- 職業/背景已有的設定（如侍祭背景的信仰經歷）要與問卷結果整合，不能衝突
- 背景故事要銜接已選背景的官方敘述，問卷提供的是「為什麼會走上這條路」的動機
- **背景故事必須整合角色卡全部已填欄位**（種族特性、職業能力、起源背景、語言、專長、法術選擇、魔寵、裝備等），不能只靠問卷結果獨立生成
- 跳過的題目：DM 根據已答題目的整體傾向 + 種族/職業/背景的刻板印象，自動補完（可偏離刻板印象以增加角色深度）

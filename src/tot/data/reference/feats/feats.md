# 專長速查表（Feats Reference）

> 資料來源：2024 PHB。四大分類：起源 → 通用 → 戰鬥風格 → 史詩祝福。
> 起源專長完整內容見 `reference/creation/options.md`（已 PDF 校對）。
> ⚠️ 通用/戰鬥風格/史詩專長以 2024 PHB 知識為基礎，詳細效果請以 PDF 為準。

---

## 專長取得時機

| 等級 | 事件 | 可選類別 |
|------|------|----------|
| Lv.1 | 背景提供 | 起源專長（Origin Feat） |
| Lv.4 | ASI / 專長 | 通用專長（General Feat）或戰鬥風格專長（Fighting Style Feat） |
| Lv.8 | ASI / 專長 | 同上 |
| Lv.12 | ASI / 專長 | 同上 |
| Lv.16 | ASI / 專長 | 同上 |
| Lv.19 | ASI / 專長 | 史詩祝福專長（Epic Boon Feat） |

> **ASI 二選一：** 每次 ASI 時可選「+2/+1 屬性提升」或「1 個專長」。2024 PHB 多數通用專長附帶 +1 屬性，減緩二選一的痛感。

## 3. 戰鬥風格專長（Fighting Style Feats）— Lv.4+

> 戰士 Lv.1、聖騎士 Lv.2、遊俠 Lv.2 免費獲得 1 個戰鬥風格。
> 其他職業（或想要第二個風格）可用 Lv.4+ ASI 選取。

| 專長 | 效果 | 適合搭配 |
|------|------|----------|
| 射術（Archery） | 遠程武器攻擊 +2 命中 | 遊俠、戰士（弓手） |
| 盲戰（Blind Fighting） | 3m 盲視（Blindsight） | 近戰全職業 |
| 防禦（Defense） | 穿甲時 AC +1 | 所有穿甲職業 |
| 決鬥（Dueling） | 單手近戰武器（另一手空或持盾）傷害 +2 | 戰士、聖騎士（劍盾流） |
| 大武器戰鬥（Great Weapon Fighting） | 雙手武器傷害骰 1/2 可重骰 | 戰士、聖騎士（雙手流） |
| 攔截（Interception） | 反應：鄰近友軍受攻擊時減傷 1d10 + 熟練加值 | 戰士、聖騎士（坦克） |
| 防護（Protection） | 反應（需持盾）：鄰近友軍被攻擊時給攻擊者劣勢 | 戰士（盾坦） |
| 投擲武器戰鬥（Thrown Weapon Fighting） | 投擲武器傷害 +2 | 戰士、遊俠 |
| 雙武器戰鬥（Two-Weapon Fighting） | 附贈攻擊加屬性修正到傷害 | 遊俠、戰士（雙持流） |
| 徒手戰鬥（Unarmed Fighting） | 徒手傷害 1d6（雙手空 = 1d8）+ 擒抱時每回合 1d4 | 武僧、戰士 |

## 4. 史詩祝福專長（Epic Boon Feats）— Lv.19+

> Lv.19 的 ASI 只能選史詩祝福。這是角色的封頂獎勵。
> 所有史詩祝福給予 +1 屬性加值（任選一項，可超過 20，上限 30）。

| 專長 | 核心效果 |
|------|----------|
| 戰鬥英才祝福（Boon of Combat Prowess） | 攻擊失手可改為命中，熟練加值次/長休 |
| 次元旅行祝福（Boon of Dimensional Travel） | 附贈動作瞬間傳送 9m + 鄰近生物可帶走 |
| 能量抗性祝福（Boon of Energy Resistance） | 選 2 種元素傷害：獲得抗性 + 反應將抗性暫時給盟友 |
| 命運祝福（Boon of Fate） | 反應：目標 d20 擲骰 +/− 2d4 |
| 強韌祝福（Boon of Fortitude） | HP 上限 +40 |
| 不可抵擋攻勢祝福（Boon of Irresistible Offense） | 攻擊忽略抗性 + 暴擊額外 2d6 力場傷害 |
| 恢復祝福（Boon of Recovery） | 附贈動作恢復等級一半 HP，1 次/長休 + 死亡豁免失敗不累積 |
| 技巧祝福（Boon of Skill） | 所有技能 +5（非熟練也加）❓ |
| 迅捷祝福（Boon of Speed） | 速度 +9m + 借機攻擊打不到你有劣勢 |
| 夜之靈祝福（Boon of the Night Spirit） | 微光中隱形 + 暗視 36m + 魔法黑暗中看得見 |
| 真視祝福（Boon of Truesight） | 真視（Truesight）36m |

## 熱門專長組合建議

### Lv.4 最搶手專長（依角色類型）

| 角色定位 | 首推專長 | 原因 |
|----------|----------|------|
| 專注施法者（魔導士/牧師/德魯伊） | **戰爭施法者** | 專注豁免優勢是遊戲改變者 |
| 近戰 STR（戰士/野蠻人） | **大武器大師** 或 **哨兵** | GWM 爆發、哨兵控場 |
| 近戰 DEX（盜賊/遊俠） | **穿刺者** 或 **哨兵** | 暴擊加傷 / 鎖人 |
| 弓手（遊俠/戰士） | **神射手** | 忽略掩蔽 + 傷害加熟練加值 |
| 長柄武器 | **長柄武器大師 + 哨兵** | 經典組合：進入 3m 就借機攻擊 + 命中速度歸 0 |
| 想兼顧屬性提升 | **韌性（CON）** | +1 CON + CON 豁免熟練 = 專注不易斷 |

# 規則速查表（Rules Quick Reference）

> 獨立速查用。戰鬥中的完整流程見 `combat_flow.md`。
> ⚠️ 資料來源：原始資料來自 D&D 5e SRD（2014），SRD 已棄用。待以 dm20（2024 PHB PDF）重新驗證。

---

## 1. 動作經濟（Action Economy）

每個角色的回合可以做：

| 類型 | 每回合次數 | 說明 |
|------|-----------|------|
| **移動（Movement）** | 1 次（可拆分） | 最多移動等於你的速度。可在動作前後拆分 |
| **動作（Action）** | 1 次 | 攻擊、施法、衝刺、脫離、躲避、躲藏、幫助、準備、使用物品 |
| **附贈動作（Bonus Action）** | 1 次 | 只有特定能力/法術才能使用。沒有 = 沒有 |
| **反應（Reaction）** | 每輪 1 次 | 在他人回合觸發。下回合開始恢復 |
| **自由互動（Free Interaction）** | 1 次 | 拔劍、開門、撿東西等小動作 |

### 常見動作一覽

| 動作 | 效果 |
|------|------|
| 攻擊（Attack） | 武器攻擊或徒手攻擊 |
| 施法（Cast a Spell） | 施放時間為「1 動作」的法術 |
| 衝刺（Dash） | 本回合額外獲得等於速度的移動距離 |
| 脫離（Disengage） | 本回合移動不觸發借機攻擊（Opportunity Attack） |
| 躲避（Dodge） | 對你的攻擊劣勢，你的敏捷（DEX）豁免優勢（到下回合開始） |
| 躲藏（Hide） | 隱匿檢定。成功後攻擊有優勢 |
| 幫助（Help） | 給友軍下次屬性檢定或攻擊擲骰優勢 |
| 準備（Ready） | 設觸發條件 + 用反應執行動作。準備法術需維持專注 |
| 使用物品（Use an Object） | 使用道具（治療藥水、卷軸等） |

### 附贈動作法術限制

> 同一回合施放了附贈動作法術 → 動作只能施放**戲法**（不能施放 1 環以上法術）。

## 2. 移動規則（Movement）

### 基礎移動

- 速度：大多數中型種族 = 9 公尺
- 可在動作前後自由拆分（例：移動 3m → 攻擊 → 移動 6m）
- 不同移動方式可在同回合混用，各自扣除速度

### 特殊移動

| 類型 | 規則 |
|------|------|
| **困難地形（Difficult Terrain）** | 每 1 公尺消耗 2 公尺速度（半速） |
| **攀爬（Climbing）** | 無攀爬速度時，每 1 公尺消耗 2 公尺速度。陡峭表面可能需運動（Athletics）檢定 |
| **游泳（Swimming）** | 無游泳速度時，每 1 公尺消耗 2 公尺速度。逆流/惡劣水域可能需運動（Athletics）檢定 |
| **爬行（Crawling）** | 每 1 公尺消耗 2 公尺速度（倒地時唯一移動方式） |
| **飛行（Flying）** | 有飛行速度的生物正常飛行。速度歸 0 時墜落（除非能懸停 Hover） |
| **擠過（Squeezing）** | 體型大一級的空間可擠過：半速 + 攻擊/敏捷豁免劣勢 + 攻擊你有優勢 |

### 站起（倒地 → 站立）

- 消耗一半速度。速度 0 時無法站起。

### 跳躍

| 類型 | 距離 |
|------|------|
| **立定跳遠** | 力量（STR）值 ÷ 2 × 0.3m（跑步助跑 = 力量值 × 0.3m） |
| **跳高** | （3 + 力量（STR）修正值）× 0.3m（助跑）；立定 = 一半 |

> 跳躍距離不能超過剩餘移動速度。

## 3. 掩蔽規則（Cover）

| 類型 | 防禦等級（AC）/敏捷（DEX）豁免加值 | 範例 |
|------|------------------------------|------|
| **半掩蔽（Half Cover）** | +2 | 矮牆、大型傢俱、另一生物 |
| **四分之三掩蔽（Three-Quarters Cover）** | +5 | 箭孔、厚樹幹 |
| **全掩蔽（Total Cover）** | 無法被直接瞄準 | 完全擋住的牆壁 |

> 掩蔽同時適用於防禦等級（AC）和敏捷（DEX）豁免（如火球術的範圍效果）。

## 4. 休息規則（Resting）

### 短休（Short Rest）— 1 小時

| 恢復項目 | 規則 |
|----------|------|
| 生命值（HP） | 消耗生命骰（Hit Dice），每顆 = 生命骰 + 體魄（CON）修正 |
| 生命骰 | 不恢復（長休才恢復一半） |
| 職業能力 | 部分能力短休恢復（見下表） |
| 法術位 | 一般施法者不恢復。**例外：魔導士（Warlock）全部恢復** |

**短休恢復的常見能力：**

| 職業 | 短休恢復 |
|------|----------|
| 魔導士（Warlock） | 全部法術位（契約魔法） |
| 戰士（Fighter） | 動作如潮（Action Surge）、二次振作（Second Wind） |
| 牧師（Cleric） | 引導神力（Channel Divinity） |
| 武僧（Monk） | 氣（Ki）點數 |
| 吟遊詩人（Bard） | 激勵骰（Bardic Inspiration）（Lv.5+） |
| 德魯伊（Druid） | 野性形態（Wild Shape）使用次數 |

### 長休（Long Rest）— 8 小時（至少 6 小時睡眠 + 2 小時輕度活動）

| 恢復項目 | 規則 |
|----------|------|
| 生命值（HP） | 全部恢復 |
| 生命骰 | 恢復一半（最少 1 顆），向下取整 |
| 法術位 | 全部恢復 |
| 職業能力 | 大多數長休恢復的能力全部回來 |
| 力竭（Exhaustion） | 降 1 級（需進食飲水） |

> **24 小時限制：** 兩次長休之間至少間隔 24 小時。

## 5. 專注規則（Concentration）

### 核心規則

- 同時只能維持 **1 個**專注法術
- 施放新的專注法術 → 舊的自動結束
- 主動取消專注不需動作（免費動作）

### 專注檢定（受傷時）

- **觸發：** 專注中的施法者受到傷害
- **豁免：** 體魄（CON）豁免
- **難度（DC）= max(10, 傷害值 ÷ 2)**
- 失敗 → 法術立即結束

| 受到傷害 | 專注 DC |
|----------|---------|
| 1-21 | 10 |
| 22 | 11 |
| 30 | 15 |
| 40 | 20 |

### 其他中斷情況

- 失能（Incapacitated）或死亡 → 自動失去專注
- DM 裁定的極端環境干擾（劇烈晃動等）

> **實戰提示：** 戰爭施法者（War Caster）專長讓專注豁免有優勢，是專注施法者的核心專長。

## 6. 死亡豁免（Death Saving Throws）

### 觸發

角色生命值（HP）降至 0（傷害未超過生命值上限的負值，否則即死）

### 流程

1. 倒地（Prone）+ 昏迷（Unconscious）
2. 每回合開始擲 **d20**（無修正值）

| 擲骰結果 | 效果 |
|----------|------|
| 10+ | 成功 1 次 |
| 2-9 | 失敗 1 次 |
| 自然 1 | **2 次失敗** |
| 自然 20 | 恢復 1 HP，**立即起身行動** |

- **成功 3 次** → 穩定（HP 0，不再骰，1d4 小時後恢復 1 HP）
- **失敗 3 次** → 角色死亡

### 昏迷中受傷

- 被攻擊命中 → 自動 1 次失敗
- 1.5 公尺內被命中 → **自動暴擊** → **2 次失敗**
- 受到傷害 ≥ HP 上限 → 即死

### 救援優先序

| 方法 | 類型 | 效果 |
|------|------|------|
| 治療之言（Healing Word） | 附贈動作・遠程 | 恢復 HP，最優先 |
| 療傷術（Cure Wounds） | 動作・觸碰 | 恢復 HP |
| 慈悲之手（Spare the Dying） | 動作・觸碰戲法 | 穩定但不恢復 HP |
| 治療藥水（他人灌） | 動作 | 恢復 HP |
| 醫療檢定 DC 10 | 動作 | 穩定 |

## 7. 視野與照明（Vision & Lighting）

### 三種照明等級

| 等級 | 效果 |
|------|------|
| **明亮光線（Bright Light）** | 正常視物 |
| **微光（Dim Light）** | 輕度遮蔽（Lightly Obscured）→ 感知（Perception）檢定劣勢 |
| **黑暗（Darkness）** | 重度遮蔽（Heavily Obscured）→ 等同目盲（Blinded） |

### 遮蔽等級

| 類型 | 效果 |
|------|------|
| **輕度遮蔽（Lightly Obscured）** | 依賴視覺的感知（WIS）檢定劣勢。如：微光、薄霧、中等植被 |
| **重度遮蔽（Heavily Obscured）** | 等同目盲：攻擊劣勢、被打優勢、視覺檢定自動失敗。如：黑暗、濃霧、茂密植被 |

### 暗視（Darkvision）

- 有暗視的生物在指定距離內：黑暗視為微光、微光視為明亮
- **重要：暗視不能把黑暗變成明亮光線。** 在黑暗中仍算微光（感知檢定仍有劣勢）
- 暗視範圍：大多數種族 18 公尺

| 有暗視的種族 | 範圍 |
|-------------|------|
| 矮人（Dwarf） | 18m |
| 精靈（Elf） | 18m |
| 侏儒（Gnome） | 18m |
| 半精靈（Half-Elf） | 18m |
| 半獸人（Half-Orc） | 18m |
| 提夫林（Tiefling） | 18m |

| 無暗視的種族 | 備註 |
|-------------|------|
| 人類（Human） | 需要火把/光源 |
| 半身人（Halfling） | 需要火把/光源 |
| 龍裔（Dragonborn） | 需要火把/光源 |

### 光源

| 光源 | 明亮範圍 | 微光範圍 | 持續時間 |
|------|----------|----------|----------|
| 蠟燭（Candle） | 1.5m | 額外 1.5m | 1 小時 |
| 火把（Torch） | 6m | 額外 6m | 1 小時 |
| 提燈・罩式（Hooded Lantern） | 9m | 額外 9m | 6 小時 |
| 提燈・牛眼（Bullseye Lantern） | 18m 錐形 | 額外 18m | 6 小時 |
| 光亮術（Light）戲法 | 6m | 額外 6m | 1 小時 |
| 舞光術（Dancing Lights）戲法 | 3m×4 | 額外 3m | 1 分鐘（專注） |

> **戰術提示：** 火把/光源讓持有者在黑暗中被遠處敵人看到，但自己只能看到光源範圍內。無暗視的角色在地下城中是戰術弱點。

## 8. 倒地 / 借機攻擊 / 夾擊

### 倒地（Prone）

| 規則 | 說明 |
|------|------|
| 移動 | 只能爬行（1m 消耗 2m 速度） |
| 站起 | 消耗一半速度。速度 0 無法站起 |
| 自己攻擊 | 劣勢 |
| 被近戰攻擊 | 1.5m 內攻擊者有**優勢** |
| 被遠程攻擊 | 攻擊者有**劣勢** |

> **戰術：** 把敵人打倒地後讓近戰角色集火（優勢），但遠程角色反而打不準。

### 借機攻擊（Opportunity Attack）

**觸發：** 敵對生物**離開**你的觸及範圍（通常 1.5m）

| 規則 | 說明 |
|------|------|
| 使用 | 消耗反應（Reaction） |
| 攻擊次數 | 1 次近戰攻擊（不是整個攻擊動作） |
| 不觸發的情況 | 脫離（Disengage）後移動、傳送、被推/拉/擊退移動 |

> **注意：** 借機攻擊是在「離開」觸及範圍時觸發，不是「經過」。在觸及範圍內移動不會觸發。

### 夾擊（Flanking）— 選用規則（Optional）

> ⚠️ 夾擊是 DMG 的選用規則（Optional Rule），非核心規則。使用前需 DM 確認。

**條件：** 兩個友方生物分別站在敵人的**正對面**兩側（穿過敵人中心畫直線）

**效果：** 兩個夾擊者對該敵人的近戰攻擊獲得**優勢**

**注意事項：**
- 只適用近戰攻擊，不適用遠程
- 需要實際站在對角位置，不是只要兩面夾攻
- 啟用夾擊會降低優勢的稀缺性，讓近戰角色更強、讓「幫助」動作貶值

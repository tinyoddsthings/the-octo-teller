# 戰鬥標準流程（Combat SOP）

> 資料來源：2024 PHB。DM 每次進入戰鬥時照此流程走。
> dm20 工具：`start_combat`、`next_turn`、`combat_action`、`roll_dice`、`end_combat`

---

## A. 戰鬥開始

### 1. 突襲判定（Surprise）

| 情境 | 處理 |
|------|------|
| 我方伏擊 | 伏擊方隱匿檢定 vs 敵方被動感知。失敗者第一輪無法使用反應（Reaction）|
| 敵方伏擊 | 敵方隱匿 vs 我方被動感知。被突襲者第一輪無反應 |
| 雙方察覺 | 無突襲，正常先攻 |

**被動感知速查：** ToT 11 ｜ 岡納 11 ｜ 莉拉 14 ｜ 乃克 **16**（專精）

### 2. 擲先攻（Roll Initiative）

- 全員：d20 + 敏捷（DEX）修正
- **同值排序：** 玩家 > 怪物 → 比敏捷（DEX）修正 → 仍同則 DM 裁定
- **先攻修正速查：** ToT +2 ｜ 岡納 +0 ｜ 莉拉 +1 ｜ 乃克 **+4**

### 3. 先攻序表（每回合印出 & 更新）

```

│ 序 │   名稱   │ 先攻 │ AC  │   HP    │ 狀態             │
├────┼──────────┼──────┼─────┼─────────┼──────────────────┤
│ 1  │ 乃克     │  22  │ 16  │  25/25  │ —                │
```

- 怪物生命值（HP）數字不對玩家顯示，用描述代替（無傷/輕傷/重傷/瀕死）
- 同類怪物用 A/B/C 區分（如哥布林A、哥布林B）
- 哭哭鱆與 ToT 同先攻，緊接在 ToT 後行動

## B. 回合結構

每位角色的回合按以下順序處理：

```
回合開始
 ├─ 1. 回合開始觸發效果（持續傷害、地形、光環）
 ├─ 2. 移動（可拆分到行動前後）
 ├─ 3. 動作（Action）— 選一
 ├─ 4. 附贈動作（Bonus Action）— 如有
 ├─ 5. 回合結束觸發（豁免重擲、效果倒數、專注持續）
 └─ → 下一位
```

**反應（Reaction）** 不在自己回合，在觸發時使用（每輪 1 次）

### DM 每回合 checklist

- [ ] 有回合開始效果嗎？（毒素 tick、灼燒、恐懼區域、脆弱詛咒額外傷害）
- [ ] 角色有專注（Concentration）中的法術嗎？→ 面板標示
- [ ] 玩家回合 → 印面板 → 問「你要怎麼做？」
- [ ] NPC 回合 → 背景跑決策管線（D-1）→ 宣布意圖 → 玩家確認 → 執行
- [ ] 怪物回合 → 描述行動（參考 F 節戰術提示）
- [ ] 受傷時：專注檢定？死亡豁免？更新生命值（HP）
- [ ] 回合結束效果？持續法術 −1 回合

## C. 玩家回合面板（固定格式）

**規則：**
- 輪到玩家角色行動前，**必須**先印出面板
- NPC 回合（岡納/莉拉/乃克由 DM 控制時）：背景跑決策管線（見 D 節）→ 宣布意圖
- 面板從角色卡 + campaign.md 即時面板動態生成
- 動作帶**編號**，讓玩家直接喊號選擇
- 不可用的選項標 ❌ 但仍列出（讓玩家知道已用過什麼）

### 面板分區子表結構

**第一段：狀態表**

| ❤️ 生命值（HP）| 暫時生命值（HP）| 🛡️ 防禦等級（AC）| 速度 | 🔮 法術欄位 | 📌 狀態 |
|----------------|-----------------|-------------------|------|------------|---------|
| {目前}/{最大} | {暫時} | {防禦等級} | {速度} | {目前}/{最大}（{環級}環）| {狀態效果} |

- 無法術的職業省略「法術欄位」欄
- 有職業特殊資源的可加欄（如魔導士 ⚡ 魔法靈巧）
- 狀態欄列出：專注法術、增益/減益、環境效果

**第二段：動作表（分區子表）**

統一編號橫跨所有子表，玩家直接喊號選擇。
每區只顯示該類型需要的欄位，避免空白。無該類型動作的職業直接跳過該子表。

**⚔️ 攻擊/戲法區（無限）**

| # | 名稱 | 命中/難度（DC）| 傷害/效果 | 備註 |
|---|------|---------------|----------|------|

> 武器攻擊和攻擊戲法排前，輔助戲法排後。
> 無消耗、無次數限制 → 不需欄位/次數/專注欄。

**🔮 法術區（消耗欄位）** — 欄位：{目前}/{最大}（{X}環）

| # | 名稱 | 命中/難度（DC）| 傷害/效果 | 消耗欄位 | 專注 | 備註 |
|---|------|---------------|----------|---|------|------|

> 專注標示：`✅⚠️換`（需專注，取消現有）/ `❌免`（不需專注）/ `🔄儀式`（可儀式施放）

**🩸 種族/限次能力**

| # | 名稱 | 命中/難度（DC）| 傷害/效果 | 剩餘 | 恢復 | 備註 |
|---|------|---------------|----------|------|------|------|

> 剩餘：`✅1/1` = 可用、`❌0/1` = 已用完
> 恢復：`長休` 或 `短休`

**── 通用動作 ──**（所有角色固定）

| # | 名稱 | 效果 |
|---|------|------|
| D | 衝刺（Dash）| 移動距離加倍 |
| G | 脫離（Disengage）| 移動不觸發借機攻擊（AoO）|
| H | 躲避（Dodge）| 被攻擊劣勢・敏捷（DEX）豁免優勢 |
| P | 準備（Ready）| 設條件 + 反應動作 |

- 角色專屬動作用數字編號（1、2、3...）
- 通用動作用字母（D/G/H/P）方便區分
- 已用完的能力標 ❌ 仍保留在表中
- 無該類型動作的職業直接跳過對應子表

**第三段：附贈動作**

分為法術型（消耗欄位）和能力型（免費/限次）兩個子表：

**🎯 附贈法術**（消耗欄位）

| 🎯 附贈動作 | 效果 | 消耗欄位 | 專注 | 備註 |
|------------|------|---|------|------|

**🎯 附贈能力**（免費 / 限次）

| 🎯 附贈動作 | 效果 | 剩餘 | 恢復 | 備註 |
|------------|------|------|------|------|

> 無附贈法術的職業跳過附贈法術子表。
> 無附贈能力的職業跳過附贈能力子表。

**第四段：反應 & 道具**

| 🛡️ 反應（每輪 1 次）| 觸發條件 | 效果 |
|----------------------|----------|------|
| {reaction_name} | {trigger} | {effect} |

| 📦 道具 | 能力 | 消耗 | 施法時間 | 備註 |
|---------|------|------|----------|------|
| {item_name} | {ability_name} | {cost} | {cast_time} | {notes} |

- 只列有主動能力的道具（武器已在動作表，不重複）
- 消耗品（治療藥水等）也列在此

**第五段：魔寵表**（有魔寵時才出現）

魔寵有自己的回合（共享先攻，在主人後行動），有自己的動作/反應。
**核心規則：魔寵平時不能攻擊。連鎖之契（Pact of the Chain）才能命令攻擊。**

| 🐙 魔寵行為 | 效果 | 消耗（魔寵）| 消耗（主人）| ⚠️ 注意事項 |
|-------------|------|-----------|-----------|------------|
| {action} | {effect} | {familiar_cost} | {player_cost} | {notes} |

**動作經濟規則：**
1. **自主行動（Help / Dash / Disengage / Hide）**：消耗魔寵自己的動作，**不消耗主人任何資源**
2. **攻擊（僅限連鎖之契）**：消耗主人的**攻擊動作（Attack Action）** + 魔寵的**反應（Reaction）**
   - ⚠️ 攻擊動作 ≠ 施法動作：魔能轟擊（EB）是「施法」不是「攻擊」，**不能同回合又讓魔寵攻擊又施法**
   - 實質上 = 主人放棄整個動作（Action）讓魔寵打一下
3. **傳遞接觸法術**：消耗主人的**施法動作** + 魔寵的**反應（Reaction）**（法術透過魔寵位置觸碰目標）
4. **鎖鏈主宰之投資（Investment of the Chain Master）祈喚**（如有）：改用主人**附贈動作**命令攻擊，主要動作空出來施法

> 無此祈喚時，讓魔寵攻擊代價極高（犧牲整個動作（Action））。大多數情況 Help 給優勢 > 讓魔寵自己打。

> **職業戰鬥 SOP：** 各職業的完整戰鬥決策框架見本資料夾內各職業 .md 檔

## D. NPC 回合處理

> NPC 隊友由 DM 操控。每次輪到 NPC 行動，DM **在背景完成完整決策管線**後，
> 向玩家宣布意圖。玩家可喊停或修改。

### D-1. NPC 決策管線（DM 背景執行）

```
NPC 回合開始
│
├─ 1. 讀角色卡：掃描可用資源
│     ├─ 生命值（HP）/ 暫時生命值（HP）
│     ├─ 法術欄位 剩餘數
│     ├─ 限次能力 剩餘（引導神力、狂怒、動作如潮…）
│     ├─ 道具（治療藥水、卷軸…）
│     └─ 專注法術是否進行中
│
├─ 2. 讀職業 SOP 決策樹
│     └─ 走 ../class_progression/{class}_combat.md B 節主流程
│        按目前資源和血量走到對應分支
│
├─ 3. 場況評估（決策樹無法覆蓋的上下文）
│     ├─ 🟥 隊友狀態
│     │   ├─ 有人 0 HP 嗎？→ 優先搶救
│     │   ├─ 有人半血以下？→ 考慮治療/防護
│     │   └─ 全員健康？→ 正常輸出
│     │
│     ├─ 🟧 敵人狀態
│     │   ├─ 誰威脅最大？（施法者 > 高傷 > 低血）
│     │   ├─ 誰快死了？（1-2 下能收頭 → 優先清掉）
│     │   └─ 敵人數量多？→ 考慮範圍效果
│     │
│     └─ 🟨 環境
│         ├─ 有掩蔽可用？→ 遠程角色利用
│         ├─ 地形限制？（窄道、高低差、橋梁）
│         └─ 有環境互動機會？（吊燈、油桶…）
│
├─ 4. 性格過濾
│     └─ 從角色卡「性格/理念/缺點」決定行為偏好
│        ├─ 例：岡納（防護戰士）→ 傾向擋在隊友前面
│        ├─ 例：莉拉（謹慎牧師）→ 預防勝於治療
│        └─ 例：乃克（機會主義盜賊）→ 偷襲 > 正面衝突
│
└─ 5. 輸出決策 → 向玩家宣布
```

### D-2. 宣布格式

**單人模式（目前）：** 所有 NPC 集中宣布，玩家一次確認。

```
DM：「輪到友軍行動——
  🛡️ 岡納：衝上前用戰斧砍哥布林A（護住莉拉的側翼）
  ✨ 莉拉：對前排三人施放祝福術（專注・1 環）
  🗡️ 乃克：繞到哥布林B 側翼偷襲（附贈躲藏）
  → OK？還是你要調整？」
```

**玩家可以：**
- ✅ 確認 → DM 執行全部，統一描述結果
- ✏️ 修改某人行動 → DM 調整後執行
- ⛔ 喊停 → DM 暫停，討論後再繼續

### D-3. 執行與描述

確認後，DM 統一執行 NPC 行動：

1. **骰檢定**（攻擊/豁免/技能）
2. **計算結果**（傷害/治療/狀態）
3. **敘事描述**（結合角色性格）
4. **更新先攻序表**（生命值、狀態）

```
> 岡納一腳踏前，戰斧劈向哥布林A——
> 攻擊骰：17 + 7 = 24 vs 防禦等級 15 → 命中！
> 傷害：1d8+5 = 11 砍擊。哥布林A 倒地。
>
> 莉拉舉起聖徽低誦禱詞，金色微光籠罩 ToT、岡納、乃克——
> 祝福術生效（專注・10 回合）：三人攻擊骰和豁免 +1d4
>
> 乃克趁亂繞到哥布林B 背後，匕首精準刺入——
> 攻擊骰：15 + 7 = 22 vs 防禦等級 15 → 命中！
> 傷害：1d4+4（匕首）+ 2d6（偷襲）= 15 穿刺。哥布林B 重傷。
```

### D-4. NPC 反應（非自己回合）

NPC 的反應由 DM 即時判斷，**不需要事先宣布**：

| 觸發 | NPC 反應決策 |
|------|-------------|
| 敵人移出觸及範圍 | 近戰 NPC → 借機攻擊（AoO）（除非有更重要的反應要保留）|
| NPC 被攻擊 | 有反應防禦？（護盾術、靈敏閃避等）→ 依 SOP G 節防禦層次 |
| 隊友被攻擊 | 有防護/哨兵等反應？→ 使用 |

> **原則：** NPC 反應不問玩家，直接用最佳判斷執行。
> 但如果是消耗珍貴資源的反應（如最後一個護盾術充能），DM 可快速告知：
> 「莉拉用反應幫 ToT 擋一下？（消耗 1 環位）」

### D-5. 多人模式

- 輪到真人玩家時：印面板 + 點名（正常 C 節流程）
- NPC 角色：同上述 D-1 至 D-4 流程

## F. 戰術提示（怪物行為模式）

### 智慧型敵人（智力 ≥ 8）
- **集火：** 優先攻擊落單、低防禦等級（AC）、或施法者
- **掩蔽：** 主動利用地形（半掩蔽 +2 防禦等級（AC）、四分之三掩蔽 +5 防禦等級（AC））
- **撤退：** 生命值（HP）< 25% 或戰況不利時嘗試撤退/投降
- **戰術配合：** 前排擋路，遠程/施法者後排輸出
- **識別威脅：** 會轉火剛剛造成大量傷害的角色

### 獸類 / 低智生物（智力 < 8）
- **本能行動：** 攻擊最近的目標，不會繞後
- **逃跑本能：** 生命值（HP）< 25% 或首領死亡時可能潰逃（感知（WIS）豁免 難度（DC）10）
- **領地意識：** 守護巢穴時士氣較高，不易潰逃
- **群體戰術：** 狼群包圍（Pack Tactics）、蟲群堆疊

### 首領 / BOSS 型
- **傳奇動作：** 在他人回合結束時使用，每輪上限見 stat block
- **護衛：** 小怪保護首領，首領不會第一個衝鋒
- **階段轉換：** 生命值（HP）低於 50% 時可能改變策略（更猛烈/更防守/召喚援軍）
- **控場優先：** BOSS 會優先用控制法術削弱最大威脅

### DM 戰術決策流程
```
該怪物智力 ≥ 8？
├─ 是 → 有遠程攻擊？
│        ├─ 是 → 找掩蔽，優先打施法者/治療者
│        └─ 否 → 衝最近的軟目標（低防禦等級 / 低生命值）
└─ 否 → 攻擊最近目標
         └─ 生命值 < 25%？→ 嘗試逃跑
```

- 失敗 → 法術立即結束

- 同時只能維持 **1 個** 專注法術

- 主動取消專注不需動作（免費）

## I. 死亡豁免（Death Saving Throws）

### 觸發條件
角色生命值（HP）降至 0 且未即死（傷害未超過生命值（HP）最大值的負值）

### 流程
1. 倒地（Prone）+ 昏迷（Unconscious）
2. 每回合開始擲 **d20 死亡豁免**（無修正）
3. **成功 3 次** → 穩定（生命值 0，不再骰）
4. **失敗 3 次** → 角色死亡
5. **自然 20** → 恢復 1 生命值（HP），立即起身
6. **自然 1** → 算 2 次失敗

### 受傷時
- 昏迷中被攻擊命中 → 自動 1 次失敗
- 1.5m 內攻擊命中 → **自動暴擊** → 2 次失敗
- 受到傷害 ≥ 生命值（HP）最大值 → 即死

### 追蹤格式
```
☠️ ToT 死亡豁免
成功：☑ ☐ ☐　失敗：☑ ☑ ☐
```

### 救援選項

| 方法 | 說明 |
|------|------|
| 治療之言（Healing Word）| 附贈動作遠程治療 → **最優先** |
| 療傷術（Cure Wounds）| 動作觸碰治療 |
| 慈悲之手（Spare the Dying）| 動作戲法，穩定但不回生命值（HP）|
| 治療藥水 | 他人灌需動作；自己喝不行（昏迷中）|
| 醫療檢定 難度（DC）10 | 穩定（無醫療熟練不建議）|

### 1. 確認結束條件
- 所有敵人倒下/逃跑/投降
- 玩家選擇撤退且脫離戰鬥

### 2. 即時處理

- [ ] 治療倒地角色（莉拉優先處理）
- [ ] 清除戰鬥專用效果（脆弱詛咒、鏡影術等短期 buff）

- 更新 campaign.md 的 XP 總計

### 4. 戰利品
- 搜索屍體/區域

### 5. 更新 campaign.md

## K. 特殊情境

### 掩蔽（Cover）

| 類型 | 防禦等級（AC）/敏捷（DEX）豁免加值 |
|------|----------------|
| 半掩蔽（Half Cover）| +2 |
| 四分之三掩蔽（Three-Quarters）| +5 |
| 全掩蔽（Total Cover）| 無法被直接瞄準 |

### 優勢 / 劣勢（Advantage / Disadvantage）
- 優勢：擲 2d20 取**高**
- 劣勢：擲 2d20 取**低**
- 同時有優勢和劣勢 → **互相抵消**，正常擲
- 多重優勢/劣勢不疊加

### 倒地攻擊判定
- 近戰攻擊倒地目標 → **優勢**
- 遠程攻擊倒地目標 → **劣勢**

# 創角/升級 呈現格式模板

> 創角和升級時的固定格式模板。日常遊玩不需載入。

---

## 創角能力值呈現

先列出該角色的種族/背景已確定的能力值加成表，再印出三種方式（標準陣列、擲骰、點數購買）。標準陣列只印**已選職業**的建議配置（不印全職業表），格式為「能力 / 數值 / 核心作用」三欄，核心作用格式為「**粗體分類標籤：** 白話解釋機制」（如「施法主屬性：決定你法術噴得準不準...」）。擲骰照 `reference/creation/abilities.md` 原文印出。點數購買印出花費表 + 備註，然後引用方式一的標準陣列當基準（指出它換算成點數購買也恰好 27 點），再提供 1 個不同面向的配法範例（DM 即時生成），格式如下：
```
其實上面標準陣列的配置換算成點數購買也恰好是 27 點。如果想走不同路線，例如 **{面向名稱}型** — {一句話核心思路}，可以這樣分配：

┌─────────────┬──────┬──────┬────────────────────────────────────┐
│ 能力        │ 數值 │ 花費 │ 核心作用（與標準陣列不同之處）     │
├─────────────┼──────┼──────┼────────────────────────────────────┤
│ {能力}（{ABR}）│ {值} │ {花費} │ {數值變化時寫影響，沒變則留空} │
│ ...         │ ...  │ ...  │ ...                              │
├─────────────┼──────┼──────┼────────────────────────────────────┤
│ 總計        │      │ 27   │                                    │
└─────────────┴──────┴──────┴────────────────────────────────────┘

只要總計在 27 點以內、每項數值 8～15，都可以自由分配成你喜歡的組合。
```
面向須與標準陣列有明顯不同的玩法取向，必須恰好 27 點且值域 8～15

## 點數購買互動格式

玩家選擇點數購買後，DM 印出以下空白工作表讓玩家填寫，不主動提供建議配置：
```
點數購買工作表（預算：27 點）

花費參考：8=0  9=1  10=2  11=3  12=4  13=5  14=7  15=9

請告訴我你想要的六項能力值（8～15），例如：
STR 10 DEX 15 CON 14 INT 8 WIS 12 CHA 10
```
玩家回覆後，DM 印出驗算結果表：
```
┌────────────┬────────┬──────┐
│ 能力       │ 數值   │ 花費 │
├────────────┼────────┼──────┤
│ 魅力（CHA）│ 15     │ 9    │
│ 敏捷（DEX）│ 14     │ 7    │
│ ...        │ ...    │ ...  │
├────────────┼────────┼──────┤
│ 總計       │        │ ○/27 │
└────────────┴────────┴──────┘
```
驗算結果三種狀態：
- `✅ 恰好 27 點，確認這個配置？`
- `⚠️ 超過 N 點，請降低某些能力值。`
- `⚠️ 還剩 N 點，省下來不會帶到其他地方。要調整嗎？`

使用以下固定格式（框線表格），不用 Markdown 表格：
```

可選能力：{能力1}（{ABR}）/ {能力2}（{ABR}）/ {能力3}（{ABR}）
配法：選一項 +2 另一項 +1，或三項各 +1

你的基礎值（標準陣列 {職業} 配置）

┌─────────────┬────────┐
│    能力     │ 基礎值 │
├─────────────┼────────┤
│ {能力}      │ {值}   │  ← 只列背景可加的三項

七種配法總覽
```

## 法術表格式

- 戲法：「#、戲法、學派、施法時間、射程、標記、效果」
- 1 環以上：「#、法術、施法時間、射程、持續、標記、效果」
- 「標記」欄填寫全名格式：專注(C) / 儀式(R) / 材料(M)，無標記填「—」
- 1 環以上「持續」欄只寫時間長度（不加 C 前綴），專注資訊統一在「標記」欄顯示
- ⚠️ **每次**印法術表都必須在表頭上方附 C/R/M 說明區塊（不可省略）：
  ```
  > **C**（Concentration / 專注）— 同一時間只能維持一個專注法術；施放新的會使舊的消失。
  > **R**（Ritual / 儀式）— 可儀式施放：+10 分鐘施法時間，不消耗法術位。
  > **M**（Material / 材料）— 需要有標價的材料成分，不可被施法法器替代。
  ```
- 資料來源：查 `reference/class_progression/{職業}.md` 取可選法術清單 + 學派 + C/R/M 標記，查 `reference/spells/{環級}.md` 取施法時間、射程、持續、效果，合併為表格

## 豁免描述格式

能力縮寫一律用「中文（英文縮寫）」格式（如「感知（WIS）豁免」），「失敗時」一律寫「對方豁免失敗時」避免新手誤會
